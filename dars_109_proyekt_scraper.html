Web Scraping Nima?

Web scraping — bu veb-sahifadan ma'lumotni avtomatik ravishda ajratib olish jarayoni. Masalan, do'kon saytidagi mahsulot nomlari va narxlarini, yangiliklar sarlavhalarini yoki havolalarni dastur yordamida yig'ib olamiz.
Bu proyektda biz requests kutubxonasi bilan sahifani yuklab olamiz, BeautifulSoup bilan undan kerakli ma'lumotni ajratamiz, so'ng natijani CSV yoki JSON faylga saqlaymiz.

Kerakli Kutubxonalar

Avval ikkita kutubxonani o'rnatamiz. Terminalda quyidagini yozing:
pip install requests beautifulsoup4
# requests — sahifani yuklaydi, beautifulsoup4 — HTML ni o'qiydi
KutubxonaVazifasi
requestsSaytga so'rov yuboradi va HTML kodini oladi
BeautifulSoupHTML ni tahlil qilib, kerakli qismni topadi
csvMa'lumotni jadval (CSV) faylga yozadi
jsonMa'lumotni JSON faylga yozadi
timeSo'rovlar orasida kutish uchun (sleep)
⚠️ Axloqiy Scraping Eslatmasi

Scraping qilishdan oldin saytning robots.txt faylini va foydalanish shartlarini tekshiring. Shaxsiy yoki himoyalangan ma'lumotni yig'manng. Serverni ortiqcha yuklamang — so'rovlar orasida kutib turing (rate limiting). Faqat ruxsat etilgan ochiq ma'lumot bilan ishlang. Hurmat bilan scraping qiling!

1-bosqich: Sahifani Yuklab Olish

requests.get() bilan sahifaning HTML kodini olamiz. Javob kelganini status_code orqali tekshiramiz (200 — muvaffaqiyat).
import requests

url = "https://example.com/mahsulotlar"
javob = requests.get(url)

print(javob.status_code)  # 200 bo'lsa yaxshi
html = javob.text  # sahifaning HTML kodi
# get() saytga so'rov yuboradi, .text esa HTML matnini beradi

2-bosqich: HTML ni Tahlil Qilish

HTML matnini BeautifulSoup ga beramiz. So'ng .find() (bitta element) yoki .find_all() (barcha elementlar) bilan kerakli qismlarni topamiz.
from bs4 import BeautifulSoup

soup = BeautifulSoup(html, "html.parser")

# class="title" bo'lgan barcha elementlarni topamiz
sarlavhalar = soup.find_all("h2", class_="title")
for s in sarlavhalar:
    print(s.text.strip())
# .text — element ichidagi matn, .strip() — chetdagi bo'sh joylarni tozalaydi

3-bosqich: Ma'lumotni Ajratib Olish

Har bir mahsulot kartasidan sarlavha, narx va havola (link) ajratib olamiz va bir lug'atga (dictionary) yig'amiz.
mahsulotlar = []

for karta in soup.find_all("div", class_="product"):
    nom = karta.find("h2").text.strip()
    narx = karta.find("span", class_="price").text.strip()
    link = karta.find("a")["href"]
    mahsulotlar.append({
        "nom": nom,
        "narx": narx,
        "link": link
    })
# Har bir karta ichidan nom, narx va link olib, lug'at ko'rinishida ro'yxatga qo'shamiz

4-bosqich: Xatolarni Tutish

Internet uzilishi, sahifa topilmasligi yoki element yo'qligi xatoga olib kelishi mumkin. try / except bilan dasturni ishonchli qilamiz.
try:
    javob = requests.get(url, timeout=10)
    javob.raise_for_status()  # xato status bo'lsa xatolik chiqaradi
except requests.exceptions.RequestException as xato:
    print("So'rovda xato:", xato)
    javob = None
# timeout — kutishning eng ko'p vaqti, except — xatoni ushlab dastur to'xtab qolmaydi

5-bosqich: Rate Limiting (So'rovlar Orasida Kutish)

Ko'p sahifani ketma-ket scraping qilganda serverni hurmat qilish uchun har so'rovdan keyin biroz kutamiz. Buni time.sleep() bilan qilamiz.
import time

for sahifa in range(1, 6):
    url = f"https://example.com/page/{sahifa}"
    javob = requests.get(url)
    # ... ma'lumotni ajratib olish ...
    time.sleep(2)  # 2 soniya kutamiz
# time.sleep(2) — keyingi so'rovdan oldin 2 soniya dam beradi, serverni yuklamaydi

6-bosqich: Faylga Saqlash (CSV)

Yig'ilgan ma'lumotni csv moduli bilan jadval faylga yozamiz. Bu faylni Excel yoki Google Sheets'da ochish mumkin.
import csv

with open("mahsulotlar.csv", "w", newline="", encoding="utf-8") as f:
    yozuvchi = csv.DictWriter(f, fieldnames=["nom", "narx", "link"])
    yozuvchi.writeheader()  # sarlavha qatori
    yozuvchi.writerows(mahsulotlar)

print("Saqlandi: mahsulotlar.csv")
# DictWriter lug'atlardan iborat ro'yxatni jadvalga aylantiradi

6-bosqich (muqobil): Faylga Saqlash (JSON)

Agar ma'lumotni boshqa dasturda ishlatmoqchi bo'lsangiz, JSON ko'rinishida saqlash qulay.
import json

with open("mahsulotlar.json", "w", encoding="utf-8") as f:
    json.dump(mahsulotlar, f, ensure_ascii=False, indent=2)
# ensure_ascii=False — o'zbekcha harflar to'g'ri saqlanadi, indent=2 — chiroyli ko'rinish
Scraper Ish Tartibi

1. requests.get() — sahifani yuklash
2. BeautifulSoup — HTML ni tahlil qilish
3. .find_all() — ma'lumotni ajratish
4. try / except — xatolarni tutish
5. time.sleep() — so'rovlar orasida kutish
6. csv / json — faylga saqlash

Misol 1: HTML dan Ma'lumot Ajratish

Quyidagi maydonga HTML kodini yozing va tugmani bosing. Demo brauzerning DOMParser orqali (BeautifulSoup kabi) sarlavha va narxlarni ajratib oladi.

Misol 2: Ajratilgan Ma'lumotni CSV / Jadval Ko'rinishida

Avvalgi misolda ajratilgan ma'lumotni jadval va CSV matni ko'rinishida ko'rsatamiz — xuddi faylga saqlangandagidek.

Misol 3: Scraping Bosqichlarini Ko'rish

Tugmani bosing — scraper qaysi bosqichdan o'tayotganini ketma-ket yonib ko'rsatadi: yuklash → tahlil → ajratish → kutish → saqlash.
