input() Nima?

input() — bu foydalanuvchidan ma'lumot olish uchun ishlatiladigan funksiya. Dastur to'xtab, foydalanuvchi klaviaturada nimadir yozib Enter bosishini kutadi.

Eng oddiy ko'rinishi:

ism = input("Ismingiz nima? ")
# Foydalanuvchidan so'raydi
Ismingiz nima? Ali
>>> ism = "Ali"

Eng Muhim Qoida

⚠️ input() HAR DOIM string (matn) qaytaradi!

Foydalanuvchi 25 deb yozsa ham, Python uni raqam emas, "25" matni sifatida saqlaydi. Raqam kerak bo'lsa, uni int() yoki float() bilan o'zgartirishimiz kerak.

yosh = input("Yoshingiz? ")
print(type(yosh))
# 25 yozsangiz ham...
>>> <class 'str'>

Matnni Songa Aylantirish

Foydalanuvchi kiritgan raqam bilan matematik amal bajarish uchun avval uni int() (butun son) yoki float() (o'nli son) ga aylantiramiz.
1-usul: Alohida qatorlarda
yosh = input("Yoshingiz? ")
yosh = int(yosh)

Avval matn olinadi, keyin u butun songa aylantiriladi.

2-usul: Bir qatorda (qisqa)
yosh = int(input("Yoshingiz? "))

input() va int() birga yoziladi — bu professional uslub.

Matnlarni Birlashtirish

Ikki matnni + belgisi bilan birlashtirish mumkin. Bu amal konkatenatsiya deb ataladi.
ism = "Ali"
familiya = "Valiyev"
print(ism + " " + familiya)
# Bo'sh joy bilan birlashtirish
>>> Ali Valiyev

Keng Tarqalgan Xato

❌ Xato: son + matn
print("Yosh: " + 25) # XATO!

Matn (str) va sonni (int) to'g'ridan-to'g'ri qo'shib bo'lmaydi.

✓ To'g'ri: sonni str() bilan
print("Yosh: " + str(25))
>>> Yosh: 25
Eslab Qoling

• input() — har doim matn (str) qaytaradi
• Raqam kerak bo'lsa: int(input())
• Matnlarni birlashtirish: +
• Son + matn = xato → str() bilan to'g'rilang
