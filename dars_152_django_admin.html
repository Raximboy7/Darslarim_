Admin Panel Nima?

Django'ning eng kuchli imkoniyatlaridan biri — tayyor admin paneli. Bu sizning ma'lumotlar bazangizni boshqarish uchun avtomatik tarzda quriladigan boshqaruv interfeysi. Hech qanday HTML yozmasdan ma'lumot qo'shish, ko'rish, tahrirlash va o'chirish mumkin (CRUD).
Nega bu zo'r?

Oddiy holatda har bir jadval uchun forma, ro'yxat sahifasi, tahrirlash sahifasi yozish kerak bo'lardi. Django admin buni avtomatik qiladi — siz faqat modellaringizni ro'yxatdan o'tkazasiz.

1-qadam: Superuser Yaratish

Admin panelga kirish uchun avval superuser (bosh foydalanuvchi) kerak. U terminalda bitta buyruq bilan yaratiladi.
python manage.py createsuperuser
# Terminalga yozasiz, keyin login va parol so'raydi
Username: admin
Email address: admin@misol.uz
Password: ********
Password (again): ********
Superuser created successfully.
Endi serverni ishga tushirib http://127.0.0.1:8000/admin/ manziliga kirsangiz, login oynasi chiqadi. Yangi yaratgan login-parolingiz bilan kirasiz.

2-qadam: Modelni Ro'yxatdan O'tkazish

Admin panelda modelingizni ko'rish uchun uni admin.py faylida ro'yxatdan o'tkazasiz. Buning uchun admin.site.register() ishlatiladi.
# admin.py
from django.contrib import admin
from .models import Maqola

admin.site.register(Maqola)
# Endi "Maqola" modeli admin panelda ko'rinadi
Eslatma

Modelni import qilishni unutmang: from .models import Maqola. Ro'yxatdan o'tkazmagan model admin panelda umuman ko'rinmaydi.

3-qadam: Admin Orqali CRUD

Model ro'yxatdan o'tgandan so'ng admin panelda to'liq CRUD imkoniyati paydo bo'ladi:
HarfAmalAdmin'da
CCreate — qo'shish"Add" tugmasi
RRead — ko'rishRo'yxat sahifasi
UUpdate — tahrirlashYozuvga bosib o'zgartirish
DDelete — o'chirish"Delete" tugmasi

4-qadam: Admin'ni Sozlash (ModelAdmin)

Admin ko'rinishini chiroyli qilish uchun ModelAdmin klassidan foydalanamiz. Eng ko'p ishlatiladiganlar:
SozlamaVazifasi
list_displayRo'yxatda qaysi ustunlar ko'rinishini belgilaydi
search_fieldsQidiruv maydoni qo'shadi
list_filterO'ng tomonda filtr paneli chiqaradi
orderingYozuvlar tartibini belgilaydi
# admin.py
from django.contrib import admin
from .models import Maqola

class MaqolaAdmin(admin.ModelAdmin):
    list_display = ("sarlavha", "sana")
    search_fields = ("sarlavha",)

admin.site.register(Maqola, MaqolaAdmin)
# list_display ustunlarni, search_fields qidiruvni qo'shadi
E'tibor bering

Sozlash bilan ro'yxatdan o'tkazishda register(Maqola, MaqolaAdmin) — ikkita argument beriladi: model va uning ModelAdmin klassi.

Modelni admin'ga qo'shish uchun register() yozishingiz kerak. Tugmani bosing va admin panelda model paydo bo'lishini ko'ring.

Misol 3: list_display Natijasi

list_display ga qo'shgan har bir maydon ro'yxatda yangi ustun bo'lib chiqadi. Ustunlarni tanlab, natijani ko'ring.
Eslab Qoling

• createsuperuser — admin'ga kirish foydalanuvchisi
• admin.site.register(Model) — modelni ro'yxatdan o'tkazadi
• ModelAdmin — admin ko'rinishini sozlaydi
• list_display — ro'yxat ustunlarini belgilaydi
• Admin orqali kod yozmasdan to'liq CRUD bajariladi
