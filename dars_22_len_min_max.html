O'rnatilgan Funksiyalar Nima?

O'rnatilgan (built-in) funksiyalar — bu Python bilan birga keladigan tayyor vositalar. Ularni o'rnatish yoki import qilish shart emas, to'g'ridan-to'g'ri ishlataverasiz. Ro'yxat, string yoki to'plam ustida tez-tez kerak bo'ladigan amallarni shu funksiyalar bajaradi: len(), min(), max(), sum().
sonlar = [4, 1, 7, 3]
print(len(sonlar))
# Ro'yxatda nechta element bor?
>>> 4

To'rt Asosiy Funksiya

FunksiyaVazifasiMisol → Natija
len(x)Elementlar soni (uzunlik)[4,1,7,3] → 4
min(x)Eng kichik element[4,1,7,3] → 1
max(x)Eng katta element[4,1,7,3] → 7
sum(x)Sonlar yig'indisi[4,1,7,3] → 15

len() — Uzunlik

len() elementlar sonini qaytaradi. U ro'yxat, string, to'plam va lug'at bilan ishlaydi. String uchun bu — harflar soni.
print(len("Salom"))
print(len([10, 20, 30]))
# 5 ta harf
# 3 ta element
>>> 5
>>> 3

min() va max() — Eng Kichik va Eng Katta

Bu funksiyalar ro'yxatdagi eng kichik va eng katta qiymatni topadi. Ular sonlar bilan ham, stringlar bilan ham ishlaydi — stringlarni alfavit tartibida solishtiradi.
baholar = [85, 92, 78, 90]
print(min(baholar))
print(max(baholar))
# Eng past baho
# Eng yuqori baho
>>> 78
>>> 92
Bir nechta argument ham mumkin

Ro'yxatsiz, to'g'ridan-to'g'ri sonlarni ham berish mumkin:
max(3, 9, 5) → 9 va min(3, 9, 5) → 3

sum() — Yig'indi

sum() ro'yxatdagi barcha sonlarni qo'shadi. U faqat sonlar bilan ishlaydi.
narxlar = [12, 8, 25]
print(sum(narxlar))
# 12 + 8 + 25
>>> 45

O'rtacha qiymat: sum() va len() birga

baholar = [85, 92, 78]
ortacha = sum(baholar) / len(baholar)
print(ortacha)
# 255 / 3
>>> 85.0

sorted() va reversed()

sorted() tartiblangan yangi nusxa qaytaradi — aslini o'zgartirmaydi. reversed() esa elementlarni teskari tartibda beradi (natijani list() ga o'rab ro'yxatga aylantiramiz).
sonlar = [3, 1, 2]
print(sorted(sonlar))
print(sonlar)
# Tartiblangan nusxa
# Asli o'zgarmadi!
>>> [1, 2, 3]
>>> [3, 1, 2]
KodNatijaIzoh
sorted([3,1,2])[1, 2, 3]O'sish bo'yicha
sorted([3,1,2], reverse=True)[3, 2, 1]Kamayish bo'yicha
list(reversed([1,2,3]))[3, 2, 1]Teskari tartib

Keng Tarqalgan Xato

❌ Xato: songa len() qo'llash
print(len(12345)) # XATO!

Son uzunlikka ega emas: TypeError: object of type 'int' has no len()

✓ To'g'ri: avval str() bilan matnga aylantiring
print(len(str(12345)))
>>> 5
⚠️ sum() faqat sonlar bilan ishlaydi

sum(["a", "b"]) xato beradi. Stringlarni qo'shish uchun "".join() dan foydalaning. Bo'sh ro'yxat uchun esa sum([]) → 0.

Eslab Qoling

• len() — elementlar soni, sum() — yig'indi
• min() / max() — eng kichik / eng katta
• O'rtacha = sum(x) / len(x)
• sorted() yangi nusxa qaytaradi — aslini o'zgartirmaydi

Misol 1: Sonlar Statistikasi

Sonlarni vergul bilan kiriting — barcha funksiyalar jonli hisoblanadi.

Misol 2: O'rtacha Baho

O'rtacha = sum(baholar) / len(baholar)

Misol 3: So'z Uzunligi

len(soz) — so'zdagi harflar soni.

Misol 4: sorted() va reversed()

Aslini o'zgartirmasdan tartiblangan nusxani ko'ring.

Misol 5: Eng Arzon va Eng Qimmat

Narxlar ro'yxatidan min() va max() ni toping.
