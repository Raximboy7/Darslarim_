Async Programming (asyncio)

Modul 15 — Concurrency va Advanced | Dars 100

Async Nima?

Async (asinxron) dasturlash — bu bitta ish kutish paytida (masalan internetdan ma'lumot kelishini kutganda) dasturning bo'sh o'tirmasdan boshqa ishni bajara olishi demakdir.
Tasavvur qiling: choy qaynashini kutyapsiz. Sinxron usulda siz faqat tikilib turasiz. Asinxron usulda esa choy qaynaguncha non kesib, idishlarni yuvib qo'yasiz. Vaqt behuda ketmaydi!
Qachon foydali?

Async ayniqsa I/O ishlarda foydali: internetdan fayl yuklab olish, sayt so'rovlari, ma'lumotlar bazasi javobini kutish — bularning hammasi "kutish" bilan o'tadi. Ana shu kutish paytida boshqa ishlarni bajarsak, jami vaqt qisqaradi.

Restoran Ofitsianti Misoli

Sinxron ofitsiant: bir stolga buyurtma oladi, oshxonada taom tayyor bo'lishini shundoq turib kutadi, keyin keltiradi, undan keyingina ikkinchi stolga o'tadi.
Asinxron ofitsiant: birinchi stoldan buyurtmani oladi, oshxonaga beradi va kutib o'tirmasdan ikkinchi, uchinchi stollarga buyurtma olib boradi. Taom tayyor bo'lganda esa olib kelib beradi. Bitta ofitsiant ko'p stolga ulguradi!
Muhim

Async — bu bir vaqtda ko'p qo'l (ko'p protsessor) emas. Bu bitta ishchining kutish vaqtini aqlli ishlatishi. Shuning uchun u I/O (kutishli) ishlar uchun ideal.

asyncio moduli

Python'da asinxron kod yozish uchun asyncio moduli ishlatiladi. U standart kutubxonada bor — o'rnatish shart emas, faqat import qilamiz.
import asyncio
# asinxron ishlar uchun asosiy modul

async def — Coroutine yaratish

Oddiy funksiya def bilan yoziladi. Asinxron funksiya esa async def bilan yoziladi. Bunday funksiyani coroutine deb ataymiz.
import asyncio

async def salom_ber():
    print("Salom!")
# async def — coroutine (asinxron funksiya)
Diqqat

Coroutine'ni oddiy funksiya kabi salom_ber() deb chaqirsangiz, u ishlamaydi — faqat coroutine obyekti yaratiladi. Uni ishga tushirish uchun event loop kerak (pastda ko'ramiz).

await — Natijani Kutish

await so'zi "shu ish tugashini kut, lekin kutish paytida dastur boshqa ishlarga o'tishi mumkin" degan ma'noni bildiradi. await faqat async def ichida ishlatiladi.
async def ish():
    print("Boshlandi")
    await asyncio.sleep(2)
    print("2 soniyadan keyin tugadi")
# await asyncio.sleep(2) — 2 soniya kutadi (bloklamasdan)
asyncio.sleep vs time.sleep

time.sleep(2) butun dasturni to'xtatib qo'yadi (hech kim ishlamaydi). await asyncio.sleep(2) esa faqat shu coroutine'ni kutdiradi, boshqalar bemalol ishlayveradi.

Event Loop — asyncio.run()

Event loop (hodisalar halqasi) — bu asinxron ishlarni boshqaradigan "dirijyor". U coroutine'larni ishga tushiradi, kim kutyaptiyu kim tayyor — kuzatib turadi. Eng oson yo'li — asyncio.run().
import asyncio

async def main():
    print("Salom asyncio!")

asyncio.run(main())
# asyncio.run(main()) — event loop'ni ishga tushiradi va main()'ni bajaradi
>>> Salom asyncio!

asyncio.gather() — Bir Nechtasini Birga

asyncio.gather() bir nechta coroutine'ni bir vaqtda (parallel) ishga tushiradi va hammasi tugashini kutadi. Bu async'ning eng kuchli tomoni!
import asyncio

async def yuklab_ol(nom, vaqt):
    await asyncio.sleep(vaqt)
    print(f"{nom} yuklab olindi")

async def main():
    await asyncio.gather(
        yuklab_ol("A", 3),
        yuklab_ol("B", 2),
        yuklab_ol("C", 1)
    )

asyncio.run(main())
# Uchchalasi birga ishlaydi. Jami vaqt eng uzuni = 3 soniya (3+2+1=6 EMAS!)
>>> C yuklab olindi
>>> B yuklab olindi
>>> A yuklab olindi

Eslab Qoling

• async def — coroutine yaratadi
• await — natijani kutadi (bloklamasdan)
• asyncio.run() — event loop'ni ishga tushiradi
• asyncio.gather() — bir nechtasini birga bajaradi
• Async I/O (kutishli) ishlar uchun eng foydali

Misol 1: Sinxron vs Asinxron Vaqt

3 ta faylni yuklab olamiz (har biri 3, 2, 1 soniya). Sinxron usulda ular birin-ketin, asinxron usulda esa birga ishlaydi. Jami vaqtni taqqoslang!

Misol 2: Kutish Paytida Boshqa Ish

Bitta coroutine 3 soniya kutadi. Sinxronda dastur shu paytda muzlab qoladi. Asinxronda esa kutish paytida boshqa ishlar bajariladi. Tugmalarni bosib ko'ring.

Misol 3: gather() bilan Parallel

Nechta ishni birga bajarishni tanlang. Har biri 2 soniyalik. gather() bilan ular bir vaqtda ishlaydi — jami vaqt baribir ~2 soniya bo'ladi!
