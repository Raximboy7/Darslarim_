O'z Xatolaringizni Yaratish

Modul 8 — Xatolar va Test | Dars 44

raise — Xatoni Qo'lda Chaqirish

Ilgari xatolar Python tomonidan o'zi chiqarilardi (masalan, nolga bo'lganda ZeroDivisionError). Lekin ba'zan xatoni o'zimiz chiqarishimiz kerak. Buning uchun raise kalit so'zi bor.
raise ValueError("Manfiy bo'lmasin")
# raise xatoni darhol chaqiradi va dasturni to'xtatadi
>>> ValueError: Manfiy bo'lmasin

Nega Bu Kerak?

raise — bu o'z qoidalaringizni majburlash usuli. Python yoshning manfiy bo'lishini bilmaydi — bu sizning qoidangiz. Agar qoida buzilsa, dasturni davom ettirishdan ko'ra darhol xato chiqarish xavfsizroq.
def yosh_kirit(yosh):
    if yosh < 0:
        raise ValueError("Yosh manfiy bo'lmasin!")
    return f"Yosh: {yosh}"

print(yosh_kirit(-5))
# Manfiy yosh mantiqsiz — shuning uchun xato chiqaramiz
>>> ValueError: Yosh manfiy bo'lmasin!
Eslatma

raise ishlagan zahoti undan keyingi kod bajarilmaydi — xuddi return kabi, lekin bu yerda xato yuboriladi.

O'z Exception Klassingizni Yaratish

ValueError kabi tayyor xatolar yetmasa, o'zingizning maxsus xato turingizni yaratishingiz mumkin. Buning uchun Exception dan meros oladigan yangi klass yozasiz.
class MeningXatom(Exception):
    pass
# Bu eng oddiy o'z xato klassi — Exception dan meros oladi
Endi bu xatoni xuddi tayyor xatolar kabi raise qila olasiz:
class PulYetmadi(Exception):
    pass

balans = 100
narx = 250
if narx > balans:
    raise PulYetmadi("Hisobingizda pul yetarli emas")
# O'z xato turimiz aniq nimani anglatishini ko'rsatadi
>>> PulYetmadi: Hisobingizda pul yetarli emas

O'z Xatongni try-except Bilan Tutish

O'z xatongizni ham try-except bilan tutib olishingiz va xabarini o'qishingiz mumkin. Xato xabarini as x orqali olib, str(x) bilan matnga aylantiramiz.
class PulYetmadi(Exception):
    pass

try:
    raise PulYetmadi("500 so'm yetishmadi")
except PulYetmadi as xato:
    print("Xato yuz berdi:", xato)
# raise xato chiqaradi, except esa uni tutib oladi
>>> Xato yuz berdi: 500 so'm yetishmadi
Xabar (message) berish

Qavs ichida yozgan matningiz — xatoning xabari. Uni except ... as x orqali olib, foydalanuvchiga tushunarli ko'rinishda ko'rsatasiz.

Eslab Qoling

• raise XatoTuri("xabar") — xatoni qo'lda chiqaradi
• class MeningXatom(Exception): pass — o'z xatongiz
• O'z xatongizni ham try-except bilan tutasiz
• except ... as x — xabarni olish uchun

Misol 1: Qoidaga Zid Qiymat → raise

Yosh kiriting. Agar manfiy bo'lsa yoki 120 dan katta bo'lsa, raise ishlab maxsus xato xabari chiqadi.

Misol 2: O'z Exception Klassi Qanday Ishlaydi

PulYetmadi nomli o'z xatomiz. Balans va narxni kiriting — agar pul yetmasa, o'z xato klassimiz ishga tushadi.

Misol 3: raise + try-except Birga

Parol uzunligini tekshiramiz. raise xato chiqaradi, try-except esa uni tutib, dasturni to'xtatmasdan chiroyli xabar ko'rsatadi.
