Tabriklaymiz! Siz Bu Yo'lni Bosib O'tdingiz

Siz Python yo'lining katta qismini yakunladingiz. O'zgaruvchilardan boshlab — shartlar, sikllar, funksiyalar, ro'yxatlar, fayllar va to'liq proyektlargacha yetib keldingiz. Bu dars — barcha bilimlarni bir joyga jamlash va "endi men nima qila olaman?" degan savolga javob berish uchun.
Eng muhim haqiqat

Dasturlash — bu kichik bloklarni birlashtirish san'ati. Siz allaqachon barcha bloklarni bilasiz. Endi ularni katta loyihalarda birga ishlatishni o'rganamiz.

Hamma Narsani Birga Ishlatish

Haqiqiy dastur — bu bitta vositadan emas, balki ko'p qismlarning birga ishlashidan tashkil topadi. Quyidagi misolda biz funksiya, ro'yxat, sikl va shart — barchasini bitta kichik proyektda birlashtiramiz.
# Talabalar bahosini hisoblaydigan dastur
def o_rtacha(baholar):
    return sum(baholar) / len(baholar)

talabalar = [
    {"ism": "Ali", "baho": [90, 85, 100]},
    {"ism": "Vali", "baho": [60, 70, 50]},
]

for t in talabalar:
    o = o_rtacha(t["baho"])
    holat = "O'tdi" if o >= 60 else "Yiqildi"
    print(t["ism"], o, holat)
# Funksiya + dictionary + ro'yxat + sikl + shart — barchasi birga
>>> Ali 91.67 O'tdi
>>> Vali 60.0 O'tdi

Katta Loyiha Tuzilishi

Kichik dasturlar bitta faylda yashaydi. Ammo loyiha o'sganda, kodni modullarga (alohida .py fayllar) va papkalarga bo'lish kerak. Bu kodni tartibli va o'qiladigan qiladi.

├── main.py # dasturni ishga tushiruvchi

│   └── malumotlar.py # ma'lumot bilan ishlash
├── malumotlar/
│   └── talabalar.txt # saqlangan fayllar
├── requirements.txt # kerakli kutubxonalar
└── README.md # loyiha tavsifi
