Iterator nima?

Iterator — bu ketma-ketlikni (ro'yxat, string, tuple...) birma-bir, elementma-element qaytaradigan maxsus obyekt. U "qaysi elementda turibman" degan joyni eslab qoladi va har safar keyingisini beradi. Bir marta o'tib bo'lgach, qaytib boshiga kelmaydi.
Oddiy taqqoslash

Iteratorni qutidagi konfetlar deb tasavvur qiling. Har safar qo'lingizni solib bitta konfet olasiz. Olingan konfet qaytib qutiga tushmaydi. Konfetlar tugaganda qo'l bo'sh chiqadi — bu Python'da StopIteration degani.

Iterable va Iterator farqi

Bu ikki so'z o'xshash, lekin boshqa narsa:
TushunchaMa'nosiMisol
iterableUstidan aylanish (sikl) mumkin bo'lgan narsalist, string, tuple, dict, set
iteratorIterable'dan iter() bilan olingan, element beruvchi obyektiter([1, 2, 3])
Eng oddiy farqi

• iterable — bu konfet solingan quti (ro'yxat).
• iterator — bu shu qutidan konfet oladigan qo'l.
Bir qutidan ko'p qo'l (iterator) yasash mumkin, har biri o'zicha ishlaydi.

iter() — iteratorga aylantiradi

iter() funksiyasi iterable'ni (masalan ro'yxatni) iteratorga aylantiradi. Endi undan elementlarni birma-bir olish mumkin bo'ladi.
mevalar = ["olma", "banan", "uzum"]
it = iter(mevalar)
print(it)
# ro'yxat (iterable) dan iterator yasadik
>>> <list_iterator object at 0x...>

next() — keyingi elementni oladi

next() funksiyasi iteratordan keyingi elementni oladi. Har chaqirilganda bir qadam oldinga yuradi.
mevalar = ["olma", "banan", "uzum"]
it = iter(mevalar)

print(next(it))
print(next(it))
print(next(it))
# har chaqirilganda keyingi mevani oladi
>>> olma
>>> banan
>>> uzum

StopIteration — elementlar tugaganda

Iteratorda boshqa element qolmasa, next() StopIteration xatosini beradi. Bu "tamom, hammasi tugadi" degan signal.
it = iter([1, 2])
print(next(it))  
print(next(it))  
print(next(it))  
# 1 → 2 → uchinchisida element yo'q!
>>> 1
>>> 2
>>> StopIteration  # xato: elementlar tugadi

for sikli aslida qanday ishlaydi?

Sir: for sikli ham ichida iter() va next() ishlatadi! U avval iter() bilan iterator yasaydi, keyin har aylanishda next() chaqiradi, StopIteration kelganda esa to'xtaydi.
for meva in mevalar:
    print(meva)
# Bu kod Python ichida quyidagicha bajariladi:
it = iter(mevalar)
while True:
    try:
        meva = next(it)
        print(meva)
    except StopIteration:
        break
# for sikli aynan shu ishni o'zi bajaradi — siz ko'rmaysiz
Eslab Qoling

• iterable — sikl qilish mumkin (ro'yxat, string...)
• iter(x) — iterable'dan iterator yasaydi
• next(it) — keyingi elementni oladi
• StopIteration — elementlar tugaganda chiqadi
• for sikli ichida aslida iter() va next() ishlaydi

Misol 1: next() bosing

Quyidagi ro'yxat ustida turing. Har next() bosishingizda keyingi element chiqadi. Elementlar tugagach StopIteration ko'rinadi.

Misol 2: for sikli qadamma-qadam

Bu yerda for sikli ichida iter()/next() qanday ishlashini bir qadamda ko'rasiz. Har "Keyingi qadam" bosilganda Python ichkarida nima qilayotgani ko'rsatiladi.

Misol 3: iterable → iterator aylanishi

Bu demo ro'yxat (iterable) ustidan iter() bilan iterator yasalishini, keyin undan ketma-ket element olinishini ko'rsatadi.

Misol 4: next() ichida default qiymat

next(it, qiymat) — agar elementlar tugagan bo'lsa, xato o'rniga berilgan qiymatni qaytaradi. Bosib ko'ring.
