Modul 17 — Qo'shimcha Mavzular | Dars 114

Open Source Nima?

Open source (ochiq kodli) — bu kodi hammaga ochiq bo'lgan loyiha. Har kim uni ko'rishi, ishlatishi, o'rganishi va hatto yaxshilashi mumkin. Python, Django, Flask — bularning barchasi open source loyihalardir.
Nega bu muhim?

Sen ham o'z kodingni ochiq qo'yib, butun dunyoga ko'rsatishing mumkin. Bu sening portfoliong bo'ladi — ish beruvchilar sening kodingni ko'rib, qobiliyatingni baholaydi.

GitHub Nima?

GitHub — bu kodlarni saqlash va ulashish uchun eng mashhur sayt. Sen loyihangni GitHub ga joylasang, u butun dunyoga ochiq bo'ladi. GitHub bepul va dasturchilar uchun eng muhim joy.
AtamaMa'nosi
Repository (repo)Loyiha papkasi — barcha fayllar shu yerda
READMELoyihani tushuntiruvchi asosiy hujjat
LICENSEKodni ishlatish ruxsati (litsenziya)
commitSaqlangan o'zgarish (qadam)
issueMuammo yoki taklif yozilgan xabar
pull requestKod qo'shish bo'yicha so'rov

Loyihani GitHub ga Joylash

Quyidagi buyruqlar bilan loyihangni GitHub ga yuborasan. Bularni terminal (buyruqlar oynasi) da yozasan.
git init
git add .
git commit -m "Birinchi versiya"
git remote add origin URL
git push -u origin main
# init — loyihani Git ga ulaydi
# add — fayllarni tayyorlaydi
# commit — o'zgarishni saqlaydi
# remote — GitHub manzilini bog'laydi
# push — GitHub ga yuboradi

Yaxshi README

README.md — loyihangizni birinchi bo'lib ko'radigan sahifa. U Markdown tilida yoziladi va loyihani tushuntiradi. Yaxshi README loyihangni professional ko'rsatadi.
# Loyiha nomi
Bu loyiha nima qilishini yozing.

## O'rnatish
pip install loyiha

## Ishlatish
python main.py
# markdown: # — sarlavha, ## — kichik sarlavha

GitHub Pages — Bepul Sayt

GitHub Pages — bu sening HTML loyihangni bepul saytga aylantiradi. Repo sozlamalarida Pages ni yoqsang, loyihang username.github.io/loyiha manzilida ochiladi.
Misol

Mana bu Python darslari ham GitHub Pages orqali bepul saytga aylanishi mumkin! HTML fayllarni repo ga joyla, Pages ni yoq — tayyor.

Jamoa va Hissa Qo'shish

Open source — bu jamoa (community). Sen boshqa loyihalarga ham hissa qo'shishing mumkin. Buning oqimi: fork → o'zgartirish → pull request.
QadamNima bo'ladi
ForkLoyihaning nusxasini o'zingga olasan
CloneKompyuteringga yuklab olasan
O'zgartirishKodni tuzatasan yoki yangilik qo'shasan
Pull RequestO'zgarishni egasiga taklif qilasan
MergeEgasi maqullasa, kodga qo'shiladi
Eslab Qoling

• Open source — kodi hammaga ochiq loyiha
• GitHub — kodni saqlash va ulashish sayti
• README va LICENSE — har repo da bo'lishi kerak
• Hissa qo'shish oqimi: fork → pull request → merge
• GitHub — bu sening portfoliong
