API Nima?

API (Application Programming Interface) — bu dasturlar o'rtasida ma'lumot almashish uchun interfeys, ya'ni maxsus eshik. Sizning dasturingiz boshqa serverdan ma'lumot so'raydi, server esa javob qaytaradi.
Oddiy taqqoslash

API — restorandagi ofitsiant kabi. Siz (dastur) menyudan taom so'raysiz, ofitsiant (API) buyurtmani oshxonaga (serverga) yetkazadi va tayyor taomni sizga olib keladi. Siz oshxona ichida nima bo'layotganini bilishingiz shart emas.

REST API va JSON API

REST API — bugungi kunda eng keng tarqalgan API turi. U oddiy HTTP so'rovlari orqali ishlaydi (GET, POST va boshqalar). Ko'pchilik REST API JSON formatida ma'lumot qaytaradi — shuning uchun ularni JSON API deb ham atashadi.

Endpoint Tushunchasi

Endpoint — bu API ning aniq bir manzili (URL). Har bir endpoint ma'lum bir ma'lumotni beradi. Masalan ob-havo API sida bitta endpoint shahar ob-havosini, boshqasi esa valyuta kursini berishi mumkin.
# Bazaviy manzil (base URL)
https://api. obhavo.uz

# Endpoint qo'shilgan to'liq manzil
https://api. obhavo.uz/weather?city=Tashkent
# ?city=Tashkent — bu parametr (qaysi shahar kerakligini bildiradi)

API Key (Kalit)

Ko'pchilik API sizdan API key (maxsus kalit) so'raydi. Bu kalit sizni tanitadi — ya'ni autentifikatsiya qiladi. Kalitsiz server so'rovingizni rad etishi mumkin.
Xavfsizlik qoidasi

API key — sizning maxfiy parolingiz kabi. Uni hech kimga bermang va kodingizni internetga joylashtirganda kalitni ko'rinadigan joyda qoldirmang.

API ga So'rov Yuborish

Python da requests kutubxonasi yordamida API ga so'rov yuboramiz. Kelgan JSON javobni .json() metodi bilan Python lug'atiga (dictionary) aylantiramiz.
import requests

url = "https://api.example.uz/weather?city=Tashkent"
javob = requests.get(url)

malumot = javob.json() # JSON -> dictionary
print(malumot)
# requests.get() so'rov yuboradi, .json() esa javobni o'qiydi
>>> {'city': 'Tashkent', 'temp': 28, 'status': 'Quyoshli'}

JSON Javobdan Maydon Olish

JSON javob Python lug'atiga aylangach, undan kerakli ma'lumotni [kalit] orqali olamiz — xuddi lug'at bilan ishlagandek.
malumot = javob.json()

shahar = malumot["city"]
harorat = malumot["temp"]

print(f"{shahar}: {harorat}°C")
# malumot["temp"] — "temp" kalitining qiymatini oladi
>>> Tashkent: 28°C

API Key bilan So'rov

import requests

url = "https://api.example.uz/currency"
parametrlar = {
    "from": "USD",
    "to": "UZS",
    "apikey": "SIZNING_KALIT"
}

javob = requests.get(url, params=parametrlar)
print(javob.json())
# params= bilan parametrlarni avtomatik URL ga qo'shadi
>>> {'from': 'USD', 'to': 'UZS', 'rate': 12650}
Eslab Qoling

• API — dasturlar o'rtasida ma'lumot almashish interfeysi
• requests.get(url) — so'rov yuboradi

• Maydonni malumot["kalit"] orqali olamiz
• API key — autentifikatsiya uchun maxfiy kalit
