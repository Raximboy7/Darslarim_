Data Pipeline Nima?

Data pipeline (ma'lumot quvuri) — bu xom ma'lumotni qadamma-qadam tayyor natijaga aylantiradigan avtomatlashtirilgan oqim. Suv quvuri kabi: bir uchidan xom ma'lumot kiradi, ichida bir nechta bosqichdan o'tadi, ikkinchi uchidan tayyor hisobot yoki grafik chiqadi.
Bu darsda biz shu paytgacha o'rgangan barcha data ko'nikmalarni — CSV, API, pandas, matplotlib — bitta to'liq loyihada birlashtiramiz.

Pipeline 5 Bosqichi

Har qanday data quvuri quyidagi 5 bosqichdan iborat. Tartib doimo bir xil:
BosqichVazifasiVosita
1. Yig'ishMa'lumotni manbadan olishcsv, requests
2. TozalashBo'sh va xato qiymatlarni to'g'rilashpandas
3. TahlilGuruhlash va statistika hisoblashgroupby, mean
4. VizualizatsiyaNatijani grafikda ko'rsatishmatplotlib
5. SaqlashHisobotni faylga yozishto_csv, savefig

Bosqich 1: Ma'lumot Yig'ish

Ma'lumotni CSV fayldan yoki API'dan olamiz. Bu yerda CSV faylni pandas bilan o'qiymiz.
import pandas as pd

# CSV faylni o'qib, jadval (DataFrame) hosil qilamiz
df = pd.read_csv("savdolar.csv")
print(df.head())
# .head() — birinchi 5 qatorni ko'rsatadi, ma'lumotni tekshirish uchun
   shahar  mahsulot  narx
0  Toshkent  olma     12000
1  Samarqand nok       9000

Bosqich 2: Tozalash

Xom ma'lumotda ko'pincha bo'sh qiymatlar va takrorlar bo'ladi. Tahlildan oldin ularni tozalaymiz.
# Bo'sh (NaN) qiymatlarni o'chiramiz
df = df.dropna()

# Takrorlangan qatorlarni olib tashlaymiz
df = df.drop_duplicates()

# Narx ustunini son turiga aylantiramiz
df["narx"] = df["narx"].astype(int)
# Toza ma'lumot = ishonchli tahlil. Bu eng muhim bosqich!

Bosqich 3: Tahlil

Endi toza ma'lumotdan ma'no chiqaramiz: shaharlar bo'yicha guruhlaymiz va o'rtacha narxni hisoblaymiz.
# Shahar bo'yicha guruhlab, o'rtacha narxni topamiz
natija = df.groupby("shahar")["narx"].mean()
print(natija)
# groupby — guruhlaydi, mean — o'rtacha qiymatni hisoblaydi
shahar
Samarqand   9500.0
Toshkent    12500.0

Bosqich 4: Vizualizatsiya

Raqamlardan grafik yasaymiz — natijani ko'z bilan tushunish ancha oson.
import matplotlib.pyplot as plt

natija.plot(kind="bar", color="green")
plt.title("Shaharlar bo'yicha o'rtacha narx")
plt.savefig("hisobot.png")
# kind="bar" — ustunli grafik. savefig — grafikni rasm qilib saqlaydi

Bosqich 5: Saqlash

Tayyor natijani faylga yozamiz — boshqalar bilan ulashish yoki keyin ishlatish uchun.
# Tahlil natijasini yangi CSV faylga yozamiz
natija.to_csv("hisobot.csv")
print("Hisobot tayyor!")
# to_csv — DataFrame'ni CSV fayl ko'rinishida saqlaydi

To'liq Pipeline (bitta funksiya)

Barcha bosqichlarni bitta funksiyaga jamlaymiz — bu avtomatlashtirilgan quvur. Bir marta yozasiz, har gal ishlatasiz.
import pandas as pd
import matplotlib.pyplot as plt

def pipeline(fayl):
    # 1. Yig'ish
    df = pd.read_csv(fayl)
    # 2. Tozalash
    df = df.dropna().drop_duplicates()
    # 3. Tahlil
    natija = df.groupby("shahar")["narx"].mean()
    # 4. Vizualizatsiya
    natija.plot(kind="bar")
    plt.savefig("hisobot.png")
    # 5. Saqlash
    natija.to_csv("hisobot.csv")
    return natija

pipeline("savdolar.csv")
# Bitta chaqiriq — to'liq oqim: yig'ish → tozalash → tahlil → grafik → saqlash
Eslab Qoling

• Pipeline = yig'ish → tozalash → tahlil → vizual → saqlash
• Tozalash bosqichi eng muhim — toza ma'lumot ishonchli natija beradi
• groupby + mean — guruhli statistika
• Barcha bosqichni bitta funksiyaga jamlab avtomatlashtiramiz
