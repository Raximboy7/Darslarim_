Tuple va O'zgarmaslik

Modul 4 — Ma'lumot tuzilmalari | Dars 17

Tuple Nima?

Tuple (o'qilishi: "teypl") — bu o'zgarmas (immutable) ro'yxat. U xuddi list kabi bir nechta qiymatni saqlaydi, lekin bir marta yaratilgach uni o'zgartirib bo'lmaydi. Tuple oddiy qavs ( ) bilan yoziladi.
koordinata = (10, 20)
print(koordinata)
# Oddiy qavs bilan yaratiladi
>>> (10, 20)

Tuple Yaratish

Qiymatlarni vergul bilan ajratamiz. Ichida son, matn yoki ikkalasi ham bo'lishi mumkin.
KodTushuntirish
(1, 2, 3)Uchta sondan iborat tuple
("ali", "bobur")Ikkita matndan iborat tuple
(2026, "yanvar", 1)Aralash qiymatlar — sana
()Bo'sh tuple
⚠️ Bitta elementli tuple — vergulni unutmang!

Bitta qiymatli tuple yozishda oxiriga vergul qo'yish SHART. Aks holda Python uni oddiy qavs deb tushunadi.

a = (5,)
b = (5)
# a — tuple, b — oddiy son (5)
>>> type(a) → <class 'tuple'>
>>> type(b) → <class 'int'>

Elementga Kirish (Indexing)

Tuple ham list kabi indeks orqali ishlaydi. Sanash 0 dan boshlanadi, manfiy indeks oxiridan sanaydi.
rang = ("qizil", "yashil", "ko'k")
print(rang[0], rang[-1])
# Birinchi va oxirgi element
>>> qizil ko'k

Tuple Qachon Ishlatiladi?

O'zgarmasligi kerak bo'lgan ma'lumotlar uchun

Agar ma'lumot dastur davomida o'zgarmasligi kerak bo'lsa — tuple ideal tanlov:

• koordinata = (41, 69) — joy koordinatasi
• sana = (2026, 6, 29) — yil, oy, kun
• rang = (255, 0, 0) — RGB qizil rang

Tuple Metodlari

Tuple o'zgarmas bo'lgani uchun unda atigi 2 ta metod bor.
MetodVazifasiMisol → Natija
.count(x)x nechta marta borligini sanaydi(1,2,2,3).count(2) → 2
.index(x)x birinchi marta qaysi indeksda(1,2,3).index(2) → 1

Packing va Unpacking

Packing — bir nechta qiymatni bitta tuplega yig'ish. Unpacking — tuple ichidagi qiymatlarni alohida o'zgaruvchilarga ajratish.
nuqta = (10, 20, 30) # packing
x, y, z = nuqta # unpacking
print(x, y, z)
# 3 qiymat → 3 o'zgaruvchi
>>> 10 20 30
💡 Foydali

Unpacking yordamida ikki o'zgaruvchini bir qatorda almashtirish mumkin: a, b = b, a

Keng Tarqalgan Xato

❌ Xato: tuple ni o'zgartirishga urinish
rang = ("qizil", "yashil")
rang[0] = "ko'k" # XATO!
>>> TypeError: 'tuple' object does not support item assignment

Tuple o'zgarmas — element qiymatini almashtirib bo'lmaydi.

✓ To'g'ri: yangi tuple yaratish
rang = ("ko'k", "yashil")

O'zgartirish kerak bo'lsa, butunlay yangi tuple yaratamiz (yoki listdan foydalanamiz).
