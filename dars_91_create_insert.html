SQL Buyruqlari Nima?

Database bilan ishlash uchun maxsus til bor — bu SQL (Structured Query Language, ya'ni "Tartiblangan So'rovlar Tili"). SQL orqali biz databasega buyruq beramiz: jadval yarat, ma'lumot qo'sh, ma'lumotni o'qi va hokazo. Bu darsda birinchi ikkita buyruqni o'rganamiz: CREATE TABLE va INSERT INTO.
Soddacha aytganda

Database — bu jadvallar to'plami. Avval bo'sh jadval yaratamiz (CREATE), keyin unga qator-qator ma'lumot qo'shamiz (INSERT). Xuddi Excel jadvali kabi: avval ustunlarni yozasiz, keyin satrlarni to'ldirasiz.

CREATE TABLE — Jadval Yaratish

CREATE TABLE buyrug'i yangi, bo'sh jadval yaratadi. Jadval nomini va uning ustunlarini yozamiz. Har bir ustun uchun nom va ma'lumot turini ko'rsatamiz.
CREATE TABLE talabalar (
    id INTEGER,
    ism TEXT,
    yosh INTEGER
);
# "talabalar" nomli jadval yaratamiz
# 3 ta ustun: id, ism, yosh
# Har ustunning turi ko'rsatilgan
>>> Jadval yaratildi (hozircha bo'sh, qatorlar yo'q)

Ma'lumot Turlari

Har bir ustun ma'lum bir turdagi ma'lumotni saqlaydi. SQLite databaseda asosiy 5 ta tur mavjud:
TurNimani saqlaydiMisol
INTEGERButun son25, 100, -7
TEXTMatn (string)"Ali", "salom"
REALKasrli son3.14, 99.5
BLOBIkkilik ma'lumot (rasm, fayl)baytlar
NULLQiymat yo'q (bo'sh)—
Diqqat

To'g'ri turni tanlash muhim. Yoshni INTEGER, ismni TEXT, narxni REAL qilib saqlaymiz. Bu ma'lumotni tartibli va to'g'ri ushlab turadi.

PRIMARY KEY — Asosiy Kalit

Ko'pincha har bir qatorga noyob raqam beramiz, masalan id. Bu raqam orqali qatorni aniq topamiz. Bunday ustunni PRIMARY KEY deb belgilaymiz — u takrorlanmaydi va har qator uchun yagona bo'ladi.
CREATE TABLE talabalar (
    id INTEGER PRIMARY KEY,
    ism TEXT,
    yosh INTEGER
);
# id — asosiy kalit (har talaba uchun yagona raqam)
# Ikki talaba bir xil id ga ega bo'la olmaydi

INSERT INTO — Ma'lumot Qo'shish

Jadval tayyor bo'lgach, unga ma'lumot qo'shamiz. INSERT INTO buyrug'i bir qator (satr) qo'shadi. Qaysi ustunlarga, qanday qiymatlar (VALUES) yozilishini ko'rsatamiz.
INSERT INTO talabalar (id, ism, yosh)
VALUES (1, 'Ali', 20);
# talabalar jadvaliga 1 ta qator qo'shadi
# id=1, ism='Ali', yosh=20
# Matn qiymatlar bitta tirnoq ichida: 'Ali'
>>> 1 qator qo'shildi

Bir Nechta Qator Qo'shish

Bitta INSERT bilan bir nechta qatorni birdaniga qo'shsa bo'ladi — har bir qatorni vergul bilan ajratamiz.
INSERT INTO talabalar (id, ism, yosh)
VALUES
    (2, 'Vali', 22),
    (3, 'Guli', 19),
    (4, 'Hasan', 21);
# 3 ta qator birdaniga qo'shildi
# Har qator qavs ichida, vergul bilan ajratilgan
>>> 3 qator qo'shildi

SQL ni Python sqlite3 da Bajarish

Endi yuqoridagi SQL buyruqlarni Python ichida ishga tushiramiz. Buning uchun sqlite3 moduli ishlatamiz. Qadamlar: databasega ulanamiz, cursor olamiz, SQL ni execute() bilan bajaramiz, so'ng commit() bilan o'zgarishlarni saqlaymiz.
import sqlite3

# databasega ulanamiz (fayl yaratiladi)
conn = sqlite3.connect("maktab.db")
cursor = conn.cursor()

# jadval yaratamiz
cursor.execute("""
    CREATE TABLE talabalar (
        id INTEGER PRIMARY KEY,
        ism TEXT,
        yosh INTEGER
    )
""")

# ma'lumot qo'shamiz
cursor.execute("INSERT INTO talabalar VALUES (1, 'Ali', 20)")

conn.commit()  # o'zgarishlarni saqlaymiz
conn.close()   # ulanishni yopamiz
# execute() — SQL buyruqni bajaradi
# commit() — o'zgarishlarni diskka yozadi (SHART!)
# commit() qilmasangiz, ma'lumot saqlanmay qoladi

Eslab Qoling

• CREATE TABLE — bo'sh jadval yaratadi
• INSERT INTO ... VALUES — qator qo'shadi
• Turlar: INTEGER, TEXT, REAL, BLOB, NULL
• PRIMARY KEY — har qator uchun yagona raqam
• commit() qilmasangiz ma'lumot saqlanmaydi

Misol 1: CREATE TABLE — Bo'sh Jadval

Ustunlar nomini yozing va "Jadval yarat" tugmasini bosing. CREATE buyrug'i hosil bo'ladi va bo'sh jadval ko'rinadi (hali bironta qator yo'q).

Misol 2: INSERT — Qator Qo'shish

Avval yuqorida jadval yarating. Keyin bu yerdan qiymatlarni kiriting va qo'shing — jadval qator-qator to'lib boradi.

Misol 3: To'g'ri Turni Tanlash

Qiymatni yozing va unga mos ma'lumot turini tanlang. Tekshiramiz — tanlovingiz to'g'rimi?
