Rekursiya Nima?

Rekursiya — bu funksiyaning o'z ichida o'zini chaqirishi. Katta masalani har safar bir oz kichikroq qismga bo'lib, oxir-oqibat eng oddiy holatga yetib kelamiz. Bu xuddi ikki oyna bir-biriga qaragandagi cheksiz aks ettirishga o'xshaydi — lekin biz uni to'xtatish shartini qo'yib boshqaramiz.
def sanagich(n):
  if n == 0:
    return
  print(n)
  sanagich(n - 1)
# funksiya o'zini chaqirmoqda
# n=0 bo'lganda to'xtaydi
>>> sanagich(3)
3
2
1

Ikkita Muhim Qism

Har bir to'g'ri rekursiyada ikkita qism bo'lishi shart:
1. Bazaviy hol (base case)

To'xtash sharti. Bu yerda funksiya o'zini boshqa chaqirmaydi, balki tayyor javobni qaytaradi. Bazaviy hol — bu zinapoyaning eng pastki pog'onasi.

2. Rekursiv hol (recursive case)

Funksiya o'zini kichikroq qiymat bilan chaqiradi va shu tariqa bazaviy holga yaqinlashadi. Har qadamda masala soddalashishi shart.

Misol: Faktorial

Faktorial n! = 1·2·3·…·n. Rekursiv ta'rifi: factorial(n) = n × factorial(n-1), va factorial(1) = 1 (bazaviy hol).
def factorial(n):
  if n == 1:
    return 1
  return n * factorial(n - 1)
# bazaviy hol: 1! = 1
#
# rekursiv hol:
# n * (n-1)!

Qadamlar qanday ochiladi

factorial(5)
= 5 * factorial(4)
= 5 * 4 * factorial(3)
= 5 * 4 * 3 * factorial(2)
= 5 * 4 * 3 * 2 * factorial(1)
= 5 * 4 * 3 * 2 * 1
>>> 120

Misol: Fibonacci

Fibonacci ketma-ketligida har bir son oldingi ikki sonning yig'indisi: 0, 1, 1, 2, 3, 5, 8, 13... Rekursiv qoidasi: fib(n) = fib(n-1) + fib(n-2).
def fib(n):
  if n < 2:
    return n
  return fib(n-1) + fib(n-2)
# ikkita bazaviy hol:
# fib(0)=0, fib(1)=1
#
# rekursiv hol
>>> fib(7)
13
>>> [fib(i) for i in range(8)]
[0, 1, 1, 2, 3, 5, 8, 13]

Keng Tarqalgan Xato

⚠️ Bazaviy hol bo'lmasa — cheksiz rekursiya!

Agar to'xtash sharti bo'lmasa yoki unga hech qachon yetib bormasa, funksiya o'zini cheksiz chaqiraveradi. Python esa buni to'xtatib RecursionError xatosini beradi (stek to'lib ketadi).

❌ Xato: bazaviy hol yo'q
def sanagich(n):
  print(n)
  sanagich(n - 1) # hech qachon to'xtamaydi
>>> RecursionError: maximum recursion depth exceeded
✓ To'g'ri: bazaviy hol bor
def sanagich(n):
  if n < 0:
    return # to'xtash sharti
  print(n)
  sanagich(n - 1)
Eslab Qoling

• Rekursiya — funksiyaning o'zini chaqirishi
• Har doim bazaviy hol (to'xtash sharti) bo'lishi shart
• Rekursiv hol masalani kichraytirib bazaviy holga yaqinlashtiradi
• Bazaviy hol bo'lmasa → RecursionError (stek to'lib ketadi)

Misol 1: Faktorial Hisoblagich

Kod: return n * factorial(n - 1)

Misol 2: Fibonacci Ketma-ketligi

Kod: fib(n) = fib(n-1) + fib(n-2)

Misol 3: Chaqiruvlar Zinapoyasi

Har chaqiruv stekka qo'shiladi, bazaviy holdan keyin javoblar qaytadi.

Misol 4: 1 dan n gacha Yig'indi

Kod: return n + yigindi(n - 1), bazaviy hol: yigindi(0) = 0

Misol 5: Matnni Teskari Aylantirish

Kod: return teskari(s[1:]) + s[0], bazaviy hol: bo'sh matn
