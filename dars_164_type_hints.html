Type Hints (Tur Belgilari)

Modul 23 — Pro daraja va Capstone | Dars 164

Type Hints Nima?

Type hints (tur belgilari) — bu o'zgaruvchi yoki funksiya qanday turdagi ma'lumot bilan ishlashini kodda yozib qo'yish usuli. Ya'ni "bu o'zgaruvchi matn (str)", "bu funksiya butun son (int) qaytaradi" deb belgilab qo'yamiz.
Eng muhim narsa: type hints — bu faqat belgi (eslatma). Python ularni ishlatmaydi va majburlamaydi. Ya'ni x: int desangiz ham, baribir matn berishingiz mumkin — Python xato bermaydi. Ular asosan dasturchi va vositalar uchun mo'ljallangan.

O'zgaruvchiga Tur Belgilash

O'zgaruvchi nomidan keyin : tur yozamiz. Bu "bu o'zgaruvchi shu turda bo'lishi kerak" degani.
ism: str = "Ali"
yosh: int = 25
bo'y: float = 1.75
student: bool = True
# ism — matn (str), yosh — butun son (int)
# bo'y — kasrli son (float), student — mantiqiy (bool)
Diqqat

: tur — tur belgisi, = qiymat — oddiy qiymat berish. Ikkalasini birga ishlatish mumkin, lekin tur belgisini tashlab, faqat ism = "Ali" deb yozsangiz ham bemalol ishlaydi.

Funksiyaga Tur Belgilash

Funksiyada har bir parametr turini, va funksiya nima qaytarishini belgilash mumkin. Qaytariladigan tur uchun -> (strelka) ishlatiladi.
def qoshish(a: int, b: int) -> int:
    return a + b

def salomlash(ism: str) -> str:
    return "Salom, " + ism
# qoshish: a va b — int, natija ham int
# salomlash: ism — str, natija ham str
Strelka -> nimani bildiradi?

-> int "bu funksiya butun son qaytaradi" degani. Agar funksiya hech narsa qaytarmasa, -> None yoziladi.

typing Moduli

str, int kabi oddiy turlar yetarli emas. Ro'yxat, lug'at kabi murakkab turlar uchun typing modulidan foydalanamiz.
from typing import List, Dict, Optional, Union

ismlar: List[str] = ["Ali", "Vali"]
ballar: Dict[str, int] = {"Ali": 90}
nick: Optional[str] = None
raqam: Union[int, float] = 3.5
# List[str] — matnlardan iborat ro'yxat
# Dict[str, int] — kalit str, qiymat int bo'lgan lug'at
# Optional[str] — str YOKI None bo'lishi mumkin
# Union[int, float] — int YOKI float bo'lishi mumkin
TurMa'nosiMisol
List[int]Sonlardan iborat ro'yxat[1, 2, 3]
Dict[str, int]str → int lug'at{"a": 1}
Optional[str]str yoki None"Ali" yoki None
Union[int, str]int yoki str5 yoki "5"
tuple[int, int]Ikki sonli tuple(10, 20)
Yangi Python (3.9+)

Hozirgi Python versiyalarida List o'rniga oddiy list[str], Dict o'rniga dict[str, int] yozsa ham bo'ladi — bu holatda typing dan import qilish shart emas.

mypy — Tur Tekshiruvchi

Python type hints ni o'zi tekshirmaydi. Lekin mypy degan maxsus vosita bor — u kodingizdagi turlarni ishga tushirmasdan tekshiradi va mos kelmasa ogohlantiradi.
# terminalda:
pip install mypy
mypy fayl.py
# mypy ni o'rnatamiz va faylni tekshiramiz
>>> fayl.py:3: error: Argument 1 to "qoshish"
>>> has incompatible type "str"; expected "int"
Esda tuting

mypy faqat ogohlantiradi — kodni baribir to'xtatmaydi. Type hints va mypy majburiy emas. Kichik dasturlarda ishlatmasangiz ham bo'ladi, lekin katta loyihalarda ular juda foydali.

Misol 1: Type Hints Bilan / Holda

Bir xil funksiyani type hints bilan va ularsiz solishtiring — qaysi biri tushunarliroq ekanini ko'ring.

Misol 2: mypy Ogohlantirishi

Funksiya int kutadi. Turli qiymat bering va mypy nima deyishini ko'ring.

Misol 3: typing Turlari

Turni tanlang — uning ma'nosi va to'g'ri/noto'g'ri misollarini ko'ring.

Misol 4: Funksiya Imzosini Yasash

Parametr va qaytarish turini tanlab, type hints bilan funksiya imzosini ko'ring.
