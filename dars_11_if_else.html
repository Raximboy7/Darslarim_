if, elif, else

Modul 3 — Shartlar va sikllar | Dars 11

Shartli Operator Nima?

Shartli operator dasturga qaror qabul qildiradi. Ya'ni dastur biror shartni tekshiradi: agar shart True (rost) bo'lsa — bir ishni bajaradi, aks holda boshqasini. Bu Pythonda if so'zi bilan yoziladi.
if yosh >= 18:
    print("Katta")
# Shart rost bo'lsa, blok ishlaydi
# yosh = 20 bo'lsa
>>> Katta

Eng Muhim Qoida: Ikki Nuqta va Bo'sh Joy

⚠️ Ikki nuqta ( : ) va 4 bo'sh joy (indentation) shart!

Shartdan keyin albatta : qo'yiladi. Shart ostidagi blok esa 4 bo'sh joy bilan ichkariga suriladi. Aynan shu surilgan satrlar shart rost bo'lganda bajariladi.

if ball > 50:
    print("O'tdingiz")
    print("Tabriklaymiz")
# 4 bo'sh joyli ikki satr — bir blok

if - else (Aks Holda)

else — "aks holda" degani. Agar if sharti True bo'lmasa, else bloki bajariladi.
if yosh >= 18:
    print("Katta")
else:
    print("Kichik")
# 18 dan kichik bo'lsa — "Kichik"
# yosh = 15 bo'lsa
>>> Kichik

if - elif - else (Bir Nechta Shart)

elif — "else if" ning qisqasi. U bir nechta shartni ketma-ket tekshirish uchun ishlatiladi. Python yuqoridan pastga qarab tekshiradi va birinchi rost bo'lgan blokni bajarib, qolganlarini o'tkazib yuboradi.
if ball >= 90:
    print("A")
elif ball >= 80:
    print("B")
elif ball >= 70:
    print("C")
else:
    print("D")
# Baho hisoblagich:
# 90+ → A, 80+ → B,
# 70+ → C, qolgani → D
# ball = 85 bo'lsa
>>> B

Eslab Qoling

• Shartdan keyin albatta : qo'yiladi
• Blok ichi 4 bo'sh joy bilan suriladi
• elif bir nechta variantni tekshiradi, birinchi rost g'olib
• else ixtiyoriy — "qolgan barcha holatlar" uchun
