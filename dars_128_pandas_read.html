Fayldan O'qish (CSV/Excel)

Modul 19 — NumPy va Pandas | Dars 128

Nima Uchun Fayldan O'qiymiz?

Haqiqiy hayotda ma'lumotlar ko'pincha fayllarda saqlanadi: jadval fayllari (CSV) yoki Excel hujjatlari (.xlsx). Pandas bu fayllarni bitta qator kod bilan o'qib, ularni DataFrame ga aylantiradi. Shundan keyin ma'lumot ustida tahlil qilish, saralash va hisoblash juda oson bo'ladi.
CSV nima?

CSV — "Comma Separated Values" (vergul bilan ajratilgan qiymatlar). Bu oddiy matn fayli bo'lib, har bir qator — jadval qatori, qiymatlar esa vergul bilan ajratiladi. Masalan: ism,yosh,shahar

1. pd.read_csv() — CSV O'qish

CSV faylni o'qish uchun pd.read_csv() funksiyasidan foydalanamiz. U faylni o'qib, tayyor DataFrame qaytaradi.
import pandas as pd

df = pd.read_csv("talabalar.csv")
print(df)
# Faylni o'qib, DataFrame ga aylantiradi
    ism  yosh   shahar
0  Ali    18  Toshkent
1 Vali    20  Samarqand
2 Guli    19   Buxoro

2. read_csv Parametrlari

read_csv() ko'plab foydali parametrlarni qabul qiladi. Ular faylni qanday o'qishni boshqaradi.
ParametrVazifasiMisol
sepAjratuvchi belgi (standart — vergul)sep=";"
headerSarlavha qaysi qatordaheader=0
namesUstunlarga o'zimiz nom beramiznames=["a","b"]
index_colQaysi ustun indeks bo'ladiindex_col=0
encodingMatn kodlash turiencoding="utf-8"
# Nuqta-vergul bilan ajratilgan fayl
df = pd.read_csv("data.csv", sep=";")

# Birinchi ustunni indeks qilamiz
df = pd.read_csv("data.csv", index_col=0)

# Ustun nomlarini o'zimiz beramiz
df = pd.read_csv("data.csv", names=["ism", "yosh"])
# Har bir parametr o'qishni boshqaradi

3. pd.read_excel() — Excel O'qish

Excel fayllarini (.xlsx) o'qish uchun pd.read_excel() ishlatiladi. Buning uchun openpyxl kutubxonasi kerak bo'ladi.
# Avval o'rnatamiz (terminalda):
# pip install openpyxl

df = pd.read_excel("hisobot.xlsx")

# Aniq varaqni tanlash
df = pd.read_excel("hisobot.xlsx", sheet_name="Yanvar")
# Excel ham bir qator kod bilan o'qiladi
Eslatma

read_excel() da sheet_name parametri orqali kerakli varaqni tanlash mumkin. Agar berilmasa, birinchi varaq o'qiladi.

4. head() — Dastlabki Qatorlarni Ko'rish

Katta fayllarda hamma qatorni ko'rsatish shart emas. head() birinchi 5 qatorni ko'rsatadi, tail() esa oxirgi 5 qatorni.
df = pd.read_csv("katta_fayl.csv")

print(df.head())
print(df.head(3))
# head() — birinchi 5, head(3) — birinchi 3 qator
    ism  yosh
0  Ali    18
1 Vali    20
2 Guli    19

5. to_csv() / to_excel() — Faylga Yozish

DataFrame ni faylga saqlash uchun to_csv() va to_excel() ishlatiladi. index=False qo'shsak, indeks ustuni faylga yozilmaydi.
# DataFrame ni CSV ga saqlash
df.to_csv("natija.csv", index=False)

# Excel ga saqlash
df.to_excel("natija.xlsx", index=False)
# index=False — chap tomondagi raqamlar yozilmaydi
Eslab Qoling

• pd.read_csv("fayl.csv") — CSV o'qish
• pd.read_excel("fayl.xlsx") — Excel o'qish (openpyxl kerak)
• Parametrlar: sep, header, names, index_col
• df.head() — dastlabki qatorlarni ko'rish
• to_csv() / to_excel() — faylga yozish
