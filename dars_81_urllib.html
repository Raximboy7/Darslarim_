urllib bilan Web

Modul 13 — Web Asoslari | Dars 81

urllib Nima?

urllib — bu Python'ning ichki (built-in) web kutubxonasi. Uni o'rnatish shart emas — Python bilan birga keladi. Uning yordamida internetdagi web sahifalarni yuklab olish, URL manzillarni tahlil qilish va web xatolarini tutib olish mumkin.
urllib 3 ta asosiy qismdan iborat

• urllib.request — web sahifani yuklab olish
• urllib.parse — URL manzilni tahlil qilish va yasash
• urllib.error — web xatolarini tutib olish

urllib.request — Sahifani Yuklab Olish

urlopen() funksiyasi berilgan URL manzilga so'rov yuboradi va serverdan javob (response) oladi. Javobni .read() bilan o'qiymiz — natija bytes (xom baytlar) bo'ladi.
from urllib.request import urlopen

javob = urlopen("https://example.com")
malumot = javob.read()
print(type(malumot))
# urlopen() sahifaga ulanadi, read() butun matnni o'qiydi
>>> <class 'bytes'>

.read() va .decode() — Matnga Aylantirish

.read() bizga bytes qaytaradi (masalan b'Salom'). Buni oddiy o'qiladigan matnga aylantirish uchun .decode("utf-8") ishlatamiz.
javob = urlopen("https://example.com")
xom = javob.read()
matn = xom.decode("utf-8")
print(matn[:60])
print("Status:", javob.status)
# decode("utf-8") baytlarni odam o'qiy oladigan matnga aylantiradi; .status — javob kodi (200)
>>> <!doctype html><html><head><title>Example Domain...
>>> Status: 200
AmalVazifasi
urlopen(url)URL'ga ulanadi, javob obyektini qaytaradi
.read()Javob tanasini bytes ko'rinishida o'qiydi
.decode("utf-8")bytes'ni matn (str) ga aylantiradi
.statusJavob kodi (200 = muvaffaqiyatli)

urllib.parse — URL'ni Tahlil Qilish

Har bir URL bir nechta qismdan iborat: sxema, host, yo'l (path) va parametrlar (query). urlparse() URL'ni shu qismlarga ajratadi.
from urllib.parse import urlparse

u = urlparse("https://shop.uz/kitob?id=5")
print(u.scheme)  # https
print(u.netloc)  # shop.uz
print(u.path)    # /kitob
print(u.query)   # id=5
# urlparse URL'ni qismlarga bo'ladi: scheme, netloc, path, query
>>> https
>>> shop.uz
>>> /kitob
>>> id=5

Query parametrlarni yasash — urlencode()

Parametrlarni qo'lda yozish o'rniga urlencode() lug'atdan to'g'ri query satr yasaydi (bo'shliq va maxsus belgilarni avtomatik to'g'rilaydi).
from urllib.parse import urlencode

params = {"q": "python kitob", "sahifa": 2}
print(urlencode(params))
# bo'shliq "+" ga aylanadi, parametrlar "&" bilan ulanadi
>>> q=python+kitob&sahifa=2

urllib.error — Xatolarni Tutish

Internet so'rovlari har doim ham muvaffaqiyatli bo'lmaydi. Sahifa topilmasligi (404) yoki tarmoq uzilishi mumkin. Shuning uchun so'rovni try / except ichida yozamiz.
Xato turiQachon yuz beradi
HTTPErrorServer javob berdi, lekin xato kod bilan (404, 500)
URLErrorServerga umuman ulana olmadik (host yo'q, internet uzilgan)
from urllib.request import urlopen
from urllib.error import HTTPError, URLError

try:
    javob = urlopen("https://example.com/yoq")
    print(javob.read())
except HTTPError as e:
    print("Xato kodi:", e.code)
except URLError as e:
    print("Ulanmadi:", e.reason)
# HTTPError'da .code (404) bor; URLError'da .reason (sabab) bor. Avval HTTPError yoziladi
>>> Xato kodi: 404
Diqqat

HTTPError aslida URLError'ning bir turi (uning farzandi). Shuning uchun HTTPError'ni doim birinchi yozing — aks holda u hech qachon ushlanmaydi.

Eslab Qoling

• urllib.request — sahifa yuklaydi (urlopen)
• .read() bytes beradi → .decode("utf-8") matnga aylantiradi
• urllib.parse — URL'ni qismlarga ajratadi va yasaydi
• urllib.error — HTTPError (404) va URLError (ulanmadi) ni tutadi
