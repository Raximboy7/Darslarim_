1-qism: OOP Masalalari

OOP (Obyektga yo'naltirilgan dasturlash) — ma'lumot va funksiyalarni bitta klass ichida birlashtiradi. Quyida uchta klassik masalani yechamiz.

Masala 1: Bank Hisobi Klassi

Shart: Bank hisobini ifodalovchi klass yozing. Unda pul qo'shish (deposit), pul yechish (withdraw) va balansni ko'rsatish bo'lsin. Balans yetarli bo'lmasa, pul yechishga ruxsat berilmasin.
class BankHisobi:
    def __init__(self, egasi, balans=0):
        self.egasi = egasi
        self.balans = balans

    def deposit(self, summa):
        self.balans += summa
        print(f"+{summa} qo'shildi")

    def withdraw(self, summa):
        if summa > self.balans:
            print("Mablag' yetarli emas!")
        else:
            self.balans -= summa
            print(f"-{summa} yechildi")

    def ko'rsat(self):
        print(f"{self.egasi}: {self.balans} so'm")

hisob = BankHisobi("Ali", 1000)
hisob.deposit(500)
hisob.withdraw(2000)
hisob.ko'rsat()
# __init__ — hisob ochilganda egasi va boshlang'ich balansni saqlaydi
# deposit — balansni oshiradi
# withdraw — avval yetarlimi tekshiradi, keyin kamaytiradi
# ko'rsat — joriy holatni chop etadi
>>> +500 qo'shildi
>>> Mablag' yetarli emas!
>>> Ali: 1500 so'm
Tushuntirish

self — har bir hisobning o'z ma'lumotini eslab qoladi. Pul yechishdan oldin if bilan tekshiruv qo'ydik — bu hisobni minusga tushib ketishdan saqlaydi.

Masala 2: Talaba Boshqaruvi Klassi

Shart: Talaba klassini yozing. Har talabaning ismi va baholar ro'yxati bo'lsin. o'rtacha() metodi o'rtacha bahoni qaytarsin, a'lochimi() metodi o'rtacha 4.5 dan yuqori bo'lsa True qaytarsin.
class Talaba:
    def __init__(self, ism, baholar):
        self.ism = ism
        self.baholar = baholar

    def o'rtacha(self):
        return sum(self.baholar) / len(self.baholar)

    def a'lochimi(self):
        return self.o'rtacha() >= 4.5

t = Talaba("Vali", [5, 4, 5, 5])
print(t.o'rtacha())
print(t.a'lochimi())
# baholar — ro'yxat (list) sifatida saqlanadi
# o'rtacha — sum() / len() formula
# a'lochimi — natijani solishtirib True/False qaytaradi
>>> 4.75
>>> True
Tushuntirish

Bir metod ichida boshqa metodni chaqirdik: self.o'rtacha(). Bu kodni takrorlamaslikka yordam beradi — o'rtachani bir joyda hisoblaymiz, hamma joyda ishlatamiz.

Masala 3: Geometrik Shakllar Yuzasi

Shart: Doira va To'rtburchak uchun alohida klasslar yozing. Har birida yuza() metodi bo'lsin. Bu — OOP da polimorfizm ning yaqqol misoli: bir xil nomli metod, har klassda boshqacha hisoblaydi.
import math

class Doira:
    def __init__(self, r):
        self.r = r
    def yuza(self):
        return math.pi * self.r ** 2

class Tortburchak:
    def __init__(self, en, boy):
        self.en = en
        self.boy = boy
    def yuza(self):
        return self.en * self.boy

shakllar = [Doira(5), Tortburchak(3, 4)]
for s in shakllar:
    print(round(s.yuza(), 2))
# Doira yuzasi = pi * r^2
# To'rtburchak yuzasi = en * boy
# for sikli har shaklning o'z yuza() metodini chaqiradi
>>> 78.54
>>> 12
Tushuntirish

for sikli ichida har shaklning yuza() metodini chaqirdik. Python o'zi biladi qaysi klassning metodi kerakligini — bu polimorfizm.

2-qism: Algoritm Masalalari

Algoritm — masalani yechish uchun aniq qadamlar ketma-ketligi. Quyida intervyularda tez-tez uchraydigan klassik algoritmlarni yechamiz.

Masala 4: Ikki Sonli Yig'indi (Two-Sum)

Shart: Sonlar ro'yxati va maqsad (target) berilgan. Yig'indisi maqsadga teng bo'lgan ikkita son indekslarini toping. Misol: [2, 7, 11, 15], maqsad 9 → natija [0, 1] (chunki 2 + 7 = 9).
def two_sum(sonlar, maqsad):
    korilgan = {}
    for i, son in enumerate(sonlar):
        kerakli = maqsad - son
        if kerakli in korilgan:
            return [korilgan[kerakli], i]
        korilgan[son] = i
    return None

print(two_sum([2, 7, 11, 15], 9))
# korilgan — dictionary: son → indeks
# kerakli = maqsad - son (juftini topish uchun)
# agar kerakli oldin ko'rilgan bo'lsa — javob topildi!
>>> [0, 1]
Tushuntirish

Dictionary ishlatib, har sonni bir marta aylanib chiqamiz. Bu tez usul — har son uchun qaytadan qidirmaymiz, balki oldingilarini korilgan da saqlab boramiz.

Masala 5: Anagram Tekshirish

Shart: Ikki so'z anagram ekanligini tekshiring. Anagram — bir xil harflardan tuzilgan so'zlar, faqat tartib boshqacha. Misol: "listen" va "silent" — anagram.
def anagrammi(s1, s2):
    s1 = s1.replace(" ", "").lower()
    s2 = s2.replace(" ", "").lower()
    return sorted(s1) == sorted(s2)

print(anagrammi("listen", "silent"))
print(anagrammi("salom", "olamx"))
# bo'shliqlarni olib tashlab, kichik harfga keltiramiz
# sorted() harflarni alifbo tartibida saralaydi
# agar saralangan ro'yxatlar teng bo'lsa — anagram!
>>> True
>>> False
Tushuntirish

Eng oddiy usul: ikki so'z harflarini sorted() bilan saralaymiz. Agar harflar bir xil bo'lsa, saralangandan keyin ular aynan teng bo'ladi.

Masala 6: Eng Uzun So'z

Shart: Gap berilgan. Undagi eng uzun so'zni toping. Misol: "men python tilini o'rganaman" → natija "o'rganaman".
def eng_uzun(gap):
    sozlar = gap.split()
    return max(sozlar, key=len)

gap = "men python tilini o'rganaman"
print(eng_uzun(gap))
# split() — gapni so'zlarga bo'ladi
# max(..., key=len) — eng uzun (eng ko'p harfli) so'zni topadi
>>> o'rganaman
Tushuntirish

max() funksiyasiga key=len berib, "eng kattani uzunligi bo'yicha topish"ni aytamiz. Bu juda qisqa va kuchli usul.

Masala 7: Oddiy Kalkulyator

Shart: Ikki son va amal (+, -, *, /) qabul qiladigan kalkulyator funksiyasini yozing. Nolga bo'lishni ham hisobga oling.
def kalkulyator(a, b, amal):
    if amal == "+":
        return a + b
    elif amal == "-":
        return a - b
    elif amal == "*":
        return a * b
    elif amal == "/":
        if b == 0:
            return "Nolga bo'lib bo'lmaydi!"
        return a / b
    return "Noma'lum amal"

print(kalkulyator(10, 2, "/"))
print(kalkulyator(5, 0, "/"))
# if / elif zanjiri amalga qarab tarmoqlanadi
# bo'lishdan oldin b == 0 ni tekshiramiz (xatolikning oldini olamiz)
>>> 5.0
>>> Nolga bo'lib bo'lmaydi!

Masala 8: Ballarni Baholash Tizimi

Shart: 0-100 oralig'idagi ballni harfli bahoga aylantiring: 90+ → A, 80+ → B, 70+ → C, 60+ → D, qolgani → F.
def baho(ball):
    if ball >= 90:
        return "A"
    elif ball >= 80:
        return "B"
    elif ball >= 70:
        return "C"
    elif ball >= 60:
        return "D"
    else:
        return "F"

for b in [95, 82, 50]:
    print(b, ->, baho(b))
# if / elif tartibi muhim: yuqoridan pastga tekshiriladi
# birinchi to'g'ri kelgan shart natija qaytaradi
>>> 95 -> A
>>> 82 -> B
>>> 50 -> F
Tushuntirish

elif zanjirida tartib muhim. Eng yuqori chegaradan boshlaymiz. 95 ball 90 dan katta — darrov "A" qaytariladi va qolgan tekshiruvlar o'tkazib yuboriladi.
