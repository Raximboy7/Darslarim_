Ma'lumotni Tozalash

Modul 19 — NumPy va Pandas | Dars 129

Nega Ma'lumotni Tozalash Kerak?

Haqiqiy hayotdagi ma'lumot deyarli har doim kir (iflos) bo'ladi: ba'zi kataklar bo'sh, ba'zi qatorlar takror, raqamlar matn ko'rinishida yozilgan, ustun nomlari noto'g'ri yozilgan bo'ladi. Tahlil yoki grafik chizishdan oldin ma'lumotni tozalash shart.
Oltin qoida

"Garbage in — garbage out" — kir ma'lumotdan kir natija chiqadi. Tahlil sifati avval ma'lumot tozaligiga bog'liq. Data olimlari vaqtining katta qismini aynan tozalashga sarflashadi.

1. Bo'sh Qiymatlar (NaN)

Pandas bo'sh katakni NaN (Not a Number) deb belgilaydi. Bo'sh qiymatlarni topish va ular bilan ishlash uchun maxsus metodlar bor.

isnull() — qaerda bo'sh qiymat borligini topish

import pandas as pd
df.isnull()
df.isnull().sum()
# isnull() — har katak bo'shmi? True/False qaytaradi
# .sum() — har ustunda nechta bo'sh qiymat bor, sanaydi

dropna() — bo'sh qatorlarni o'chirish

df.dropna()
df.dropna(subset=["yosh"])
# dropna() — bo'sh qiymati bor barcha qatorlarni o'chiradi
# subset — faqat "yosh" ustuni bo'sh bo'lsa o'chiradi

fillna() — bo'sh qiymatni to'ldirish

df["yosh"].fillna(0)
df["yosh"].fillna(df["yosh"].mean())
# fillna(0) — bo'sh joyni 0 bilan to'ldiradi
# fillna(mean()) — o'rtacha qiymat bilan to'ldiradi
Qaysi birini tanlash?

• Ma'lumot ko'p bo'lsa va bo'shlari oz bo'lsa — dropna() bilan o'chir.
• Har qator qimmatli bo'lsa — fillna() bilan to'ldir (o'rtacha, 0 yoki "Noma'lum").

2. Takror Qatorlar

Bir xil qator bir necha marta takrorlanishi tahlilni buzadi. duplicated() takrorlarni topadi, drop_duplicates() ularni olib tashlaydi.
df.duplicated()
df.duplicated().sum()
df.drop_duplicates()
# duplicated() — qator takrormi? True/False
# .sum() — nechta takror bor
# drop_duplicates() — takrorlarni o'chiradi (birinchisi qoladi)
df.drop_duplicates(subset=["ism"], keep="last")
# subset — faqat "ism" ustuniga qarab takror tekshiradi
# keep="last" — oxirgisini qoldiradi (default: "first")

3. Tur O'zgartirish (astype)

Ba'zan raqamlar matn (string) ko'rinishida saqlangan bo'ladi. Ular bilan hisob qilish uchun avval to'g'ri turga o'tkazish kerak. astype() shu ishni bajaradi.
df["yosh"] = df["yosh"].astype(int)
df["narx"] = df["narx"].astype(float)
df["id"] = df["id"].astype(str)
# str → int: "25" → 25 (butun son)
# str → float: "9.5" → 9.5 (kasr son)
# int → str: 100 → "100" (matn)
>>> df.dtypes yosh int64 narx float64 id object

4. Ustun Nomini O'zgartirish (rename)

Ustun nomlari ba'zan noqulay bo'ladi (bo'shliq, katta harf, ingliz tili). rename() ularni tushunarli nomlarga o'zgartiradi.
df = df.rename(columns={
    "Name": "ism",
    "Age": "yosh"
})
# columns={eski_nom: yangi_nom}
# "Name" → "ism", "Age" → "yosh"

5. Qiymat Almashtirish (replace)

Bir xil narsa har xil yozilgan bo'lishi mumkin ("male", "M", "Erkak"). replace() ularni bitta standart qiymatga keltiradi.
df["jins"] = df["jins"].replace({
    "M": "Erkak",
    "F": "Ayol"
})
# {eski: yangi} — "M" → "Erkak", "F" → "Ayol"
# Endi barcha qiymatlar bir xil ko'rinishda
Eslab Qoling

• isnull().sum() — qancha bo'sh qiymat borligini ko'rsatadi
• dropna() o'chiradi, fillna() to'ldiradi
• drop_duplicates() — takror qatorlarni olib tashlaydi
• astype() — turni o'zgartiradi (str → int)
• rename() nomni, replace() qiymatni almashtiradi

Misol 1: Bo'sh Qiymatlarni Tozalash (NaN)

Quyidagi jadvalda ba'zi yosh qiymatlari bo'sh (NaN, qizil). dropna() bo'sh qatorni o'chiradi, fillna() to'ldiradi.

Misol 2: Takror Qatorlarni Olib Tashlash

Sariq qatorlar — takror (duplicate). drop_duplicates() tugmasini bosing.

Misol 3: Oldin va Keyin (Kir → Toza)

"Tozalash" tugmasini bosing — to'liq jarayonni ko'ring: bo'sh to'ldiriladi, takror o'chiriladi, jins qiymati standartlanadi, tur to'g'rilanadi.
