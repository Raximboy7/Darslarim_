Modul 13 — Web Asoslari | Dars 87

Forma Nima?

Forma (form) — bu foydalanuvchidan ma'lumot oladigan veb sahifa qismi. Masalan: ism kiritish maydoni, parol maydoni, "Yuborish" tugmasi. Foydalanuvchi formani to'ldirib tugmani bosganda, ma'lumot serverga (Flask ga) yuboriladi.
Oddiy misol: ro'yxatdan o'tish formasi. Foydalanuvchi ismini yozadi, "Yuborish" ni bosadi, Flask esa bu ismni qabul qilib ekranga chiqaradi. Ana shu jarayonni shu darsda o'rganamiz.

HTML Forma: <form> Tegi

Forma HTML da <form> tegi bilan yoziladi. Uning ichida <input> (kiritish maydoni) va submit (yuborish) tugmasi bo'ladi.
<form method="POST">
    <input type="text" name="ism">
    <button type="submit">Yuborish</button>
</form>
# method="POST" — ma'lumot qanday yuborilishi
# name="ism" — maydonning nomi (Flask shu nom orqali oladi)
# type="submit" — bosilganda formani yuboradigan tugma
Eng muhimi: name atributi

Har bir <input> da name bo'lishi SHART. Flask aynan shu nom bilan kiritilgan qiymatni topadi. name bo'lmasa, ma'lumot serverga umuman yetib bormaydi.

POST Metodi

method="POST" — bu forma ma'lumotini serverga yashirin tarzda (URL ko'rinmaydi) yuborish usuli. Parol, ism kabi ma'lumotlar uchun POST ishlatiladi.

Sodda qoida

Foydalanuvchi ma'lumot yuborsa (ism, parol, izoh) → POST.
Foydalanuvchi ma'lumot so'rasa (qidiruv natijasi) → GET.

Flask da Forma Ma'lumotini Olish

Flask da forma qiymatini olish uchun avval request ni import qilamiz, so'ng request.form['nom'] bilan qiymatni olamiz. Bu yerda 'nom' — HTML dagi name atributining qiymati.
from flask import Flask, request, render_template

app = Flask(__name__)

@app.route("/", methods=["GET", "POST"])
def index():
    if request.method == "POST":
        ism = request.form["ism"]
        return f"Salom, {ism}!"
    return '''<form method="POST">
        <input name="ism">
        <button>Yuborish</button>
    </form>'''
# request — yuborilgan ma'lumotni o'qiydigan obyekt
# methods=["GET","POST"] — ikkala metodga ruxsat
# request.method — hozir qaysi metod kelganini bildiradi
# request.form["ism"] — formadagi "ism" maydonidan qiymat
>>> Foydalanuvchi "Ali" yozib yuborsa:
>>> Salom, Ali!

methods=["GET", "POST"] nima uchun kerak?

Odatdagi @app.route("/") faqat GET ga ruxsat beradi. Agar formani POST bilan yuborsangiz, Flask "405 Method Not Allowed" xatosini beradi. Shuning uchun POST formali sahifaga ikkala metodni ham qo'shamiz:
# Faqat sahifani ko'rsatish — GET
@app.route("/")

# Forma yuborish ham bo'lsa — GET va POST
@app.route("/", methods=["GET", "POST"])
# GET — sahifa birinchi marta ochilganda (forma ko'rsatiladi)
# POST — forma to'ldirilib yuborilganda (ma'lumot qabul qilinadi)

Forma Validatsiyasi (Tekshirish)

Validatsiya — foydalanuvchi kiritgan ma'lumotni tekshirish. Eng oddiy tekshirish: maydon bo'sh emasligini tekshirish. Bo'sh bo'lsa, xato xabari beramiz va ma'lumotni qabul qilmaymiz.
if request.method == "POST":
    ism = request.form["ism"]
    if ism == "":
        return "Xato: ism bo'sh bo'lishi mumkin emas!"
    return f"Salom, {ism}!"
# ism == "" — maydon bo'shmi tekshiramiz
# bo'sh bo'lsa — xato xabari qaytaramiz
# to'ldirilgan bo'lsa — salom beramiz
Maslahat: .strip() ishlating

Foydalanuvchi faqat bo'sh joy (probel) kiritishi mumkin. Shuning uchun ism = request.form["ism"].strip() deb yozish yaxshi — u chetlardagi bo'sh joylarni olib tashlaydi, so'ng tekshiramiz.

Misol 1: request.form Qanday Ishlaydi

Quyida HTML forma to'ldirilganda Flask request.form orqali qanday ma'lumot olishini ko'rasiz. Maydonlarni to'ldiring va "Yuborish" ni bosing.

Misol 2: GET va POST Farqi

Bir xil ma'lumotni GET va POST bilan yuborib, farqni ko'ring. GET da ma'lumot URL da ko'rinadi, POST da esa yashirin yuboriladi.

Misol 3: Forma Validatsiyasi

Maydonni bo'sh qoldirib yuborib ko'ring — Flask uni rad etadi. To'ldirib yuborsangiz, qabul qilinadi. Bu eng oddiy validatsiya.
Eslab Qoling

• Forma HTML da <form method="POST"> bilan yoziladi
• Har <input> da name bo'lishi SHART
• Flask da qiymat: request.form["nom"]
• POST formali sahifada methods=["GET","POST"] kerak
• Validatsiya: bo'sh maydonni if ism == "" bilan tekshiring
