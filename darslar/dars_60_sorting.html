Saralash Algoritmlari (Sorting)

Modul 11 — Algoritmlar | Dars 60

Saralash Nima?

Saralash (Sorting) — bu ma'lumotlarni ma'lum bir tartibda joylashtirish demakdir. Masalan, raqamlarni kichikdan kattaga yoki ismlarni alifbo tartibida joylashtirish.
Saralash juda muhim, chunki tartiblangan ma'lumotda biror narsani topish ancha oson va tez bo'ladi. Telefon kontaktlari, do'kondagi narxlar, o'quvchilar reytingi — barchasi saralash yordamida tartibga solinadi.
Hayotiy misol

Kutubxonadagi kitoblar alifbo bo'yicha terilgan. Agar ular tartibsiz bo'lsa, kerakli kitobni topish uchun har bir kitobni ko'rib chiqishga to'g'ri kelardi. Saralash aynan shu ishni avtomatlashtiradi.

Python'da Tayyor Saralash

Python'da saralash uchun tayyor vositalar bor: sorted() funksiyasi va .sort() metodi. Lekin algoritmlar ichida qanday ishlashini bilish — yaxshi dasturchi bo'lish uchun zarur.
raqamlar = [5, 2, 8, 1, 9]
natija = sorted(raqamlar)
print(natija)
# sorted() yangi tartiblangan ro'yxat qaytaradi
>>> [1, 2, 5, 8, 9]
raqamlar = [5, 2, 8, 1]
raqamlar.sort(reverse=True)
print(raqamlar)
# .sort() ro'yxatning o'zini o'zgartiradi; reverse=True — kattadan kichikka
>>> [8, 5, 2, 1]

1. Bubble Sort (Pufakcha Saralash)

Eng sodda algoritm. Yonma-yon turgan ikki elementni taqqoslaydi va agar chap tarafdagisi katta bo'lsa, ularni almashtiradi. Bu jarayon takrorlanaveradi: eng katta raqam har bir aylanishda oxirga "suzib" chiqadi — xuddi pufakcha suvda yuqoriga ko'tarilgandek.
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr
# Yonma-yon elementlarni taqqoslab almashtiradi.
# arr[j], arr[j+1] = arr[j+1], arr[j] — bir qatorda almashtirish
Qadamlar (5, 2, 8, 1)

1-bosqich: 5>2 → 2,5,8,1 · 5<8 (qoladi) · 8>1 → 2,5,1,8
2-bosqich: 2<5 (qoladi) · 5>1 → 2,1,5,8
3-bosqich: 2>1 → 1,2,5,8 ✓ tartiblandi

2. Selection Sort (Tanlash bilan Saralash)

Har bir aylanishda eng kichik elementni topadi va uni boshqa o'rinlardan boshiga olib qo'yadi. Ya'ni: tartiblanmagan qismdan eng kichigini tanlab, tartiblangan qismga qo'shadi.
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        eng_kichik = i
        for j in range(i + 1, n):
            if arr[j] < arr[eng_kichik]:
                eng_kichik = j
        arr[i], arr[eng_kichik] = arr[eng_kichik], arr[i]
    return arr
# Har aylanishda eng kichik elementni topib, boshiga qo'yadi

3. Insertion Sort (Joylash bilan Saralash)

Xuddi qartani qo'lda terib chiqqandek ishlaydi. Har bir yangi elementni olib, uni tartiblangan qismdagi to'g'ri joyiga qo'yadi (joylaydi). Kichik ro'yxatlar va deyarli tartiblangan ma'lumot uchun juda tez.
def insertion_sort(arr):
    for i in range(1, len(arr)):
        kalit = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > kalit:
            arr[j + 1] = arr[j]
            j = j - 1
        arr[j + 1] = kalit
    return arr
# Har elementni tartiblangan qismdagi to'g'ri joyiga joylaydi

4. Merge Sort va Quick Sort (G'oyasi)

Bu ikki algoritm "bo'l va hukmronlik qil" (divide and conquer) tamoyiliga asoslanadi: katta masalani kichik bo'laklarga bo'lib yechadi. Shuning uchun ular yuqoridagi uchta sodda algoritmdan ancha tezroq.

Merge Sort (Birlashtirib Saralash)

Ro'yxatni ikkiga bo'ladi, har bir yarmini alohida saralaydi, so'ng ikki tartiblangan yarmni birlashtiradi. Bu bo'lish toki bitta elementgacha davom etadi.
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    orta = len(arr) // 2
    chap = merge_sort(arr[:orta])
    ong = merge_sort(arr[orta:])
    return birlashtir(chap, ong)
# Ro'yxatni o'rtasidan bo'lib, har yarmini saralab, qayta birlashtiradi

Quick Sort (Tez Saralash)

Bitta element "tayanch" (pivot) sifatida tanlanadi. Undan kichiklar chapga, kattalar o'ngga ajratiladi. So'ng har ikki tomon uchun shu jarayon takrorlanadi.
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    tayanch = arr[len(arr) // 2]
    chap = [x for x in arr if x < tayanch]
    orta = [x for x in arr if x == tayanch]
    ong = [x for x in arr if x > tayanch]
    return quick_sort(chap) + orta + quick_sort(ong)
# Tayanchdan kichiklar chapga, kattalar o'ngga; so'ng har tomonni qayta saralaydi

Big O bilan Taqqoslash

Big O — algoritm elementlar soni (n) oshganda qancha sekinlashishini ko'rsatadi. O(n²) sekin (n katta bo'lsa juda sekin), O(n log n) esa ancha tez.

Sodda tushuntirish

n = 1000 element uchun:
• O(n²) ≈ 1 000 000 qadam (sekin)
• O(n log n) ≈ 10 000 qadam (100 marta tezroq!)
Shuning uchun katta ma'lumotda Merge yoki Quick Sort ishlatiladi.
