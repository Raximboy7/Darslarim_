Vektor Amallar — Bir Vaqtning O'zida

NumPy array bilan amal qilganingizda, amal har bir elementga alohida qo'llaniladi. Oddiy list uchun bunday qila olmaysiz — NumPy esa buni juda tez va sodda bajaradi. Bunga vektor amal deyiladi.

Array + Array

import numpy as np
a = np.array([1, 2, 3])
b = np.array([10, 20, 30])
print(a + b)
# Har element o'z jufti bilan qo'shiladi: 1+10, 2+20, 3+30
>>> [11 22 33]

Array * Son (skalyar amal)

a = np.array([1, 2, 3])
print(a * 2)
print(a + 100)
# Son har bir elementga alohida qo'llaniladi
>>> [2 4 6]
>>> [101 102 103]

Broadcasting — Sodda Tushuncha

Broadcasting — NumPy o'lchami har xil bo'lgan narsalarni avtomatik moslab amal qilishi. Masalan, a * 2 da 2 bitta son, lekin NumPy uni "cho'zib" har bir elementga qo'llaydi. Shuning uchun siz sikl yozishingiz shart emas.
narxlar = np.array([100, 200, 300])
chegirma = narxlar * 0.9 # 10% chegirma
print(chegirma)
# 0.9 soni har bir narxga ko'paytiriladi
>>> [ 90. 180. 270.]
Eslab qoling

Vektor amallar uchun for sikl kerak emas. NumPy hammasini bir o'zi, juda tez bajaradi.

Indexing — Bitta Elementni Olish

Xuddi list kabi, array elementlari 0 dan sanaladi. Manfiy indeks oxiridan sanaydi.
arr = np.array([10, 20, 30, 40])
print(arr[0]) # birinchi
print(arr[-1]) # oxirgi
# arr[0] = 10, arr[-1] = 40
>>> 10
>>> 40

Slicing — Bo'lakni Kesib Olish

arr[boshi:oxiri] — boshidan oxirigacha (oxiri kirmaydi) kesib oladi.
KodNatijaIzoh
arr[1:3][20 30]1 va 2-indekslar
arr[:2][10 20]boshidan 2 gacha
arr[2:][30 40]2 dan oxirigacha
arr[::2][10 30]har 2-element

Boolean Indexing — Shartga Mos Elementlar

Eng kuchli imkoniyatlardan biri! Array ichidan shartga mos elementlarnigina ajratib olishingiz mumkin. arr > 5 har element uchun True/False qaytaradi, keyin shu maskani array ichiga qo'yasiz.
arr = np.array([3, 8, 1, 9, 5])
print(arr > 5)
print(arr[arr > 5])
# Birinchi qator: maska (True/False), ikkinchi qator: faqat mos elementlar
>>> [False True False True False]
>>> [8 9]

2D Array — Matritsa (Jadval)

2D array — bu satr va ustunlardan iborat jadval (matritsa). Element olish uchun arr[satr, ustun] ishlatiladi.
m = np.array([[1, 2, 3],
            [4, 5, 6]])
print(m[0, 1]) # 0-satr, 1-ustun
print(m[1, 2]) # 1-satr, 2-ustun
# m[0,1] = 2 (birinchi satr, ikkinchi son), m[1,2] = 6
>>> 2
>>> 6
Maslahat

Butun satrni olish: m[0] → [1 2 3]. Butun ustunni olish: m[:, 0] → [1 4].

Statistik Funksiyalar

NumPy bitta buyruq bilan butun array bo'yicha hisob-kitob qiladi.
FunksiyaVazifasiMisol [1,2,3,4]

baholar = np.array([85, 90, 70, 95])
print(baholar.mean()) # o'rtacha baho
print(baholar.max()) # eng yuqori
# Sikl yozmasdan butun array bo'yicha hisoblaymiz
>>> 85.0
>>> 95
