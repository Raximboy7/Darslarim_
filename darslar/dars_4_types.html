O'zgaruvchi Nima?

O'zgaruvchi — bu xotirada qiymat saqlash uchun nomi berilgan joy. Kompyuter o'zgaruvchini ko'rganda, uning qiymatini eslaydi va qo'llaydi.

O'zgaruvchi yaratish sodda:

ism = "Ali"

Bu degani: ism nomli o'zgaruvchiga "Ali" qiymatini qo'ydik.

Ma'lumot Turlari (Data Types)

Python'da turli xil ma'lumot turlari bor. Kompyuter har birni boshqacha ishlab chiqaradi.

1️⃣ int (Butun son)

int — Butun son

Minus (-), nol (0), va musbat (+) raqamlar.

yosh = 25 # int turi
>>> yosh = 25
>>> type(yosh) # <class 'int'>

Misollar: 10, -5, 0, 1000

2️⃣ float (O'nli Kasr Raqam)

float - O'nli kasr raqam

Nuqta (.) bilan o'nli kasr raqamlar.

bo'yi = 1.75 # float turi
>>> bo'yi = 1.75
>>> type(bo'yi) # <class 'float'>

Misollar: 3.14, -2.5, 0.5

3️⃣ str (String - Matn)

str - String (matn)

Tirnoq ichidagi barcha narsalar (harflar, belgilar, hatto raqamlar).

ism = "Muhammad" # str turi
>>> ism = "Muhammad"
>>> type(ism) # <class 'str'>

Misollar: "Salom", "123" (tirnoq ichida), "" (bo'sh)

4️⃣ bool (Boolean - Rost/Yolg'on)

bool — Boolean (Rost yoki Yolg'on)

Faqat ikkita qiymat: True yoki False

faol = True # bool turi
>>> faol = True
>>> type(faol) # <class 'bool'>

Misollar: True, False (katta harfda!)

Turlarni Taqqoslash

Tur Nomi Misol Tirnoq?
int Butun son 42, -10, 0 YO'Q
float O'nli kasr 3.14, -2.5 YO'Q
str Matn "Salom", "123" HA!
bool Rost/Yolg'on True, False YO'Q

Turni Bilish: type()

O'zgaruvchining turi nima ekanligini bilish uchun type() funksiyasini ishlating.
print(type(42))
>>> <class 'int'>
print(type("Salom"))
>>> <class 'str'>

Turlarni Almashtirish (Type Conversion)

Ba'zida bir turni boshqa turga o'tkazish kerak. Buni int(), float(), str() funksiyalari bilan qilamiz.

String-dan int-ga

son = int("25") # "25" soni bo'ladi
>>> son = 25 (int turi)

int-dan string-ga

matn = str(42) # 42 matni bo'ladi
>>> matn = "42" (str turi)

float-dan int-ga

son = int(3.99) # Kasr qismini butunlashtiradi
>>> son = 3 (o'nli qismi o'chiriladi!)
Eslaydigan Narsa

• int — raqam (butun)
• float — raqam (o'nli)
• str — matn (TIRNOQ ICHIDA!)
• bool — True yoki False

Misol 1: int Turi

Misol 2: float Turi

Misol 3: str Turi

Misol 4: bool Turi

Misol 5: Turni Almashtirish

String → int
son = int("42")
int → str
matn = str(42)
float → int
son = int(3.99) # 3 bo'ladi

Qo'shish: Raqam va String

❌ Xato: int + str
print(42 + "5") # XATO!

Xato: Raqam va matnni to'g'ridan-to'g'ri qo'sha olmaysiz!

✓ To'g'ri: Turni almashtirib
print(42 + int("5")) # 47
>>> 47
