Proyekt 3 — Ob-havo Ilovasi

Modul 16 — Yakuniy Proyektlar | Dars 106

Biz Nima Quramiz?

Bu darsda Ob-havo ilovasi quramiz. Foydalanuvchi shahar nomini kiritadi, dasturimiz esa internetdan o'sha shaharning ob-havosini olib keladi va chiroyli ko'rinishda chiqaradi: harorat, holat (quyoshli, yomg'irli...), namlik va shamol.
Nima o'rganamiz?

• Internetdan ma'lumot olish — requests kutubxonasi
• API va API kalit (key) nima ekani
• JSON javobni tahlil qilish (harorat, holatni ajratib olish)
• Natijani chiroyli ko'rsatish
• Xatolarni tutish: shahar topilmadi, internet yo'q

API Nima?

API — bu boshqa server bilan gaplashish usuli. Biz serverga "Toshkent ob-havosini ber" deb so'rov yuboramiz, server esa bizga javob qaytaradi. Bu xuddi restoranda ofitsiantga buyurtma berishga o'xshaydi: siz so'raysiz — ular keltiradi.
Ko'pchilik API'lar API kalit (key) talab qiladi. Bu — sizning shaxsiy parolingiz. Ob-havo saytida ro'yxatdan o'tib, bepul kalit olasiz va uni so'rovga qo'shasiz. Kalitsiz server sizni tanimaydi.
requests kutubxonasi

Python'da internetga so'rov yuborish uchun requests kutubxonasidan foydalanamiz. Uni o'rnatish:
pip install requests

1-qadam: Kutubxonani Ulash va Kalit

Avval kerakli vositalarni tayyorlaymiz: requestsni ulaymiz va o'z API kalitimizni o'zgaruvchiga yozamiz.
import requests

API_KEY = "sizning_kalitingiz"  # ob-havo saytidan olingan
BASE_URL = "https://api.openweathermap.org/data/2.5/weather"
# requests bilan internetga chiqamiz. API_KEY — bizning shaxsiy kalitimiz.

2-qadam: So'rov Yuborish

Endi shahar nomini so'raymiz va serverga so'rov yuboramiz. params — bu so'rovga qo'shadigan qo'shimcha ma'lumotlar (shahar, kalit, o'lchov birligi).
shahar = input("Shahar nomini kiriting: ")

params = {
    "q": shahar,
    "appid": API_KEY,
    "units": "metric",  # Selsiy uchun
    "lang": "uz"
}

javob = requests.get(BASE_URL, params=params)
# units="metric" — haroratni Selsiyda beradi. javob ichida server qaytargan natija bor.

3-qadam: JSON Javobni Tahlil Qilish

Server javobni JSON formatida qaytaradi — bu Python lug'atiga juda o'xshash. .json() bilan uni lug'atga aylantiramiz, so'ng kerakli qiymatlarni kalit orqali ajratib olamiz.
malumot = javob.json()  # JSON -> lug'at

harorat = malumot["main"]["temp"]
holat = malumot["weather"][0]["description"]
namlik = malumot["main"]["humidity"]
shamol = malumot["wind"]["speed"]
# JSON ichida lug'atlar va ro'yxatlar bor. "weather" ro'yxat, shuning uchun [0] olamiz.
JSON javob namunasi:
{
  "main": {"temp": 24.5, "humidity": 40},
  "weather": [{"description": "ochiq osmon"}],
  "wind": {"speed": 3.2}
}

4-qadam: Natijani Chiroyli Ko'rsatish

Endi olgan ma'lumotlarimizni foydalanuvchiga tushunarli ko'rinishda chiqaramiz.
print(f"📍 {shahar} ob-havosi")
print(f"🌡️ Harorat: {harorat}°C")
print(f"☁️ Holat: {holat}")
print(f"💧 Namlik: {namlik}%")
print(f"💨 Shamol: {shamol} m/s")
# f-string bilan qiymatlarni matn ichiga joylaymiz. Emoji natijani jonli qiladi.
>>> 📍 Toshkent ob-havosi
>>> 🌡️ Harorat: 24.5°C
>>> ☁️ Holat: ochiq osmon
>>> 💧 Namlik: 40%
>>> 💨 Shamol: 3.2 m/s

5-qadam: Xatolarni Tutish

Nima bo'lishi mumkin? Shahar nomi xato bo'lsa, internet uzilgan bo'lsa yoki kalit noto'g'ri bo'lsa — dastur "yiqilmasligi" kerak. Buning uchun try / except ishlatamiz va server qaytargan status_codeni tekshiramiz.
try:
    javob = requests.get(BASE_URL, params=params)
    if javob.status_code == 200:
        malumot = javob.json()
        print(malumot["main"]["temp"])
    elif javob.status_code == 404:
        print("❌ Bunday shahar topilmadi!")
    else:
        print("⚠️ Xatolik:", javob.status_code)
except requests.exceptions.ConnectionError:
    print("📡 Internet aloqasi yo'q!")
# 200 = muvaffaqiyatli, 404 = shahar topilmadi. ConnectionError = internet yo'q.
Status kodlar nimani anglatadi?

• 200 — Hammasi joyida, ma'lumot keldi
• 404 — Shahar topilmadi
• 401 — Kalit noto'g'ri
• ConnectionError — Internet yo'q

Misol 1: Ob-havoni Ko'rsatish

Quyida tayyor ilovamizning ishlashini sinab ko'ring. Shahar nomini yozing yoki tezkor tugmalardan birini bosing (bu yerda soxta ma'lumotlar ishlatilgan).

Misol 2: JSON Javobni Tahlil Qilish

Server JSON qaytaradi. Tugmani bosing — JSON ichidan har bir qiymat qanday kalit orqali ajratib olinishini ko'rasiz.

Misol 3: Xatoni Tutish

Mavjud bo'lmagan shahar nomini yozsangiz, server 404 qaytaradi. Quyida xato qanday ushlanishini sinab ko'ring.

To'liq Dastur

import requests

API_KEY = "sizning_kalitingiz"
URL = "https://api.openweathermap.org/data/2.5/weather"

shahar = input("Shahar: ")
params = {"q": shahar, "appid": API_KEY, "units": "metric"}

try:
    javob = requests.get(URL, params=params)
    if javob.status_code == 200:
        m = javob.json()
        print(f"🌡️ {m['main']['temp']}°C")
        print(f"☁️ {m['weather'][0]['description']}")
    elif javob.status_code == 404:
        print("❌ Shahar topilmadi")
except requests.exceptions.ConnectionError:
    print("📡 Internet yo'q")
# Bu to'liq, ishlaydigan dastur. Faqat o'z API kalitingizni qo'ying.
