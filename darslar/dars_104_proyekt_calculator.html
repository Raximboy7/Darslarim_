Proyekt 1 — Kalkulyator

Modul 16 — Yakuniy Proyektlar | Dars 104

Bu Darsda Nima Yasaymiz?

Endi o'rgangan bilimlaringizni birlashtirib, haqiqiy kalkulyator yasaymiz! Kalkulyator to'rt amalni bajaradi: qo'shish, ayirish, ko'paytirish va bo'lish. Avval sodda konsol versiyasini, keyin esa tugmalari bor oyna (GUI) versiyasini tushunamiz.
Proyekt nima?

Proyekt — bu bir nechta tushunchani birlashtirib, to'liq ishlaydigan dastur yasash. Bu yerda funksiyalar, shartlar, sikllar va xatolarni tutishni bir joyda ishlatamiz.

1-bosqich: Hisoblash Logikasi

Kalkulyatorning yuragi — bu hisoblash. Har bir amal uchun alohida funksiya yozamiz. Funksiya ikkita son oladi va natijani qaytaradi.
def qoshish(a, b):
    return a + b

def ayirish(a, b):
    return a - b

def kopaytirish(a, b):
    return a * b

def bolish(a, b):
    return a / b
# Har amal — alohida funksiya. Ikki son oladi, natija qaytaradi.
>>> qoshish(2, 3)
5
>>> kopaytirish(4, 5)
20

2-bosqich: Nolga Bo'lishni Tekshirish

Bo'lishda katta muammo bor: songa 0 ga bo'lib bo'lmaydi! Buni tekshirmasak, dastur xato bilan to'xtaydi. Shuning uchun try / except bilan xatoni tutamiz.
def bolish(a, b):
    try:
        return a / b
    except ZeroDivisionError:
        return "Xato: nolga bo'lib bo'lmaydi!"
# b = 0 bo'lsa, dastur yiqilmaydi — chiroyli xabar qaytaradi.
>>> bolish(10, 2)
5.0
>>> bolish(10, 0)
Xato: nolga bo'lib bo'lmaydi!

3-bosqich: Konsol Versiyasi (To'liq)

Endi hammasini birlashtiramiz. Foydalanuvchidan ikkita son va amalni input() bilan so'raymiz, keyin natijani chiqaramiz.
# Amalga qarab to'g'ri funksiyani tanlaydi
def hisobla(a, b, amal):
    if amal == "+":
        return a + b
    elif amal == "-":
        return a - b
    elif amal == "*":
        return a * b
    elif amal == "/":
        if b == 0:
            return "Nolga bo'lib bo'lmaydi!"
        return a / b
    else:
        return "Noma'lum amal!"

son1 = float(input("Birinchi son: "))
amal = input("Amal (+ - * /): ")
son2 = float(input("Ikkinchi son: "))
print("Natija:", hisobla(son1, son2, amal))
# float() — kasr sonlar uchun. hisobla() amalni tanlaydi va natijani qaytaradi.
Birinchi son: 12
Amal (+ - * /): *
Ikkinchi son: 3
Natija: 36.0

4-bosqich: Oyna (GUI) Versiyasi — tkinter

Konsol yaxshi, lekin haqiqiy kalkulyatorda tugmalar bo'ladi. Python'da oyna yasash uchun tkinter kutubxonasi ishlatiladi. U Python bilan birga keladi — alohida o'rnatish kerak emas.
Elementtkinter'daVazifasi
Tk()OynaAsosiy oyna (window)
EntryDisplayNatija ko'rsatadigan maydon
ButtonTugmaRaqam yoki amal tugmasi
.grid()JoylashuvTugmalarni qatorlarga joylaydi
import tkinter as tk

oyna = tk.Tk()
oyna.title("Kalkulyator")

# Display — natija shu yerda ko'rinadi
display = tk.Entry(oyna, font=("Arial", 20))
display.grid(row=0, column=0, columnspan=4)

# Tugma bosilganda display'ga belgi qo'shadi
def bos(belgi):
    display.insert(tk.END, belgi)

tk.Button(oyna, text="7",
    command=lambda: bos("7")).grid(row=1, column=0)

oyna.mainloop()
# Tk() oyna yaratadi, Entry display, Button tugma. mainloop() oynani ochiq tutadi.
= tugmasi qanday ishlaydi?

Foydalanuvchi "12+3" yozadi. "=" bosilganda eval("12+3") ifodani hisoblaydi va natijani (15) display'ga yozadi. Xato bo'lsa "Xato" deb ko'rsatamiz.

Ikkita son va amalni tanlang — kalkulyatorning ichida qanday funksiya chaqirilishini va natija qanday chiqishini ko'ring.

Misol 3: Nolga Bo'lish Xatosini Tutish

Ikkinchi songa 0 yozib, bo'lishga harakat qiling. try / except xatoni qanday ushlab, dasturni yiqilishdan saqlashini ko'ring.
Eslab Qoling

• Har amalni alohida funksiya qiling — kod tartibli bo'ladi
• Nolga bo'lishni try/except yoki if b == 0 bilan tuting
• input() matn beradi — float() bilan songa aylantiring
• GUI uchun tkinter: Tk() oyna, Entry display, Button tugma
