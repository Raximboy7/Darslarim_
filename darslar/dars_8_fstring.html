f-string Nima?

f-string — bu matn ichiga o'zgaruvchilarni qo'yishning eng oson va zamonaviy usuli. Matndan oldin f harfi qo'yiladi, o'zgaruvchilar esa { } qavslar ichida yoziladi.
ism = "Ali"
print(f"Salom, {ism}!")
# {ism} → Ali bilan almashadi
>>> Salom, Ali!

Eski Usul vs f-string

Ilgari matnni + bilan birlashtirib, sonlarni str() ga aylantirish kerak edi. f-string buni ancha soddalashtiradi.
UsulKod
Eski (+)"Yosh: " + str(yosh)
format()"Yosh: {}".format(yosh)
f-string ✓f"Yosh: {yosh}"

{ } Ichida Ifoda Yozish

Qavslar ichiga shunchaki o'zgaruvchi emas, butun ifoda (hisob-kitob) ham yozish mumkin.
a = 5; b = 3
print(f"Yig'indi: {a + b}")
# {a + b} → 8 hisoblanadi
>>> Yig'indi: 8

Sonlarni Formatlash

O'nli kasrni yaxlitlash: :.2f
narx = 3.14159
print(f"{narx:.2f}")

Nuqtadan keyin 2 ta raqam qoldiradi → 3.14

Nol bilan to'ldirish: :05d
raqam = 42
print(f"{raqam:05d}")

5 xonali qilib, oldini nol bilan to'ldiradi → 00042

Formatlash Belgilarini Solishtirish

FormatMisol (qiymat)Natija
{x:.2f}3.141593.14
{x:.0f}3.74
{x:05d}4200042
{x:,}10000001,000,000
{x:.1%}0.25625.6%
