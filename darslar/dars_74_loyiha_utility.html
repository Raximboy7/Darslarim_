Loyiha: Funksiyalar kutubxonasi

Bu loyihada biz bir nechta foydali funksiyalardan iborat kichik kutubxona (utils.py) yaratamiz. Har bir funksiya bitta aniq vazifani bajaradi va boshqa loyihalarda qayta ishlatilishi mumkin.

Maqsad — o'rganilgan barcha bilimlarni (default parametr, *args, toza funksiya, comprehension) birlashtirish.

1. Matn bilan ishlash funksiyalari

def sozlarni_sana(matn):
    """Matndagi so'zlar sonini qaytaradi."""
    return len(matn.split())

def teskari(matn):
    """Matnni teskari qaytaradi."""
    return matn[::-1]

print(sozlarni_sana("salom dunyo bugun"))
print(teskari("python"))
# split() so'zlarga ajratadi, [::-1] teskari o'giradi
>>> 3
>>> nohtyp

2. Sonlar bilan ishlash funksiyalari

def ortacha(*sonlar):
    """Istalgan sondagi argumentlar o'rtachasi."""
    if not sonlar:
        return 0
    return sum(sonlar) / len(sonlar)

def tub_mi(n):
    """Son tub bo'lsa True qaytaradi."""
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

print(ortacha(10, 20, 30))
print(tub_mi(7), tub_mi(8))
# *sonlar — ixtiyoriy son argument; tub_mi bo'luvchilarni tekshiradi
>>> 20.0
>>> True False

3. Ro'yxat bilan ishlash funksiyalari

def noyob(royxat):
    """Takrorlarni o'chirib, tartibni saqlaydi."""
    korilgan = []
    for x in royxat:
        if x not in korilgan:
            korilgan.append(x)
    return korilgan

def juftlar(royxat):
    """Faqat juft sonlarni qaytaradi."""
    return [x for x in royxat if x % 2 == 0]

print(noyob([1, 2, 2, 3, 1]))
print(juftlar([1, 2, 3, 4, 5, 6]))
# juftlar — list comprehension bilan filtrlash
>>> [1, 2, 3]
>>> [2, 4, 6]

4. Kutubxonani ishlatish

Barcha funksiyalar tayyor. Endi ularni birga qo'llaymiz: 

matn = "python juda qiziqarli til python"

print("So'zlar soni:", sozlarni_sana(matn))
print("Noyob so'zlar:", noyob(matn.split()))
print("O'rtacha (3 son):", ortacha(4, 8, 12))
# funksiyalarni birlashtirib ishlatish
>>> So'zlar soni: 5
>>> Noyob so'zlar: ['python', 'juda', 'qiziqarli', 'til']
>>> O'rtacha (3 son): 8.0

Amaliyot va xulosa

Kutubxonani kengaytiring

• eng_katta(*sonlar) — eng katta sonni qaytaruvchi funksiya qo'shing.
• bosh_harf(matn) — har bir so'z bosh harfini katta qiladigan funksiya yozing.
• Har bir funksiya bitta vazifa bajarsin va toza bo'lsin.

Nimalarni ishlatdik

Default parametr, *args, list comprehension, toza funksiyalar va docstring ("""izoh""") — hammasi bitta loyihada.
