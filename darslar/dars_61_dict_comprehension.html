Dict comprehension

Modul 5 — Comprehensions | Dars 61

Dict comprehension nima?

Dict comprehension — bu bitta qatorda lug'at (dictionary) yaratishning qisqa va tez usuli. Xuddi list comprehension'ga o'xshaydi, lekin natijada kalit: qiymat juftliklari hosil bo'ladi.

Umumiy shakli: {kalit: qiymat for element in ketma-ketlik}. Ya'ni har bir element uchun bitta kalit va bitta qiymat aniqlanadi.

Oddiy tsikldan comprehension'ga

Avval odatdagi for yordamida sonlar va ularning kvadratlaridan lug'at yasaymiz: 

# Oddiy usul
kvadratlar = {}
for son in range(1, 6):
    kvadratlar[son] = son ** 2

print(kvadratlar)
# 4 qator kod, natija bir xil

Endi xuddi shu narsani dict comprehension bilan bir qatorda: 

kvadratlar = {son: son**2 for son in range(1, 6)}
print(kvadratlar)
# son — kalit, son**2 — qiymat
>>> {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

Shart bilan filtrlash

Comprehension oxiriga if qo'shib, faqat kerakli elementlarni tanlashimiz mumkin. Quyida faqat juft sonlar olinadi: 

juft_kvadrat = {n: n**2 for n in range(1, 7) if n % 2 == 0}
print(juft_kvadrat)
# if n % 2 == 0 — faqat juft sonlar o'tadi
>>> {2: 4, 4: 16, 6: 36}
Ikki ro'yxatni birlashtirish

zip() yordamida ikki ro'yxatdan lug'at yasash juda qulay: 

nomlar = ["olma", "nok", "uzum"]
narxlar = [12, 9, 20]

menyu = {nom: narx for nom, narx in zip(nomlar, narxlar)}
print(menyu)
# zip juftlab beradi: ("olma", 12) ...
>>> {'olma': 12, 'nok': 9, 'uzum': 20}

Amaliyot

O'zingiz sinab ko'ring

1. 1 dan 10 gacha sonlar va ularning kublari (n**3) lug'atini tuzing.
2. Berilgan so'zdagi har bir harf necha marta uchrashini sanang: 

soz = "salom"
harflar = {h: soz.count(h) for h in soz}
print(harflar)
# har bir harf uchun count() chaqiriladi
>>> {'s': 1, 'a': 1, 'l': 1, 'o': 1, 'm': 1}

Xulosa

Eslab qoling

• Shakli: {kalit: qiymat for x in ketma-ketlik}
• if bilan elementlarni filtrlash mumkin
• zip() ikki ro'yxatni lug'atga aylantiradi
• Kod qisqa, tez va o'qishga oson bo'ladi
