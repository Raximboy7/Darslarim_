Parametr va Argument Nima?

Parametr — bu funksiya e'lon qilinganda qavs ichida yoziladigan o'zgaruvchi nomi. Argument — bu funksiyani chaqirganda unga beriladigan haqiqiy qiymat. Oddiy qilib aytganda: parametr — bu "quti", argument — bu "qutiga solinadigan narsa".
def salomlash(ism):
    print("Salom, " + ism)

salomlash("Ali")
# ism — parametr
# "Ali" — argument
>>> Salom, Ali

1. Positional (Tartib Bo'yicha)

Argumentlar tartib bo'yicha beriladi: birinchi qiymat birinchi parametrga, ikkinchi qiymat ikkinchi parametrga tushadi. Tartibni adashtirsangiz, natija ham o'zgaradi.
def bolish(birinchi, ikkinchi):
    return birinchi / ikkinchi

print(bolish(10, 2))
# birinchi = 10
# ikkinchi = 2
# 10 / 2
>>> 5.0

2. Keyword (Nom Bilan)

Argumentni nom=qiymat ko'rinishida bersangiz, tartib muhim emas. Python qaysi qiymat qaysi parametrga tegishli ekanini nomidan biladi.
def tanishtir(ism, shahar):
    print(ism, "—", shahar)

tanishtir(shahar="Toshkent", ism="Ali")
# tartib teskari,
# lekin nom bilan
# to'g'ri tushadi
>>> Ali — Toshkent

3. Standart (Default) Qiymat

Parametrga standart qiymat berib qo'yish mumkin. Agar argument berilmasa, Python standart qiymatdan foydalanadi. Bu parametrni "ixtiyoriy" qiladi.
def salom(ism, til="uz"):
    if til == "uz":
        print("Salom, " + ism)
    else:
        print("Hello, " + ism)

salom("Ali")
salom("Bob", "en")
# til berilmadi → "uz"





# 1-chaqiruv
# 2-chaqiruv: til = "en"
>>> Salom, Ali
>>> Hello, Bob
⚠️ Muhim qoida

Standart qiymatli parametrlar har doim oxirida turishi lozim. Majburiy parametr (standart qiymatsiz) standartlidan keyin kelolmaydi.

4. *args — Ko'p Pozitsion Argument

Funksiyaga nechta argument kelishini oldindan bilmasangiz, *args ishlatiladi. Barcha qo'shimcha argumentlar bitta tuple ichiga yig'iladi.
def qoshish(*sonlar):
    return sum(sonlar)

print(qoshish(1, 2, 3, 4, 5))
# sonlar = (1, 2, 3, 4, 5)
# sum() yig'indini
# qaytaradi
>>> 15
Eslatma

args shunchaki nom — istalgan nom qo'yish mumkin. Muhimi — oldidagi bitta yulduzcha *. Lekin an'anaga ko'ra *args deb yoziladi.

5. **kwargs — Nomli Argumentlar

Ikkita yulduzcha **kwargs — bu nomli argumentlarni dictionary (lug'at) ichiga yig'adi. Har bir nom kalit, qiymat esa qiymat bo'ladi.
def info(**malumotlar):
    for k, v in malumotlar.items():
        print(k, "=", v)

info(ism="Ali", yosh=20)
# malumotlar =
# {"ism": "Ali",
# "yosh": 20}
>>> ism = Ali
>>> yosh = 20

To'g'ri tartib

Bir funksiyada barchasini birga ishlatganda tartib quyidagicha bo'lishi lozim:

def f(a, b=2, *args, **kwargs):

oddiy → standart → *args → **kwargs

Keng Tarqalgan Xato

❌ Xato: keyword'dan keyin positional
tanishtir(ism="Ali", "Toshkent") # XATO!

Nomli argumentdan keyin tartib bo'yicha argument berib bo'lmaydi. Python SyntaxError beradi.

✓ To'g'ri: avval positional, keyin keyword
tanishtir("Ali", shahar="Toshkent")
>>> Ali — Toshkent
Eslab Qoling

• Positional — tartib bo'yicha, keyword — nom bilan
• Standart qiymatli parametrlar oxirida turadi
• *args → tuple, **kwargs → dict
• Tartib: oddiy → standart → *args → **kwargs

Misol 1: *args Yig'indisi

Kod: def qoshish(*sonlar): return sum(sonlar)

Misol 2: Keyword Argument Tartibi

Tartibni o'zgartirsangiz ham nom bilan natija bir xil.

Misol 3: **kwargs Ma'lumot Kartasi

Kod: def info(**malumotlar): for k,v in malumotlar.items()

Misol 4: Standart Qiymat

Kod: def takrorla(ism, karra=1)

Misol 5: Eng Katta Sonni Topish

Kod: def eng_katta(*sonlar): return max(sonlar)
