Saralash — 2: Merge, Quick

Modul 12 — Algoritmlar va Strukturalar | Dars 134

Kirish — nima uchun tezroq saralash?

Bubble va Selection sort oddiy, lekin O(n^2) — katta ma'lumotda juda sekin. Merge sort va Quick sort esa O(n log n) darajasida ishlaydi va "bo'l va hukmronlik qil" (divide and conquer) g'oyasiga asoslanadi.

G'oya oddiy: katta masalani kichik bo'laklarga bo'lamiz, har bir bo'lakni yechamiz, keyin natijalarni birlashtiramiz.

Merge Sort — birlashtirib saralash

Ro'yxatni ikkiga bo'lamiz, har yarmini rekursiv saralaymiz, so'ng ikkita saralangan yarmni birlashtiramiz. 

def merge_sort(a):
    if len(a) <= 1:            # bir elementli — allaqachon saralangan
        return a
    orta = len(a) // 2
    chap = merge_sort(a[:orta])   # chap yarmini saralaymiz
    ong = merge_sort(a[orta:])    # o'ng yarmini saralaymiz
    return birlashtir(chap, ong)

def birlashtir(chap, ong):
    natija = []
    i = j = 0
    while i < len(chap) and j < len(ong):
        if chap[i] <= ong[j]:      # kichigini oldin qo'yamiz
            natija.append(chap[i]); i += 1
        else:
            natija.append(ong[j]); j += 1
    natija += chap[i:]            # qolganlarini qo'shamiz
    natija += ong[j:]
    return natija

print(merge_sort([5, 2, 9, 1, 7]))
Bo'lish log n marta, har darajada birlashtirish n ta amal → jami O(n log n). Barqaror (stable) va bashorat qilinadigan.
>>> [1, 2, 5, 7, 9]

Quick Sort — tayanch element bilan

Bitta tayanch (pivot) element tanlaymiz, ro'yxatni undan kichik va katta qismlarga ajratamiz, har qismni rekursiv saralaymiz. 

def quick_sort(a):
    if len(a) <= 1:
        return a
    tayanch = a[len(a) // 2]        # o'rtadagi elementni tanladik
    kichik = [x for x in a if x < tayanch]
    teng   = [x for x in a if x == tayanch]
    katta  = [x for x in a if x > tayanch]
    return quick_sort(kichik) + teng + quick_sort(katta)

print(quick_sort([5, 2, 9, 1, 7]))
O'rtacha O(n log n), lekin eng yomon holatda (yomon tayanch) O(n^2). Amalda juda tez va kam xotira ishlatadi.
>>> [1, 2, 5, 7, 9]

Amaliyot

Mashqlar

1. merge_sort ni satrlar ro'yxatiga qo'llang.
2. Quick sort da tayanch sifatida birinchi elementni tanlang va farqni ko'ring.
3. Har ikkala funksiyaga print qo'shib, rekursiya qadamlarini kuzating.

Xulosa

Esda saqlang

• Ikkalasi ham "bo'l va hukmronlik qil" g'oyasi.
• Merge — barqaror, kafolatli O(n log n), ko'proq xotira.
• Quick — amalda tezroq, kam xotira, lekin eng yomon holat O(n^2).
• Real loyihada o'zingiz yozmang — sorted() dan foydalaning.
