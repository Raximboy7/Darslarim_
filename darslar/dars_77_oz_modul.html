O'z modulingizni yaratish

O'tgan darsda modul — bu oddiy .py fayl ekanini bildik. Endi o'zimiz modul yaratib, uni boshqa fayldan import qilib ishlatamiz.

Modul yaratish uchun hech qanday maxsus buyruq shart emas — shunchaki funksiyalaringizni alohida faylga yozing, tamom.

1-qadam: modul faylini yozish

Yangi fayl yaratamiz: matematika.py

# matematika.py

def qosh(a, b):
    return a + b

def kopaytir(a, b):
    return a * b

PI = 3.14159
# Bu fayl ichida ikkita funksiya va bitta o'zgaruvchi bor. Bu — bizning modulimiz.

2-qadam: modulni import qilish

Xuddi shu papkada main.py yaratamiz va modulni chaqiramiz. E'tibor bering: .py yozilmaydi.

# main.py
import matematika

natija = matematika.qosh(5, 3)
print(natija)

print(matematika.PI)
# import matematika — butun modulni yuklaydi. Undagi narsalarga modul.nomi orqali murojaat qilamiz.
>>> 8
>>> 3.14159

Amaliyot: kalkulyator moduli

Keling, kichik kalkulyator.py moduli yasab, uni ishlatamiz:

# kalkulyator.py
def bol(a, b):
    if b == 0:
        return "Nolga bo'lib bo'lmaydi"
    return a / b
# Modul ichida shartlar, sikllar — istalgan kod bo'lishi mumkin.
# main.py
import kalkulyator

print(kalkulyator.bol(10, 2))
print(kalkulyator.bol(10, 0))
# Modulni bir marta yozib, xohlagancha marta ishlatasiz.
>>> 5.0
>>> Nolga bo'lib bo'lmaydi

Muhim: __name__ == "__main__"

Nega kerak?

Modulni import qilganda, undagi barcha kod ishga tushadi. Agar modul faylini alohida sinash uchun kod yozsangiz, u import paytida ham ishlab qolmasligi uchun shunday yozing:

# matematika.py
def qosh(a, b):
    return a + b

if __name__ == "__main__":
    # faqat fayl to'g'ridan-to'g'ri ishga tushsa bajariladi
    print(qosh(2, 2))
# import qilinganda bu blok ishlamaydi, to'g'ridan-to'g'ri "python matematika.py" qilinganda ishlaydi.

Xulosa

Eslab qoling

• Modul yaratish = funksiyalarni .py faylga yozish
• import matematika orqali yuklanadi (.py yozilmaydi)
• Ichidagilarga matematika.qosh(...) tarzida murojaat qilinadi
• Modul va uni ishlatuvchi fayl bir papkada bo'lsin

Keyingi darsda import ning turli usullarini o'rganamiz.
