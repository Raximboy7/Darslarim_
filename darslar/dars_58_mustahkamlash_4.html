Mashq 1 — Takrorlanmas elementlar (set)

Masala: Berilgan ro'yxatdan takrorlanuvchi elementlarni olib tashlab, faqat noyob qiymatlarni chiqaring. 
sonlar = [1, 2, 2, 3, 3, 3, 4, 5, 5]

noyob = list(set(sonlar))
noyob.sort()

print("Noyob elementlar:", noyob)
print("Soni:", len(noyob))
set() takrorlarni avtomatik o'chiradi. Keyin list() bilan qayta ro'yxatga aylantirib, sort() bilan tartiblaymiz.
>>> Noyob elementlar: [1, 2, 3, 4, 5]
>>> Soni: 5

Mashq 2 — Lug'atlar ro'yxatini saralash

Masala: Talabalarni bahosi bo'yicha kamayish tartibida saralang (list of dicts). 
talabalar = [
    {"ism": "Ali", "baho": 4},
    {"ism": "Vali", "baho": 5},
    {"ism": "Guli", "baho": 3}
]

talabalar.sort(key=lambda t: t["baho"], reverse=True)

for t in talabalar:
    print(t["ism"], "-", t["baho"])
key=lambda t: t["baho"] — har bir lug'atning "baho" qiymatiga qarab saralaydi. reverse=True — kattadan kichikka.
>>> Vali - 5
>>> Ali - 4
>>> Guli - 3

Mashq 3 — Matritsa yig'indisi

Masala: Nested list (matritsa) dagi barcha sonlar yig'indisini toping. 
matritsa = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

jami = 0
for qator in matritsa:
    for son in qator:
        jami += son

print("Barcha sonlar yig'indisi:", jami)
Ichma-ich for: tashqi loop qatorlarni, ichki loop har qatordagi sonlarni aylanadi va jamiga qo'shadi.
>>> Barcha sonlar yig'indisi: 45

Mashq 4 — So'zlar chastotasi (dict)

Masala: Matndagi har bir so'z necha marta uchraganini hisoblang. 
matn = "olma anor olma uzum anor olma"

sozlar = matn.split()
chastota = {}

for soz in sozlar:
    chastota[soz] = chastota.get(soz, 0) + 1

for soz, son in chastota.items():
    print(soz, ":", son)
get(soz, 0) — agar so'z lug'atda yo'q bo'lsa 0 qaytaradi, bor bo'lsa mavjud sonini. Har uchraganda +1 qilinadi.
>>> olma : 3
>>> anor : 2
>>> uzum : 1

Mashq 5 — Slicing va tuple unpacking

Masala: Ro'yxatning birinchi, oxirgi elementini va o'rtadagilarni ajrating. 
raqamlar = [10, 20, 30, 40, 50]

birinchi = raqamlar[0]
oxirgi = raqamlar[-1]
ortasi = raqamlar[1:-1]

print("Birinchi:", birinchi)
print("Oxirgi:", oxirgi)
print("O'rtasi:", ortasi)

# teskari tartib slicing bilan
print("Teskari:", raqamlar[::-1])
raqamlar[1:-1] — birinchi va oxirgidan tashqari hammasi. [::-1] — ro'yxatni teskari aylantiradi.
>>> Birinchi: 10
>>> Oxirgi: 50
>>> O'rtasi: [20, 30, 40]
>>> Teskari: [50, 40, 30, 20, 10]

Mashq 6 — Set operatsiyalari

Masala: Ikki guruhdagi umumiy va faqat bittasida bor talabalarni toping. 
python = {"Ali", "Vali", "Guli"}
matematika = {"Vali", "Guli", "Hasan"}

print("Ikkalasida:", python & matematika)   # intersection
print("Hammasi:", python | matematika)       # union
print("Faqat Python:", python - matematika)  # difference
& — ikkala to'plamda bor elementlar, | — barcha elementlar (takrorsiz), - — birinchisida bor, ikkinchisida yo'q.
>>> Ikkalasida: {'Vali', 'Guli'}
>>> Hammasi: {'Ali', 'Vali', 'Guli', 'Hasan'}
>>> Faqat Python: {'Ali'}
