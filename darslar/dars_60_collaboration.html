Modul 11 — Git va version control | Dars 60

Jamoa Bo'lib Ishlash Nima?

Katta loyihalar bitta odam tomonidan emas, balki jamoa bo'lib yoziladi. Git va GitHub yordamida bir nechta dasturchi bir loyiha ustida bir vaqtda ishlay oladi — har kim o'z qismini yozadi, keyin hammasi birlashtiriladi. Bu darsda boshqaning loyihasiga qanday hissa qo'shishni o'rganamiz.
Hayotiy misol

Tasavvur qiling: do'stingiz GitHub'da ochiq loyiha qilgan. Siz unga yangi imkoniyat qo'shmoqchisiz. Lekin uning loyihasiga to'g'ridan-to'g'ri yoza olmaysiz. Buning uchun maxsus oqim bor: fork → branch → pull request.

Asosiy Tushunchalar

TushunchaMa'nosi
ForkNusxa olish — boshqaning repo'sini o'z GitHub hisobingga nusxalash
BranchTarmoq — o'zgarishlarni alohida joyda yozish
CommitO'zgarishni saqlash (foto olish)
PushO'zgarishni GitHub'ga yuborish

Code ReviewKodni ko'rib chiqish va izoh berish
MergeTasdiqlangan o'zgarishni asosiy loyihaga qo'shish

Fork Nima?

Fork — boshqaning loyihasini butunlay o'z hisobingga nusxalash. Endi sizda o'sha loyihaning shaxsiy nusxasi bo'ladi. Unda istagancha o'zgartirish kiritasiz — asl loyihaga zarar yetmaydi.
# GitHub saytida "Fork" tugmasini bosasiz
# Keyin o'z nusxangizni kompyuterga olasiz:
git clone https://github.com/men/loyiha.git
# Fork qilingan nusxa endi sizning hisobingizda

Pull Request (PR) Nima?

Pull Request — "Iltimos, mening o'zgarishlarimni asosiy loyihaga qo'shing" degan rasmiy taklif. Siz kodni yozasiz, push qilasiz, keyin GitHub'da PR ochasiz. Loyiha egasi uni ko'rib chiqadi va rozi bo'lsa merge qiladi.
PR — muloqot joyi

Pull Request faqat kod emas — bu suhbat ham. Loyiha egasi savol berishi, tuzatish so'rashi yoki maqtashi mumkin. Bu jarayon Code Review deyiladi.

Hamkorlik Oqimi (Qadamlar)

To'liq hamkorlik oqimi 7 qadamdan iborat. Quyida ularning ketma-ketligi:
#QadamBuyruq / Harakat
1ForkGitHub'da "Fork" tugmasi
2Branchgit checkout -b yangi-imkoniyat
3Commitgit commit -m "Yangi tugma"
4Pushgit push origin yangi-imkoniyat
5Pull RequestGitHub'da "New Pull Request"
6ReviewEgasi kodni ko'rib chiqadi
7MergeEgasi "Merge" tugmasini bosadi
Eslab Qoling

• Fork — boshqaning repo'sini o'zingga nusxalash
• Pull Request — o'zgarishni qo'shish taklifi
• Code Review — kodni ko'rib chiqish va izoh berish
• Oqim: fork → branch → commit → push → PR → review → merge
