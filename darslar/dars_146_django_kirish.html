Django Nima?

Django — bu Python tilida yozilgan to'liq web framework. "Framework" — bu tayyor qurilish materiallari to'plami: siz har bir narsani noldan yozmaysiz, balki tayyor qismlardan foydalanib tez sayt quryasiz. Django katta va kuchli loyihalar uchun mo'ljallangan.
"Batareyalari bilan keladi" (batteries included)

Django ichida hamma narsa tayyor: ma'lumotlar bazasi bilan ishlash, foydalanuvchini ro'yxatdan o'tkazish, admin paneli, xavfsizlik himoyasi va boshqalar. Shuning uchun u Flask dan kattaroq va to'liqroq.

Django'ni O'rnatish

Django'ni pip orqali o'rnatamiz. Bu boshqa kutubxonalarni o'rnatishga juda o'xshaydi.
pip install django
# Terminalda yozing — Django o'rnatiladi
django-admin --version
# O'rnatilganini tekshirish — versiya raqami chiqadi
>>> 5.0.6

Loyiha Yaratish

Yangi Django loyihasini django-admin startproject buyrug'i bilan yaratamiz. U barcha kerakli papka va fayllarni avtomatik tayyorlaydi.
django-admin startproject mysite
# "mysite" nomli yangi loyiha yaratiladi
Eslatma

mysite — bu loyiha nomi. Uni xohlagancha o'zgartirishingiz mumkin. Buyruq ishlagach, yangi mysite/ papkasi paydo bo'ladi.

Serverni Ishga Tushirish

Loyiha papkasi ichiga kirib, manage.py orqali test serverini ishga tushiramiz. Bu sizning kompyuteringizda kichik web-server ochadi.
cd mysite
python manage.py runserver
# Loyihaga kiramiz va serverni ishga tushiramiz
>>> Starting development server at http://127.0.0.1:8000/
>>> Quit the server with CONTROL-C.
Saytni ko'rish

Brauzerda http://127.0.0.1:8000/ manzilini oching. Django'ning xush kelibsiz sahifasi (raketa rasmi bilan) chiqadi. Tabriklaymiz — sayt ishlamoqda!

MVT Arxitektura

Django MVT tartibida ishlaydi: Model — View — Template. Bu — kodni uchta vazifaga ajratib, tartibli saqlash usuli.
QismVazifasiSodda izoh
ModelMa'lumotlar bazasi bilan ishlaydiMa'lumot qayerda saqlanadi
ViewMantiq (logika) ni boshqaradiNima qilish kerakligini hal qiladi
TemplateHTML sahifani chiqaradiFoydalanuvchi nimani ko'radi
Oqim (qanday ishlaydi)

Foydalanuvchi saytga so'rov yuboradi → View uni qabul qiladi → kerak bo'lsa Model dan ma'lumot oladi → natijani Template ga joylab, foydalanuvchiga qaytaradi.

Flask vs Django Farqi

Ikkalasi ham Python web framework, lekin Flask kichik (mikro), Django katta (to'liq). Flask — yengil, lekin ko'p narsani o'zingiz qo'shasiz. Django — og'irroq, lekin hammasi tayyor.
XususiyatFlaskDjango
HajmiKichik (mikro)Katta (to'liq)
Admin panelYo'q (o'zingiz qilasiz)Tayyor bor
Ma'lumotlar bazasi (ORM)Qo'shimcha kerakIchida bor
TuzilmaErkinQat'iy (MVT)
Qachon yaxshiKichik loyihaKatta loyiha
Eslab Qoling

• Django — to'liq web framework, Flask dan kattaroq
• O'rnatish: pip install django
• Loyiha: django-admin startproject mysite
• Server: python manage.py runserver
• Arxitektura: Model — View — Template (MVT)
