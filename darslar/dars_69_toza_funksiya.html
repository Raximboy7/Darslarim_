Toza funksiya yozish

Modul 6 — Funksiyalar | Dars 69

Toza funksiya (pure function) nima?

Toza funksiya — bu ikki qoidaga bo'ysunadigan funksiya:
1. Bir xil kirish uchun har doim bir xil natija qaytaradi.
2. Tashqi olamga ta'sir (side-effect) qilmaydi — global o'zgaruvchini o'zgartirmaydi, fayl yozmaydi, ekranga chop etmaydi.

Toza funksiyalar bashorat qilinadigan, test qilishga oson va xatosi kam bo'ladi.

Toza va nopok funksiya

Quyidagi funksiya toza: faqat kirishga tayanadi va natija qaytaradi: 

# TOZA
def kvadrat(x):
    return x * x

print(kvadrat(4))
print(kvadrat(4))
# bir xil kirish -> bir xil natija, hech narsani o'zgartirmaydi
>>> 16
>>> 16

Bu esa nopok: global o'zgaruvchiga tegadi (side-effect): 

# NOPOK
jami = 0

def qosh(x):
    global jami
    jami += x
    return jami

print(qosh(5))
print(qosh(5))
# bir xil kirish (5), lekin natija har safar boshqacha!
>>> 5
>>> 10

Bitta funksiya — bitta vazifa

Yaxshi qoida

Har bir funksiya faqat bitta ish qilsin. Bu kodni o'qish va qayta ishlatishni osonlashtiradi.

Aralashgan funksiya o'rniga, uni mayda toza funksiyalarga bo'lamiz: 

def ortacha(sonlar):
    return sum(sonlar) / len(sonlar)

def formatla(natija):
    return f"O'rtacha: {natija:.2f}"

sonlar = [10, 20, 30]
print(formatla(ortacha(sonlar)))
# ortacha hisoblaydi, formatla matnga aylantiradi — har biri bitta vazifa
>>> O'rtacha: 20.00

Ro'yxatni buzmaslik

Toza funksiya kirish argumentini o'zgartirmaydi — yangi obyekt qaytaradi: 

def saralangan(royxat):
    return sorted(royxat)

asl = [3, 1, 2]
yangi = saralangan(asl)
print(asl)
print(yangi)
# sorted() yangi ro'yxat qaytaradi, asl o'zgarmaydi
>>> [3, 1, 2]
>>> [1, 2, 3]

Xulosa

Eslab qoling

• Toza funksiya = bir xil kirish → bir xil natija
• Side-effect yo'q (global, fayl, print o'zgartirmaslik)
• Bitta funksiya — bitta vazifa
• Kirish argumentini buzmang, yangi qiymat qaytaring
