Closure va Lambda

Modul 9 — Iterator va Generator | Dars 53

Bu Darsda Nimani O'rganamiz?

Bu darsda Python'ning ikkita kuchli imkoniyatini ko'ramiz: closure (ichki funksiya tashqi o'zgaruvchini eslab qolishi) va lambda (qisqa, nomsiz funksiya). So'ngra ularni map(), filter() va reduce() bilan birga ishlatishni o'rganamiz.

Closure Nima?

Closure — bu ichki funksiya o'zini o'rab turgan tashqi funksiyaning o'zgaruvchisini "eslab qoladi". Tashqi funksiya o'z ishini tugatib, yo'qolib ketsa ham, ichki funksiya o'sha o'zgaruvchini saqlab qoladi.
def kopaytuvchi_yasa(son):
    def ichki(x):
        return x * son
    return ichki

ikkiga = kopaytuvchi_yasa(2)
print(ikkiga(5))
# ichki() funksiya "son=2" ni eslab qoldi
# ikkiga(5) → 5 * 2 = 10
>>> 10
Closure uchun 3 ta shart

1. Funksiya ichida boshqa funksiya bo'lishi kerak (ichki funksiya).
2. Ichki funksiya tashqi funksiyaning o'zgaruvchisidan foydalanishi kerak.
3. Tashqi funksiya ichki funksiyani return qilishi kerak.

Lambda — Qisqa, Nomsiz Funksiya

Lambda — bu bitta qatorda yoziladigan, nomi yo'q kichik funksiya. U lambda parametr: ifoda ko'rinishida yoziladi. Natijani return yozmasdan, avtomatik qaytaradi.
// Oddiy funksiya:
def ikki_baravar(x):
    return x * 2

// Xuddi shu lambda bilan:
ikki_baravar = lambda x: x * 2
print(ikki_baravar(4))
# ikkalasi ham bir xil ishlaydi
# lambda — bir qatorda, qisqa
>>> 8

Bir nechta parametr bilan

qoshish = lambda a, b: a + b
print(qoshish(3, 7))
# bir nechta parametr vergul bilan ajratiladi
>>> 10

Qoidaning mohiyati

Lambda'ni asosan map(), filter(), sorted() kabi funksiyalarga qisqa qoida berish uchun ishlatamiz. Agar mantiq murakkab bo'lsa — oddiy def dan foydalaning, kod tushunarliroq bo'ladi.

map() — Har Elementga Funksiya Qo'llash

map(funksiya, royxat) — ro'yxatning har bir elementiga funksiyani qo'llaydi va yangi natijalar beradi.
sonlar = [1, 2, 3, 4]
kvadrat = list(map(lambda x: x ** 2, sonlar))
print(kvadrat)
# har bir sonni kvadratga oshiradi
# 1→1, 2→4, 3→9, 4→16
>>> [1, 4, 9, 16]

filter() — Shartga Mosini Saqlash

filter(shart, royxat) — faqat shartni bajaradigan (True qaytaradigan) elementlarni saqlab qoladi.
sonlar = [1, 2, 3, 4, 5, 6]
juftlar = list(filter(lambda x: x % 2 == 0, sonlar))
print(juftlar)
# faqat juft sonlarni saqlaydi
# shart True bo'lganlar qoladi
>>> [2, 4, 6]

reduce() — Hammasini Bitta Qiymatga Yig'ish

reduce(funksiya, royxat) — ro'yxatdagi elementlarni juft-juft qilib birlashtiradi va oxir-oqibat bitta qiymat qaytaradi. U functools modulida joylashgan, shuning uchun avval import qilamiz.
from functools import reduce

sonlar = [1, 2, 3, 4]
yigindi = reduce(lambda a, b: a + b, sonlar)
print(yigindi)
# ((1+2)+3)+4 = 10
# bosqichma-bosqich yig'ib boradi
>>> 10
Uchalasini eslab qoling

• map() — har elementni o'zgartiradi (soni o'zgarmaydi)
• filter() — kerakligini saqlaydi (soni kamayadi)
• reduce() — hammasini bitta qiymatga yig'adi

Misol 1: Lambda Funksiyani Sinab Ko'rish

Bir son kiriting va lambda qoidani tanlang — natijani darhol ko'ring.

Misol 2: map() va filter() Natijasi

Vergul bilan sonlar yozing, so'ng amalni tanlang — map yoki filter natijasini ko'ring.

Misol 3: Closure — Tashqi O'zgaruvchini Eslab Qolish

Avval ko'paytuvchi yasang (masalan 3 ga), so'ng turli sonlar kiriting. E'tibor bering: yasalgan funksiya o'z ko'paytuvchisini eslab qoladi.

Misol 4: reduce() — Yig'indi va Ko'paytma

Sonlarni yozing va reduce bilan ularni bitta qiymatga yig'ing.
