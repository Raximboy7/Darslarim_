Linked List (Bog'langan Ro'yxat)

Modul 12 — Algoritmlar va optimizatsiya | Dars 67

Linked List Nima?

Linked List (bog'langan ro'yxat) — bu ma'lumotlarni saqlashning maxsus usuli. Unda har bir element keyingi elementga ishora qiladi (uning manzilini biladi). Ya'ni elementlar bir-biriga zanjirdek ulangan.
Vagon-poyezd analogiyasi 🚂

Linked List'ni poyezd deb tasavvur qiling. Har bir vagon — bu bitta element (node). Har vagonda ikki narsa bor: ichidagi yuk (ma'lumot) va keyingi vagonga ulangan ilgak (ishora). Oxirgi vagonning ilgagi bo'sh — u None ga ishora qiladi, demak poyezd tugadi.

Node — Bitta Vagon

Node (tugun) — Linked List'ning bitta bo'lagi. Har node ikki qismdan iborat: data (saqlanadigan ma'lumot) va next (keyingi node'ga ishora). Yangi node yaratilganda next hali hech kimga ishora qilmaydi — shuning uchun u None bo'ladi.
class Node:
    def __init__(self, data):
        self.data = data    
        self.next = None
# data — vagon ichidagi yuk (ma'lumot)
# next — keyingi vagonga ilgak, boshda bo'sh (None)

Linked List vs Oddiy List

Python'dagi oddiy list elementlarni xotirada yonma-yon saqlaydi va indeks orqali tez topadi. Linked List esa elementlarni ishoralar bilan bog'laydi — boshqa joyda turishlari mumkin.

Linked List Klassi

Endi node'larni boshqaradigan LinkedList klassini yozamiz. Unda bitta muhim narsa bor — head (boshlang'ich node). Head — poyezdning birinchi vagoni. Undan boshlab butun zanjirga kira olamiz.
class LinkedList:
    def __init__(self):
        self.head = None  

    def add_last(self, data):
        yangi = Node(data)
        if self.head is None:
            self.head = yangi
            return
        hozir = self.head
        while hozir.next:
            hozir = hozir.next
        hozir.next = yangi
# head — birinchi node (boshda bo'sh)
# add_last — oxiriga yangi node qo'shadi
# ro'yxat bo'sh bo'lsa, yangi node head bo'ladi
# aks holda oxirgi node'gacha yurib, ilgakni ulaymiz

Boshiga Qo'shish (add_first)

Boshiga qo'shish eng tez amal: yangi node'ning ilgagini eski head'ga ulaymiz, so'ng head'ni yangi node qilamiz. Tartibni o'zgartirmaslik uchun ilgakni avval ulash muhim.
def add_first(self, data):
    yangi = Node(data)
    yangi.next = self.head
    self.head = yangi
# yangi node'ni eski head'ga ulaymiz
# head endi yangi node'ga ishora qiladi

O'chirish (delete)

Element o'chirish uchun o'chiriladigan node'ning oldingi node'ini topamiz va uning ilgagini o'chiriladigan node'dan oshirib, undan keyingisiga ulaymiz. Shunda o'rtadagi vagon zanjirdan chiqib ketadi.
def delete(self, qiymat):
    hozir = self.head
    if hozir and hozir.data == qiymat:
        self.head = hozir.next
        return
    while hozir.next:
        if hozir.next.data == qiymat:
            hozir.next = hozir.next.next
            return
        hozir = hozir.next
# head'ning o'zini o'chirsak, head'ni keyingisiga suramiz
# aks holda o'chiriladigandan oldingi node'ni topamiz
# uning ilgagini bittadan oshirib ulaymiz — vagon chiqib ketadi

Traversal — Boshdan Oxirgacha Yurish

Traversal — bu Linked List'ning hamma node'larini birma-bir ziyorat qilish. Head'dan boshlaymiz va har safar next orqali keyingi node'ga o'tamiz, toki None ga yetguncha.
def show(self):
    hozir = self.head
    while hozir:
        print(hozir.data, end=" -> ")
        hozir = hozir.next
    print("None")
# head'dan boshlaymiz
# har node'ni chop etib, keyingisiga o'tamiz
# None ga yetganda to'xtaymiz
>>> 10 -> 20 -> 30 -> None
Eslab Qoling

• Har node = data + next
• head — birinchi node, undan butun zanjirga kiriladi
• Oxirgi node'ning next = None
• Boshiga qo'shish tez, indeks bilan olish sekin
• Traversal — har doim head'dan boshlanadi

Misol 1: Element Qo'shish (Zanjir Qurish)

Quyida har qo'shilgan node zanjir ko'rinishida chiqadi. Qiymat yozing va boshiga yoki oxiriga qo'shing.

Misol 2: Element O'chirish

O'chiriladigan qiymatni yozing. Avval u node xira bo'lib ko'rsatiladi, so'ng zanjirdan chiqib ketadi.

Misol 3: Traversal (Har Node'ni Ziyorat Qilish)

Tugmani bosing — kursor head'dan boshlab har node'ni birma-bir yoritadi, xuddi while sikli kabi.
