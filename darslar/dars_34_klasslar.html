Modul 7 — OOP (Obyektga asoslangan dasturlash) | Dars 34

Klass Nima?

Klass (class) — bu qolip yoki shablon. U biror narsani qanday bo'lishini tasvirlaydi, lekin o'zi hali tayyor narsa emas. Klassdan biz haqiqiy narsalarni — obyektlarni yasaymiz.
Hayotiy analogiya 🍪

Klass — bu pechenye uchun qolip (forma). Qolipning o'zi pechenye emas! Lekin undan istagancha pechenye yasash mumkin. Har bir yasalgan pechenye — bu obyekt. Ular bir qolipdan chiqsa ham, har biri alohida narsa.

Eng Oddiy Klass Yaratish

Klass class kalit so'zi bilan yaratiladi. Klass nomi odatda katta harf bilan boshlanadi.
class Kishi:
    pass
# Bo'sh klass (qolip). pass — "hozircha hech narsa yo'q" degani

Obyekt Yaratish (Instance)

Klassdan obyekt yasash uchun klass nomini qavs bilan chaqiramiz. Hosil bo'lgan obyektga instance ham deyiladi.
class Kishi:
    pass

ali = Kishi()
vali = Kishi()
# ali va vali — bitta qolipdan yasalgan ikki alohida obyekt

__init__ — Konstruktor

__init__ — bu maxsus metod. Obyekt yaratilgan paytda avtomatik ishlaydi va obyektga boshlang'ich ma'lumotlarni beradi. Uni konstruktor deb ataymiz ("quruvchi" degani).
class Kishi:
    def __init__(self, ism, yosh):
        self.ism = ism
        self.yosh = yosh

ali = Kishi("Ali", 20)
print(ali.ism, ali.yosh)
# Kishi("Ali", 20) chaqirilganda __init__ ishlaydi.
# self.ism = ism → obyektga "Ali" ni yozadi
>>> Ali 20

self Nima va Nega Kerak?

self — bu "ayni shu obyekt" degani. Har bir metodda birinchi parametr sifatida yoziladi. U orqali obyekt o'zining ma'lumotlariga murojaat qiladi.
Oddiy tushuntirish

self.ism degani — "shu obyektning ismi". Ali uchun self = ali, Vali uchun self = vali. Shuning uchun ali.ism = "Ali", vali.ism = "Vali" bo'ladi. Agar self bo'lmasa, Python qaysi obyekt haqida gapirayotganimizni bilmaydi.

Xususiyat (Attribute) vs Metod (Method)

Xususiyat — bu obyektning ma'lumoti (ism, yosh). Metod — bu obyektning ichidagi funksiya (harakat qiladi).
class Kishi:
    def __init__(self, ism):
        self.ism = ism    

    def salomlash(self):
        return "Salom, men " + self.ism

ali = Kishi("Ali")
print(ali.ism)
print(ali.salomlash())
# ali.ism → xususiyat (qavssiz)
# ali.salomlash() → metod (qavs bilan chaqiriladi)
>>> Ali
>>> Salom, men Ali

Eslab Qoling

• class — qolip yaratadi, obyekt — undan yasalgan narsa
• __init__ obyekt yaratilganda avtomatik ishlaydi
• self — "ayni shu obyekt" degani
• Xususiyat — ma'lumot (qavssiz), metod — harakat (qavs bilan)

Misol 1: Obyekt Yaratuvchi

Ism va yoshni kiriting, tugmani bosing — Kishi obyekti yaratiladi va uning xususiyatlari ko'rsatiladi.

Misol 2: __init__ Qadamma-Qadam

Kishi("Guli", 18) chaqirilganda __init__ ichida nima sodir bo'lishini qadam-qadam kuzating.
1-qadam — obyekt yaratishguli = Kishi("Guli", 18)
2-qadam — __init__ chaqiriladidef __init__(self, ism, yosh)
3-qadam — self = guli obyektiself → guli
4-qadam — ism yoziladiself.ism = "Guli"
5-qadam — yosh yoziladiself.yosh = 18
Tayyor! obyekt qaytariladiguli.ism = "Guli", guli.yosh = 18

Misol 3: Bir Nechta Obyekt

Bitta qolipdan ko'p obyekt yasaymiz. Har biri o'z xususiyatiga ega ekanini ko'ring — ular bir-biriga ta'sir qilmaydi.
