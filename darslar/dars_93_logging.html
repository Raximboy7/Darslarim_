Logging (Jurnal Yuritish)

Modul 8 — Xatolar, Test va Debug | Dars 93

Logging Nima?

Logging (jurnal yuritish) — bu dasturning ishlashi davomida nima bo'layotganini yozib borish demakdir. Dastur qachon ishga tushdi, qaysi amal bajarildi, qayerda xato chiqdi — bularning hammasi maxsus jurnal (log) sifatida saqlanadi.
Avval biz xabarlarni ko'rsatish uchun print() dan foydalandik. Lekin katta dasturlarda print() yetarli emas. Aynan shu yerda logging moduli yordamga keladi. U xabarlarni darajalarga ajratadi, vaqtini yozadi va faylga saqlash imkonini beradi.
Oddiy taqqoslash

print() — bu shunchaki ekranga gapirish.
logging — bu daftarga tartib bilan, vaqti bilan yozib borish. Keyin daftarni qaytadan o'qib, nima bo'lganini bilib olasiz.

logging Modulini Ulash

Logging — Python ichidagi tayyor modul. Uni o'rnatish shart emas, faqat import qilish kifoya.
import logging

logging.warning("Bu ogohlantirish xabari")
logging.error("Bu xato xabari")
# logging modulini chaqirdik va ikkita xabar yozdik
>>> WARNING:root:Bu ogohlantirish xabari
>>> ERROR:root:Bu xato xabari
Diqqat

Standart holatda logging faqat WARNING va undan yuqori darajadagi xabarlarni ko'rsatadi. DEBUG va INFO xabarlari ko'rinmaydi — buni sozlash kerak (pastda ko'ramiz).

Log Darajalari (Levels)

Har bir log xabarining darajasi bor. Daraja xabarning naqadar muhim ekanini bildiradi. Pastdan yuqoriga qarab muhimligi oshib boradi.
DarajaQachon ishlatiladiMisol
DEBUGMayda-chuyda tekshiruv ma'lumotlari (faqat dasturchi uchun)"x qiymati = 5"
INFODastur normal ishlayapti, oddiy xabar"Foydalanuvchi tizimga kirdi"
WARNINGMuammo bor, lekin dastur hali ishlayapti"Disk joyi kamayyapti"
ERRORJiddiy xato, biror amal bajarilmadi"Fayl topilmadi"
CRITICALHalokatli xato, dastur to'xtashi mumkin"Ma'lumotlar bazasi o'chdi"
import logging

logging.debug("Tekshiruv: x = 5")
logging.info("Dastur ishga tushdi")
logging.warning("Disk joyi kam")
logging.error("Fayl topilmadi")
logging.critical("Tizim ishdan chiqdi")
# Beshta daraja — eng past DEBUG, eng yuqori CRITICAL

basicConfig — Loggingni Sozlash

logging.basicConfig() — bu loggingni sozlaydigan asosiy funksiya. U bilan biz qaysi darajadan boshlab xabar ko'rsatishni va xabar ko'rinishini (formatini) belgilaymiz.

level — daraja chegarasi

import logging

logging.basicConfig(level=logging.DEBUG)

logging.debug("Endi bu ham ko'rinadi")
logging.info("Bu ham ko'rinadi")
# level=DEBUG -> DEBUG va undan yuqori hammasi ko'rinadi
>>> DEBUG:root:Endi bu ham ko'rinadi
>>> INFO:root:Bu ham ko'rinadi

format — xabar ko'rinishi

format orqali har bir xabar oldidan vaqt, daraja va matnni qo'shamiz.
import logging

logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s - %(levelname)s - %(message)s"
)

logging.info("Dastur boshlandi")
# asctime = vaqt, levelname = daraja, message = matn
>>> 2026-06-30 10:15:42,531 - INFO - Dastur boshlandi

Faylga Log Yozish

Eng kuchli xususiyat — loglarni faylga saqlash. Buning uchun filename= parametrini beramiz. Endi xabarlar ekranga emas, faylga yoziladi va keyin ham o'qishingiz mumkin.
import logging

logging.basicConfig(
    filename="dastur.log",
    level=logging.INFO,
    format="%(asctime)s - %(levelname)s - %(message)s"
)

logging.info("Foydalanuvchi tizimga kirdi")
logging.error("Parol noto'g'ri")
# Endi xabarlar ekranga emas, "dastur.log" fayliga yoziladi
# dastur.log fayli ichida:
2026-06-30 10:20:01,114 - INFO - Foydalanuvchi tizimga kirdi
2026-06-30 10:20:01,118 - ERROR - Parol noto'g'ri
Eslatma

Fayl avtomatik yaratiladi. Har safar dastur ishlaganda yangi xabarlar fayl oxiriga qo'shiladi — eskilari o'chmaydi. Shuning uchun dastur tarixini har doim ko'rib chiqishingiz mumkin.

Asosiy fikr

print() — o'rganish va tezkor sinash uchun yaxshi. Lekin haqiqiy dasturlarda nima bo'layotganini kuzatish uchun logging ishlatiladi. Bitta level ni o'zgartirib, barcha keraksiz xabarlarni yashirib qo'yasiz — kodni o'chirmaysiz.

Misol 1: Daraja Tanlash va Xabarlar

Pastdan darajani tanlang. Faqat tanlangan darajadan yuqori yoki teng bo'lgan xabarlar ko'rinadi.

Misol 2: Log Xabari Formati

Xabaringizni yozing va darajani tanlang. Logging xabarni vaqt + daraja + matn ko'rinishida shakllantirib beradi.

Misol 3: print() vs logging

Bitta xabarni ikki usulda ko'ring: oddiy print() va tartibli logging bilan. Farqni sezing.

Misol 4: Faylga Yozishni Tasavvur Qilish

Bir necha xabar qo'shing — ular dastur.log fayliga qanday saqlanishini ko'ring.
