Modul 23 — Pro daraja va Capstone | Dars 170

Design Pattern Nima?

Design pattern (dizayn andozasi) — bu dasturlashda tez-tez uchraydigan muammolarga tayyor, sinab ko'rilgan yechimlar. Ular kod emas, balki g'oya — "shu muammoni shunday hal qilish qulay" degan tavsiya. Tajribali dasturchilar yillar davomida eng yaxshi yechimlarni to'plab, ularga nom berishgan.
Patternlarni bilish ikki foyda beradi: birinchidan, g'ildirakni qaytadan ixtiro qilmasdan tayyor yechimdan foydalanasiz; ikkinchidan, boshqa dasturchilar bilan bir tilda gaplashasiz — "bu yerda Singleton kerak" deganda hamma nimani nazarda tutganingizni tushunadi.
Asosiy g'oya

Pattern — bu retsept emas, balki tamoyil. Bir muammoni har xil tilda, har xil kod bilan yozish mumkin, lekin g'oya bir xil bo'ladi.

1. Singleton — Bitta Nusxa

Singleton — klassdan butun dastur davomida faqat bitta nusxa bo'lishini ta'minlaydi. Necha marta chaqirsangiz ham, har doim o'sha bitta obyekt qaytadi. Sozlamalar yoki ma'lumotlar bazasiga ulanish kabi narsalar uchun ideal.
class Sozlama:
    _nusxa = None  # yagona nusxa shu yerda saqlanadi

    def __new__(cls):
        if cls._nusxa is None:
            cls._nusxa = super().__new__(cls)
        return cls._nusxa

a = Sozlama()
b = Sozlama()
print(a is b)
# __new__ obyekt yaratilishini boshqaradi: agar nusxa bor bo'lsa, # yangisini yaratmay o'shani qaytaradi. a is b → True
>>> True

2. Factory — Obyekt Yaratuvchi

Factory (fabrika) — obyektni to'g'ridan-to'g'ri yaratish o'rniga, maxsus funksiya orqali yaratadi. Bu funksiyaga "menga shunday hayvon kerak" deysiz, u esa kerakli turdagi obyektni qaytaradi. Klass turini shartga qarab tanlash kerak bo'lganda juda qulay.
class It:
    def ovoz(self): return "Vov!"

class Mushuk:
    def ovoz(self): return "Miyov!"

def hayvon_yarat(turi):
    if turi == "it": return It()
    return Mushuk()

h = hayvon_yarat("it")
print(h.ovoz())
# hayvon_yarat() — bu fabrika. Qaysi klass kerakligini o'zi hal qiladi, # siz faqat "it" yoki "mushuk" deb nom berasiz.
>>> Vov!

3. Observer — Kuzatuvchi

Observer (kuzatuvchi) — bir obyekt o'zgarganda, unga "obuna bo'lgan" boshqa obyektlarni avtomatik xabardor qiladi. YouTube kanaliga o'xshaydi: kanal yangi video chiqarsa, barcha obunachilar bildirishnoma oladi.
class Kanal:
    def __init__(self):
        self.obunachilar = []

    def obuna(self, kishi):
        self.obunachilar.append(kishi)

    def video_chiqar(self, nom):
        for k in self.obunachilar:
            print(f"{k}: {nom} chiqdi!")

c = Kanal()
c.obuna("Ali")
c.obuna("Vali")
c.video_chiqar("Python")
# video_chiqar() chaqirilganda barcha obunachilarga bir vaqtda xabar boradi.
>>> Ali: Python chiqdi!
>>> Vali: Python chiqdi!

4. Strategy — Almashtiriladigan Algoritm

Strategy (strategiya) — bir vazifani bajarishning bir necha usuli bo'lganda, kerakli usulni tashqaridan beriladi. Masalan to'lov: naqd, karta yoki onlayn. Asosiy kodni o'zgartirmasdan algoritmni almashtirasiz.
def naqd(summa): return f"Naqd: {summa} so'm"
def karta(summa): return f"Karta: {summa} so'm"

def tolov(summa, usul):
    return usul(summa)

print(tolov(100, naqd))
print(tolov(100, karta))
# tolov() o'zi qaysi usul ekanini bilmaydi. Usulni funksiya sifatida # uzatamiz — shu sababli osongina almashtirish mumkin.
>>> Naqd: 100 so'm
>>> Karta: 100 so'm

5. Decorator — Bezak (eslatma)

Decorator (bezovchi) — mavjud funksiyaning kodini o'zgartirmasdan, unga qo'shimcha imkoniyat qo'shadi. Buni siz @ belgisi orqali ko'p ishlatgansiz (masalan @property). Bu — funksiyani boshqa funksiya ichiga "o'rab", oldidan yoki keyinidan amal qo'shish.
def qalin(func):
    def ichki():
        return "<b>" + func() + "</b>"
    return ichki

@qalin
def salom():
    return "Salom"

print(salom())
# @qalin — salom() funksiyasini "o'raydi". Asl funksiya o'zgarmaydi, # lekin natija avtomatik <b> teglar bilan bezatiladi.
>>> <b>Salom</b>
Eslab Qoling

• Singleton — butun dasturda bitta nusxa
• Factory — obyektni funksiya orqali yaratish
• Observer — o'zgarishdan kuzatuvchilarni xabardor qilish
• Strategy — algoritmni tashqaridan tanlash
• Decorator — funksiyani o'zgartirmay kengaytirish

Misol 1: Singleton — Bitta Nusxa

Tugmani bosing — necha marta "yangi" Sozlama yaratsangiz ham, har doim bir xil obyekt (bir xil ID) qaytishini ko'rasiz.

Misol 2: Factory — Obyekt Yaratuvchi

Hayvon turini tanlang va "Yarat" tugmasini bosing — fabrika kerakli klassni yaratib, uning ovozini qaytaradi.

Misol 3: Observer — Xabardor Qilish

Obunachi qo'shing, so'ng video nomini yozib "Chiqar" tugmasini bosing — barcha obunachilarga bir vaqtda bildirishnoma boradi.
