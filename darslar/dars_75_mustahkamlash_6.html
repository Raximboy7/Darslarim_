Modul 6 — Mustahkamlash

Bu darsda Funksiyalar moduli bo'yicha o'rganganlarimizni mashqlar orqali mustahkamlaymiz: funksiya e'lon qilish, return, default parametr, *args / **kwargs, lambda va toza funksiyalar.

Har bir mashqni avval o'zingiz yechishga urining, so'ng yechim bilan solishtiring.

Mashq 1: Salomlashuv (default parametr)

Vazifa

Ism va vaqt (vaqt="tong") qabul qilib, mos salomni qaytaruvchi funksiya yozing.

def salomlash(ism, vaqt="tong"):
    salomlar = {"tong": "Xayrli tong", "kech": "Xayrli kech"}
    return f"{salomlar[vaqt]}, {ism}!"

print(salomlash("Ali"))
print(salomlash("Vali", "kech"))
# vaqt berilmasa "tong" ishlatiladi
>>> Xayrli tong, Ali!
>>> Xayrli kech, Vali!

Mashq 2: Ko'paytma (*args)

Vazifa

Istalgan sondagi argumentni qabul qilib, ularning ko'paytmasini qaytaruvchi funksiya yozing.

def kopaytma(*sonlar):
    natija = 1
    for s in sonlar:
        natija *= s
    return natija

print(kopaytma(2, 3, 4))
print(kopaytma(5, 5))
# natijani 1 dan boshlab, har birini ko'paytiramiz
>>> 24
>>> 25

Mashq 3: Filtrlash (lambda + filter)

Vazifa

Sonlar ro'yxatidan faqat 3 ga bo'linadiganlarni ajrating.

sonlar = [3, 5, 9, 10, 12, 14, 15]
uchga = list(filter(lambda x: x % 3 == 0, sonlar))
print(uchga)
# x % 3 == 0 sharti True bo'lgan sonlar
>>> [3, 9, 12, 15]

Mashq 4: Toza funksiya (o'rtacha va baho)

Vazifa

Ballar ro'yxatining o'rtachasini hisoblab, unga qarab bahoni qaytaruvchi toza funksiyalar yozing (side-effect'siz).

def ortacha(ballar):
    return sum(ballar) / len(ballar)

def baho(oz):
    if oz >= 90:
        return "A'lo"
    elif oz >= 70:
        return "Yaxshi"
    return "Qoniqarli"

ballar = [80, 95, 100]
o = ortacha(ballar)
print(o, baho(o))
# har biri bitta vazifa, hech narsani o'zgartirmaydi
>>> 91.66666666666667 A'lo

Mashq 5: Comprehension bilan lug'at

Vazifa

1 dan 5 gacha sonlar uchun {son: son_faktoriali} lug'atini tuzing.

def faktorial(n):
    natija = 1
    for i in range(2, n + 1):
        natija *= i
    return natija

jadval = {n: faktorial(n) for n in range(1, 6)}
print(jadval)
# dict comprehension + funksiyani birga ishlatish
>>> {1: 1, 2: 2, 3: 6, 4: 24, 5: 120}

• Default parametr, *args, **kwargs
• lambda bilan map, filter, sorted
• Toza funksiya va comprehension'larni birlashtirish
