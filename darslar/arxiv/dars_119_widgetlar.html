Widget Nima?

Widget — bu oyna ichidagi har bir element: yozuv, tugma, matn kiritish maydoni va boshqalar. Oyna o'zi bo'sh ramka, undagi barcha ko'rinadigan narsalar — widgetlar. Boshqacha aytganda, widgetlar — oynaning g'ishtlari.
Asosiy widgetlar

• Label — matn yoki rasm ko'rsatadi
• Button — bosiladigan tugma
• Entry — bir qatorli matn kiritish maydoni

Label — Matn Ko'rsatish

Label oynada matn chiqaradi. Foydalanuvchi uni o'zgartira olmaydi — bu shunchaki ekrandagi yozuv.
import tkinter as tk

oyna = tk.Tk()
yozuv = tk.Label(oyna, text="Salom dunyo!")
yozuv.pack()
oyna.mainloop()
# Label yaratdik va oynaga qo'shdik (pack)
>>> Oynada "Salom dunyo!" yozuvi paydo bo'ladi

Button — Tugma

Button — bosilganda biror amal bajaradigan tugma. command parametriga bosilganda chaqiriladigan funksiya nomini beramiz (qavssiz!).
import tkinter as tk

def salom_ber():
    print("Tugma bosildi!")

oyna = tk.Tk()
tugma = tk.Button(oyna, text="Bos", command=salom_ber)
tugma.pack()
oyna.mainloop()
# command=salom_ber — tugma bosilganda funksiya ishlaydi (qavs YO'Q)
>>> Har bosganda: Tugma bosildi!
Diqqat

command=salom_ber deb yozamiz — salom_ber() emas! Qavs qo'ysangiz, funksiya tugma bosilishini kutmay, darrov bir marta ishlab ketadi.

Entry — Matn Kiritish

Entry — foydalanuvchi matn yozadigan bir qatorli maydon. Yozilgan matnni .get() bilan o'qib olamiz.
maydon = tk.Entry(oyna)
maydon.pack()

matn = maydon.get()  # yozilgan matnni oladi
# .get() — Entry ichidagi matnni qaytaradi

Widget Parametrlari

Har bir widgetni yaratayotganda uning ko'rinishini parametrlar orqali sozlaymiz. Quyida eng ko'p ishlatiladiganlari:

    oyna,
    text="Chiroyli matn",
    font=("Arial", 18),

)
yozuv.pack()
# Sariq fon, ko'k matn, Arial 18 shrift

Widgetni Oynaga Qo'shish — pack()

Widget yaratilgani bilan o'zi ko'rinmaydi. Uni oynaga joylash kerak. Eng oddiy usul — .pack(). U widgetlarni yuqoridan pastga qatorlab joylaydi.
yozuv.pack()    # 1-qator
tugma.pack()    # 2-qator
maydon.pack()  # 3-qator
# pack() chaqirilmasa, widget ekranda ko'rinmaydi!
Eslab Qoling

Widget yaratish 2 qadam: 1) widgetni yaratish, 2) .pack() bilan oynaga qo'shish. Ikkalasi ham bo'lishi shart.
