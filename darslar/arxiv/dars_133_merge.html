Nima Uchun Birlashtirish Kerak?

Ko'pincha ma'lumot bitta jadvalda emas, bir nechta jadvalda tarqalgan bo'ladi. Masalan, bir jadvalda talabalar ismi, ikkinchisida ularning baholari. Bularni bitta jadvalga birlashtirish uchun pandas pd.merge() va pd.concat() ni beradi.
Ikki yo'l bor

• pd.merge() — umumiy ustun bo'yicha jadvallarni moslab birlashtiradi (SQL JOIN kabi).
• pd.concat() — jadvallarni shunchaki ustma-ust yoki yonma-yon qo'shadi.

pd.merge() — Umumiy Ustun Bo'yicha

pd.merge(df1, df2, on='ustun') — ikki DataFrame ni umumiy ustun qiymatlari bir xil bo'lgan qatorlarni moslab birlashtiradi.
import pandas as pd

ismlar = pd.DataFrame({
    "id": [1, 2, 3],
    "ism": ["Ali", "Vali", "Guli"]
})
baho = pd.DataFrame({
    "id": [1, 2, 3],
    "baho": [90, 75, 88]
})

natija = pd.merge(ismlar, baho, on="id")
print(natija)
# "id" ustuni umumiy — shu bo'yicha moslaymiz
>>>    id   ism  baho     0   1   Ali    90     1   2  Vali    75     2   3  Guli    88

how Parametri — JOIN Turlari

Agar ikki jadvaldagi id lar to'liq mos kelmasa, qaysi qatorlar qolishini how parametri belgilaydi. Bu SQL dagi JOIN turlariga to'g'ri keladi.
howMa'nosiQaysi qatorlar qoladi
"inner"Faqat moslar (standart)Ikkala jadvalda ham bor id lar
"left"Chap jadval to'liqdf1 dagi barcha qatorlar
"right"O'ng jadval to'liqdf2 dagi barcha qatorlar
"outer"HammasiIkkala jadvaldagi barcha qatorlar
natija = pd.merge(ismlar, baho, on="id", how="left")
# Chap jadvaldagi (ismlar) barcha qatorlar saqlanadi. # Mos baho topilmasa, o'sha joyga NaN (bo'sh) yoziladi.

pd.concat() — Ustma-ust va Yonma-yon

pd.concat() moslamaydi — u jadvallarni shunchaki ulab tashlaydi. axis=0 (standart) — ustma-ust (qatorlarni qo'shadi), axis=1 — yonma-yon (ustunlarni qo'shadi).
guruh_a = pd.DataFrame({"ism": ["Ali", "Vali"]})
guruh_b = pd.DataFrame({"ism": ["Guli", "Hasan"]})

hammasi = pd.concat([guruh_a, guruh_b])
print(hammasi)
# Ikki guruh ustma-ust bitta jadvalga qo'shildi
>>>      ism     0   Ali     1  Vali     0  Guli     1 Hasan

Sodda qoida

Ikki jadvalda bir xil ustun bor va shu bo'yicha bog'lamoqchimisiz? → merge.
Bir xil tuzilishdagi jadvallarni bir-biriga qo'shmoqchimisiz? → concat.

Eslab Qoling

• pd.merge(df1, df2, on="ustun") — umumiy ustun bo'yicha moslaydi
• how: inner / left / right / outer
• pd.concat([df1, df2]) — shunchaki ulaydi (axis=0 yoki 1)
• merge = bog'lash, concat = qo'shish
