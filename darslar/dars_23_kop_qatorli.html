Ko'p Qatorli Stringlar

Ba'zan bir necha qatordan iborat uzun matn yozish kerak bo'ladi. Buning uchun Pythonda uch qo'shtirnoq ("""...""" yoki '''...''') ishlatiladi. Bunda matn qanday yozilsa, xuddi shunday saqlanadi — \n yozishning hojati yo'q.

Uch Qo'shtirnoq Bilan Yozish

matn = """Salom!
Bu ko'p qatorli matn.
Uch qatordan iborat."""

print(matn)
# qatorlar qanday yozilsa, shunday chiqadi
>>> Salom!
>>> Bu ko'p qatorli matn.
>>> Uch qatordan iborat.
Qulaylik

Oddiy stringda uch qatorni yozish uchun "a\nb\nc" deyish kerak edi. Uch qo'shtirnoq bilan esa matnni tabiiy ko'rinishda, qatorma-qator yozamiz.

Solishtirish: Oddiy vs Ko'p Qatorli

# Oddiy string — \n bilan
a = "Bir\nIkki\nUch"

# Ko'p qatorli — tabiiy
b = """Bir
Ikki
Uch"""
# ikkalasi bir xil natija beradi
>>> Bir
>>> Ikki
>>> Uch

Docstring — Funksiya Hujjati

Docstring — funksiya, klass yoki fayl boshiga yoziladigan ko'p qatorli izoh. U funksiya nima qilishini tushuntiradi. Odatda uch qo'shtirnoq bilan yoziladi. 

def salomlash(ism):
    """Berilgan ismga salom aytadi.

    ism: foydalanuvchi ismi (string)
    """
    print(f"Salom, {ism}!")

salomlash("Ali")
# """...""" — funksiya nima qilishini tushuntiradi
>>> Salom, Ali!

Docstring'ni O'qish — help() va __doc__

Docstring shunchaki izoh emas — Python uni saqlaydi. Uni __doc__ orqali o'qish mumkin. 

def kvadrat(x):
    """Sonning kvadratini qaytaradi."""
    return x * x

print(kvadrat.__doc__)
# funksiyaning docstring'ini chiqaradi
>>> Sonning kvadratini qaytaradi.
Maslahat

Har bir funksiyaga qisqa docstring yozish yaxshi odat. Bu kodni boshqalar (va kelajakdagi siz) tushunishini osonlashtiradi. help(funksiya) ham docstring'ni ko'rsatadi.

Amaliy Mashq

Sinab ko'ring

1. Uch qo'shtirnoq bilan o'zingiz haqingizda 3 qatorli matn yozing va chiqaring.
2. Bir funksiya yozing va unga docstring qo'shing.
3. O'sha funksiyaning __doc__ ini chiqaring.

Xulosa

Uch qo'shtirnoq (""" yoki ''') ko'p qatorli matnni tabiiy ko'rinishda yozish imkonini beradi. Funksiya boshiga yozilgan ko'p qatorli string — docstring deb ataladi va funksiya nima qilishini tushuntiradi. Uni __doc__ yoki help() orqali o'qish mumkin. Bu professional kod yozishning muhim qismi.
