Polimorfizm Nima?

Polimorfizm — grekcha so'z bo'lib, "ko'p shakl" degan ma'noni anglatadi (poli = ko'p, morf = shakl). Dasturlashda bu shuni anglatadiki: turli klasslar bir xil metod nomiga ega bo'lishi mumkin, lekin har biri o'zicha, o'zgacha ishlaydi.
Tasavvur qiling: barcha hayvonlar ovoz() chiqaradi. Lekin it "Vov", mushuk esa "Miyov" deydi. Metod nomi bir xil — natija har xil. Ana shu polimorfizmdir.

Bir Nomli — Turli Natija

Quyida ikki xil hayvon klassi bor. Ikkalasida ham ovoz() metodi bor, lekin har biri boshqa narsa qaytaradi.
class It:
    def ovoz(self):
        return "Vov!"

class Mushuk:
    def ovoz(self):
        return "Miyov!"
# Ikkala klassda ham ovoz() bor
# Lekin natija har xil — bu polimorfizm
it = It()
mushuk = Mushuk()
print(it.ovoz())
print(mushuk.ovoz())
# Bir xil metodni chaqiramiz, natija boshqacha
>>> Vov!
>>> Miyov!

For Sikl bilan Polimorfizm

Polimorfizmning eng kuchli tomoni shu: bitta for sikl turli obyektlarni aylanib chiqib, har biriga bir xil metodni chaqirishi mumkin. Har bir obyekt o'zicha javob beradi.
hayvonlar = [It(), Mushuk(), It()]

for hayvon in hayvonlar:
    print(hayvon.ovoz())
# Ro'yxatda turli obyektlar bor
# for sikl har biriga ovoz() ni chaqiradi
# Python o'zi qaysi klass ekanini aniqlaydi
>>> Vov!
>>> Miyov!
>>> Vov!
Diqqat qiling

Biz hayvon.ovoz() deb yozdik — bu it ekanmi yoki mushukmi, deb tekshirmadik! Python o'zi har bir obyekt uchun to'g'ri metodni tanlaydi. Kod soda va tushunarli bo'ladi.

Duck Typing (O'rdak qoidasi)

Pythonda mashhur bir ibora bor: "Agar u o'rdakdek yursa va o'rdakdek qichqirsa — u o'rdakdir."
Buni soddaroq aytsak: Pythonga obyekt qaysi klassdan ekani muhim emas. Muhimi shuki — agar obyektda ovoz() metodi bo'lsa, Python uni chaqiraveradi. "Sen aslida itmi yoki mushukmi?" deb so'ramaydi. "Senda ovoz() bormi? Bo'lsa, chaqiraman" deydi.
class Robot:
    def ovoz(self):
        return "Biip-bip!"

def gapirtir(narsa):
    print(narsa.ovoz())

gapirtir(Robot())
# Robot hayvon emas, lekin ovoz() metodi bor
# Shuning uchun gapirtir() ishlayveradi
>>> Biip-bip!
