Modul 9 — nimalar o'rgandik?

Bu darsda 9-modul (OOP) bo'yicha bilimlarni mashqlar orqali mustahkamlaymiz. Har mashq uchun avval o'zingiz yechishga urinib ko'ring, keyin yechimni oching.
Qamrab olingan mavzular 📚

• Klass, obyekt, __init__, self
• Inkapsulyatsiya, @property
• Meros, super(), override
• Polimorfizm, abstract klass (ABC)
• Kompozitsiya va SOLID tamoyillari

Mashq 1: Kitob klassi

Kitob klassini yozing: nom va muallif xususiyatlariga ega bo'lsin. malumot() metodi "Nom — Muallif" ni chiqarsin.
class Kitob:
    def __init__(self, nom, muallif):
        self.nom = nom
        self.muallif = muallif

    def malumot(self):
        return f"{self.nom} — {self.muallif}"

k = Kitob("O'tkan kunlar", "Qodiriy")
print(k.malumot())
# Yechim: oddiy klass, __init__ va bitta metod
>>> O'tkan kunlar — Qodiriy

Mashq 2: Meros — Talaba

Odam klassidan meros oluvchi Talaba klassi yozing. Talabada qo'shimcha guruh xususiyati bo'lsin, super() ishlating.
class Odam:
    def __init__(self, ism):
        self.ism = ism

class Talaba(Odam):
    def __init__(self, ism, guruh):
        super().__init__(ism)
        self.guruh = guruh

t = Talaba("Ali", "IT-21")
print(t.ism, t.guruh)
# super().__init__(ism) → ism ni Odam o'rnatadi, guruh ni Talaba qo'shadi
>>> Ali IT-21

Mashq 3: Polimorfizm

Uch xil hayvon klassi yozing (Mushuk, It, Sigir), har biri ovoz() metodini o'zicha bajarsin. Ro'yxatda aylanib, har birining ovozini chiqaring.
class Mushuk:
    def ovoz(self): return "Miyov"
class It:
    def ovoz(self): return "Vov"
class Sigir:
    def ovoz(self): return "Mu"

for h in [Mushuk(), It(), Sigir()]:
    print(h.ovoz())
# Bir xil ovoz() chaqiruvi — har obyekt o'zicha javob beradi (polimorfizm)
>>> Miyov
>>> Vov
>>> Mu

Mashq 4: @property bilan Termometr

Termometr klassi harorat saqlasin, lekin -273 dan past qiymat qabul qilmasin (mutlaq nol). @property va setter ishlating.
class Termometr:
    def __init__(self):
        self._t = 0

    @property
    def harorat(self):
        return self._t

    @harorat.setter
    def harorat(self, v):
        if v < -273:
            raise ValueError("Juda past!")
        self._t = v

term = Termometr()
term.harorat = 25
print(term.harorat)
# setter tekshiradi: -273 dan past bo'lsa xato beradi
>>> 25

Mashq 5: Abstract klass — To'lov

To'lov abstract klassi yozing: amalga_oshir() majburiy metod bo'lsin. Karta klassi undan meros olib, metodni yozsin.
from abc import ABC, abstractmethod

class To'lov(ABC):
    @abstractmethod
    def amalga_oshir(self, summa): pass

class Karta(To'lov):
    def amalga_oshir(self, summa):
        print(f"Kartadan {summa} so'm to'landi")

Karta().amalga_oshir(50000)
# Karta amalga_oshir() ni yozgani uchun obyekt yaratilishi mumkin
>>> Kartadan 50000 so'm to'landi
