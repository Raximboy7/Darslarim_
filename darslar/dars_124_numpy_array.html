NumPy Nima?

NumPy (Numerical Python) — bu Python uchun raqamli hisoblar kutubxonasi. Uning yuragida array (massiv) deb ataladigan maxsus ma'lumotlar turi yotadi. Array oddiy list'ga o'xshaydi, lekin u juda tez ishlaydi va minglab raqamlar ustida bir vaqtda amal bajara oladi.
NumPy ilm-fan, sun'iy intellekt, ma'lumotlar tahlili va muhandislik hisoblarining asosidir. Pandas, scikit-learn, TensorFlow kabi mashhur kutubxonalar ham ichida NumPy'dan foydalanadi.

O'rnatish va Import

NumPy Python bilan birga kelmaydi — uni alohida o'rnatish lozim. Terminalda quyidagi buyruqni yozing:
pip install numpy
# Terminalda yoziladi (bir marta o'rnatiladi)
Dasturda esa uni import qilamiz. An'anaga ko'ra NumPy doim np qisqartmasi bilan import qilinadi:
import numpy as np
# Endi numpy o'rniga qisqacha np yozamiz

Array Yaratish

Eng oddiy usul — oddiy list'ni np.array() ichiga berish:
import numpy as np

a = np.array([1, 2, 3, 4])
print(a)
print(type(a))
# List'dan array yasadik
>>> [1 2 3 4]
>>> <class 'numpy.ndarray'>
E'tibor bering: array chop etilganda elementlar orasida vergul bo'lmaydi — faqat bo'sh joy bor. Uning turi esa ndarray (n-o'lchovli array).

List vs Array — Asosiy Farq

Tashqaridan o'xshasa-da, list va array ichida juda farq qiladi.
XususiyatListNumPy Array
TezlikSekinroqJuda tez
Element turiAralash bo'lishi mumkinBir xil tur (masalan, hammasi son)
Matematik amalQo'lda sikl kerakButun arrayga birdan
XotiraKo'proqKamroq, ixcham
Eng kuchli farq — matematik amal. List'ni 2 ga ko'paytirsangiz u cho'ziladi, array esa har bir elementni 2 ga ko'paytiradi:
ro'yxat = [1, 2, 3]
massiv = np.array([1, 2, 3])

print(ro'yxat * 2)
print(massiv * 2)
# List cho'ziladi, array elementlari ko'payadi
>>> [1, 2, 3, 1, 2, 3]
>>> [2 4 6]

Array Xususiyatlari

Har bir array o'zi haqida ma'lumot beradi. Bular metod emas — qavssiz yoziladigan atributlar:
AtributVazifasiMisol natija
.shapeShakli (qator, ustun)(2, 3)
.ndimO'lchamlar soni2
.sizeJami elementlar soni6
.dtypeElementlar turiint64
m = np.array([[1, 2, 3], [4, 5, 6]])

print(m.shape)
print(m.ndim)
print(m.size)
print(m.dtype)
# 2 qator, 3 ustunli array haqida ma'lumot
>>> (2, 3)
>>> 2
>>> 6
>>> int64

Tayyor Array'lar Yaratish

NumPy'da array'ni qo'lda yozmasdan, tez yaratuvchi funksiyalar bor:

np.zeros() — nollardan array

print(np.zeros(4))
# 4 ta noldan iborat array
>>> [0. 0. 0. 0.]

np.ones() — birlardan array

print(np.ones(3))
# 3 ta birdan iborat array
>>> [1. 1. 1.]

np.arange() — oraliqdagi sonlar

print(np.arange(0, 10, 2))
# 0 dan 10 gacha, 2 qadam bilan (10 kirmaydi)
>>> [0 2 4 6 8]

np.linspace() — teng bo'lingan sonlar

print(np.linspace(0, 1, 5))
# 0 dan 1 gacha teng 5 ta son (chetlari kiradi)
>>> [0. 0.25 0.5 0.75 1. ]
