Modul 3 — Shartlar va sikllar | Dars 9

Taqqoslash Nima?

Taqqoslash operatorlari ikki qiymatni o'zaro solishtiradi. Solishtirish natijasi HAR DOIM Boolean qiymat bo'ladi: True (rost) yoki False (yolg'on). Bu operatorlar shartlarni tekshirishda — masalan, "agar yosh 18 dan katta bo'lsa" — asos bo'lib xizmat qiladi.
print(5 == 5)
print(5 > 3)
# Natija har doim True yoki False
>>> True
>>> True

Oltita Asosiy Operator

OperatorMa'nosiMisol → Natija
==Teng5 == 5 → True
!=Teng emas5 != 3 → True
>Katta5 > 3 → True
<Kichik5 < 3 → False
>=Katta yoki teng5 >= 5 → True
<=Kichik yoki teng3 <= 5 → True

Sonlarni Solishtirish

Sonlar matematikadagidek solishtiriladi. Natijani print() bilan chiqarib ko'rishingiz mumkin.
yosh = 20
print(yosh >= 18)
# 20 katta yoki teng 18 mi?
>>> True

Matnlarni Solishtirish

Stringlarni ham solishtirish mumkin. Eng ko'p ishlatiladigani — == bilan ikki matn bir xilmi yoki yo'qligini tekshirish. E'tibor bering: solishtirish katta-kichik harfga sezgir — "Ali" va "ali" bir xil emas.
print("ali" == "ali")
print("ali" != "bobur")
print("Ali" == "ali")
# Bir xilmi?
# Boshqami?
# Katta harf farq qiladi
>>> True
>>> True
>>> False

Eng Muhim Farq: == va =

⚠️ == (taqqoslash) va = (qiymat berish) — ikki boshqa narsa!

= belgisi qiymat beradi (saqlaydi).
== belgisi ikki qiymatni solishtiradi va True/False qaytaradi.

= bitta tenglik — qiymat berish
yosh = 18

"yosh o'zgaruvchisiga 18 ni saqla" degani.

== ikkita tenglik — solishtirish
print(yosh == 18)

"yosh 18 ga tengmi?" degani — natija True yoki False.

Keng Tarqalgan Xato

❌ Xato: solishtirishda = ishlatish
print(yosh = 18) # SyntaxError!

Solishtirmoqchi bo'lsangiz = emas, ikkita teng belgisi kerak.

✓ To'g'ri: solishtirishda == ishlatish
print(yosh == 18)
>>> True
Eslab Qoling

• Taqqoslash natijasi True yoki False
• Operatorlar: == != > < >= <=
• = qiymat beradi, == solishtiradi
• Matn solishtirish katta-kichik harfga sezgir: "Ali" ≠ "ali"
