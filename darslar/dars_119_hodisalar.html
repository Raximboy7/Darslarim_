Hodisalar (Events)

Modul 18 — GUI (tkinter) | Dars 119

Hodisa (Event) Nima?

Hodisa (event) — bu foydalanuvchining dasturdagi har qanday harakati: tugmani bosish, klaviaturada tugma bosish, sichqonchani harakatlantirish yoki oynani yopish. Dastur shu harakatlarga javob beradi.
Oddiy dastur tepadan pastga bir marta ishlaydi va tugaydi. GUI dastur esa kutib turadi — siz nimadir qilguningizcha jim turadi, harakat qilsangiz — javob beradi. Bu kutish jarayoni mainloop() ichida bo'ladi.

Callback Funksiya

Hodisa ro'y berganda chaqiriladigan funksiya — bu callback (qaytma funksiya) deyiladi. Siz funksiyani yozasiz, ammo uni o'zingiz chaqirmaysiz — tkinter hodisa bo'lganda o'zi chaqiradi.
def salomlash():
    print("Tugma bosildi!")
# Bu callback funksiya — tugma bosilganda chaqiriladi
Diqqat: qavssiz yoziladi!

Callback nomini qavssiz beramiz: salomlash — bu "shu funksiyani keyin chaqir" degani.
Agar salomlash() deb qavs bilan yozsangiz, funksiya darrov ishlaydi (xato!).

1-usul: Button command

Tugmaning eng oddiy hodisasi — bosilishi. Buni command= parametri bilan bog'laymiz.
import tkinter as tk

def bosildi():
    natija.config(text="Salom!")

oyna = tk.Tk()
natija = tk.Label(oyna, text="...")
natija.pack()

tugma = tk.Button(oyna, text="Bos", command=bosildi)
tugma.pack()
oyna.mainloop()
# command=bosildi → tugma bosilganda "bosildi" funksiyasi ishlaydi
>>> Tugma bosilsa, Label "Salom!" ga o'zgaradi

2-usul: bind() bilan Hodisa Bog'lash

command= faqat tugma bosilishi uchun. Boshqa hodisalar uchun — sichqoncha, klaviatura, sichqoncha harakati — bind() ishlatamiz.
widget.bind("<Button-1>", callback)
# bind(hodisa_nomi, chaqiriladigan_funksiya)

Eng ko'p ishlatiladigan hodisalar

HodisaMa'nosi
<Button-1>Sichqonchaning chap tugmasi bosildi
<Button-3>Sichqonchaning o'ng tugmasi bosildi
<Double-Button-1>Ikki marta tez bosish (double click)
<Motion>Sichqoncha harakatlanmoqda
<Key>Klaviaturada istalgan tugma bosildi
<Return>Enter tugmasi bosildi
<Enter>Sichqoncha widget ustiga keldi
<Leave>Sichqoncha widgetdan chiqib ketdi
Esda tuting

Hodisa nomi har doim < > burchakli qavslar ichida yoziladi.

Event Obyekti

bind() orqali chaqirilgan callback bitta argument oladi — event obyekti. Unda hodisa haqida ma'lumot bor: bosilgan joy, bosilgan tugma va h.k.
def bosildi(event):
    print("X:", event.x, "Y:", event.y)

oyna.bind("<Button-1>", bosildi)
# event.x va event.y → sichqoncha bosilgan joyning koordinatasi
MaydonNimani beradi
event.xSichqonchaning X (gorizontal) koordinatasi
event.ySichqonchaning Y (vertikal) koordinatasi
event.charKlaviaturada bosilgan harf
event.keysymTugma nomi (masalan "Return", "a", "space")
event.widgetHodisa ro'y bergan widget
Muhim farq

• command= callback'i argument olmaydi: def f():
• bind() callback'i event oladi: def f(event):

Misol 3: Hodisa → Callback Oqimi

Hodisa qanday ishlaydi? Tugmani bosing va bosqichlar yonib-yonib ketishini kuzating: foydalanuvchi harakati → hodisa hosil bo'ladi → callback chaqiriladi → natija ko'rinadi.
👆 Harakat
(tugma bosildi)
→
⚡ Hodisa
(event hosil bo'ldi)
→
🔁 Callback
(funksiya ishladi)
→
✅ Natija
(ekran o'zgardi)
Eslab Qoling

• Hodisa (event) — foydalanuvchining harakati
• command=funksiya — tugma bosilishi uchun (qavssiz!)
• bind("<hodisa>", funksiya) — boshqa hodisalar uchun
• bind callback'i event argumentini oladi
• event.x, event.char — hodisa haqida ma'lumot
