Modul 18 — GUI (tkinter) | Dars 123

Bu Darsda Nima Qilamiz?

Bu modulning yakuniy proyekti! Endi biz tkinter bilan o'rgangan hamma narsani bitta to'liq ilovada birlashtiramiz — To'do (Vazifalar) Ilovasi.

Bu ilovada: oyna ochish, widgetlar (Entry, Button, Label), layout (joylashuv), hodisalar (tugma bosish), messagebox (ogohlantirish) — barchasi bir joyda ishlaydi.
Maqsad

Foydalanuvchi vazifa yozadi → tugmani bosadi → vazifa ro'yxatga qo'shiladi. Vazifani bajarilgan deb belgilash va o'chirish ham mumkin bo'ladi. Real ishlaydigan, foydali dastur!

1-Bosqich: Oyna va Asosiy Tuzilma

Har bir GUI ilova bitta asosiy oynadan boshlanadi. Biz uni tkinter.Tk() bilan yaratamiz va sozlaymiz.
import tkinter as tk
from tkinter import messagebox

oyna = tk.Tk()
oyna.title("Mening Vazifalarim")
oyna.geometry("360x500")
oyna.config(bg="#f8fafc")
# Tk() — asosiy oyna
# title() — sarlavha matni
# geometry() — o'lcham (kenglik x balandlik)
# config(bg=...) — fon rangi

2-Bosqich: Widgetlarni Yaratish

Endi oynaga elementlar (widgetlar) qo'shamiz: yozish uchun Entry, qo'shish uchun Button, vazifalar uchun Listbox.
kiritish = tk.Entry(oyna, font=("Segoe UI", 13))
qoshish_tugma = tk.Button(oyna, text="Qo'shish")
royxat = tk.Listbox(oyna, font=("Segoe UI", 12))
# Entry — bir qatorli matn maydoni
# Button — bosiladigan tugma
# Listbox — vazifalar ro'yxati

3-Bosqich: Hodisa (Funksiya) Yozish

Tugma bosilganda nima bo'lishini funksiyada yozamiz. Bu funksiya kiritilgan matnni o'qiydi va ro'yxatga qo'shadi. Agar maydon bo'sh bo'lsa — messagebox bilan ogohlantirish chiqaramiz.
def qoshish():
    matn = kiritish.get()  # maydondagi matn
    if matn.strip() == "":
        messagebox.showwarning("Xato", "Bo'sh!")
        return
    royxat.insert(tk.END, matn)  # oxiriga qo'sh
    kiritish.delete(0, tk.END)  # maydonni tozala
# get() — Entry ichidagi matnni oladi
# strip() == "" — bo'sh emasligini tekshirish
# insert(END, ...) — ro'yxat oxiriga qo'shish
# delete(0, END) — maydonni bo'shatish

4-Bosqich: Bog'lash, Joylashtirish va Ishga Tushirish

Tugmani funksiyaga bog'laymiz (command=qoshish), widgetlarni joylashtiramiz (pack) va oynani ishga tushiramiz (mainloop).
qoshish_tugma.config(command=qoshish)

kiritish.pack(pady=10, padx=20, fill="x")
qoshish_tugma.pack(pady=5)
royxat.pack(pady=10, padx=20, fill="both", expand=True)

oyna.mainloop()  # oynani ko'rsatib turish
# command=qoshish — tugmani funksiyaga bog'lash
# pack() — widgetni oynaga joylash
# mainloop() — ilovani ishlatib turadi (eng oxirda!)
Diqqat

mainloop() har doim eng oxirgi qator bo'lishi kerak. U oynani yopilguncha ochiq saqlaydi va tugma bosishlarni "tinglab" turadi.

    To'liq Kod Bir Joyda

    Yuqoridagi 4 bosqichni birlashtirsak, mana butun ilova kodi:
    import tkinter as tk
    from tkinter import messagebox

    def qoshish():
        matn = kiritish.get()
        if matn.strip() == "":
            messagebox.showwarning("Xato", "Maydon bo'sh!")
            return
        royxat.insert(tk.END, matn)
        kiritish.delete(0, tk.END)

    def ochirish():
        tanlangan = royxat.curselection()
        if tanlangan:
            royxat.delete(tanlangan[0])

    oyna = tk.Tk()
    oyna.title("Mening Vazifalarim")
    oyna.geometry("360x500")

    kiritish = tk.Entry(oyna, font=("Segoe UI", 13))
    kiritish.pack(pady=10, padx=20, fill="x")

    tk.Button(oyna, text="Qo'shish", command=qoshish).pack()
    tk.Button(oyna, text="O'chirish", command=ochirish).pack(pady=5)

    royxat = tk.Listbox(oyna, font=("Segoe UI", 12))
    royxat.pack(pady=10, padx=20, fill="both", expand=True)

    oyna.mainloop()
    # Butun ilova ~25 qator kod!
    # qoshish() — vazifa qo'shadi
    # ochirish() — tanlangan vazifani o'chiradi
    # curselection() — tanlangan qatorni qaytaradi
    # mainloop() — eng oxirida, ilovani ishlatadi

    Asosiy Metodlar Jadvali

    MetodVazifasi

    .get()Entry ichidagi matnni oladi
    .insert(END, x)Ro'yxat oxiriga element qo'shadi
    .delete(0, END)Maydonni yoki elementni o'chiradi

    Eslab Qoling

    • Ilova = oyna + widgetlar + hodisalar
    • Tugma command=funksiya orqali ishga tushadi
    • messagebox bilan foydalanuvchini ogohlantiring
    • mainloop() har doim eng oxirgi qator
