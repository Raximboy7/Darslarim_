Bu Darsda Nima O'rganamiz?

Haqiqiy dasturlar foydalanuvchi bilan oynalar orqali gaplashadi: "Saqlaysizmi?" deb so'raydi, xatolik haqida ogohlantiradi, fayl tanlash oynasini ochadi. Bu darsda tkinter'ning uchta muhim qismini ko'ramiz: messagebox (xabar oynalari), filedialog (fayl tanlash oynasi) va Menu (oyna tepasidagi menyu).

1. messagebox — Xabar Oynalari

messagebox — bu kichik oyna ochib, foydalanuvchiga xabar ko'rsatadi yoki savol beradi. Avval uni import qilamiz.
from tkinter import messagebox

messagebox.showinfo("Ma'lumot", "Ish bajarildi!")
# Birinchi argument — sarlavha, ikkinchisi — matn

messagebox turlari

FunksiyaVazifasiQaytaradi
showinfo()Oddiy ma'lumot beradi (ko'k "i")"ok"
showwarning()Ogohlantiradi (sariq "!")"ok"
showerror()Xatolik haqida xabar (qizil "x")"ok"
askyesno()Ha/Yo'q savol beradiTrue / False
javob = messagebox.askyesno("Tasdiq", "Faylni o'chirasizmi?")
if javob:
    print("O'chirildi")
else:
    print("Bekor qilindi")
# askyesno True (Ha) yoki False (Yo'q) qaytaradi
>>> O'chirildi

2. filedialog — Fayl Tanlash Oynasi

filedialog — kompyuterdan fayl tanlash yoki saqlash oynasini ochadi. U tanlangan faylning manzilini (yo'lini) matn ko'rinishida qaytaradi.
from tkinter import filedialog

# Faylni ochish uchun tanlash
yol = filedialog.askopenfilename()
print(yol)

# Faylni saqlash uchun joy tanlash
joy = filedialog.asksaveasfilename()
# askopenfilename — ochish, asksaveasfilename — saqlash oynasi
FunksiyaVazifasi
askopenfilename()Mavjud faylni ochish uchun tanlaydi
asksaveasfilename()Faylni saqlash joyini tanlaydi
askdirectory()Papka (jild) tanlaydi
Eslatma

Agar foydalanuvchi hech narsa tanlamay oynani yopsa, bu funksiyalar bo'sh matn "" qaytaradi. Shuning uchun har doim tanlangan-tanlanmaganini tekshirib turing.

3. Menu — Oyna Menyusi

Menu — oyna tepasidagi File, Edit kabi menyularni yasaydi. Har bir menyu ochilganda ichida buyruqlar (masalan "Ochish", "Saqlash") ko'rinadi.
import tkinter as tk

oyna = tk.Tk()
menyu = tk.Menu(oyna)

# File menyusini yaratamiz
fayl_menyu = tk.Menu(menyu, tearoff=0)
fayl_menyu.add_command(label="Ochish", command=ochish)
fayl_menyu.add_command(label="Saqlash", command=saqlash)
fayl_menyu.add_separator()
fayl_menyu.add_command(label="Chiqish", command=oyna.quit)

# File menyusini asosiy menyuga ulaymiz
menyu.add_cascade(label="File", menu=fayl_menyu)
oyna.config(menu=menyu)
# add_command — buyruq qo'shadi, add_cascade — menyuni asosiy panelga ulaydi
MetodVazifasi
Menu(oyna)Yangi menyu obyektini yaratadi
add_command(label, command)Bosiladigan buyruq qo'shadi
add_cascade(label, menu)Ichki menyuni asosiy panelga ulaydi
add_separator()Ajratuvchi chiziq qo'yadi
config(menu=...)Menyuni oynaga biriktiradi
Eslab Qoling

• showinfo / showwarning / showerror — xabar ko'rsatadi
• askyesno — True yoki False qaytaradi
• filedialog — fayl manzilini matn qilib qaytaradi
• add_command buyruq, add_cascade menyu ulaydi
