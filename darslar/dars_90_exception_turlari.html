Built-in Exception Turlari

Modul 8 — Xatolar, Test va Debug | Dars 90

Built-in Exception Nima?

Python ichida juda ko'p tayyor (built-in) xato turlari bor. Dasturda biror narsa noto'g'ri bo'lganda Python o'zi mos xato turini "tashlaydi" (raises). Har bir xatoning nomi bor — masalan NameError, TypeError, ValueError. Xato nomini bilsangiz, muammoni tezroq topasiz.
Asosiy g'oya

Xato turini tanish — bu kasallikni tashxis qilishga o'xshaydi. Python sizga "qaysi turdagi xato yuz berdi" deb aytadi, siz esa shu nom bo'yicha sababni topasiz va tuzatasiz.

NameError — Noma'lum o'zgaruvchi

O'zgaruvchini e'lon qilmasdan ishlatsangiz yoki nomini noto'g'ri yozsangiz yuz beradi.
print(yosh)
# "yosh" o'zgaruvchisi hech qachon yaratilmagan
NameError: name 'yosh' is not defined

TypeError — Noto'g'ri tur

Bir-biriga mos kelmaydigan turlar ustida amal qilsangiz yuz beradi. Masalan matn (string) va son (int) ni + bilan qo'shsangiz.
natija = "2" + 2
# String va int ni qo'shib bo'lmaydi
TypeError: can only concatenate str (not "int") to str

ValueError — Noto'g'ri qiymat

Tur to'g'ri, ammo qiymat amalga to'g'ri kelmaydi. Masalan int() ga raqam bo'lmagan matn bersangiz.
son = int("salom")
# "salom" ni butun songa aylantirib bo'lmaydi
ValueError: invalid literal for int() with base 10: 'salom'

IndexError — Yo'q indeks

Ro'yxatda mavjud bo'lmagan indeksga murojaat qilsangiz yuz beradi.
ronxat = [10, 20, 30]
print(ronxat[5])
# Faqat 0, 1, 2 indekslar bor — 5 yo'q
IndexError: list index out of range

KeyError — Yo'q kalit

Lug'atda (dictionary) mavjud bo'lmagan kalitni so'rasangiz yuz beradi.
odam = {"ism": "Ali"}
print(odam["yosh"])
# Lug'atda faqat "ism" bor, "yosh" yo'q
KeyError: 'yosh'

ZeroDivisionError — Nolga bo'lish

Matematikada sonni nolga bo'lib bo'lmaydi. Python ham buni xato deb hisoblaydi.
natija = 10 / 0
# Nolga bo'lib bo'lmaydi
ZeroDivisionError: division by zero

FileNotFoundError — Fayl yo'q

Mavjud bo'lmagan faylni ochmoqchi bo'lsangiz yuz beradi.
fayl = open("yoq.txt")
# Bunday nomli fayl mavjud emas
FileNotFoundError: [Errno 2] No such file or directory: 'yoq.txt'
Eslab Qoling

• NameError — o'zgaruvchi yo'q yoki nomi xato
• TypeError — turlar mos emas
• ValueError — qiymat noto'g'ri
• IndexError / KeyError — yo'q indeks / kalit
• ZeroDivisionError — nolga bo'lish
• FileNotFoundError — fayl topilmadi

Misol 1: Xato Turini Tanlang

Xato turini tanlang — uni keltirib chiqaradigan kod va Python beradigan xabar ko'rsatiladi.

Misol 2: Qaysi Xato Chiqadi?

Kod parchasini tanlang va Python qaysi xatoni beradi — taxmin qilib, tugmani bosing.

Misol 3: Xato → Sabab Moslashtirish

Chap ustundan xato nomini, o'ng ustundan unga mos sababni bosing. To'g'ri juftliklar yashil bo'ladi.
