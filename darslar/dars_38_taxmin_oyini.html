Loyiha: Taxmin o'yini

Modul 3 — Shartlar va Sikllar | Dars 38

Loyiha haqida

Taxmin o'yini — bu kompyuter tasodifiy son o'ylab qo'yadigan, foydalanuvchi esa uni topishga harakat qiladigan klassik o'yin. Har taxmindan keyin dastur "katta" yoki "kichik" degan ishora beradi. Bu loyiha random, while va if ni birga ishlatishni o'rgatadi.

random moduli

Tasodifiy son olish uchun Python ning random modulidan foydalanamiz. randint(a, b) — a va b oralig'idagi (ikkalasini ham qamrab) tasodifiy butun son beradi: 

import random

yashirin = random.randint(1, 100)
print("Kompyuter 1-100 orasida son o'yladi")
import random modulni chaqiradi. randint(1, 100) 1 dan 100 gacha tasodifiy son qaytaradi.

while sikli bilan takrorlash

Foydalanuvchi to'g'ri sonni topmaguncha o'yin davom etishi kerak. Buning uchun while True cheksiz siklidan foydalanamiz va son topilganda break bilan chiqamiz: 

import random

yashirin = random.randint(1, 100)
urinishlar = 0

while True:
    taxmin = int(input("Taxmininingiz: "))
    urinishlar += 1

    if taxmin < yashirin:
        print("Kattaroq son ayting!")
    elif taxmin > yashirin:
        print("Kichikroq son ayting!")
    else:
        print("Tabriklaymiz! Topdingiz!")
        print("Urinishlar soni:", urinishlar)
        break
Har taxmindan keyin uch holat: kichik, katta yoki teng. Teng bo'lsa g'olib bo'ldi va break siklni to'xtatadi. urinishlar += 1 har safar hisobni oshiradi.
>>> Taxmininingiz: 50
>>> Kattaroq son ayting!
>>> Taxmininingiz: 75
>>> Kichikroq son ayting!
>>> Taxmininingiz: 62
>>> Tabriklaymiz! Topdingiz!
>>> Urinishlar soni: 3

Urinishlar sonini cheklash

O'yinni qiziqarli qilish uchun urinishlar sonini cheklaymiz. Masalan, 7 ta imkoniyat beramiz: 

import random

yashirin = random.randint(1, 100)
imkoniyat = 7

while imkoniyat > 0:
    taxmin = int(input("Taxmin: "))
    imkoniyat -= 1

    if taxmin == yashirin:
        print("Topdingiz!")
        break
    elif taxmin < yashirin:
        print("Kattaroq. Qolgan:", imkoniyat)
    else:
        print("Kichikroq. Qolgan:", imkoniyat)
else:
    print("Imkoniyatlar tugadi! Son edi:", yashirin)
while ... else — sikl break siz tugasa (ya'ni imkoniyat 0 ga yetsa) else ishlaydi.

Amaliyot va xulosa

Loyihani rivojlantiring

1. Diapazonni o'zgartiring (1-1000) va urinishlar limitini moslang.
2. Foydalanuvchi noto'g'ri (raqam bo'lmagan) qiymat kiritsa xatoni ushlang.
3. O'yin tugagach "Yana o'ynaysizmi?" deb so'rab, qaytadan boshlash imkonini bering.

Nimalarni o'rgandik

• import random va randint() bilan tasodifiy son.
• while True + break — takrorlanuvchi o'yin sikli.
• if/elif/else bilan taxminni solishtirish va ishora berish.
