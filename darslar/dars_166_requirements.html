requirements.txt Nima?

requirements.txt — bu loyiha ishlashi uchun kerak bo'lgan barcha kutubxonalar va ularning versiyalari yozilgan oddiy matn fayl. U loyihaning "xarid ro'yxati" — kim loyihani ko'chirib olsa, bir buyruq bilan barcha kerakli kutubxonalarni o'rnata oladi.
Nega kerak?

Sizning loyihangiz flask, requests kabi kutubxonalarga tayanadi. Do'stingiz loyihani ochganda ularni qo'lda birma-bir o'rnatishi shart emas — requirements.txt borligi uchun hammasini bitta buyruqda oladi.

Fayl Ichi Qanday Ko'rinadi?

Har bir qatorda bitta kutubxona nomi va (ixtiyoriy) versiyasi yoziladi. Versiyani == orqali aniq belgilaymiz.
flask==3.0.0
requests==2.31.0
python-dotenv==1.0.0
# Har qator: kutubxona==versiya. Bu loyiha aynan shu versiyalar bilan sinovdan o'tgan
YozuvMa'nosi
flask==3.0.0Aynan 3.0.0 versiyasi (eng ishonchli)
flask>=3.0.03.0.0 va undan yuqori versiya
flaskHar qanday (eng oxirgi) versiya
Maslahat

Real loyihalarda ko'pincha == bilan aniq versiya yoziladi. Shunda kod bugun ham, bir yildan keyin ham bir xil ishlaydi — kutubxona yangilanib qolib xato chiqarmaydi.

1. Faylni Avtomatik Yaratish — pip freeze

Faylni qo'lda yozish shart emas. pip freeze buyrug'i joriy muhitda o'rnatilgan hamma kutubxonani versiyasi bilan ko'rsatadi. Uni > belgisi bilan faylga yo'naltiramiz.
pip freeze > requirements.txt
# pip freeze — o'rnatilganlar ro'yxatini chiqaradi; "> requirements.txt" — natijani shu faylga yozadi
# pip freeze buyrug'i chiqishi:
flask==3.0.0
Jinja2==3.1.2
Werkzeug==3.0.1
... (barchasi requirements.txt ga yoziladi)

2. Fayldan O'rnatish — pip install -r

Boshqa kompyuterda (yoki yangi venv da) loyihani ishga tushirish uchun, -r (requirements) bayrog'i bilan faylni ko'rsatamiz. pip barcha kutubxonani birma-bir kerakli versiyada o'rnatadi.
pip install -r requirements.txt
# -r requirements.txt — fayldagi barcha kutubxonani belgilangan versiyalarda o'rnatadi
To'liq oqim

1) venv yarating va faollashtiring → 2) pip install -r requirements.txt → 3) loyiha darrov ishga tayyor. Kutubxonalarni qidirib yurish kerak emas.

Amaliyot

1
Kutubxona o'rnating

Faol venv da pip install flask requests bajaring

2
Faylni yarating

pip freeze > requirements.txt buyrug'ini bajaring

3
Faylni tekshiring

requirements.txt ni oching — kutubxonalar va versiyalar yozilganini ko'ring

4
Qayta o'rnatib ko'ring

Yangi venv da pip install -r requirements.txt bilan hammasini tiklang

Xulosa

Eslab qoling

• requirements.txt — loyiha kutubxonalari va versiyalari ro'yxati
• kutubxona==versiya — aniq versiyani belgilaydi
• pip freeze > requirements.txt — faylni avtomatik yaratadi
• pip install -r requirements.txt — fayldan hammasini o'rnatadi
• venv + requirements.txt — loyihani boshqa joyda oson ishga tushirish kaliti
