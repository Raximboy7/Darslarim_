Kirish: Iterator nima?

Iterator — bu qiymatlarni bittalab qaytarib beradigan obyekt. for sikli aslida har bir list, str yoki range ustidan aynan iterator yordamida yuradi. Bu darsda biz Python'ga o'z iteratorimizni yasab beramiz.

Har qanday obyektni iterator qilish uchun ikkita "sehrli" (dunder) metod kerak:

  • __iter__(self) — iterator obyektining o'zini qaytaradi.
  • __next__(self) — keyingi qiymatni qaytaradi, tugasa StopIteration chaqiradi.

Iterator protokoli qanday ishlaydi?

for x in obj: yozganingizda Python ichki tomondan quyidagini bajaradi:

1
iter(obj)

Obyektning __iter__ metodi chaqiriladi va iterator olinadi.

2
next(iterator)

Har aylanishda __next__ chaqirilib keyingi qiymat olinadi.

3
StopIteration

Qiymatlar tugaganda bu xatolik chaqiriladi — for uni ushlab, sikldan chiqadi.

1-misol: Sanaydigan iterator

1 dan berilgan songacha sanaydigan o'z klassimizni yozamiz:

class Sanagich:
    def __init__(self, chegara):
        self.chegara = chegara
        self.joriy = 0

    def __iter__(self):
        return self          # iteratorning o'zi

    def __next__(self):
        if self.joriy >= self.chegara:
            raise StopIteration   # tugadi
        self.joriy += 1
        return self.joriy

for son in Sanagich(5):
    print(son)
__init__ boshlang'ich holatni saqlaydi. __next__ har chaqirilganda joriyni birga oshirib qaytaradi, chegaraga yetganda StopIteration chaqiradi.
>>> 1
>>> 2
>>> 3
>>> 4
>>> 5

2-misol: Fibonachchi iteratori

Iterator ichida murakkabroq mantiq ham bo'lishi mumkin:

class Fibonachchi:
    def __init__(self, nechta):
        self.nechta = nechta
        self.hisob = 0
        self.a, self.b = 0, 1

    def __iter__(self):
        return self

    def __next__(self):
        if self.hisob >= self.nechta:
            raise StopIteration
        natija = self.a
        self.a, self.b = self.b, self.a + self.b
        self.hisob += 1
        return natija

print(list(Fibonachchi(8)))
Har __next__ da a va b almashtiriladi — bu Fibonachchi qatorini hosil qiladi. list() ham iterator ustidan yuradi, shuning uchun ishlaydi.
>>> [0, 1, 1, 2, 3, 5, 8, 13]

Amaliyot

Mustaqil bajaring

1. Teskari nomli klass yozing — berilgan ro'yxatni oxiridan boshiga qarab qaytarsin.
2. Juftlar iteratori yozing — 0, 2, 4, 6 ... kabi faqat juft sonlar bersin.
3. iter() va next() ni qo'lda chaqirib, StopIterationni try/except bilan ushlab ko'ring.

Xulosa

Esda tuting

• Iterator = __iter__ + __next__ bo'lgan obyekt.
• __iter__ ko'pincha return self qaytaradi.
• Qiymatlar tugaganda albatta StopIteration chaqiring.
• for, list(), sum() — hammasi shu protokolga tayanadi.
