else va finally

Modul 8 — Xatolar, Test va Debug | Dars 89

try/except ni to'ldiruvchi qismlar

Siz try/except ni bilasiz: xatoni ushlash uchun. Endi unga else va finally qismlarini qo'shamiz. Ular kodni yanada tartibli va ishonchli qiladi.
QismQachon ishlaydi
tryHar doim — sinaladigan kod
exceptFaqat xato yuz bersa
elseFaqat xato yuz bermasa
finallyHar doim — xato bo'lsa ham, bo'lmasa ham

else — xato bo'lmaganda

else bloki faqat try muvaffaqiyatli o'tsa ishlaydi. Xatosiz o'tgandagina bajariladigan kodni shu yerga qo'yamiz.

try:
    son = int(input("Son: "))
except ValueError:
    print("Bu son emas!")
else:
    print("Rahmat! Son:", son)
# Foydalanuvchi to'g'ri son kiritsa → else ishlaydi. Xato kiritsa → except ishlaydi, else esa o'tkazib yuboriladi.
Kirish: 5
>>> Rahmat! Son: 5

Kirish: salom
>>> Bu son emas!

finally — har doim ishlaydi

finally bloki har qanday holatda bajariladi. Ko'pincha resurslarni yopish (fayl, ulanish) uchun ishlatiladi.

try:
    natija = 10 / 0
except ZeroDivisionError:
    print("Nolga bo'lib bo'lmaydi")
finally:
    print("Bu qism har doim ishlaydi")
# Xato bo'ldi → except ishladi, keyin finally ham ishladi. Agar xato bo'lmaganida ham finally baribir ishlar edi.
>>> Nolga bo'lib bo'lmaydi
>>> Bu qism har doim ishlaydi

To'liq misol: to'rttasi birga

def bol(a, b):
    try:
        natija = a / b
    except ZeroDivisionError:
        print("Xato: nolga bo'lish")
    else:
        print("Natija:", natija)
    finally:
        print("Amal tugadi")

bol(10, 2)
print("---")
bol(10, 0)
# 1-chaqiruv: xato yo'q → else + finally. 2-chaqiruv: xato bor → except + finally. finally har ikkisida ham ishlaydi.
>>> Natija: 5.0
>>> Amal tugadi
>>> ---
>>> Xato: nolga bo'lish
>>> Amal tugadi

Amaliy foyda: fayl bilan ishlash

fayl = open("malumot.txt")
try:
    matn = fayl.read()
    print(matn)
finally:
    fayl.close()
    print("Fayl yopildi")
# O'qishda xato bo'lsa ham finally faylni yopadi. Bu resurs oqib ketishining oldini oladi.
Nega else kerak?

Muvaffaqiyat kodini try ichiga emas, else ga qo'ysangiz — kutilmagan xatoni noto'g'ri ushlab qolmaysiz. Kod nima qilayotgani ham aniqroq ko'rinadi.

Xulosa

Eslab qoling

• try — sinaladigan kod
• except — xato bo'lsa
• else — xato bo'lmasa
• finally — har doim (tozalash uchun)
• Tartib har doim: try → except → else → finally
