Python sort() ichida — Timsort

Modul 12 — Algoritmlar va Strukturalar | Dars 135

sorted() ichida nima bor?

Python'da sorted() va list.sort() Timsort algoritmidan foydalanadi. Uni 2002-yilda Tim Peters yozgan. Bu Merge sort va Insertion sort ni birlashtirgan aqlli gibrid algoritm.

Timsort real hayotdagi ma'lumotlar ko'pincha qisman saralangan bo'lishidan foydalanadi va O(n log n) kafolat beradi, eng yaxshi holatda esa O(n).

sort() va sorted() farqi

a = [3, 1, 2]

a.sort()          # ro'yxatni O'ZINI o'zgartiradi, None qaytaradi
print(a)          # [1, 2, 3]

b = [3, 1, 2]
yangi = sorted(b) # YANGI ro'yxat qaytaradi, b o'zgarmaydi
print(b)          # [3, 1, 2]
print(yangi)      # [1, 2, 3]
sort() — faqat ro'yxatlarda, joyida o'zgartiradi. sorted() — har qanday iteratsiyalanuvchida ishlaydi va yangi ro'yxat beradi.

Barqarorlik (stability)

Timsort barqaror: teng qiymatli elementlarning dastlabki tartibi saqlanadi. Bu bir nechta mezon bo'yicha saralashda juda muhim. 

talabalar = [
    ("Ali", 90),
    ("Vali", 85),
    ("Guli", 90),
]
# Avval ismi bo'yicha, keyin ball bo'yicha saralaymiz
talabalar.sort(key=lambda t: t[0])
talabalar.sort(key=lambda t: t[1], reverse=True)
print(talabalar)
90 balli Ali va Guli o'zaro alifbo tartibida qoladi — chunki oldingi saralash buzilmaydi. Bu barqarorlikning kuchi.
>>> [('Ali', 90), ('Guli', 90), ('Vali', 85)]

key parametri

key — har element uchun taqqoslash qiymatini qaytaruvchi funksiya. Bu Timsort'ni juda moslashuvchan qiladi. 

sozlar = ["olma", " nok", "anor", "uzum"]

# uzunligi bo'yicha
print(sorted(sozlar, key=len))

# oxirgi harfi bo'yicha
print(sorted(sozlar, key=lambda s: s[-1]))

# katta-kichik harfga befarq
print(sorted(["Bob", "ali"], key=str.lower))
key har element uchun BIR marta chaqiriladi (tez). Solishtirish funksiyasidan (cmp) ancha samarali.

Timsort qanday ishlaydi (soddalashtirilgan)

1
"Run" larni topadi

Ro'yxatdan allaqachon saralangan ketma-ketliklarni (o'suvchi yoki kamayuvchi) qidiradi.

2
Kichik bo'laklarni Insertion sort qiladi

Qisqa runlar Insertion sort bilan minimal uzunlikka to'ldiriladi — kichik massivda u tez.

3
Runlarni Merge qiladi

Saralangan runlar Merge sort uslubida bosqichma-bosqich birlashtiriladi.

Natija Qisman saralangan ma'lumotda deyarli O(n), umumiy holatda O(n log n), barqaror.

Amaliyot

Mashqlar

1. Lug'atlar ro'yxatini key=lambda d: d["yosh"] bilan saralang.
2. Bir ro'yxatni avval kamayish, keyin o'sish tartibida saralang (reverse).
3. Barqarorlikni tekshirish uchun teng kalitli elementlar bilan tajriba qiling.

Xulosa

Esda saqlang

• Python sort()/sorted() = Timsort, O(n log n), barqaror.
• sort() joyida o'zgartiradi, sorted() yangi ro'yxat qaytaradi.
• key bilan istalgan mezon bo'yicha saralang.
• reverse=True — teskari tartib.
• O'zingiz saralash yozmang — Timsort deyarli har doim tezroq.
