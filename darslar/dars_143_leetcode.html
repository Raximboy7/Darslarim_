Masala 1 — Two Sum (ikki sonning yig'indisi)

Shart: Sonlar ro'yxati va target berilgan. Yig'indisi target ga teng bo'lgan ikki sonning indekslarini qaytaring. 

# Optimal yechim — lug'at (hash map), O(n)
def two_sum(sonlar, target):
    korilgan = {}                     # qiymat -> indeks
    for i, son in enumerate(sonlar):
        kerak = target - son          # juftlik uchun kerakli son
        if kerak in korilgan:        # avval ko'rgan bo'lsak — topdik
            return [korilgan[kerak], i]
        korilgan[son] = i             # hozirgi sonni eslab qolamiz
    return []

print(two_sum([2, 7, 11, 15], 9))
Ikki ichma-ich sikl (O(n^2)) o'rniga lug'atdan foydalanamiz — bir yurishda, O(n). 2 + 7 = 9, indekslari 0 va 1.
>>> [0, 1]

Masala 2 — Palindrom tekshirish

Shart: Satr oldidan va orqadan bir xil o'qilsa — palindrom (masalan "radar", "shalash"). 

# 1-usul: teskarisi bilan solishtirish
def palindrom(s):
    return s == s[::-1]        # s[::-1] — teskari satr

# 2-usul: ikki ko'rsatkich (two pointers), O(n)
def palindrom2(s):
    chap, ong = 0, len(s) - 1
    while chap < ong:
        if s[chap] != s[ong]:     # chetlar teng emas — palindrom emas
            return False
        chap += 1
        ong -= 1
    return True

print(palindrom("radar"), palindrom("salom"))
1-usul qisqa va pythonik. 2-usul esa ikki ko'rsatkich texnikasini ko'rsatadi — u chetlardan markazga qarab yuradi.
>>> True False

Masala 3 — Anagram tekshirish

Shart: Ikki satr bir xil harflardan (bir xil sonda) tuzilgan bo'lsa — anagram (masalan "tarozi" va "roziat"). 

from collections import Counter

# 1-usul: saralab solishtirish, O(n log n)
def anagram(a, b):
    return sorted(a) == sorted(b)

# 2-usul: harflarni sanash, O(n)
def anagram2(a, b):
    return Counter(a) == Counter(b)   # har harf nechtadan

print(anagram("listen", "silent"))
print(anagram2("olma", "malo"))
Counter har harf nechta marta uchraganini sanaydi. Ikki Counter teng bo'lsa — anagram. Bu saralashdan tezroq (O(n)).
>>> True >>> True

Umumiy strategiya

1
Shartni tushun

Kirish va chiqishni aniq belgila, chegara holatlarini (bo'sh, bitta element) o'yla.

2
Sodda yechim yoz

Avval ishlaydigan, hatto sekin (O(n^2)) yechimni tuz.

3
Optimallashtir

Lug'at (hash map), ikki ko'rsatkich yoki Counter bilan tezligini oshir.

Amaliyot

Mashqlar

1. Ro'yxatda takrorlanuvchi son bor-yo'qligini set bilan toping.
2. Satrda birinchi takrorlanmaydigan harfni toping.
3. Two Sum ni uch son yig'indisiga (Three Sum) kengaytiring.

Xulosa

Esda saqlang

• Lug'at (hash map) — izlashni O(n^2) dan O(n) ga tushiradi.
• Ikki ko'rsatkich — palindrom, saralangan ro'yxatlarda kuchli.
• Counter — sanash masalalarini soddalashtiradi.
• Avval ishlat, keyin optimallashtir.
• Chegara holatlarini (bo'sh, bitta element) doim tekshir.
