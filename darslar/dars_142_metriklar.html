Model Baholash (Metriklar)

Modul 20 — Vizualizatsiya va ML | Dars 142

Metrika Nima?

Modelni o'rgatdik — endi u qanchalik yaxshi ishlaydi? degan savol tug'iladi. Metrika — bu modelning sifatini o'lchaydigan raqam. U modelning bashoratlarini haqiqiy javoblar bilan solishtiradi va bizga ball beradi.
Oddiy misol

100 ta rasmni model ko'rdi. 90 tasini to'g'ri topdi. Demak modelning to'g'riligi (accuracy) = 90%. Mana shu — metrika.

Klassifikatsiya Metrikalari

Klassifikatsiya — bu modelning sinflarga ajratishi (masalan: kasal / sog'lom, spam / oddiy xat). Bunday modellarni baholashda 4 ta asosiy metrika ishlatiladi.
MetrikaO'zbekchaNimani o'lchaydi?
accuracyTo'g'rilikUmuman nechta javob to'g'ri chiqdi
precisionAniqlik"Ha" deganlarning nechtasi rost edi
recallTopuvchanlikBor "Ha" larning nechtasini topdi
F1MuvozanatPrecision va recall o'rtachasi

Accuracy — To'g'rilik

Eng sodda metrika. To'g'ri javoblar sonini umumiy javoblar soniga bo'lamiz.
accuracy = to'g'ri / umumiy
# 90 ta to'g'ri, 100 ta jami
accuracy = 90 / 100 = 0.9
# 0.9 = 90% to'g'rilik
Diqqat!

Accuracy har doim ham yetarli emas. Agar 100 ta xatdan 99 tasi oddiy bo'lsa, model hammasini "oddiy" desa ham 99% accuracy chiqadi — lekin u bironta spamni topmaydi! Shuning uchun precision va recall ham kerak.

Precision va Recall — Sodda Tushuntirish

Tasavvur qiling: model kasallikni aniqlaydi.
MetrikaSavolFormula
precisionModel "kasal" degan odamlarning nechta foizi haqiqatan kasal?TP / (TP + FP)
recallHaqiqatdan kasal odamlarning nechta foizini model topdi?TP / (TP + FN)
F1Ikkalasini birlashtirgan muvozanatli ball2·P·R / (P + R)
Eslab qoling

• precision yuqori bo'lsa — model kam yolg'on "ha" aytadi.
• recall yuqori bo'lsa — model haqiqiy holatlarni o'tkazib yubormaydi.
• F1 ikkalasi o'rtasidagi muvozanat.

Confusion Matrix (Chalkashlik Matritsasi)

Confusion matrix — bu modelning bashoratlarini jadval ko'rinishida ko'rsatadi. U 4 ta katakdan iborat: nimani to'g'ri, nimani xato topganini ko'rsatadi.
Bashorat: Ha
Bashorat: Yo'q
Haqiqat: Ha
TPTo'g'ri Ha
FNO'tkazib yuborildi
Haqiqat: Yo'q
FPYolg'on signal
TNTo'g'ri Yo'q
4 ta tushuncha

• TP (True Positive) — to'g'ri "Ha" topdi
• TN (True Negative) — to'g'ri "Yo'q" topdi
• FP (False Positive) — yolg'ondan "Ha" dedi
• FN (False Negative) — bor "Ha" ni o'tkazib yubordi

Regressiya Metrikalari

Regressiya — bu modelning raqam bashorat qilishi (masalan: uy narxi, harorat). Bu yerda accuracy emas, balki xatolik qancha katta ekanini o'lchaymiz.
MetrikaTo'liq nomiMa'nosi

MSEMean Squared ErrorXatoning kvadrati o'rtachasi (katta xatoni jazolaydi)
R2R-squaredModel qanchalik mos (1 = mukammal, 0 = yomon)

sklearn Bilan Baholash

Amalda biz formulani qo'lda hisoblamaymiz. sklearn.metrics moduli tayyor funksiyalarni beradi.
from sklearn.metrics import accuracy_score

haqiqiy = [1, 0, 1, 1, 0]
bashorat = [1, 0, 0, 1, 0]

ball = accuracy_score(haqiqiy, bashorat)
print(ball)
# 5 tadan 4 tasi to'g'ri → 0.8
>>> 0.8
Boshqa tayyor funksiyalar

• precision_score(haqiqiy, bashorat)
• recall_score(haqiqiy, bashorat)
• f1_score(haqiqiy, bashorat)
• confusion_matrix(haqiqiy, bashorat)
