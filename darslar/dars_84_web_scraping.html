Web Scraping Nima?

Web scraping — bu web sahifadan kerakli ma'lumotlarni avtomatik ravishda ajratib olish jarayoni. Masalan, do'kon saytidan mahsulot narxlarini, yangiliklar saytidan sarlavhalarni yoki ob-havo saytidan harorat ma'lumotini dasturingiz bilan to'plab olishingiz mumkin.
Oddiy qilib aytganda

Brauzer web sahifani ko'rsatadi, biz uni ko'z bilan o'qiymiz. Web scraping esa dastur orqali shu sahifani o'qib, kerakli qismlarini avtomatik ajratib oladi.

HTML — Sahifaning Skeleti

Har bir web sahifa HTML tegidan tashkil topgan. Ma'lumot ajratib olish uchun avval HTML qanday tuzilganini bilish kerak. Teglar <teg>...</teg> ko'rinishida bo'ladi.
<h1>Sarlavha</h1>
<p class="narx">100 000 so'm</p>
<a href="https://sayt.uz">Batafsil</a>
# h1 — sarlavha, p — matn, a — havola (link)
# class va href — bular atributlar

BeautifulSoup Kutubxonasi

Python'da web scraping uchun eng mashhur kutubxona — BeautifulSoup. U HTML matnni o'qib, undan teg va ma'lumotlarni oson topishga yordam beradi.

O'rnatish

pip install beautifulsoup4
# Terminalda bir marta o'rnatamiz

HTML'ni o'qish (parsing)

from bs4 import BeautifulSoup

html = "<h1>Salom</h1>"
soup = BeautifulSoup(html, "html.parser")
# html.parser — Python'ning ichki HTML o'quvchisi
# soup endi HTML'ni "tushunadigan" obyekt

Element Topish

BeautifulSoup bilan elementni topishning bir necha usuli bor:
MetodVazifasi
.find("p")Birinchi <p> tegini topadi
.find_all("a")Barcha <a> teglarini ro'yxat qilib qaytaradi
.select(".narx")CSS selektor bilan topadi (class = narx)
.select_one("#id")CSS selektor bilan birinchisini topadi

.find() — birinchi elementni topish

sarlavha = soup.find("h1")
print(sarlavha.text)
# Birinchi h1 tegini topadi
# .text — teg ichidagi matnni oladi
>>> Salom

.find_all() — barcha elementlarni topish

linklar = soup.find_all("a")
for link in linklar:
    print(link["href"])
# Barcha havolalarni topadi
# link["href"] — atribut qiymatini oladi

Matn va Atribut Olish

Element topilgach, undan ikki narsa olamiz: .text (teg ichidagi yozuv) va ['atribut'] (teg xususiyati, masalan href).
html = '<a href="https://uz.wikipedia.org">Wiki</a>'
soup = BeautifulSoup(html, "html.parser")
link = soup.find("a")

print(link.text)# teg ichidagi matn
print(link["href"])# href atributi
# .text → Wiki
# ['href'] → https://uz.wikipedia.org
>>> Wiki
>>> https://uz.wikipedia.org

CSS Selektorlar (.select)

.select() CSS uslubidagi tanlovchilar bilan ishlaydi. Bu juda kuchli: nuqta . — class uchun, panjara # — id uchun ishlatiladi.
SelektorNimani tanlaydi
pBarcha <p> teglar
.narxclass="narx" bo'lgan elementlar
#asosiyid="asosiy" bo'lgan element
div pdiv ichidagi p teglar
narxlar = soup.select(".narx")
for n in narxlar:
    print(n.text)
# class="narx" bo'lgan barcha elementlar matni
Axloqiy Scraping — Muhim Qoidalar

• robots.txt ni hurmat qiling — sayt qaysi sahifalarni yig'ishga ruxsat berishini ko'rsatadi (masalan: sayt.uz/robots.txt).
• Ortiqcha so'rov yubormang — soniyada o'nlab so'rov saytni sekinlashtiradi. So'rovlar orasiga time.sleep() qo'ying.
• Saytning foydalanish shartlarini buzmang.
• Shaxsiy yoki mualliflik huquqi himoyalangan ma'lumotni ruxsatsiz olmang.

Eslab Qoling

• pip install beautifulsoup4 — o'rnatish
• BeautifulSoup(html, "html.parser") — o'qish
• .find() bitta, .find_all() hammasi
• .text matn, ['href'] atribut
• .select() CSS selektorlar bilan ishlaydi

Misol 1: Teg Matnini Ajratib Olish

Quyiga HTML yozing va teg nomini kiriting. Demo birinchi shu tegni topib, ichidagi matnni ajratib beradi (xuddi soup.find("teg").text kabi).

Misol 2: Barcha Linklarni Topish (find_all)

HTML ichidagi barcha <a> teglarini topib, ularning matni va href atributini chiqaramiz — xuddi find_all("a") kabi.

Misol 3: CSS Selektor bilan Tanlash (.select)

CSS selektor kiriting (masalan .narx yoki li) va mos keladigan barcha elementlar matnini ajratib oling — xuddi soup.select(".narx") kabi.

Misol 4: To'liq Scraping Jarayoni

Quyidagi tugmalar bir scraping jarayonini qadamma-qadam ko'rsatadi: HTML olish → o'qish (parse) → element topish → ma'lumot ajratish.
