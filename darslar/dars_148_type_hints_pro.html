Type hints nima?

Type hints (tur ko'rsatmalari) — o'zgaruvchi yoki funksiya qanday turdagi qiymat kutishini belgilash. Python ularni majburlamaydi (kod baribir ishlaydi), lekin ular kodni o'qishni osonlashtiradi, xatolarni oldindan topadi va IDE yordamini kuchaytiradi. 
def salomlash(ism: str) -> str:   # ism — str, qaytaradi — str
    return "Salom, " + ism

yosh: int = 25                       # o'zgaruvchiga ham qo'yish mumkin
narx: float = 19.99
: str — argument turi, -> str — qaytish turi. Bu shunchaki hujjat emas, IDE va mypy uchun aniq ma'lumot.

List, Dict, Tuple, Set

Konteynerlar ichidagi tur ham ko'rsatiladi. Zamonaviy Python (3.9+) da to'g'ridan-to'g'ri list, dict ishlatiladi. 

def ortacha(sonlar: list[float]) -> float:
    return sum(sonlar) / len(sonlar)

# lug'at: kalit str, qiymat int
ballar: dict[str, int] = {"Ali": 90, "Vali": 85}

# o'zgarmas juftlik
nuqta: tuple[int, int] = (3, 5)

# takrorlanmas to'plam
teglar: set[str] = {"python", "kod"}
Eski Python (3.8 va undan past) da from typing import List, Dict va List[float] yoziladi.

Optional — bo'lishi yoki None bo'lishi mumkin

Optional[X] qiymat X turida yoki None bo'lishi mumkinligini bildiradi. U aslida Union[X, None] ning qisqartmasi. 

from typing import Optional

def foydalanuvchi_top(id: int) -> Optional[str]:
    baza = {1: "Ali", 2: "Vali"}
    if id in baza:
        return baza[id]      # str qaytaradi
    return None              # topilmasa None

# Python 3.10+ da yangi yozuv:
def yosh_top(id: int) -> str | None:
    ...
Funksiya None ham qaytarishi mumkin bo'lsa — Optional ishlating. Bu chaqiruvchini None holatini tekshirishga majbur qiladi.

Union — bir nechta turdan biri

Union[A, B] qiymat A yoki B turida bo'lishini bildiradi. Python 3.10+ da A | B deb yoziladi. 

from typing import Union

def ikkilantir(x: Union[int, float]) -> float:
    return x * 2

# yangi, qisqa yozuv (3.10+)
def formatla(qiymat: int | str) -> str:
    return f"Qiymat: {qiymat}"

print(ikkilantir(5))       # 10
print(formatla("salom"))    # Qiymat: salom
Union[int, float] — int yoki float. int | str — bir xil ma'no, yangi sintaksis.
>>> 10 >>> Qiymat: salom

Diqqat Type hints Python tomonidan tekshirilmaydi — kod noto'g'ri tur bilan ham ishlayveradi. Ularni tekshirish uchun mypy kerak (keyingi dars).

Amaliyot

Mashqlar

1. bolish(a: int, b: int) -> float | None — b=0 bo'lsa None qaytarsin.
2. Talabalar ro'yxatini list[dict[str, int]] deb belgilang.
3. Optional qaytaruvchi funksiyani chaqirib, natijani None ga tekshiring.

Xulosa

Esda saqlang

• : tur va -> tur — argument va qaytish turlari.
• Optional[X] = X yoki None.
• Union[A, B] = A yoki B (3.10+ da A | B).
• list[int], dict[str, int] — konteyner ichidagi tur.
• Type hints kodni ishga tushirmaydi — ular hujjat va mypy uchun.
