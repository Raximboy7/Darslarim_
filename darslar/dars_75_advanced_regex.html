Bu Darsda Nima O'rganamiz?

Oldingi darsda oddiy patternlarni topishni ko'rdik. Endi esa eng kuchli qism — guruhlar (groups) bilan ishlashni o'rganamiz. Guruhlar yordamida matndan kerakli bo'laklarni ajratib olamiz: masalan sanadan kun, oy va yilni; yoki telefon raqamidan kodni. Bu regexning eng foydali imkoniyatlaridan biridir.

1. Guruhlash — Qavslar ( )

Pattern ichidagi ( ) qavslar bir bo'lakni guruh qilib belgilaydi. Topilgandan keyin har bir guruhni alohida .group(1), .group(2) orqali olamiz. .group(0) esa butun moslikni qaytaradi.
import re
matn = "2024-06-30"
natija = re.search(r"(\d{4})-(\d{2})-(\d{2})", matn)
print(natija.group(0)) # butun moslik
print(natija.group(1)) # 1-guruh: yil
print(natija.group(2)) # 2-guruh: oy
print(natija.group(3)) # 3-guruh: kun
# Har bir ( ) ichidagi qism alohida guruhga aylanadi. Sanash 1 dan boshlanadi.
>>> 2024-06-30
>>> 2024
>>> 06
>>> 30
Diqqat

Guruhlar 1 dan sanaladi (0 emas!). .group(0) har doim butun moslikni beradi.

2. Nomli Guruhlar (?P<nom>...)

Raqam bilan eslab qolish qiyin. Shuning uchun guruhga nom berish mumkin: (?P<nom>...). Keyin uni .group("nom") orqali olamiz. Bu kodni ancha o'qishli qiladi.
import re
matn = "2024-06-30"
p = r"(?P<yil>\d{4})-(?P<oy>\d{2})-(?P<kun>\d{2})"
natija = re.search(p, matn)
print(natija.group("yil"))
print(natija.group("oy"))
print(natija.group("kun"))
# (?P<nom>...) — guruhga nom beradi. Endi raqam emas, nom bilan chaqiramiz.
>>> 2024
>>> 06
>>> 30

3. Non-capturing Guruhlar (?:...)

Ba'zan qavs faqat guruhlash uchun kerak, lekin uni alohida saqlashni xohlamaymiz. Bunda (?:...) ishlatamiz — "saqlamaydigan guruh". U mosliklarni guruhlaydi, lekin .group() raqamiga kirmaydi.
import re
matn = "https://example.com"
# http yoki https — lekin uni saqlashga hojat yo'q
p = r"(?:https?)://(\w+\.\w+)"
natija = re.search(p, matn)
print(natija.group(1)) # faqat domen
# (?:...) guruhlaydi, ammo .group() ga raqam sifatida kirmaydi. Domen 1-guruh bo'ladi.
>>> example.com

4. Lookahead va Lookbehind (g'oya)

Ba'zida biz "oldidan/orqasidan nima kelishini" tekshirmoqchimiz, ammo uni natijaga olmaymiz. Bu — lookahead va lookbehind.
SintaksisNomiMa'nosi
(?=...)LookaheadOldinda shu narsa bo'lsa mos keladi
(?<=...)LookbehindOrqada shu narsa bo'lsa mos keladi
import re
matn = "narx: 5000 so'm"
# "so'm" dan oldingi raqamni ol (so'm natijaga kirmaydi)
natija = re.search(r"\d+(?= so'm)", matn)
print(natija.group())
# (?= so'm) — raqamdan keyin " so'm" kelishini tekshiradi, lekin uni olmaydi.
>>> 5000
Oddiy tushuntirish

Lookahead — "shartni tekshir, lekin natijaga qo'shma". Boshlovchilar uchun: hozircha guruhlarni mukammal o'rganing, lookahead esa keyin kerak bo'ladi.

5. Guruhlar Bilan Ma'lumot Ajratish

Guruhlarning eng katta foydasi — bir matndan bir nechta bo'lakni bir vaqtda ajratib olish. .groups() hamma guruhlarni tuple ko'rinishida qaytaradi.
import re
tel = "+998 90 123 45 67"
p = r"\+(\d{3}) (\d{2}) (\d{3}) (\d{2}) (\d{2})"
m = re.search(p, tel)
print(m.groups()) # hamma guruhlar
davlat, kod = m.group(1), m.group(2)
print(f"Davlat: {davlat}, Kod: {kod}")
# .groups() hamma guruhni tuple qilib qaytaradi: ('998', '90', '123', '45', '67')
>>> ('998', '90', '123', '45', '67')
>>> Davlat: 998, Kod: 90
Eslab Qoling

• ( ) — guruh yaratadi, .group(1) bilan olinadi
• (?P<nom>...) — nomli guruh, .group("nom")
• (?:...) — guruhlaydi, lekin saqlamaydi
• (?=...) / (?<=...) — lookahead / lookbehind
• .groups() — barcha guruhlarni tuple qilib beradi
