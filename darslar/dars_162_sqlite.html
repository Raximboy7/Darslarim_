Database Nima?

Database (ma'lumotlar bazasi) — bu ma'lumotlarni tartibli, ishonchli va tez topiladigan shaklda saqlaydigan ombor. Uni katta, aqlli bir daftar deb tasavvur qiling: u yerda har bir ma'lumot o'z joyida turadi va istalgan paytda osongina topiladi.
Oddiy misol

Tasavvur qiling, sizda 10 000 ta foydalanuvchi bor. Ularning ismi, yoshi va emailini oddiy matnli faylga yozsangiz, kerakli odamni topish juda qiyin bo'ladi. Database esa "yoshi 20 dan katta foydalanuvchilarni ber" desangiz, bir lahzada javob beradi.

Excel Jadvaliga O'xshaydi

Agar siz Excel yoki Google Sheets ko'rgan bo'lsangiz, database tushunchasini allaqachon bilasiz! Database ham xuddi Excel kabi ustun va qatorlardan tashkil topgan jadvallardir.

SQL Nima?

SQL (Structured Query Language) — bu database bilan gaplashish tili. SQL orqali siz databasega buyruq berasiz: "ma'lumot qo'sh", "ma'lumotni ko'rsat", "o'zgartir" yoki "o'chir".
SQL buyrug'iVazifasi
CREATE TABLEYangi jadval yaratish
INSERTYangi qator (ma'lumot) qo'shish
SELECTMa'lumotlarni o'qish / ko'rish
UPDATEMavjud ma'lumotni o'zgartirish
DELETEMa'lumotni o'chirish
Eslatma

SQL deyarli barcha databaselar uchun bir xil. Bir marta o'rgansangiz, ko'p joyda ishlatasiz.

import sqlite3
# Modulni chaqirdik — o'rnatish shart emas, Python ichida bor

Ulanish: connect va cursor

Database bilan ishlash uchun avval unga ulanamiz (connect), so'ng buyruq berish uchun kursor (cursor) olamiz. Kursorni database ichida buyruqlarni bajaradigan "qo'l" deb tasavvur qiling.
import sqlite3

# 1. Bazaga ulanamiz (fayl yo'q bo'lsa, yaratiladi)
ulanish = sqlite3.connect("baza.db")

# 2. Buyruq beruvchi kursor olamiz
kursor = ulanish.cursor()

# 3. Ishimiz tugagach, ulanishni yopamiz
ulanish.close()
# connect — bazaga ulanadi, cursor — buyruq beradi, close — yopadi
commit() haqida

Ma'lumot qo'shsangiz yoki o'zgartirsangiz, uni saqlash uchun ulanish.commit() yozish kerak. Aks holda o'zgarishlar bazaga yozilmaydi. Bu darslarda buni keyinroq batafsil ko'ramiz.

Misol 2: Database vs Oddiy Fayl

Ma'lumotni oddiy matnli faylga ham yozish mumkin. Lekin database nima uchun yaxshiroq? Tugmani bosib taqqoslang.

📄 Oddiy fayl (.txt)

  • Ma'lumot tartibsiz matn ko'rinishida
  • Qidirish sekin va qiyin
  • Ko'p odam birga ishlasa, chalkashadi
  • Xato yozuvdan himoya yo'q

🗄️ Database

  • Ma'lumot jadval ko'rinishida tartibli
  • Qidirish juda tez (SELECT bilan)
  • Ko'p foydalanuvchi xavfsiz ishlaydi
  • Qoidalar buzilsa, xatoni qaytaradi
