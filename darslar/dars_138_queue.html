Queue (FIFO) — deque

Modul 12 — Algoritmlar va Strukturalar | Dars 138

Queue nima?

Queue (navbat) — FIFO tamoyili bilan ishlaydigan struktura: First In, First Out — birinchi kirgan birinchi chiqadi. Xuddi kassa oldidagi navbat kabi: birinchi kelgan odam birinchi xizmat oladi.

Stack (LIFO) da esa oxirgi kirgan birinchi chiqadi. Queue undan farq qiladi — tartib saqlanadi.

Nega oddiy list emas?

list bilan ham navbat qilsa bo'ladi, lekin boshidan element o'chirish (pop(0)) sekin — O(n), chunki qolgan hamma element bir pozitsiya siljiydi. 

navbat = ["Ali", "Vali", "Guli"]
navbat.pop(0)   # O(n) — sekin! Hamma element siljiydi
Katta navbatlarda bu jiddiy sekinlashuv. Yechim — collections.deque.

collections.deque

deque ("dek" deb o'qiladi, double-ended queue) — ikkala uchidan ham O(1) da qo'shish/o'chirish imkonini beradi. Navbat uchun ideal. 

from collections import deque

navbat = deque()
navbat.append("Ali")      # oxiriga qo'shish (navbatga kelish)
navbat.append("Vali")
navbat.append("Guli")
print(navbat)              # deque(['Ali', 'Vali', 'Guli'])

birinchi = navbat.popleft()  # boshidan olish (xizmat qilish) — O(1)
print(birinchi)            # Ali
print(navbat)              # deque(['Vali', 'Guli'])
append() — navbatga qo'shadi, popleft() — boshidan oladi. Ikkalasi ham O(1) — juda tez.
>>> Ali >>> deque(['Vali', 'Guli'])

Maslahat Klassik navbat uchun juftlik: append (kirish) + popleft (chiqish).

Amaliy misol — chipta navbati

from collections import deque

navbat = deque(["Ali", "Vali", "Guli"])

while navbat:                 # navbat bo'sh bo'lguncha
    mijoz = navbat.popleft()
    print(f"{mijoz} ga xizmat ko'rsatildi")

print("Navbat tugadi!")
Mijozlar aynan kelgan tartibda xizmat oladi — bu FIFO ning amaliy ko'rinishi.
>>> Ali ga xizmat ko'rsatildi >>> Vali ga xizmat ko'rsatildi >>> Guli ga xizmat ko'rsatildi >>> Navbat tugadi!

Amaliyot

Mashqlar

1. Bosh og'irlik cheklovi bilan navbat yarating: deque(maxlen=3) ni sinang.
2. appendleft va pop bilan ikki tomonlama navbat qiling.
3. Printer navbatini modellashtiring: hujjatlar kelgan tartibda chop etilsin.

Xulosa

Esda saqlang

• Queue = FIFO: birinchi kirgan birinchi chiqadi.
• list.pop(0) sekin (O(n)) — navbatga ishlatmang.
• collections.deque — ikki uchdan ham O(1).
• Navbat: append + popleft.
• while navbat: — navbat bo'sh bo'lguncha aylantiradi.
