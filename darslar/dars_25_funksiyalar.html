Funksiya Nima?

Funksiya — bu biror vazifani bajaradigan, nom berilgan kod bloki. U bir marta yoziladi va keyin istalgancha qayta ishlatiladi. Funksiya kodni qisqartiradi, tartibga soladi va takrorlanishning oldini oladi.
Oddiy hayotiy taqqos

Funksiya — bu "retsept" kabi. Bir marta retseptni yozasiz, keyin uni xohlagancha qo'llab, har safar bir xil natijaga erishasiz.

Funksiya Yaratish: def

Funksiya def kalit so'zi bilan yaratiladi. Nomdan keyin () qavslar va : ikki nuqta qo'yiladi. Funksiya ichidagi kod 4 bo'sh joy bilan chap tomondan suriladi (indentation).
def salom():
    print("Salom!")
# "salom" nomli funksiya yaratildi
# ichidagi kod 4 bo'sh joy bilan suriladi
⚠️ Diqqat

Funksiyani yaratish — uni ishga tushirmaydi. Bu faqat "retseptni yozish". Kod faqat funksiyani chaqirganda bajariladi.

Funksiyani Chaqirish

Funksiyani ishga tushirish uchun uning nomini yozib, oxiriga () qavs qo'yamiz. Bunga chaqirish (call) deyiladi.
def salom():
    print("Salom!")

salom()
salom()
# funksiyani 2 marta chaqirdik
# har chaqiriqda ichidagi kod ishlaydi
>>> Salom!
>>> Salom!

Parametr — Funksiyaga Ma'lumot Berish

Parametr — bu funksiya qavslari ichidagi o'zgaruvchi. U funksiyaga tashqaridan ma'lumot uzatish imkonini beradi. Shunda bitta funksiya turli qiymatlar bilan ishlay oladi.
def salomlash(ism):
    print(f"Salom, {ism}!")

salomlash("Ali")
salomlash("Vali")
# ism — parametr
# "Ali", "Vali" — argument (real qiymat)
>>> Salom, Ali!
>>> Salom, Vali!

return — Qiymat Qaytarish

return funksiyadan natija qaytaradi. Bu natijani o'zgaruvchiga saqlab, keyin ishlatish mumkin. print() faqat ekranga chiqaradi, return esa qiymatni qaytaradi.
def qoshish(a, b):
    return a + b

natija = qoshish(5, 3)
print(natija)
# a + b natijasini qaytaradi
# natija = 8 saqlanadi
>>> 8

Standart (Default) Parametr

Parametrga standart qiymat berish mumkin. Agar funksiya chaqirilganda argument berilmasa, standart qiymat ishlatiladi.
def salom(ism="Dunyo"):
    print(f"Salom, {ism}!")

salom()
salom("Ali")
# argument berilmadi → "Dunyo"
# argument berildi → "Ali"
>>> Salom, Dunyo!
>>> Salom, Ali!

Keng Tarqalgan Xato

❌ Xato: ikki nuqta va indentation yo'q
def salom() # : yo'q!
print("Salom!") # surilmagan!

Qavsdan keyin : bo'lishi va ichki kod 4 bo'sh joy bilan surilishi shart. Aks holda SyntaxError bo'ladi.

✓ To'g'ri
def salom():
    print("Salom!")
>>> Salom!

Yana Bir Xato

❌ Xato: funksiyani chaqirishni unutish
def salom():
    print("Salom!")
# salom() yozilmadi → hech narsa chiqmaydi

Faqat def yozsangiz funksiya ishlamaydi. Uni salom() deb chaqirishingiz kerak.

Eslab Qoling

• Funksiya def nom(): bilan yaratiladi
• Ichki kod 4 bo'sh joy bilan suriladi (indentation)
• Chaqirish: nom() — qavslar bilan
• return qiymat qaytaradi, print() ekranga chiqaradi

Misol 1: salomlash() Funksiyasi

Kod: def salomlash(ism): print(f"Salom, {ism}!")

Misol 2: qoshish() — return

return a + b natijani qaytaradi va saqlaydi.

Misol 3: Standart Parametr

Bo'sh qoldirsangiz ism="Dunyo" ishlatiladi.

Misol 4: Funksiya Oqimi (Vizual)

Kod: def kvadrat(n): return n * n

Misol 5: To'liq Funksiya Qurish

Yozgan ma'lumotingizdan haqiqiy funksiya kodi hosil bo'ladi.
