Python Nima?

Python — bu oddiy, tushunarli va kuchli dasturlash tili. Google, Instagram, Netflix, Spotify va boshqa katta kompaniyalar Python-dan foydalanadilar.

Python nima uchun yaxshi?

  • Oson: Sintaksisi ingliz tiliga o'xshash
  • Tez: Tez yozish va o'rganish mumkin
  • Kuchli: Web, AI, Data Science uchun ishlatiladi
  • Bepul: Tekin va ochiq kodli (open source)

Python O'rnatish (Windows)

Python-ni o'rnatish oson! Quyidagi qadamlarni bajaring:
1
Python.org saytiga kiring

python.org saytiga kirib, Python o'rnatuvchi dasturini yuklab oling

2
O'rnatish faylini ishga tushiring

Yuklab olingan faylni ochib, o'rnatish oynasini ishga tushiring

3
Add Python to PATH ni belgilang

O'rnatish davomida "Add Python to PATH" ni belgilang (MUHIM!)

4
O'rnatishni tugatish

"Install Now" tugmasini bosing va o'rnatish tugaguni kuting

5
Tekshirish

Terminal/CMD da python --version yozing

Windows CMD-ni Qanday Ochish Kerak?

Windows tugmachasini bosib, "cmd" yozib, Enter bosing.
Yoki: Win + R → "cmd" → Enter

Tekshiruv: Python O'rnatildimi?

Python-ni o'rnatganingizdan keyin, Terminal/CMD da quyidagini yozib Enter bosing:
python --version

Agar ko'rinsa: Python 3.10.0 (yoki boshqa versiya) — o'rnatish muvaffaqiyatli!
Agar xato bo'lsa: Python to'g'ri o'rnatilmadi. "Add Python to PATH" ni tekshiring.

Kod Yozish Uchun Editor

Python kodini yozish uchun maxsus editor kerak. Quyidagi variantlardan birini tanlang:
VS Code (Tavsiya etiladi)

Eng mashhur, bepul va kuchli muharrir (editor).

PyCharm

Python uchun maxsus editor. Katta proyektlar uchun yaxshi.

Thonny

Yangi dasturchilar uchun eng oson muharrir. Kod bajarilishini animatsiya qiladi!

IDLE

Python bilan birga o'rnatiladi. Sodda, lekin foydali.

Eslatma:

Notepad++ yoki oddiy matn muharriri FAQAT oddiy matnlar uchun. Python uchun maxsus editor kerak! (Rang belgilash, xatolarni ko'rsatish uchun)

Birinchi Kod Yozish

Python o'rnatildi va editor tayyor. Endi birinchi kodni yozamiz!
1
Yangi fayl yaratish

Editorni ochib, yangi fayl yarating. Fayl nomi: hello.py

2
Kodni yozish

print("Assalomu alaikum, Dunyo!")

3
Faylni saqlash

Ctrl + S yoki File → Save

4
Kodni bajarish

Terminal/CMD da python hello.py yozib Enter bosing

Natija:

Ekranda ko'rinadi: Assalomu alaikum, Dunyo!

Eslatma!

Fayl nomi .py bilan tugashi kerak. Masalan: hello.py, my_code.py, program.py

Birinchi Kod Misollar

Quyida birinchi kod misollarini ko'rasiz. Siz ham shunday yozib sinab ko'ring!

Misol 1: Salomni Chiqarish

📄 hello.py
print("Salom!")

Natija: Salom!

Misol 2: Raqamlarni Chiqarish

📄 numbers.py
print(5)
print(10)
print(15)

Natija:
5
10
15

Misol 3: Qo'shish

📄 add.py
print(5 + 3)

Natija: 8

Terminal/CMD Bilan Ishlash

Terminal — bu kompyuterga buyruq beradigan joy. Bunda Python-ni ishga tushiramiz.

Muhim Buyruqlar:

# Faylni bajarish
python fayl_nomi.py

# Hozirgi joyni bilish
pwd

# Fayllarni ro'yxat ko'rish
dir (Windows) yoki ls (Mac/Linux)

# Joyni o'zgartirish
cd Desktop
Terminal Maslahat:

Python faylini bajarish uchun, avval Terminal/CMD da faylning joyiga boring. Masalan, Desktop'da hello.py fayl bo'lsa:
cd Desktop → python hello.py
