datetime moduli nima?

datetime — sana va vaqt bilan ishlash uchun standart modul. Hozirgi vaqtni olish, sanalarni formatlash, ikki sana orasidagi farqni hisoblash — hammasi shu modul orqali. 
from datetime import datetime, timedelta
# datetime moduli ichida datetime klassi bor (nom bir xil bo'lgani uchun ba'zan chalkashtiradi).

now() — hozirgi sana va vaqt

from datetime import datetime

hozir = datetime.now()
print(hozir)

print(hozir.year)
print(hozir.month)
print(hozir.day)
# now() shu ondagi to'liq sana-vaqtni beradi. .year, .month, .day, .hour, .minute — alohida qismlariga murojaat qilamiz.
>>> 2026-07-01 14:30:52.123456
>>> 2026
>>> 7
>>> 1

strftime() — sanani chiroyli formatlash

strftime (string format time) sanani biz xohlagan ko'rinishga o'giradi.

from datetime import datetime

hozir = datetime.now()

print(hozir.strftime("%d.%m.%Y"))
print(hozir.strftime("%H:%M"))
print(hozir.strftime("%d-%m-%Y %H:%M:%S"))
# %d kun, %m oy, %Y to'liq yil, %H soat, %M daqiqa, %S soniya. Belgilardan xohlagancha shakl yasaysiz.
>>> 01.07.2026
>>> 14:30
>>> 01-07-2026 14:30:52

timedelta — vaqt oralig'i

timedelta sanaga vaqt qo'shish yoki ayirish uchun ishlatiladi.

from datetime import datetime, timedelta

bugun = datetime.now()
ertaga = bugun + timedelta(days=1)
bir_hafta = bugun + timedelta(weeks=1)

print(ertaga.strftime("%d.%m.%Y"))
print(bir_hafta.strftime("%d.%m.%Y"))
# timedelta(days=1) — bir kun. Sanaga qo'shsangiz kelajak, ayirsangiz o'tmish. days, weeks, hours, minutes ishlatiladi.
>>> 02.07.2026
>>> 08.07.2026

Amaliyot: yosh yoki qolgan kunlar

from datetime import datetime

yangi_yil = datetime(2027, 1, 1)
bugun = datetime.now()

farq = yangi_yil - bugun
print("Yangi yilgacha:", farq.days, "kun")
# Ikki datetime ni ayirsak, timedelta chiqadi. Uning .days xususiyati orqali necha kun qolganini bilamiz.
>>> Yangi yilgacha: 184 kun

Xulosa

Eslab qoling

• datetime.now() — hozirgi sana va vaqt
• .year, .month, .day — qismlariga murojaat
• strftime("%d.%m.%Y") — chiroyli formatlash
• timedelta(days=1) — vaqt qo'shish/ayirish
• Ikki sana ayirmasi — necha kun/soat farqi
