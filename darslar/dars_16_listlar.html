Listlar (Lists)

Modul 4 — Ma'lumot tuzilmalari | Dars 16

List Nima?

List (ro'yxat) — bu bir nechta qiymatni bitta o'zgaruvchida saqlash uchun ishlatiladigan tuzilma. Qiymatlar [ ] kvadrat qavslar ichida, vergul bilan ajratib yoziladi. Listdagi qiymatlar tartibli bo'ladi va ularni o'zgartirish mumkin.
sonlar = [1, 2, 3, 4, 5]
print(sonlar)
# 5 ta sonni bitta o'zgaruvchida saqladik
>>> [1, 2, 3, 4, 5]

List Yaratish

List ichida istalgan turdagi qiymatlar bo'lishi mumkin: sonlar, matnlar, hatto aralash.
ismlar = ["ali", "bobur"]
bahar = [3.5, 4, 5]
bosh = []
# matnlar ro'yxati
# sonlar ro'yxati
# bo'sh list (hozircha hech narsa yo'q)
Bo'sh list

Ko'pincha listni [] bo'sh holda yaratib, keyin unga qiymatlarni qo'shib boramiz. Bu juda keng tarqalgan usul.

Indeks (Indexing)

Har bir element o'z raqamiga (indeksiga) ega. Sanash 0 dan boshlanadi. Manfiy indeks oxiridan sanaydi: -1 — oxirgi element.
meva = ["olma", "banan", "uzum"]
print(meva[0])
print(meva[-1])

# birinchi element
# oxirgi element
>>> olma
>>> uzum

Kesim (Slicing)

list[boshi:oxiri] — boshidan oxirigacha (oxiri kirmaydi) bo'lakni qaytaradi.
KodNatijaIzoh
[10,20,30,40][1:3][20, 30]1 va 2-indeks
[10,20,30,40][:2][10, 20]boshidan 2 gacha
[10,20,30,40][2:][30, 40]2 dan oxirigacha

List Metodlari

Metodlar listni boshqarish uchun ishlatiladi. Ular list.metod() ko'rinishida chaqiriladi.
MetodVazifasiMisol → Natija
.append(x)Oxiriga qo'shadi[1,2].append(3) → [1, 2, 3]
.insert(i,x)i-o'ringa qo'shadi[1,3].insert(1,2) → [1, 2, 3]
.remove(x)x qiymatni o'chiradi[1,2,3].remove(2) → [1, 3]
.pop(i)i-indeksni o'chiradi[1,2,3].pop(0) → [2, 3]
.sort()Tartiblaydi[3,1,2].sort() → [1, 2, 3]
.reverse()Teskari qiladi[1,2,3].reverse() → [3, 2, 1]
.clear()Hammasini tozalaydi[1,2,3].clear() → []
.copy()Nusxa oladi[1,2].copy() → [1, 2]

append() va pop() amalda

savat = ["non"]
savat.append("sut")
savat.pop()
print(savat)

# oxiriga "sut" qo'shildi
# oxirgisi o'chirildi
>>> ['non']

len() va for bilan Aylanish

len(list) listdagi elementlar sonini qaytaradi. for sikli bilan har bir elementni birma-bir aylanib chiqamiz.
meva = ["olma", "banan", "uzum"]
print(len(meva))
for m in meva:
    print(m)

# nechta element bor
# har bir mevani chiqaramiz
>>> 3
>>> olma
>>> banan
>>> uzum

Keng Tarqalgan Xato

❌ Xato: mavjud bo'lmagan indeks
meva = ["olma", "banan"]
print(meva[2]) # XATO!

Listda 2 ta element bor (indekslar 0 va 1). 2-indeks yo'q, shuning uchun IndexError xatosi chiqadi.

✓ To'g'ri: oxirgini -1 bilan oling
meva = ["olma", "banan"]
print(meva[-1])
>>> banan
