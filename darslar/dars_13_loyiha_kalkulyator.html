Mini-loyiha: Kalkulyator (CLI)

Modul 1 — Asoslar | Dars 13

Loyihamiz: oddiy kalkulyator

Endi o'rgangan bilimlarimizni birlashtirib, haqiqiy ishlaydigan kalkulyator yozamiz! U foydalanuvchidan ikkita son va bitta amal (+, −, ×, ÷) so'raydi, keyin natijani chiqaradi. Bu CLI (buyruq qatori) dasturi bo'ladi.

Loyihada quyidagilarni ishlatamiz: input(), float(), if/elif/else va arifmetik operatorlar.

1-qadam: Sonlarni olish

Avval foydalanuvchidan ikkita sonni olamiz. input() matn qaytargani uchun darhol float() ga o'giramiz (kasr sonlar ham ishlashi uchun).
son1 = float(input("Birinchi son: "))
son2 = float(input("Ikkinchi son: "))
# float() bilan 3.5 kabi sonlar ham ishlaydi

2-qadam: Amalni so'rash

Endi qaysi amalni bajarishni so'raymiz. Foydalanuvchi +, -, *, / belgilaridan birini kiritadi.
amal = input("Amal (+, -, *, /): ")
# amal — matn bo'lib qoladi, o'girish shart emas

3-qadam: Amalni bajarish (if/elif/else)

if/elif/else yordamida qaysi belgi kiritilganini tekshiramiz va mos amalni bajaramiz. Nolga bo'lishni ham alohida tekshiramiz.
if amal == "+":
    natija = son1 + son2
elif amal == "-":
    natija = son1 - son2
elif amal == "*":
    natija = son1 * son2
elif amal == "/":
    if son2 == 0:
        natija = "Nolga bo'lib bo'lmaydi!"
    else:
        natija = son1 / son2
else:
    natija = "Noma'lum amal!"
# har bir amalni alohida tekshiramiz
# / uchun son2 == 0 ni tekshirish MUHIM

4-qadam: Natijani chiqarish

print("Natija:", natija)
# print bilan javobni ekranga chiqaramiz

To'liq dastur

Hamma qadamlarni birlashtiramiz. Mana tayyor, ishlaydigan kalkulyator:
print("=== KALKULYATOR ===")

son1 = float(input("Birinchi son: "))
son2 = float(input("Ikkinchi son: "))
amal = input("Amal (+, -, *, /): ")

if amal == "+":
    natija = son1 + son2
elif amal == "-":
    natija = son1 - son2
elif amal == "*":
    natija = son1 * son2
elif amal == "/":
    if son2 == 0:
        natija = "Nolga bo'lib bo'lmaydi!"
    else:
        natija = son1 / son2
else:
    natija = "Noma'lum amal!"

print("Natija:", natija)
# to'liq ishlaydigan kalkulyator dasturi
=== KALKULYATOR ===
Birinchi son: 12
Ikkinchi son: 4
Amal (+, -, *, /): *
>>> Natija: 48.0

Sinab ko'ring: brauzer kalkulyatori

Quyida yuqoridagi mantiq bilan ishlaydigan jonli kalkulyator. Sonlarni kiriting va amalni tanlang!

• if/elif/else bilan tanlov qilish
• Nolga bo'lishni oldindan tekshirish muhim
• Kichik qadamlarni birlashtirib to'liq dastur yasash
