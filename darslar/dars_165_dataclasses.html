Dataclass Nima?

Dataclass — bu asosan ma'lumot saqlovchi klasslarni yozishni soddalashtiradigan vosita. Ya'ni vazifasi faqat bir nechta qiymatni (ism, yosh, narx...) birga ushlab turish bo'lgan klasslar uchun. Odatda bunday klassda ko'p takroriy kod yozasiz — __init__, __repr__, __eq__ — dataclass esa bularni siz uchun avtomatik yozadi.
Sodda qilib aytganda

Dataclass — "kamroq yozib, ko'proq ish bajaradigan" klass. Siz faqat qanday maydonlar borligini aytasiz, qolganini Python o'zi qiladi.

@dataclass Dekoratori

Dataclass yaratish uchun avval uni dataclasses modulidan import qilamiz, so'ng klass ustiga @dataclass dekoratorini qo'yamiz.
from dataclasses import dataclass

@dataclass
class Talaba:
    ism: str
    yosh: int = 0
# @dataclass dekoratori klassni "dataclass" qiladi
# ism va yosh — maydonlar (fields)
# yosh ning standart qiymati 0
Maydonlarni ism: tur ko'rinishida yozamiz (avvalgi darsdagi type hint). Standart qiymat berish uchun = qiymat qo'shamiz.

Avtomatik Tayyorlanadigan Narsalar

@dataclass dekoratorini qo'yganingizdan keyin Python uchta muhim metodni siz uchun yozib beradi:
MetodVazifasi
__init__Obyekt yaratganda maydonlarni qabul qiladi: Talaba("Ali", 20)
__repr__Obyektni chiroyli matn ko'rinishida chiqaradi: Talaba(ism='Ali', yosh=20)
__eq__Ikki obyektni == orqali taqqoslash imkonini beradi

Oddiy Klass vs Dataclass

Quyida bir xil natija beradigan ikkita yozuvni solishtiramiz. Oddiy klassda hamma narsani qo'lda yozish kerak:

Oddiy klass (uzun)

class Talaba:
    def __init__(self, ism, yosh):
        self.ism = ism
        self.yosh = yosh
    def __repr__(self):
        return f"Talaba({self.ism}, {self.yosh})"
    def __eq__(self, other):
        return self.ism == other.ism and self.yosh == other.yosh
# 8 qator kod — hammasi qo'lda yozilgan

Dataclass (qisqa)

from dataclasses import dataclass

@dataclass
class Talaba:
    ism: str
    yosh: int
# Faqat 3 qator! __init__, __repr__, __eq__ avtomatik tayyor
Natija bir xil, kod kamroq

Ikkala holatda ham obyekt bir xil ishlaydi, lekin dataclass bilan ancha kam kod yozasiz va xatolik ehtimoli kamayadi.

Obyekt Yaratish va Ishlatish

t1 = Talaba("Ali", 20)
t2 = Talaba("Ali", 20)

print(t1)
print(t1.ism)
print(t1 == t2)
# Obyekt yaratish va taqqoslash
>>> Talaba(ism='Ali', yosh=20)
>>> Ali
>>> True
E'tibor bering: oddiy klassda t1 == t2 odatda False bo'lardi (chunki ikki alohida obyekt), lekin dataclass maydonlar qiymatini taqqoslab True beradi.

field() va default_factory

Agar maydonning standart qiymati ro'yxat (list) yoki lug'at (dict) kabi o'zgaruvchan bo'lsa, uni to'g'ridan-to'g'ri = [] deb yozish xato. Buning o'rniga field(default_factory=list) ishlatamiz.
from dataclasses import dataclass, field

@dataclass
class Guruh:
    nom: str
    talabalar: list = field(default_factory=list)
# talabalar — har bir yangi obyektda bo'sh ro'yxat bo'lib boshlanadi
# default_factory=list → har safar yangi [] yaratadi
>>> g = Guruh("Python-1")
>>> g.talabalar.append("Ali")
>>> print(g)
>>> Guruh(nom='Python-1', talabalar=['Ali'])
Nega field() kerak?

Agar barcha obyektlar bir xil [] ni baham ko'rsa, bittasiga qo'shilgan element boshqalariga ham tegadi. default_factory har bir obyektga alohida yangi ro'yxat beradi.

Eslab Qoling

• from dataclasses import dataclass — import qiling
• Klass ustiga @dataclass qo'ying
• Maydonlarni ism: tur = qiymat ko'rinishida yozing
• __init__, __repr__, __eq__ avtomatik tayyor bo'ladi
• Ro'yxat/lug'at standart qiymati uchun field(default_factory=list) ishlating
