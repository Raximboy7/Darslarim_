Matn Analitikasi Nima?

Matn analitikasi (text analytics) — bu katta hajmdagi matndan foydali ma'lumot ajratib olish san'ati. Masalan: qaysi so'z eng ko'p uchraydi, matn ijobiymi yoki salbiy, nechta so'z bor. Bu usullar izlash tizimlari, spam filtrlari va izoh tahlili kabi dasturlarda ishlatiladi.

Ushbu darsda 4 ta asosiy bosqichni o'rganamiz: tozalash → tokenization → so'z chastotasi → sentiment.

1-bosqich: Matnni Tozalash

Matnni tahlil qilishdan oldin uni tozalash kerak. Buning ikki asosiy qadami bor: barcha harflarni kichik harfga aylantirish va tinish belgilarini olib tashlash. Aks holda Salom, salom va salom! uchta turli so'z deb hisoblanadi.
import string

matn = "Salom, Dunyo! Salom?"
matn = matn.lower()  # kichik harf
for belgi in string.punctuation:
    matn = matn.replace(belgi, "")
print(matn)
# string.punctuation — barcha tinish belgilari: ! " # $ % & ' ( ) * ... va h.k.
>>> salom dunyo salom
string.punctuation nima?

Bu Python'ning string modulidagi tayyor o'zgaruvchi. U barcha tinish belgilarini bitta matn ko'rinishida saqlaydi: !"#$%&'()*+,-./:;<=>?@[\]^_`{|}~

2-bosqich: Tokenization (So'zlarga Bo'lish)

Tokenization — bu matnni alohida bo'laklarga (odatda so'zlarga) ajratish. Eng oddiy usuli — .split() metodi. U bo'sh joy bo'yicha matnni so'zlar ro'yxatiga aylantiradi. Har bir so'z "token" deb ataladi.
matn = "salom dunyo salom"
sozlar = matn.split()
print(sozlar)
print("Jami:", len(sozlar))
# .split() bo'sh joy bo'yicha bo'ladi va so'zlar ro'yxatini qaytaradi
>>> ['salom', 'dunyo', 'salom']
>>> Jami: 3

3-bosqich: So'z Chastotasi (Word Frequency)

So'z chastotasi — har bir so'z necha marta uchraganini sanash. Bu eng ko'p ishlatilgan so'zlarni topishga yordam beradi. Buni qo'lda dictionary bilan ham qilish mumkin, lekin Python'da buning uchun maxsus tayyor vosita bor: collections.Counter.

Qo'lda — dictionary bilan

sozlar = ["olma", "olma", "nok"]
hisob = {}
for soz in sozlar:
    hisob[soz] = hisob.get(soz, 0) + 1
print(hisob)
# .get(soz, 0) — agar so'z hali yo'q bo'lsa 0 dan boshlaydi
>>> {'olma': 2, 'nok': 1}

Oson yo'l — collections.Counter

Counter aynan shu ishni bitta qatorda bajaradi. Uning .most_common(n) metodi eng ko'p uchragan n ta so'zni qaytaradi.
from collections import Counter

sozlar = "olma nok olma uzum olma nok".split()
hisob = Counter(sozlar)
print(hisob.most_common(2))
# most_common(2) — eng ko'p uchragan 2 ta so'zni (so'z, son) juftlik shaklida beradi
>>> [('olma', 3), ('nok', 2)]

4-bosqich: Sodda Sentiment Analizi

Sentiment analizi — matnning his-tuyg'usini aniqlash: ijobiy, salbiy yoki neytral. Eng sodda usuli: oldindan tayyor ijobiy so'zlar va salbiy so'zlar ro'yxatini tuzib, matndagi har bir so'zni ular bilan solishtirish va ball berish.
ijobiy = {"yaxshi", "zo'r", "ajoyib"}
salbiy = {"yomon", "xunuk", "dahshat"}

matn = "film juda yaxshi va ajoyib edi"
ball = 0
for soz in matn.split():
    if soz in ijobiy:
        ball += 1
    elif soz in salbiy:
        ball -= 1

print("Ijobiy" if ball > 0 else "Salbiy")
# Ijobiy so'z uchun +1, salbiy uchun -1. Yakuniy ball belgisi natijani aniqlaydi.
>>> Ijobiy
Diqqat

Bu juda sodda usul. Haqiqiy sentiment analizi "yomon emas" kabi iboralarni (inkor) tushunmaydi. Murakkab loyihalarda mashinaviy o'rganish (machine learning) kutubxonalari ishlatiladi. Lekin asos shu yerdan boshlanadi.

Eslab Qoling

• Tahlildan oldin matnni tozalang (kichik harf + tinish belgisiz)
• .split() matnni so'zlarga (token) bo'ladi
• Counter so'z chastotasini bir qatorda hisoblaydi
• .most_common(n) eng ko'p uchragan so'zlarni beradi
• Sentiment — so'zlarni ijobiy/salbiy ro'yxat bilan solishtirish
