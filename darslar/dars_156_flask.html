Flask Nima?

Flask — bu Python tilida web sayt va web ilova yaratish uchun ishlatiladigan yengil (mikro) web framework. "Yengil" degani — u sizga faqat eng kerakli narsalarni beradi, qolganini siz xohlagancha qurasiz. Flask yordamida brauzerga sahifa qaytaradigan dasturni bir necha qatorda yozish mumkin.
Oddiy qilib aytganda

Siz Python kodi yozasiz, Flask uni server ga aylantiradi. Brauzer biror manzilni so'raganda (masalan localhost:5000), Flask sizning kodingizni ishga tushiradi va natijani brauzerga qaytaradi.

O'rnatish

Flask Python bilan birga kelmaydi — uni alohida o'rnatish kerak. Buni terminal (buyruq qatori) orqali pip bilan qilamiz.
pip install flask
# Flask kutubxonasini kompyuterga o'rnatadi
Tekshirish

O'rnatilganini tekshirish uchun pip show flask deb yozing. U Flask versiyasi haqida ma'lumot ko'rsatsa — hammasi joyida.

Birinchi Flask Dasturi

Eng kichik Flask dasturi quyidagicha bo'ladi. Uni app.py nomli faylga yozamiz.
from flask import Flask

app = Flask(__name__)

@app.route("/")
def bosh_sahifa():
    return "Salom, Dunyo!"

if __name__ == "__main__":
    app.run(debug=True)
# Flask ilovasini yaratdik, bitta sahifa qo'shdik va serverni ishga tushirdik
Ishga tushirish

Terminalda python app.py deb yozing. So'ng brauzerda http://localhost:5000 manzilini oching — u yerda "Salom, Dunyo!" chiqadi.

Kodni Qism-Qism Tushunamiz

1. Flask ni import qilish

from flask import Flask
# flask kutubxonasidan Flask klassini olamiz

2. Ilovani yaratish

app = Flask(__name__)
# app — bizning web ilovamiz. __name__ Flask ga fayl qayerdaligini bildiradi

3. Route (manzil)

@app.route("/")
# "/" — bu bosh sahifa manzili. Brauzer shu manzilni so'rasa, pastdagi funksiya ishlaydi

4. View funksiyasi

def bosh_sahifa():
    return "Salom, Dunyo!"
# View funksiyasi brauzerga ko'rsatiladigan javobni qaytaradi (return)

5. Serverni ishga tushirish

app.run(debug=True)
# Serverni ishga tushiradi. debug=True — xatolarni ekranga chiqaradi va kod o'zgarsa qayta yuklaydi

Bir Nechta Route

Har bir @app.route(...) alohida sahifani bildiradi. Shunday qilib bitta dasturda bir nechta sahifa bo'lishi mumkin.
@app.route("/")
def bosh():
    return "Bosh sahifa"

@app.route("/haqida")
def haqida():
    return "Biz haqimizda"
# localhost:5000/ -> "Bosh sahifa", localhost:5000/haqida -> "Biz haqimizda"

Eslab Qoling

• pip install flask — avval o'rnatamiz
• app = Flask(__name__) — ilova yaratamiz
• @app.route("/") — manzil belgilaymiz
• View funksiya return bilan javob qaytaradi
• app.run() serverni ishga tushiradi (localhost:5000)
