Sonlar: int va float

Modul 1 — Asoslar | Dars 5

Sonlar — Python'ning ikki turi

Python'da sonlar bilan ishlash uchun ikkita asosiy tur bor: int (butun sonlar) va float (kasr sonlar). Bu ikkalasi bilan hisob-kitob qilamiz, o'lchaymiz va dasturlarimizni "aqlli" qilamiz.

Farqi juda oddiy:

  • int — nuqtasiz butun sonlar: 5, 100, -7, 0
  • float — nuqtali kasr sonlar: 3.14, -0.5, 2.0
yosh = 20
narx = 19.99
print(type(yosh))
print(type(narx))
# type() sonning turini ko'rsatadi
>>> <class 'int'>
>>> <class 'float'>

Arifmetik amallar

Sonlar ustida oddiy matematik amallarni bajarish mumkin. Python'ni katta kalkulyator deb tasavvur qiling.
print(7 + 3)
print(7 - 3)
print(7 * 3)
print(7 / 3)
# qo'shish, ayirish, ko'paytirish, bo'lish
>>> 10
>>> 4
>>> 21
>>> 2.3333333333333335
Diqqat! Oddiy bo'lish / har doim float qaytaradi. Masalan 6 / 2 natijasi 3.0 bo'ladi, 3 emas.

Maxsus amallar: // , % , **

Butun bo'lish: //

Bo'lib, faqat butun qismini oladi (qoldiqni tashlab yuboradi).

print(17 // 5)
# 17 ni 5 ga bo'lganda butun qismi
>>> 3
Qoldiq (modul): %

Bo'lishdan qolgan qoldiqni qaytaradi. Juft/toq tekshirishda juda qo'l keladi.

print(17 % 5)
# 17 = 3*5 + 2, qoldiq 2
>>> 2
Daraja: **

Sonni darajaga ko'taradi.

print(2 ** 10)
# 2 ning 10-darajasi
>>> 1024

Foydali funksiyalar: round() va abs()

Python'da tayyor funksiyalar bor. round() sonni yaxlitlaydi, abs() esa sonning musbat (mutlaq) qiymatini qaytaradi.
print(round(3.14159, 2))
print(round(7.8))
print(abs(-15))
print(abs(15))
# round(son, nechta_raqam) — yaxlitlash
# abs() — manfiy belgini olib tashlaydi
>>> 3.14
>>> 8
>>> 15
>>> 15

Amaliy misol: do'kon cheki

Bir mahsulotdan bir nechta olsak, umumiy narx va o'rtacha narxni hisoblaymiz.
narx = 19.99
soni = 3
jami = narx * soni
print("Jami:", round(jami, 2))
print("O'rtacha:", round(jami / soni, 2))
# narxni songa ko'paytiramiz
# natijani 2 xonagacha yaxlitlaymiz
>>> Jami: 59.97
>>> O'rtacha: 19.99

Asosiy xulosalar

Esda tuting

• int — butun son, float — kasr son
• / har doim float, // butun qismini beradi
• % — qoldiq, ** — daraja
• round() — yaxlitlash, abs() — mutlaq qiymat
• Sonning turini bilish uchun type() dan foydalaning
