Bu Darsda Nima Quramiz?

Bu darsda biz to'liq Todo (vazifalar ro'yxati) ilovasi quramiz. Bu ilova sizning kunlik ishlaringizni yozib boruvchi haqiqiy dasturdir. Ilova quyidagilarni biladi: vazifa qo'shish, ro'yxatni ko'rsatish, vazifani bajarildi deb belgilash va vazifani o'chirish. Eng muhimi — vazifalar faylga saqlanadi, shuning uchun dasturni yopib qayta ochsangiz ham ro'yxatingiz joyida qoladi.
CRUD nima?

Deyarli har bir ilova 4 ta asosiy amalni bajaradi. Ularning birinchi harflari CRUD deyiladi:
• Create — yangi ma'lumot qo'shish (vazifa qo'shish)
• Read — ma'lumotni o'qish/ko'rsatish (ro'yxatni ko'rish)
• Update — ma'lumotni o'zgartirish (bajarildi belgilash)
• Delete — ma'lumotni o'chirish (vazifani o'chirish)

1-bosqich: Vazifani Saqlash Uchun Tuzilma

Har bir vazifa ikki narsadan iborat: matn (nima qilish kerak) va holat (bajarildimi yoki yo'q). Buni saqlash uchun dictionary ishlatamiz, hamma vazifalarni esa list ichida saqlaymiz.
# Har bir vazifa — dictionary
vazifa = {"matn": "Kitob o'qish", "bajarildi": False}

# Hamma vazifalar — list ichida
vazifalar = []
vazifalar.append(vazifa)
# "matn" — nima qilish kerak, "bajarildi" — True yoki False
# Boshida ro'yxat bo'sh, append() bilan yangi vazifa qo'shamiz

2-bosqich: Vazifa Qo'shish (Create)

Foydalanuvchidan matn so'raymiz va uni yangi vazifa qilib ro'yxatga qo'shamiz. Yangi vazifa har doim bajarildi: False bilan boshlanadi.
def qoshish(vazifalar, matn):
    yangi = {"matn": matn, "bajarildi": False}
    vazifalar.append(yangi)
    print("Qo'shildi:", matn)
# matn — yangi vazifa nomi
# dictionary yasaymiz va ro'yxat oxiriga qo'shamiz
>>> qoshish(vazifalar, "Kitob o'qish")
Qo'shildi: Kitob o'qish

3-bosqich: Ro'yxatni Ko'rsatish (Read)

Vazifalarni raqamlab ekranga chiqaramiz. Bajarilgan vazifalarga [x], bajarilmaganlarga [ ] belgisini qo'yamiz. Buning uchun enumerate() dan foydalanamiz.
def korsatish(vazifalar):
    if not vazifalar:
        print("Ro'yxat bo'sh.")
        return
    for i, v in enumerate(vazifalar, 1):
        belgi = "x" if v["bajarildi"] else " "
        print(f"{i}. [{belgi}] {v['matn']}")
# Agar ro'yxat bo'sh bo'lsa, xabar berib chiqamiz
# enumerate(..., 1) — raqamlash 1 dan boshlanadi
# bajarildi True bo'lsa "x", aks holda bo'sh joy
>>> korsatish(vazifalar)
1. [ ] Kitob o'qish
2. [x] Sport bilan shug'ullanish

4-bosqich: Bajarildi Belgilash (Update)

Foydalanuvchi vazifa raqamini beradi, biz uning bajarildi holatini True qilamiz. Ro'yxatda indeks 0 dan boshlanadi, shuning uchun raqamdan 1 ni ayiramiz.
def belgilash(vazifalar, raqam):
    i = raqam - 1
    if 0 <= i < len(vazifalar):
        vazifalar[i]["bajarildi"] = True
        print("Bajarildi deb belgilandi!")
    else:
        print("Bunday raqam yo'q.")
# raqam - 1 = ro'yxatdagi indeks
# Tekshirish: raqam ro'yxat ichida bormi?
# bor bo'lsa holatni True qilamiz

5-bosqich: Vazifani O'chirish (Delete)

Tanlangan vazifani ro'yxatdan butunlay olib tashlaymiz. Buning uchun .pop(indeks) metodidan foydalanamiz.
def ochirish(vazifalar, raqam):
    i = raqam - 1
    if 0 <= i < len(vazifalar):
        olingan = vazifalar.pop(i)
        print("O'chirildi:", olingan["matn"])
    else:
        print("Bunday raqam yo'q.")
# pop(i) — i-indeksdagi vazifani olib tashlaydi
# va olib tashlangan vazifani qaytaradi

6-bosqich: Faylga Saqlash (JSON)

Dastur yopilganda ro'yxat yo'qolmasligi uchun uni faylga yozamiz. Bizning ma'lumotimiz list va dictionary lardan iborat — buni JSON formatida saqlash juda qulay. json.dump() ma'lumotni faylga yozadi, json.load() esa fayldan qayta o'qiydi.
import json

def saqlash(vazifalar):
    with open("vazifalar.json", "w") as f:
        json.dump(vazifalar, f, ensure_ascii=False)
# "w" — yozish rejimi (write)
# json.dump — list/dict ni faylga JSON qilib yozadi
# ensure_ascii=False — o'zbekcha harflar to'g'ri saqlanadi

7-bosqich: Fayldan Yuklash

Dastur ochilganda saqlangan vazifalarni qayta yuklaymiz. Agar fayl hali mavjud bo'lmasa (birinchi marta ishga tushirilsa), bo'sh ro'yxat qaytaramiz. Buni try / except bilan ushlaymiz.
def yuklash():
    try:
        with open("vazifalar.json", "r") as f:
            return json.load(f)
    except FileNotFoundError:
        return []
# "r" — o'qish rejimi (read)
# json.load — fayldan o'qib list/dict ga aylantiradi
# Fayl topilmasa — bo'sh ro'yxat qaytaramiz

8-bosqich: Asosiy Dastur (Menyu)

Endi hammasini birlashtiramiz. while True sikli bilan cheksiz menyu yasaymiz: foydalanuvchi raqam tanlaydi, biz mos amalni bajaramiz. Har o'zgarishdan keyin saqlash() ni chaqiramiz.
vazifalar = yuklash() # saqlanganlarni yuklaymiz

while True:
    print("\n1-Qo'shish 2-Ko'rish 3-Belgilash 4-O'chirish 5-Chiqish")
    tanlov = input("Tanlang: ")
    if tanlov == "1":
        qoshish(vazifalar, input("Vazifa: "))
    elif tanlov == "2":
        korsatish(vazifalar)
    elif tanlov == "3":
        belgilash(vazifalar, int(input("Raqam: ")))
    elif tanlov == "4":
        ochirish(vazifalar, int(input("Raqam: ")))
    elif tanlov == "5":
        break
    saqlash(vazifalar) # har amaldan keyin saqlaymiz
# Boshida yuklash() — eski vazifalar qaytadi
# while True — foydalanuvchi 5 ni tanlaguncha davom etadi
# Har siklning oxirida saqlash() — ma'lumot yo'qolmaydi
Eslab Qoling

• Vazifa = dictionary, ro'yxat = list of dict
• CRUD — qo'shish, ko'rish, belgilash, o'chirish
• json.dump() saqlaydi, json.load() yuklaydi
• Har o'zgarishdan keyin saqlash kerak — aks holda ma'lumot yo'qoladi
• Fayl topilmasa try/except bilan bo'sh ro'yxat beramiz

Misol 1: Ishlaydigan Todo Ro'yxati

Mana haqiqiy ishlaydigan Todo! Vazifa yozing va Qo'shish ni bosing. Aylana belgisini bosib bajarildi deb belgilang. × tugmasi vazifani o'chiradi. Bu xuddi yuqoridagi Python kod kabi ishlaydi.

    Misol 2: JSON ga Saqlash va Yuklash

    Yuqoridagi ro'yxat faylga qanday saqlanishini ko'ring. Saqlash ni bossangiz, ro'yxat JSON matniga aylanadi (xuddi json.dump kabi). Yuklash esa o'sha JSON dan ro'yxatni qayta tiklaydi (xuddi json.load kabi).

    Misol 3: CRUD Amallarini Sinash

    Har bir CRUD amalini bosib ko'ring. Demo bir vazifa ustida amalni bajaradi va natijani Python uslubida ko'rsatadi.
