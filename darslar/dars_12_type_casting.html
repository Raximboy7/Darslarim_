Type casting — int(), float(), str(), bool()

Modul 1 — Asoslar | Dars 12

Type casting nima?

Type casting (tur o'girish) — bu bir turdagi qiymatni boshqa turga aylantirish. Masalan, matn ko'rinishidagi "25" ni haqiqiy son 25 ga o'girish. Bu ayniqsa foydalanuvchidan ma'lumot olganda juda muhim.

Asosiy o'girish funksiyalari:

FunksiyaNima qiladiMisolNatija
int()butun songaint("25")25
float()kasr songafloat("3.5")3.5
str()matngastr(100)"100"
bool()mantiqiy turgabool(0)False

type() — turni tekshirish

O'girishdan oldin qiymatning hozirgi turini bilish uchun type() ishlatamiz.
a = "25"
b = 25
print(type(a))
print(type(b))
# "25" — matn (str)
# 25 — son (int)
>>> <class 'str'>
>>> <class 'int'>

int() va float()

int() matn yoki kasr sonni butun songa o'giradi. float() esa kasr songa aylantiradi.
print(int("42"))
print(int(3.9))
print(float("3.14"))
print(float(7))
# int(3.9) kasrni TASHLAYDI (yaxlitlamaydi!)
# float(7) -> 7.0
>>> 42
>>> 3
>>> 3.14
>>> 7.0
Diqqat! int(3.9) natijasi 4 emas, 3 bo'ladi — kasr qismi shunchaki olib tashlanadi.

str() va bool()

str() sonni matnga o'giradi — sonni matn bilan birlashtirmoqchi bo'lganda kerak. bool() qiymatni True/False ga aylantiradi.
yosh = 20
print("Yoshim: " + str(yosh))
print(bool(1))
print(bool(0))
# str(yosh) bo'lmasa xatolik chiqadi
# bool(0) -> False, boshqasi -> True
>>> Yoshim: 20
>>> True
>>> False

input() bilan real misol

input() har doim matn (str) qaytaradi — hatto son yozsangiz ham! Shuning uchun hisob-kitob qilishdan oldin uni int() yoki float() ga o'girish shart.
yosh = input("Yoshingizni kiriting: ")
yosh = int(yosh)
keyingi_yil = yosh + 1
print("Kelasi yil:", keyingi_yil)
# input() -> "20" (matn)
# int() bilan songa o'girdik
# endi + 1 qila olamiz
Yoshingizni kiriting: 20
>>> Kelasi yil: 21
Xato: int() ni unutsangiz Agar int() qilmasangiz, "20" + 1 TypeError beradi — matn va sonni qo'shib bo'lmaydi.

O'girish xatoliklari (ValueError)

Har qanday matnni songa o'girib bo'lmaydi. int("salom") xatolik beradi, chunki "salom" son emas.
son = int("salom")
# "salom" sonli matn emas!
>>> ValueError: invalid literal for int()
>>> with base 10: 'salom'
Eslatma Faqat sonli ko'rinishdagi matn ("123") int() ga o'giriladi. int("12.5") ham xatolik beradi — buning uchun float("12.5") ishlating.

Asosiy xulosalar

Esda tuting

• int() — butun son, float() — kasr son
• str() — matn, bool() — True/False
• int(3.9) = 3 (kasr tashlanadi, yaxlitlanmaydi)
• input() doim matn qaytaradi — int() ga o'giring
• Sonli bo'lmagan matnni o'girsa ValueError chiqadi
