Machine Learning Nima?

Machine Learning (ML) — "mashinaning o'rganishi" degani. Bunda biz kompyuterga aniq qoidalarni qo'lda yozib bermaymiz. Buning o'rniga kompyuter ko'plab ma'lumotni ko'rib, o'zi qoidalarni topadi va o'rganadi.
Oddiy dasturda biz shunday yozamiz: "agar narx 100 dan katta bo'lsa, qimmat de". ML da esa biz minglab misol beramiz va kompyuterning o'zi "qimmat" nima ekanini ma'lumotdan o'rganadi. Inson misollar bilan o'rganganidek, kompyuter ham misollar bilan o'rganadi.
Sodda taqqoslash

Bola olmani ko'p marta ko'rib, "bu olma" deb o'rganadi. Keyin yangi olmani ko'rsa, tanaydi. ML ham xuddi shunday: ko'p misol → o'rganish → yangi narsani tanish.

ML ning 3 Turi

TuriMa'nosiQanday o'rganadi
Supervised O'qituvchili o'rganish Savol va to'g'ri javob birga beriladi (misol: rasm + "bu mushuk")
Unsupervised O'qituvchisiz o'rganish Faqat ma'lumot beriladi, javob yo'q. Kompyuter o'zi guruhlarni topadi
Reinforcement Mukofotli o'rganish To'g'ri qilsa mukofot, xato qilsa jarima. Sinab-yanglishib o'rganadi
Eng ko'p ishlatiladigani

Boshlovchilar uchun eng muhimi — Supervised (o'qituvchili) o'rganish. Bu darsda asosan shunga e'tibor beramiz.

Supervised: Regressiya vs Klassifikatsiya

O'qituvchili o'rganish ikkiga bo'linadi. Farqi — nimani bashorat qilamiz: son yoki sinf (toifa)?

Javob son bo'lsa → Regressiya.
Javob nom/toifa bo'lsa → Klassifikatsiya.

Features va Label

Har bir ML masalada ikki tushuncha bor: features (xususiyatlar — kirish ma'lumoti) va label (natija — biz bilmoqchi bo'lgan javob).
# Uy narxini bashorat qilish
features = [maydon, xonalar, joylashuv]
label    = narx
# features = kompyuterga beradigan ma'lumot
# label = kompyuter topishi kerak bo'lgan javob

Model Qanday O'rganadi?

Model o'rgatish (train) — modelga ko'p misol (features + label) berib, u xususiyatlar bilan natija o'rtasidagi bog'liqlikni topishini ta'minlash. Keyin yangi ma'lumot kelganda model bashorat qiladi.
1. Ma'lumot
Features + Label bilan ko'p misol yig'amiz
→
2. Train
Model misollardan qoidalarni o'rganadi
→
3. Bashorat
Yangi ma'lumotga javob beradi
# G'oya (soddalashtirilgan)
model.fit(features, label)   # o'rgatish
natija = model.predict(yangi) # bashorat
# fit() = o'rgatish (train)
# predict() = yangi ma'lumotga bashorat

Eslab Qoling

• ML = kompyuter ma'lumotdan o'zi o'rganadi
• 3 tur: supervised, unsupervised, reinforcement
• Supervised: son → regressiya, toifa → klassifikatsiya
• features = kirish, label = javob
• Oqim: ma'lumot → train (fit) → bashorat (predict)

Misol 1: Regressiya yoki Klassifikatsiya?

Vazifani tanlang — bu son bashorat qilishmi (regressiya) yoki toifa bashorat qilishmi (klassifikatsiya)?

Misol 2: Features → Model → Bashorat

Uy maydoni va xonalar sonini kiriting. Model o'rgangan oddiy qoida bo'yicha narxni bashorat qiladi (har m² uchun ~4 mln, har xona uchun +10 mln).

Misol 3: Supervised vs Unsupervised

Bir tugma — ma'lumot to'g'ri javob (label) bilan keladi. Ikkinchisi — faqat ma'lumot, javob yo'q. Farqni ko'ring.
