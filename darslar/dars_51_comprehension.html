Comprehension (To'liq)

Modul 9 — Iterator va Generator | Dars 51

Comprehension Nima?

Comprehension — bu ro'yxat, lug'at yoki to'plamni bitta qatorda, qisqa va tushunarli yozish usuli. Oddiy for sikl bilan bir necha qator yoziladigan ishni comprehension bir qatorga jamlaydi — kodni qisqaroq va o'qishga oson qiladi.
Asosiy g'oya

Bo'sh ro'yxat ochib, sikl ichida append() qilishning o'rniga — natijani to'g'ridan-to'g'ri kvadrat qavslar [ ] ichida tasvirlaymiz.

List Comprehension — Asosi

Umumiy shakl: [ifoda for element in ketma-ketlik]. O'ngdan chapga o'qing: "ketma-ketlikdagi har bir element uchun ifodani ol".
sonlar = [x for x in range(5)]
print(sonlar)
# 0 dan 4 gacha sonlarni ro'yxatga yig'adi
>>> [0, 1, 2, 3, 4]

Elementni O'zgartirish

Boshidagi ifoda qismida har bir elementni xohlagancha o'zgartirish mumkin — masalan ikkiga ko'paytirish.
ikki = [x * 2 for x in range(5)]
print(ikki)
# Har bir sonni 2 ga ko'paytiradi
>>> [0, 2, 4, 6, 8]

Shart Bilan (if)

Oxiriga if shart qo'shsak — faqat shartga mos elementlar ro'yxatga tushadi. Quyida faqat juft sonlar olinadi.
juft = [x for x in range(10) if x % 2 == 0]
print(juft)
# x % 2 == 0 bo'lsa — son juft
>>> [0, 2, 4, 6, 8]

Oddiy for vs Comprehension

Ikkisi bir xil natija beradi. Comprehension shunchaki qisqaroq. Quyida bir xil masala ikki usulda yozilgan.
Oddiy for siklComprehension
natija = []
for x in range(5):
    natija.append(x*x)		 natija = [x*x for x in range(5)]
4 qator 1 qator

Dictionary Comprehension

Lug'at ham bir qatorda yasaladi. Shakli: {kalit: qiymat for ...}. Farqi — kvadrat qavs emas, jingalak qavs va kalit:qiymat juftligi.
kvadrat = {x: x**2 for x in range(5)}
print(kvadrat)
# Kalit — son, qiymat — uning kvadrati
>>> {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

Set Comprehension

To'plam (set) ham jingalak qavs bilan yasaladi, lekin kalit:qiymat yo'q — faqat qiymat. To'plam takrorlanmaydigan elementlarni saqlaydi.
toplam = {x % 3 for x in range(10)}
print(toplam)
# Qoldiqlar takrorlanmaydi: 0, 1, 2
>>> {0, 1, 2}

Generator Expression — Qavs Farqi

Agar kvadrat qavs [ ] o'rniga oddiy qavs ( ) ishlatsak — bu generator bo'ladi. Generator barcha qiymatlarni xotirada saqlamaydi, balki kerak bo'lganda bittadan hosil qiladi. Katta ma'lumotlarda xotirani tejaydi.
YozuvTuriXotira
[x for x in range(5)]listHammasini saqlaydi
{x for x in range(5)}setHammasini saqlaydi
{x: x for x in range(5)}dictHammasini saqlaydi
(x for x in range(5))generatorBittadan hosil qiladi
gen = (x * 2 for x in range(5))
print(sum(gen))
# Generator — sum() bilan birga ishlatish qulay
>>> 20
Eslab Qoling

• List: [ifoda for x in ...]
• Dict: {kalit: qiymat for x in ...}
• Set: {ifoda for x in ...}
• Generator: (ifoda for x in ...) — xotirani tejaydi
• Shart: oxiriga if shart qo'shiladi
