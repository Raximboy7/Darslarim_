Xatolik turlari — syntax vs runtime

Modul 1 — Asoslar | Dars 15

Xatoliklar — bu me'yoriy holat

Dasturlashda xatolikka yo'l qo'yish — bu tabiiy. Hatto tajribali dasturchilar ham doim xatoga uchraydi. Muhimi — xatolik turini tushunish va uni tez topib tuzatish. Python xatoliklarni uch guruhga bo'lamiz: syntax, runtime va logic xatolar.
TurQachon yuzaga keladiPython ogohlantiradi?
SyntaxKod yozilishi noto'g'riHa, ishga tushmaydi
RuntimeDastur ishlayotgandaHa, to'xtab qoladi
LogicKod ishlaydi, lekin javob xatoYo'q! O'zingiz topasiz

1. SyntaxError — yozuv xatosi

SyntaxError — kodni yozish qoidasi buzilganda chiqadi. Masalan, qavs yopilmagan, ikki nuqta unutilgan yoki qo'shtirnoq to'liq emas. Bunday holda dastur umuman ishga tushmaydi.
print("Salom"

if yosh > 18
    print("Katta")
# 1-qatorda ) yopilmagan
# 3-qatorda oxiriga : qo'yilmagan
>>> SyntaxError: '(' was never closed
>>> SyntaxError: expected ':'
Yechim Xato ko'rsatilgan qatorni diqqat bilan tekshiring: qavslar, qo'shtirnoqlar va : joyida ekaniga ishonch hosil qiling.

2. Runtime xatolar — ishlash paytida

Runtime xatolar kod to'g'ri yozilgan, lekin ishlayotganda muammoga uchraganda chiqadi. Dastur boshlanadi, keyin xato joyida to'xtab qoladi. Eng ko'p uchraydigan uchtasini ko'ramiz.

NameError — aniqlanmagan nom

print(natija)
# "natija" hech qayerda yaratilmagan
>>> NameError: name 'natija' is not defined

TypeError — noto'g'ri turlar

yosh = 20
print("Yosh: " + yosh)
# matn va sonni + bilan qo'shib bo'lmaydi
>>> TypeError: can only concatenate str
>>> (not "int") to str

ZeroDivisionError — nolga bo'lish

natija = 10 / 0
# matematikada nolga bo'lib bo'lmaydi
>>> ZeroDivisionError: division by zero

3. Logic xatolar — eng ayyori

Logic (mantiqiy) xato — bu eng qiyin turi. Dastur ishlaydi va hech qanday xato bermaydi, lekin natija noto'g'ri! Python bu xatoni topa olmaydi — faqat siz o'zingiz sezasiz.
# O'rtachani topmoqchimiz
a = 10
b = 20
ortacha = a + b / 2
print(ortacha)
# kutgan javob: 15
# aslida: b/2 birinchi hisoblanadi!
>>> 20.0
# 15 emas! Qavs kerak edi: (a + b) / 2
Yechim Logic xatoni topish uchun print() bilan oraliq qiymatlarni tekshiring va natijani qo'lda hisoblab solishtiring.

Xatolikni qanday o'qish kerak

Python xato xabari (traceback) ni ekranga chiqaradi. Uni pastdan yuqoriga o'qing: oxirgi qator xato turini va sababini aytadi, yuqoridagi qator esa qaysi joyda ekanini ko'rsatadi.
1
Oxirgi qatorni o'qing

Xato turi: NameError, TypeError, ...

2
Qator raqamiga qarang

"line 5" — 5-qatorda muammo bor

3
Sababini tushuning

"is not defined", "division by zero" ...

4
Tuzating va qayta ishga tushiring

Bitta-bitta xatoni bartaraf eting

Asosiy xulosalar

Esda tuting

• SyntaxError — yozuv xatosi, dastur ishga tushmaydi
• Runtime — ishlash paytida to'xtaydi (NameError, TypeError, ZeroDivisionError)
• Logic — kod ishlaydi, lekin javob xato
• Xato xabarini pastdan yuqoriga o'qing
• Xatolik — o'rganishning bir qismi, undan qo'rqmang!
