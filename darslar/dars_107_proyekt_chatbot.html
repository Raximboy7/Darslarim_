Chatbot Nima?

Chatbot — bu odam bilan suhbatlashadigan dastur. Foydalanuvchi matn yozadi, chatbot esa o'sha matnga qarab javob qaytaradi. Bizning chatbotimiz aqlli emas — u faqat kalit so'zlarni qidiradi va shu so'zga mos javobni tanlaydi. Lekin shu oddiy mantiq bilan ham haqiqiy suhbatga o'xshash dastur yasash mumkin!

Chatbot Qanday Ishlaydi?

Chatbotning asosiy g'oyasi juda sodda. U doimiy ravishda quyidagi 4 qadamni takrorlaydi:
1
Eshitadi

Foydalanuvchidan matn oladi (input)

2
Tushunadi

Matnda kalit so'zlarni qidiradi

3
Javob tanlaydi

Topilgan kalit so'zga mos javobni oladi

4
Javob beradi

Javobni ekranga chiqaradi

1-bosqich: Javoblarni dict ga saqlash

Chatbotning "miyasi" — bu dict (lug'at). Unda har bir kalit so'z uchun bitta javob saqlanadi. Kalit so'z — bu dictning kaliti, javob — uning qiymati.
javoblar = {
    "salom": "Salom! Qalaysiz?",
    "ism": "Mening ismim PyBot.",
    "yordam": "Albatta yordam beraman!",
    "xayr": "Xayr! Yana keling."
}
# Kalit so'z : javob ko'rinishida saqlaymiz

2-bosqich: Kalit so'zni topish (string matching)

Foydalanuvchi "Salom, qalaysan?" deb yozishi mumkin. Bizning kalit so'zimiz esa faqat "salom". Shuning uchun biz in operatori bilan tekshiramiz: agar kalit so'z matn ichida bo'lsa, javobni tanlaymiz. Avval matnni .lower() bilan kichik harfga aylantiramiz — shunda "SALOM" ham, "Salom" ham bir xil topiladi.
matn = input("Siz: ").lower()

for kalit in javoblar:
    if kalit in matn:
        print(javoblar[kalit])
        break
# Har bir kalit so'zni matn ichida qidiramiz
# Topilsa — javob beramiz va to'xtaymiz (break)
in operatori nima qiladi?

"salom" in "salom dunyo" → True (chunki "salom" matn ichida bor)
"xayr" in "salom dunyo" → False (chunki "xayr" yo'q)

3-bosqich: Tushunmasa nima qiladi?

Agar hech bir kalit so'z topilmasa, chatbot standart javob berishi kerak. Buni for...else bilan qilamiz: agar for sikli breaksiz tugasa (ya'ni hech narsa topilmasa), else ishlaydi.
for kalit in javoblar:
    if kalit in matn:
        print(javoblar[kalit])
        break
else:
    print("Kechirasiz, tushunmadim.")
# else faqat break ishlamasa bajariladi
# Ya'ni hech qaysi kalit so'z topilmasa

4-bosqich: Suhbat tarixini saqlash (list)

Yaxshi chatbot butun suhbatni eslab qoladi. Buning uchun bizga list kerak. Har bir xabar (foydalanuvchidan ham, botdan ham) listga .append() bilan qo'shiladi.
tarix = [] # bo'sh list

tarix.append("Siz: " + matn)
tarix.append("Bot: " + javob)

print("Jami xabarlar:", len(tarix))
# Har bir xabarni tarix listiga qo'shamiz
# len(tarix) — nechta xabar borligini ko'rsatadi

5-bosqich: Hammasini birlashtirish

Endi barcha bosqichlarni bitta while sikliga joylaymiz. Sikl foydalanuvchi "xayr" deganicha davom etadi.
javoblar = {"salom": "Salom!", "xayr": "Xayr!"}
tarix = []

while True:
    matn = input("Siz: ").lower()
    tarix.append(matn)
    
    for kalit in javoblar:
        if kalit in matn:
            javob = javoblar[kalit]
            break
    else:
        javob = "Tushunmadim."
    
    print("Bot:", javob)
    if "xayr" in matn:
        break
# while True — cheksiz sikl
# "xayr" yozilsa siklni to'xtatamiz
Eslab Qoling

• Chatbotning miyasi — dict (kalit so'z : javob)
• Javob tanlash — in operatori bilan string matching
• Tushunmasa — for...else standart javob beradi
• Suhbat tarixi — list ga .append()

Misol 1: Ishlaydigan Chatbot

Quyida haqiqiy ishlaydigan chatbot! Xabar yozing va Yuborish tugmasini bosing. Bot kalit so'zga qarab javob beradi. Sinab ko'ring: salom, ism, yordam, vaqt, rahmat, xayr.

Misol 2: String Matching Qanday Ishlaydi?

Bu demo botning "miyasi" ichini ko'rsatadi. Matn yozing — har bir kalit so'z uchun kalit in matn tekshiruvi True yoki False qaytarishini ko'rasiz. Birinchi True bo'lgan kalit javob tanlaydi.

Misol 3: Suhbat Tarixi
