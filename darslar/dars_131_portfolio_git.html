Amaliy: Portfolioni GitHub'ga joylash

Modul 11 — Git va Version Control | Dars 131

Maqsad

Bu amaliy darsda o'z portfolio saytingizni (yoki istalgan loyihangizni) GitHub'ga joylab, uni GitHub Pages orqali internetda jonli havola sifatida ochamiz. Yakunda sizda https://username.github.io/portfolio ko'rinishidagi haqiqiy manzil bo'ladi.
Kerak bo'ladi

• O'rnatilgan Git (git --version ishlashi kerak).
• GitHub akkaunti (github.com da bepul).
• Portfolio papkasi ichida kamida index.html fayli.

1-qadam: Loyihani Git repozitoriysiga aylantirish

Terminalda portfolio papkangizga kiring va init qiling:

cd portfolio
git init
cd — papkaga kiradi. git init — shu papkada bo'sh Git repozitoriysi (yashirin .git papkasi) yaratadi.
$ git init
Initialized empty Git repository in /home/user/portfolio/.git/

2-qadam: Fayllarni qo'shish va commit qilish

git add .
git commit -m "Birinchi commit: portfolio sayti"
git add . — papkadagi barcha fayllarni kuzatuvga (staging) qo'shadi. git commit -m "..." — ularni izoh bilan tarixga saqlaydi.
$ git commit -m "Birinchi commit: portfolio sayti"
[main (root-commit) a1b2c3d] Birinchi commit: portfolio sayti
 3 files changed, 120 insertions(+)

3-qadam: GitHub'da repository yaratish

1
github.com ga kiring

O'ng yuqoridagi + tugmasidan New repository ni tanlang.

2
Nom bering

Masalan portfolio. Public qoldiring (Pages bepul bo'lishi uchun).

3
README qo'shmang

"Add a README" ni belgilamang — bizda allaqachon fayllar bor. Create repository bosing.

4-qadam: Kodni GitHub'ga yuklash (push)

GitHub bergan havolani remote sifatida ulaymiz va yuklaymiz:

git remote add origin https://github.com/USERNAME/portfolio.git
git branch -M main
git push -u origin main
remote add origin — GitHub manzilini "origin" nomi bilan ulaydi (USERNAME o'rniga o'z ismingizni yozing). branch -M main — asosiy shoxni main deb nomlaydi. push — kodni serverga yuklaydi.
$ git push -u origin main
Enumerating objects: 5, done.
To https://github.com/USERNAME/portfolio.git
 * [new branch]     main -> main

5-qadam: GitHub Pages'ni yoqish

1
Settings → Pages

Repository sahifasida Settings, so'ng chap menyudan Pages ni oching.

2
Branch tanlang

"Branch" bo'limida main va / (root) ni tanlab Save bosing.

3
Havolani oching

Bir necha daqiqada sayt tayyor: https://USERNAME.github.io/portfolio

Keyingi o'zgarishlarni yuklash

Saytni yangilaganingizda faqat shu uch buyruqni takrorlaysiz:

git add .
git commit -m "Yangi loyiha qo'shildi"
git push
push dan bir necha daqiqa keyin jonli sayt avtomatik yangilanadi.

Xulosa

Bajarilgan qadamlar

1. git init — repo yaratildi.
2. git add . + git commit — o'zgarishlar saqlandi.
3. GitHub'da repository ochildi.
4. git remote add + git push — kod yuklandi.
5. GitHub Pages yoqilib, sayt internetga chiqdi.
Endi bu havolani rezyumengizga qo'shishingiz mumkin!
