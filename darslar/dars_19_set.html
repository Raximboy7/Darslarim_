Set (To'plam)

Modul 4 — Ma'lumot tuzilmalari | Dars 19

Set Nima?

Set (to'plam) — bu takrorlanmaydigan elementlar to'plami. Ya'ni bir xil qiymat to'plamda faqat bir marta saqlanadi. Set { } qavslar ichida, elementlar vergul bilan yoziladi. Elementlar tartibsiz saqlanadi.
sonlar = {1, 2, 3, 4, 5}
print(sonlar)
# Takrorlanmas elementlar to'plami
>>> {1, 2, 3, 4, 5}

Takror Avtomatik O'chiriladi

Agar to'plamga bir xil qiymat bir necha marta yozsangiz, Python uni bittaga qoldiradi — takrorlar o'zi o'chiriladi.
s = {1, 2, 2, 3}
print(s)
# 2 ikki marta yozildi, lekin bitta qoldi
>>> {1, 2, 3}

Bo'sh To'plam Yaratish

⚠️ Diqqat: {} bo'sh to'plam EMAS!

Bo'sh {} — bu to'plam emas, balki dictionary (lug'at). Bo'sh to'plam yaratish uchun set() dan foydalaning.

a = set()
b = {}
print(type(a), type(b))
# a — to'plam, b — dictionary
>>> <class 'set'> <class 'dict'>

Asosiy Metodlar

MetodVazifasiMisol
.add(x)Element qo'shadi{1,2}.add(3) → {1,2,3}
.remove(x)O'chiradi (yo'q bo'lsa xato beradi){1,2,3}.remove(2) → {1,3}
.discard(x)O'chiradi (yo'q bo'lsa xatosiz){1,2}.discard(9) → {1,2}
.clear()Hammasini tozalaydi{1,2,3}.clear() → set()
len(s)Elementlar sonini beradilen({1,2,3}) → 3

remove() va discard() farqi

s = {1, 2, 3}
s.discard(9) # xatosiz
s.remove(9) # KeyError!
# discard() — xavfsiz, remove() — yo'q bo'lsa xato

To'plamlar Ustida Amallar

To'plamlarni xuddi matematikadagi kabi birlashtirish, kesishma va farq amallari bilan ishlatish mumkin.
AmalBelgiMetodMa'nosi
Birlashma|.union()Ikkalasining barcha elementlari
Kesishma&.intersection()Faqat umumiy elementlar
Farq-.difference()Birinchisida bor, ikkinchisida yo'q
a = {1, 2, 3}
b = {3, 4, 5}
print(a | b)
print(a & b)
print(a - b)
# birlashma
# kesishma
# farq
>>> {1, 2, 3, 4, 5}
>>> {3}
>>> {1, 2}

Ro'yxatdan Takrorlarni Olib Tashlash

Set ning eng foydali amaliy ishi — ro'yxatdagi (list) takror qiymatlarni tezda yo'qotish. set() ga aylantirib, keyin yana list() ga qaytaramiz.
ismlar = ["Ali", "Vali", "Ali", "Guli"]
noyob = list(set(ismlar))
print(noyob)
# Takror "Ali" olib tashlandi
>>> ['Ali', 'Vali', 'Guli']

Keng Tarqalgan Xato

❌ Xato: bo'sh to'plamni {} bilan yaratish
s = {}
s.add(5) # XATO! {} — dictionary

{} dictionary bo'lgani uchun unda .add() metodi ishlamaydi.

✓ To'g'ri: set() bilan yarating
s = set()
s.add(5)
print(s)
>>> {5}
