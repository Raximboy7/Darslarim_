To'rt Tuzilma — Bir Maqsad

Python'da ma'lumotni saqlashning 4 ta asosiy tuzilmasi bor: list, tuple, dict va set. Ularning hammasi bir nechta qiymatni saqlaydi, lekin har biri o'ziga xos maqsad uchun yaratilgan. To'g'ri tuzilmani tanlash — dasturingizni tez, sodda va xatosiz qiladi.

Har birining ko'rinishi

List — tartibli, o'zgaruvchan [ ]

mevalar = ["olma", "banan", "olma"]
mevalar.append("uzum")
# Tartibli, takror mumkin, o'zgartirsa bo'ladi
>>> ['olma', 'banan', 'olma', 'uzum']

Tuple — tartibli, O'ZGARMAS ( )

nuqta = (10, 20)
print(nuqta[0])
# Yaratilgach o'zgartirib bo'lmaydi
>>> 10

Dictionary — key-value juftliklar { }

talaba = {"ism": "Ali", "yosh": 20}
print(talaba["ism"])
# Kalit orqali qiymatga murojaat
>>> Ali

Set — tartibsiz, takrorlanmaydigan { }

raqamlar = {1, 2, 2, 3, 3}
print(raqamlar)
# Takrorlar avtomatik o'chiriladi
>>> {1, 2, 3}

Samaradorlik va Xotira

Tezlik haqida qisqacha

• set va dict da "bormi?" tekshiruvi (in) juda tez — millionta elementda ham bir xil tez.

• list da in tekshiruvi har bir elementni ketma-ket ko'radi — ro'yxat kattalashsa, sekinlashadi.

• tuple list'dan biroz kam xotira oladi va o'zgarmasligi xatolardan saqlaydi.

Keng Tarqalgan Xato

❌ Xato: tuple ni o'zgartirishga urinish
nuqta = (10, 20)
nuqta[0] = 99 # XATO!
>>> TypeError: 'tuple' object does not
    support item assignment

Tuple o'zgarmas (immutable). O'zgartirish kerak bo'lsa list ishlating.

✓ To'g'ri: o'zgaradigan ma'lumot uchun list
nuqta = [10, 20]
nuqta[0] = 99
print(nuqta)
>>> [99, 20]
❌ Xato: set elementlari tartiblangan deb o'ylash
s = {3, 1, 2}
print(s[0]) # XATO! indeks yo'q

Set tartibsiz — indeks bilan murojaat qilib bo'lmaydi. Tartib kerak bo'lsa list ishlating.

Eslab Qoling

• list [ ] — tartibli, o'zgaruvchan, takror mumkin
• tuple ( ) — tartibli, O'ZGARMAS
• dict {k:v} — kalit-qiymat juftliklar
• set { } — tartibsiz, takrorlanmaydi, tez qidirish
