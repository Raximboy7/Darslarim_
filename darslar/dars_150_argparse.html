argparse nima?

argparse — Python'ning standart kutubxonasi bo'lib, buyruq qatori (CLI — Command Line Interface) dasturlari yaratishga yordam beradi. U foydalanuvchi terminalda bergan argumentlarni o'qiydi, tekshiradi va avtomatik yordam (help) matnini yaratadi.

Masalan: python salom.py --ism Ali --marta 3 — bu yerda --ism va --marta argumentlar.

Eng oddiy misol

# salom.py
import argparse

parser = argparse.ArgumentParser(description="Salomlashish dasturi")
parser.add_argument("ism")          # pozitsion argument (majburiy)
args = parser.parse_args()

print(f"Salom, {args.ism}!")
ArgumentParser — asosiy obyekt. add_argument — argument qo'shadi. parse_args — terminaldan qiymatlarni o'qiydi.

Terminalda:

python salom.py Ali
"Ali" — pozitsion argument sifatida args.ism ga tushadi.
Salom, Ali!

Pozitsion va ixtiyoriy argumentlar

Pozitsion argument majburiy va tartibi muhim. --flag ko'rinishidagi argumentlar ixtiyoriy (optional). 

import argparse

parser = argparse.ArgumentParser()
parser.add_argument("ism")                       # majburiy
parser.add_argument("--marta", type=int, default=1)  # ixtiyoriy, int, standart 1
parser.add_argument("--baland", action="store_true")  # bayroq: bor/yo'q
args = parser.parse_args()

matn = f"Salom, {args.ism}!"
if args.baland:
    matn = matn.upper()
for _ in range(args.marta):
    print(matn)
type=int — qiymatni songa aylantiradi. default — berilmasa ishlatiladigan qiymat. action="store_true" — bayroq borligi True/False sifatida saqlanadi.

Terminalda:

python salom.py Ali --marta 2 --baland
Ali ismini 2 marta, katta harflarda chiqaradi.
SALOM, ALI! SALOM, ALI!

Avtomatik yordam (--help)

argparse har bir dasturga -h / --help ni avtomatik qo'shadi. Buni siz yozishingiz shart emas. 

python salom.py --help
Barcha argumentlar, turlari va tavsiflari ko'rsatiladi.
usage: salom.py [-h] [--marta MARTA] [--baland] ism positional arguments: ism options: -h, --help show this help message and exit --marta MARTA --baland
