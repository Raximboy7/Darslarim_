assert Nima?

assert — bu shartni tekshiradigan maxsus kalit so'z. U sizning taxminingiz to'g'ri ekanligini tasdiqlaydi. Agar shart True bo'lsa — hech narsa bo'lmaydi, dastur davom etadi. Agar shart False bo'lsa — Python darhol AssertionError xatosini chiqaradi va dasturni to'xtatadi.
assert 5 > 3
print("Hammasi joyida")
# 5 > 3 — bu True, demak dastur davom etadi
>>> Hammasi joyida

Sintaksis

assert ikki ko'rinishda yoziladi: faqat shart bilan, yoki shart va xato xabari bilan birga.
assert shart
assert shart, "xabar"
# 1-ko'rinish: faqat shart # 2-ko'rinish: shart yolg'on bo'lsa "xabar" ko'rsatiladi
x = -5
assert x > 0, "X ijobiy bo'lishi kerak"
# x = -5 manfiy, shart yolg'on # natija: AssertionError: X ijobiy bo'lishi kerak
>>> AssertionError: X ijobiy bo'lishi kerak

Eslatma

assert — bu dasturchining yordamchisi. U kod ishlash davomida "men shunday deb o'ylagandim" degan taxminlarni avtomatik tekshiradi. Agar taxmin xato bo'lsa — siz darhol bilasiz.

assert vs try-except — Farqi

Bu ikkala vosita turli maqsadlar uchun. Ularni adashtirmang!

Oltin qoida

Foydalanuvchi noto'g'ri ma'lumot kiritishi mumkin bo'lsa — try-except ishlat. Faqat dasturchi xatosi (kodda mantiqiy nosozlik) bo'lsa — assert ishlat.

Misol 1: Shartni Tekshirish

Qiymat kiriting. Agar son 0 dan katta bo'lsa, assert o'tadi. Aks holda AssertionError chiqadi.

Misol 2: assert Bilan / Holda Farqi

Bir xil noto'g'ri qiymat. Tugmani bosib, assert borligi va yo'qligi qanday natija berishini solishtiring.

Misol 3: Xabarli assert

Yoshni kiriting. assert xabar bilan ishlaydi — shart yolg'on bo'lsa, siz yozgan xabar AssertionError bilan birga ko'rsatiladi.

Misol 4: Funksiya Natijasini Tekshirish

Ikkita sonni qo'shamiz. Kutilgan natijani kiriting — assert haqiqiy natija bilan solishtiradi. Test yozishda aynan shunday ishlatiladi.
Eslab Qoling

• assert shart — shart yolg'on bo'lsa AssertionError chiqaradi
• assert shart, "xabar" — xato bilan birga xabar ko'rsatadi
• assert dasturchi xatosini topadi, foydalanuvchi xatosi uchun emas
• Foydalanuvchi xatosi uchun try-except ishlating
• assert debug va test yozishda juda foydali
