Bool va mantiq — True/False

Modul 1 — Asoslar | Dars 7

Bool nima?

Bool (mantiqiy tur) — bu faqat ikkita qiymatga ega bo'lgan tur: True (rost) va False (yolg'on). Kompyuter "ha/yo'q", "to'g'ri/noto'g'ri" degan savollarga aynan shu qiymatlar bilan javob beradi.
yomgir_yogyapti = True
dars_tugadi = False
print(yomgir_yogyapti)
print(type(dars_tugadi))
# True va False — katta harf bilan yoziladi!
>>> True
>>> <class 'bool'>
Diqqat! True va False katta harf bilan boshlanadi. true yoki FALSE deб yozsangiz — xatolik bo'ladi.

Taqqoslashdan bool hosil bo'ladi

Ikki qiymatni solishtirsak, natija har doim True yoki False bo'ladi. Bu Python'da qaror qabul qilishning asosidir.
OperatorMa'nosiMisolNatija
==teng5 == 5True
!=teng emas5 != 3True
>katta10 > 3True
<kichik2 < 1False
>=katta yoki teng5 >= 5True
<=kichik yoki teng4 <= 3False
yosh = 18
print(yosh >= 18)
print(yosh == 20)
# 18 katta yoki teng 18 -> True
# 18 teng 20 mi? -> False
>>> True
>>> False

Truthy va Falsy

Python'da har qanday qiymatni "rostga o'xshash" (truthy) yoki "yolg'onga o'xshash" (falsy) deb ko'rish mumkin. Ba'zi qiymatlar False deb hisoblanadi:
  • 0 va 0.0 — falsy
  • "" (bo'sh string) — falsy
  • None — falsy
  • Bo'sh ro'yxat, bo'sh lug'at — falsy

Qolgan hamma narsa (nol bo'lmagan son, bo'sh bo'lmagan matn) — truthy.

Oddiy qoida "Bo'sh yoki nol" — False. "To'la yoki bor" — True.

bool() bilan o'girish

bool() funksiyasi istalgan qiymatning truthy yoki falsy ekanini ko'rsatadi.
print(bool(0))
print(bool(42))
print(bool(""))
print(bool("Salom"))
# 0 va bo'sh matn -> False
# boshqa hamma narsa -> True
>>> False
>>> True
>>> False
>>> True

Amaliy misol: kirish tekshiruvi

Foydalanuvchi biror narsa yozdimi yoki bo'sh qoldirdimi — buni bool bilan tekshiramiz.
ism = "Ali"
parol = ""
print("Ism kiritildi:", bool(ism))
print("Parol kiritildi:", bool(parol))
print("Ikkalasi ham to'ldirilgan:", bool(ism) and bool(parol))
# bo'sh parol -> False
# and: ikkalasi True bo'lsagina True
>>> Ism kiritildi: True
>>> Parol kiritildi: False
>>> Ikkalasi ham to'ldirilgan: False

Asosiy xulosalar

Esda tuting

• Bool faqat True yoki False (katta harf!)
• Taqqoslash (==, !=, >, <) natijasi doim bool
• 0, "", None — falsy (False kabi)
• Boshqa qiymatlar — truthy (True kabi)
• bool() — qiymatni True/False ga o'giradi
