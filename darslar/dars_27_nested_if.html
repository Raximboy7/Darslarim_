Ichma-ich shartlar (nested if)

Modul 3 — Shartlar va Sikllar | Dars 27

Nested if nima?

Ichma-ich shart (nested if) — bu bir if blokining ichida yana boshqa if shartini yozish demakdir. Ya'ni birinchi shart bajarilgandan keyin, ichkarida yana qo'shimcha tekshiruv o'tkaziladi.

Buni hayotdagi misolga o'xshatish mumkin: avval "Ob-havo yaxshimi?" deb so'raymiz. Agar yaxshi bo'lsa, ichkarida yana "Vaqtim bormi?" deb tekshiramiz. Ikkala shart ham rost bo'lsagina sayrga chiqamiz.

Diqqat: Otступ (indentatsiya)

Ichki if tashqi if dan yana bir pog'ona ichkariroqda (odatda 4 ta bo'sh joy) yoziladi. Python bloklarni aynan shu bo'shliqlar orqali ajratadi.

Oddiy nested if misoli

Foydalanuvchi yoshini so'raymiz. Agar 18 dan katta bo'lsa, ichkarida haydovchilik guvohnomasi bor-yo'qligini tekshiramiz. 

age = int(input("Yoshingiz: "))

if age >= 18:
    print("Siz balog'atga yetgansiz.")
    guvohnoma = input("Guvohnomangiz bormi? (ha/yo'q): ")
    if guvohnoma == "ha":
        print("Mashina hayday olasiz!")
    else:
        print("Avval guvohnoma oling.")
else:
    print("Siz hali voyaga yetmagansiz.")
Tashqi if yoshni tekshiradi. Faqat yosh 18 dan katta bo'lsagina, ichki if guvohnomani tekshiradi.
>>> Yoshingiz: 20
>>> Siz balog'atga yetgansiz.
>>> Guvohnomangiz bormi? (ha/yo'q): ha
>>> Mashina hayday olasiz!

Baho aniqlash misoli

Nested if ko'pincha bir necha bosqichli qarorlar uchun ishlatiladi. Quyida talaba imtihondan o'tdimi, o'tsa bahosi qanday ekanini aniqlaymiz. 

ball = 85

if ball >= 60:
    print("Imtihondan o'tdingiz!")
    if ball >= 90:
        print("Baho: A'lo")
    elif ball >= 75:
        print("Baho: Yaxshi")
    else:
        print("Baho: Qoniqarli")
else:
    print("Afsuski, yiqildingiz.")
Avval o'tdi/yiqildi aniqlanadi, keyin ichkarida aniq baho beriladi.
>>> Imtihondan o'tdingiz!
>>> Baho: Yaxshi

Amaliyot

O'zingiz sinab ko'ring

1. Foydalanuvchidan son so'rang. Agar musbat bo'lsa, ichkarida u juftmi yoki toqmi tekshiring.
2. Ikki sonni solishtiring: avval tengmi tekshiring, teng bo'lmasa qaysi biri kattaligini aniqlang.
3. Login tizimi: avval login to'g'rimi, keyin ichkarida parol to'g'rimi tekshiring. 

son = int(input("Son kiriting: "))

if son > 0:
    if son % 2 == 0:
        print("Musbat va juft")
    else:
        print("Musbat va toq")
else:
    print("Son musbat emas")
Bu yerda ikki daraja: tashqi shart ishorani, ichki shart juft/toqlikni tekshiradi.

Xulosa

Nimalarni o'rgandik

• Nested if — bu if ichida yana if.
• Ichki blok qo'shimcha bo'shliq bilan yoziladi.
• U bir necha bosqichli qarorlar qabul qilishga yordam beradi.
• Juda ko'p ichma-ichlik kodni chalkashtiradi — imkon bo'lsa elif yoki and bilan soddalashtiring.
