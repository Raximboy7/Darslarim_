Uch Xil Metod Bor

Klass ichida uch turdagi metod yozish mumkin. Ularning farqi — birinchi argument nima bo'lishida: self, cls yoki umuman hech narsa.
TuriBirinchi argumentNima bilan ishlaydi
Instance metodselfAniq obyekt (masalan: bitta talaba)
@classmethodclsKlassning o'zi (barcha obyektlar)
@staticmethod(yo'q)Hech narsa — mustaqil yordamchi

1. Instance Metod (eslatma)

Bu biz odatda yozadigan metod. Birinchi argumenti self — u aniq obyektni bildiradi. Shu obyektning ma'lumotlari bilan ishlaydi.
class Talaba:
    def __init__(self, ism):
        self.ism = ism

    def salomlash(self):
        return f"Salom, {self.ism}!"
# self orqali aynan shu obyektning ismiga murojaat qilamiz
# Talaba("Ali").salomlash() → "Salom, Ali!"

2. @staticmethod — Mustaqil Yordamchi

Ba'zan metod klass ichida joylashgan, lekin na obyektga (self), na klassga (cls) muhtoj bo'ladi. U shunchaki mustaqil yordamchi funksiya — faqat o'ziga kelgan argumentlar bilan ishlaydi. Ustiga @staticmethod yoziladi.
class Matematika:
    @staticmethod
    def qoshish(a, b):
        return a + b

print(Matematika.qoshish(2, 3))
# self yo'q — obyekt yaratmasdan to'g'ridan-to'g'ri chaqiramiz
# Klass shunchaki funksiyalarni bir joyga to'playdi
>>> 5
Qachon ishlatiladi?

Metod mantiqan klassga taalluqli, lekin obyektning ma'lumotlariga kerak bo'lmasa. Masalan: hisoblash, tekshirish, formatlash kabi yordamchi vazifalar.

3. Klass va Instance O'zgaruvchisi

Klass o'zgaruvchisi (class variable) — klass ichida, metodlardan tashqarida yoziladi va barcha obyektlar uchun umumiy.
Instance o'zgaruvchisi — self. bilan yoziladi va har obyektga alohida tegishli.
class Talaba:
    maktab = "1-maktab"    

    def __init__(self, ism):
        self.ism = ism    
# maktab — klass o'zgaruvchisi (hammaga umumiy)
# self.ism — instance o'zgaruvchisi (har kimga alohida)

4. @classmethod — Klass Bilan Ishlaydi

@classmethod ning birinchi argumenti cls — bu klassning o'zi. U aniq bir obyekt bilan emas, balki butun klass darajasida ish ko'radi. Masalan: yaratilgan obyektlar sonini sanash yoki klass o'zgaruvchisini o'zgartirish.
class Talaba:
    soni = 0

    def __init__(self, ism):
        self.ism = ism
        Talaba.soni += 1

    @classmethod
    def nechta(cls):
        return cls.soni
# Har yangi obyekt yaratilganda soni 1 ga oshadi
# cls.soni — klass o'zgaruvchisini o'qiydi
# Talaba.nechta() → jami nechta talaba borligini qaytaradi
>>> Talaba("Ali"); Talaba("Vali")
>>> Talaba.nechta()
2

Qaysi Birini Qachon?

HolatTanlov
Obyektning shaxsiy ma'lumoti bilan ishlashself (instance metod)
Butun klass darajasidagi ish (soni, umumiy sozlama)cls (@classmethod)
Mustaqil yordamchi (na self, na cls kerak)(yo'q) (@staticmethod)
Eslab Qoling

• self → bitta obyekt bilan ishlaydi
• @classmethod + cls → klassning o'zi bilan ishlaydi
• @staticmethod → na self, na cls — mustaqil funksiya
• Klass o'zgaruvchisi hammaga umumiy, instance o'zgaruvchisi har kimga alohida

Misol 1: @staticmethod Yordamchi

Matematika.ozgar(a, b) — obyekt yaratmasdan ishlaydigan statik metod. Ikki son kiriting va amalni tanlang.

Misol 2: Klass O'zgaruvchisi Umumiy

maktab — klass o'zgaruvchisi. Uni o'zgartirsangiz, barcha obyektlarda birdaniga o'zgaradi. Quyida 3 ta talaba bor.

Misol 3: @classmethod — Obyektlarni Sanash

Har "Yangi talaba" tugmasini bossangiz, obyekt yaratiladi va klass o'zgaruvchisi soni 1 ga oshadi. Talaba.nechta() jamini qaytaradi.
