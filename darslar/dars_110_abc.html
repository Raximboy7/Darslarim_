Abstract Base Class (ABC)

Modul 9 — OOP | Dars 110

Abstract klass nima?

Abstract Base Class (ABC) — bu to'g'ridan-to'g'ri obyekt yasab bo'lmaydigan klass. U faqat qolip / shartnoma vazifasini bajaradi: undan meros olgan har bir klass ma'lum metodlarni albatta yozishi shart.
Nega kerak? 📋

Tasavvur qiling — barcha to'lov usullari (Karta, Naqd, Click) to'la() metodiga ega bo'lishi kerak. Abstract klass buni majburiy qiladi: agar dasturchi to'la() ni yozishni unutsa, dastur xato beradi.

ABC va abstractmethod import qilish

Abstract klass yaratish uchun abc modulidan ABC va abstractmethod ni import qilamiz.
from abc import ABC, abstractmethod

class Shakl(ABC):
    @abstractmethod
    def yuza(self):
        pass
# Shakl(ABC) — abstract klass. yuza() — majburiy metod, tanasi bo'sh

Abstract klassdan obyekt yasab bo'lmaydi

Abstract klassni to'g'ridan-to'g'ri chaqirsak, Python xato beradi — chunki u faqat qolip.
s = Shakl()  # XATO!
# Abstract klassdan bevosita obyekt yaratib bo'lmaydi
>>> TypeError: Can't instantiate abstract class Shakl
>>> with abstract method yuza

Meros olgan klass metodni yozishi shart

Shakl dan meros olgan klasslar yuza() ni albatta yozishi kerak. Yozgandan keyingina obyekt yasash mumkin bo'ladi.
class Kvadrat(Shakl):
    def __init__(self, tomon):
        self.tomon = tomon
    def yuza(self):  # majburiy — yozildi
        return self.tomon ** 2

k = Kvadrat(4)
print(k.yuza())
# yuza() yozilgani uchun endi Kvadrat obyektini yasash mumkin
>>> 16

Metodni yozishni unutsak?

Agar meros olgan klass yuza() ni yozmasa, obyekt yasaganda Python xato beradi — bu xatoni erta ushlashga yordam beradi.
class Doira(Shakl):
    pass  # yuza() yozilmadi!

d = Doira()  # XATO
# yuza() yozilmagani uchun Doira() obyekti yaratilmaydi
>>> TypeError: Can't instantiate abstract class Doira
>>> with abstract method yuza

Xulosa

Eslab qoling

• from abc import ABC, abstractmethod
• Abstract klass — class X(ABC), undan obyekt yasab bo'lmaydi
• @abstractmethod — meros oluvchi klass uchun majburiy metod
• Bu "shartnoma" — barcha bola klasslar bir xil interfeysga ega bo'ladi
