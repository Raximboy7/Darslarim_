Entry Nima?

Entry — bu foydalanuvchi bir qatorli matn yozadigan maydon. Ism, parol, raqam kabi qisqa ma'lumotlarni kiritish uchun ishlatiladi. Buni saytlardagi "input" oynachasiga o'xshating.
import tkinter as tk

oyna = tk.Tk()
maydon = tk.Entry(oyna)
maydon.pack()

oyna.mainloop()
# Entry(oyna) — yozish maydonini yaratadi
# pack() — uni oynaga joylaydi

.get() — Qiymatni Olish

Foydalanuvchi maydonga nima yozganini olish uchun .get() metodini ishlatamiz. U yozilgan matnni string sifatida qaytaradi.
maydon = tk.Entry(oyna)
maydon.pack()

def korsat():
    matn = maydon.get()  # maydondagi yozuvni oladi
    print("Siz yozdingiz:", matn)

tugma = tk.Button(oyna, text="Ko'rsat", command=korsat)
tugma.pack()
# Tugma bosilganda korsat() ishlaydi
# .get() har doim string qaytaradi
Siz yozdingiz: Ali

.insert() va .delete()

Dastur orqali maydonga matn yozish yoki uni tozalash mumkin. .insert() matn qo'shadi, .delete() matnni o'chiradi.
maydon = tk.Entry(oyna)
maydon.pack()

maydon.insert(0, "Salom")  # 0-pozitsiyaga yozadi

maydon.delete(0, tk.END)  # boshidan oxirigacha o'chiradi
# insert(0, ...) — boshiga yozadi
# delete(0, tk.END) — hammasini tozalaydi
MetodVazifasi
.get()Maydondagi matnni oladi
.insert(0, "matn")Berilgan joyga matn qo'shadi
.delete(0, tk.END)Maydonni butunlay tozalaydi

Text — Ko'p Qatorli Maydon

Entry faqat bir qator uchun. Agar uzun matn, izoh yoki bir nechta qator kerak bo'lsa Text widgetidan foydalanamiz.
matn_maydon = tk.Text(oyna, height=5, width=30)
matn_maydon.pack()

natija = matn_maydon.get("1.0", tk.END)
# Text — ko'p qatorli yozish maydoni
# "1.0" — 1-qator, 0-belgi (boshi)

StringVar — Qiymatni Bog'lash

StringVar — bu maxsus "o'zgaruvchi quti". Uni Entry ga bog'lasak, maydon o'zgarganda quti ham o'zgaradi. Bu qiymatni boshqarishni osonlashtiradi.
ism = tk.StringVar()
maydon = tk.Entry(oyna, textvariable=ism)
maydon.pack()

ism.set("Ali")  # maydonga "Ali" yozadi
print(ism.get())  # maydondagi qiymatni oladi
# textvariable=ism — maydonni qutiga bog'laydi
# .set() yozadi, .get() o'qiydi

Checkbutton va Radiobutton

Checkbutton — ha/yo'q (bayroqcha) tanlovi: bir nechtasini birga belgilash mumkin. Radiobutton — bir nechta variantdan faqat bittasini tanlash.
roziman = tk.BooleanVar()
tk.Checkbutton(oyna, text="Roziman", variable=roziman).pack()

jins = tk.StringVar()
tk.Radiobutton(oyna, text="Erkak", variable=jins, value="E").pack()
tk.Radiobutton(oyna, text="Ayol", variable=jins, value="A").pack()
# Checkbutton — belgilash (True/False)
# Radiobutton — bir guruhdan bittasi (value bilan)

Oddiy Forma Yasash

Forma — bu Label + Entry + Button kombinatsiyasi. Label nima yozish kerakligini tushuntiradi, Entry ma'lumot oladi, Button esa uni qayta ishlaydi.
import tkinter as tk

oyna = tk.Tk()

tk.Label(oyna, text="Ismingiz:").pack()
ism = tk.Entry(oyna)
ism.pack()

natija = tk.Label(oyna, text="")
natija.pack()

def salomla():
    natija.config(text="Salom, " + ism.get())

tk.Button(oyna, text="Yubor", command=salomla).pack()

oyna.mainloop()
# Label — yo'riqnoma matni
# Entry — foydalanuvchi yozadi
# Button bosilganda natija Label ga yoziladi

Eslab Qoling

• Entry — bir qatorli, Text — ko'p qatorli
• .get() qiymatni oladi (har doim string)
• .insert() yozadi, .delete() o'chiradi
• Checkbutton — bir nechta tanlov, Radiobutton — faqat bitta
• Forma = Label + Entry + Button

Misol 1: Ishlaydigan Forma

Misol 3: Checkbox va Radio Tanlovi

Bayroqchani belgilang va bittagina variantni tanlang — tugma bosilganda tanlovlaringiz ko'rsatiladi.
