Scope va O'zgaruvchilar

Modul 5 — Funksiyalar va modullar | Dars 27

Scope Nima?

Scope (ko'rinish doirasi) — bu o'zgaruvchi qayerda ko'rinadi va ishlatiladi degani. Har bir o'zgaruvchining o'z "hududi" bor: ba'zilari hamma joyda ishlaydi, ba'zilari esa faqat bitta funksiya ichida yashaydi.

Python'da ikkita asosiy scope mavjud: global (umumiy) va local (mahalliy).

Global Scope

Funksiyalardan tashqarida e'lon qilingan o'zgaruvchi — global o'zgaruvchi. U dasturning hamma joyida, jumladan funksiyalar ichida ham ko'rinadi.
ism = "Ali" # global

def salom():
    print("Salom, " + ism)

salom()
# ism funksiya ichida ham ko'rinadi
>>> Salom, Ali

Local Scope

Funksiya ichida e'lon qilingan o'zgaruvchi — local o'zgaruvchi. U faqat shu funksiya ichida yashaydi. Funksiya tugashi bilan u o'chiriladi va tashqarida ko'rinmaydi.
def hisobla():
    natija = 10 # local
    print(natija)

hisobla()
print(natija) # XATO!
# natija faqat funksiya ichida mavjud
>>> 10
NameError: name 'natija' is not defined

global Kalit So'zi

Odatda funksiya ichida global o'zgaruvchini o'zgartirib bo'lmaydi — Python yangi local o'zgaruvchi yaratadi. Agar funksiya ichidan global o'zgaruvchining qiymatini o'zgartirmoqchi bo'lsak, global kalit so'zidan foydalanamiz.
hisob = 0

def oshir():
    global hisob
    hisob = hisob + 1

oshir()
oshir()
print(hisob)
# global hisob ni o'zgartiramiz
>>> 2
⚠️ global bilan ehtiyot bo'ling!

global dan ko'p foydalanish kodni chalkash qiladi. Yaxshiroq usul — qiymatni return bilan qaytarish. global ni faqat zarur bo'lganda ishlating.

Nested (Ichma-ich) Funksiyalar

Funksiya ichida yana funksiya bo'lishi mumkin. Ichki funksiya tashqi funksiyaning o'zgaruvchilarini o'qiy oladi. Bu hodisa enclosing scope deb ataladi.
def tashqi():
    xabar = "Salom"
    def ichki():
        print(xabar)
    ichki()

tashqi()
# ichki, tashqi'ning xabar'ini ko'radi
>>> Salom

O'zgaruvchini Qidirish Tartibi (LEGB)

Python o'zgaruvchini quyidagi tartibda qidiradi:
BosqichQayerda
L — LocalJoriy funksiya ichida
E — EnclosingTashqi funksiya ichida
G — GlobalFayl darajasida
B — Built-inPython'ning o'zida (print, len)

Keng Tarqalgan Xato

❌ Xato: local o'zgaruvchiga tashqaridan kirish
def hisobla():
    yigindi = 5 + 3

hisobla()
print(yigindi) # NameError!

yigindi faqat funksiya ichida mavjud, tashqarida yo'q.

✓ To'g'ri: qiymatni return bilan qaytaring
def hisobla():
    yigindi = 5 + 3
    return yigindi

natija = hisobla()
print(natija)
>>> 8
Eslab Qoling

• Global — funksiyadan tashqarida, hamma joyda ko'rinadi
• Local — funksiya ichida, faqat o'sha yerda yashaydi
• Funksiya ichidagi o'zgaruvchi tashqarida ko'rinmaydi
• global kalit so'zi global o'zgaruvchini o'zgartiradi
• Eng yaxshi usul — qiymatni return bilan qaytarish

Misol 1: Local va Global

Funksiya ichidagi local o'zgaruvchiga tashqaridan kirib bo'lmaydi. Tugmani bosing va natijani ko'ring.

Misol 2: global Kalit So'zi

global bilan funksiya ichidan umumiy hisoblagichni oshiramiz. "Oshir" tugmasini bir necha marta bosing.

Misol 3: Scope Vizualizatori

Qaysi o'zgaruvchi qaysi doirada yashashini ko'ring.

Misol 4: Scope Viktorina

Quyidagi kod uchun print(x) tashqarida nima chiqaradi?

x = 5 # global
def f():
    x = 99 # local
f()
print(x)
