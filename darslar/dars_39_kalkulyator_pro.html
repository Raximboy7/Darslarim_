Loyiha: Kengaytirilgan kalkulyator

Modul 3 — Shartlar va Sikllar | Dars 39

Loyiha haqida

Bu loyihada biz uzluksiz ishlaydigan kalkulyator yaratamiz. Oddiy kalkulyatordan farqi — u bir marta hisoblab to'xtamaydi, balki foydalanuvchi "Chiqish" ni tanlaguncha menyuni qayta-qayta ko'rsatadi. Bu yerda modul 3 da o'rgangan hamma narsa birlashadi: while, if/elif/else va menyu.

Takrorlanuvchi menyu skeleti

Avval menyu while True ichida bo'lib, "5" tanlanganda chiqadigan skeletni yasaymiz: 

while True:
    print("\n=== KALKULYATOR PRO ===")
    print("1 - Qo'shish")
    print("2 - Ayirish")
    print("3 - Ko'paytirish")
    print("4 - Bo'lish")
    print("5 - Chiqish")

    tanlov = input("Tanlov: ")

    if tanlov == "5":
        print("Dastur tugadi. Xayr!")
        break
Menyu har aylanishda qayta chiqadi. "5" tanlansa break siklni to'xtatadi va dastur tugaydi.

To'liq kalkulyator dasturi

Endi barcha amallarni qo'shamiz. Nolga bo'lishni ham tekshiramiz, noto'g'ri tanlovni ham ushlaymiz: 

while True:
    print("\n=== KALKULYATOR PRO ===")
    print("1-Qo'shish  2-Ayirish  3-Ko'paytirish")
    print("4-Bo'lish   5-Chiqish")

    tanlov = input("Tanlov: ")

    if tanlov == "5":
        print("Xayr!")
        break

    if tanlov not in ["1", "2", "3", "4"]:
        print("Noto'g'ri tanlov!")
        continue

    a = float(input("1-son: "))
    b = float(input("2-son: "))

    if tanlov == "1":
        print("Natija =", a + b)
    elif tanlov == "2":
        print("Natija =", a - b)
    elif tanlov == "3":
        print("Natija =", a * b)
    elif tanlov == "4":
        if b == 0:
            print("Xato: nolga bo'lib bo'lmaydi!")
        else:
            print("Natija =", a / b)
continue noto'g'ri tanlovda son so'ramasdan menyuga qaytadi. Nolga bo'lish ichki if bilan himoyalangan.
>>> === KALKULYATOR PRO ===
>>> Tanlov: 1
>>> 1-son: 12
>>> 2-son: 8
>>> Natija = 20.0
>>> Tanlov: 5
>>> Xayr!

Kod qanday ishlaydi?

1
Menyu ko'rsatiladi

Har aylanishda variantlar qayta chiqadi.

2
Tanlov tekshiriladi

"5" bo'lsa chiqadi, noto'g'ri bo'lsa continue bilan qaytadi.

3
Sonlar so'raladi

Amal uchun ikkita son kiritiladi.

4
Natija chiqadi

if/elif tanlangan amalni bajaradi va sikl qaytadan boshlanadi.

Amaliyot va xulosa

Loyihani rivojlantiring

1. Daraja (**) va qoldiq (%) amallarini qo'shing.
2. Oxirgi natijani eslab qolib, keyingi amalda ishlatish imkonini bering.
3. Barcha bajarilgan amallar tarixini ro'yxatga yozib boring.

Nimalarni o'rgandik

• while True menyuni uzluksiz takrorlaydi.
• break chiqish uchun, continue noto'g'ri tanlovni qaytarish uchun ishlatiladi.
• if/elif/else amallarni tanlaydi va xatolardan (nolga bo'lish) himoya qiladi.
