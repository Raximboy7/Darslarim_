scikit-learn Nima?

scikit-learn (qisqacha sklearn) — bu Python uchun eng mashhur mashinaviy o'rganish (ML) kutubxonasi. U yordamida kompyuterga ma'lumotlardan qoidalarni o'rgatamiz va keyin yangi holatlar uchun bashorat qildiramiz — masalan uy o'lchamiga qarab narxini taxmin qilish.
Nima uchun sklearn?

• Yuzlab tayyor modellar (regressiya, klassifikatsiya, klasterlash)
• Hamma modellar bir xil sodda API (interfeys) bilan ishlaydi
• Bepul, ochiq kodli va juda yaxshi hujjatlangan

O'rnatish

scikit-learn standart Pythonda yo'q, uni pip orqali o'rnatamiz. Buyruqni terminalga (komandalar qatoriga) yozasiz:
pip install scikit-learn
# Terminalga yoziladi (Python ichiga emas)
Kodda esa nomi sklearn bo'ladi. Tekshirish:
import sklearn
print(sklearn.__version__)
# Versiya chiqsa — o'rnatildi
>>> 1.5.0

ML Ish Oqimi (4 bosqich)

Deyarli har qanday ML masalasi bir xil tartibda yechiladi. Buni ish oqimi (workflow) deymiz:
BosqichNima qilamizKod
1. Ma'lumotX (belgilar) va y (javob) tayyorlaymizX, y
2. Model tanlashMos model klassini olamizmodel = ModelClass()
3. O'rgatish (fit)Modelga ma'lumotni ko'rsatamizmodel.fit(X, y)
4. Bashorat (predict)Yangi ma'lumot uchun javob olamizmodel.predict(X_new)

Umumiy API (eng muhim qism)

sklearn ning go'zalligi shundaki — barcha modellar bir xil 3 ta amaldan iborat. Modelni o'zgartirsangiz ham kod deyarli o'zgarmaydi:
from sklearn.linear_model import LinearRegression

model = LinearRegression()  # 1) model yaratish
model.fit(X, y)            # 2) o'rgatish
natija = model.predict(X_new) # 3) bashorat
# fit() o'rgatadi, predict() bashorat qiladi
Sehrli formula

model = ModelClass() → yarat
model.fit(X, y) → o'rgat
model.predict(X_new) → bashorat qil

X va y Formati

sklearn ikki narsa kutadi:
• X — belgilar (features): kirish ma'lumotlari, har doim 2 o'lchovli jadval (qatorlar = misollar, ustunlar = belgilar).
• y — javob (label/target): biz bashorat qilmoqchi bo'lgan natija, 1 o'lchovli ro'yxat.
# Uy o'lchami (m2) -> narx (ming $)
X = [[50], [80], [120]]  # 3 misol, 1 belgi
y = [100, 160, 240]     # har biriga narx
# X — ichma-ich ro'yxat (2D), y — oddiy ro'yxat (1D)
Diqqat

X har doim ikki kvadrat qavs bilan: [[50], [80]]. Bitta misol ham 2 o'lchovli bo'lishi kerak: [[100]], oddiy [100] emas!

To'liq Misol: O'lcham → Narx

from sklearn.linear_model import LinearRegression

X = [[50], [80], [120]] # o'lcham
y = [100, 160, 240]   # narx

model = LinearRegression()
model.fit(X, y)

print(model.predict([[100]]))
# 100 m2 uy uchun narx bashorati
>>> [200.]
Eslab Qoling

• pip install scikit-learn — o'rnatish, kodda nomi sklearn
• Ish oqimi: ma'lumot → model → fit → predict
• X — belgilar (2D), y — javob (1D)
• Hamma modellar bir xil API: fit() va predict()

Misol 1: fit → predict Oqimi

Misol 2: X / y Ma'lumot Formati

Quyiga belgi (X) va javob (y) qiymatlarini kiriting. Kod jadvalga aylanib, sklearn kutadigan format ko'rinadi.

Misol 3: Oddiy Bashorat (O'lcham → Narx)

Model bizning ma'lumotdan oddiy qoidani o'rgangan: narx = 2 × o'lcham. O'lchamni kiriting va predict natijasini ko'ring.

Misol 4: Model Almashtirish

API bir xil bo'lgani uchun model klassini almashtirsangiz, qolgan kod o'zgarmaydi. Modelni tanlang va kod qanday ko'rinishini ko'ring.
