mypy nima?

mypy — Python kodidagi type hints larni tekshiruvchi dastur (static type checker). Kodni ishga tushirmasdan tur xatolarini topadi. Python o'zi type hints ni tekshirmaydi — mypy aynan shu bo'shliqni to'ldiradi.

Natijada xatolar dastur foydalanuvchiga yetib bormasdan, hali kod yozish paytida topiladi.

O'rnatish

mypy pip orqali o'rnatiladi: 

pip install mypy

mypy --version   # o'rnatilganini tekshirish
Bu terminal (buyruq qatori) buyrug'i, Python kodi emas. Bir marta o'rnatiladi.

Ishlatish — xatoni topish

Faraz qilaylik hisob.py faylida quyidagi kod bor: 

# hisob.py
def qoshish(a: int, b: int) -> int:
    return a + b

natija = qoshish(5, "salom")   # XATO: str ni int deb bermoqchimiz
Bu kod Python'da ishga tushganda runtime xato beradi. Lekin mypy uni oldindan, ishga tushirmasdan topadi.

Terminalda tekshiramiz: 

mypy hisob.py
Fayldagi barcha type hints ni tekshiradi.
hisob.py:5: error: Argument 2 to "qoshish" has incompatible type "str"; expected "int" [arg-type] Found 1 error in 1 file (checked 1 source file)

To'g'ri kod — xatosiz natija

# hisob.py — to'g'irlangan
def qoshish(a: int, b: int) -> int:
    return a + b

natija = qoshish(5, 3)   # endi ikkalasi ham int
Endi mypy hech qanday xato topmaydi.
Success: no issues found in 1 source file

Ko'p uchraydigan xatolar

Holatmypy xabari
Noto'g'ri argument turiincompatible type ... expected ...
None ustida amalItem "None" has no attribute ...
Qaytish turi mos emasIncompatible return value type
E'lon qilinmagan o'zgaruvchiName "..." is not defined
None xatosi misoli Agar funksiya Optional[str] qaytarsa va siz uni None ga tekshirmasdan .upper() chaqirsangiz — mypy ogohlantiradi. Bu haqiqiy nosozliklarni oldini oladi.

Foydali flaglar

mypy loyiha/          # butun papkani tekshirish
mypy --strict fayl.py # qattiq rejim — hamma tekshiruv yoqiladi
mypy --ignore-missing-imports fayl.py  # kutubxona turlari yo'q bo'lsa
--strict yangi loyiha uchun juda tavsiya etiladi: type hints yo'q joylarni ham topadi.

Amaliyot

Mashqlar

1. Type hints li kichik fayl yozing va unga ataylab tur xatosi kiriting, mypy bilan toping.
2. Optional[str] qaytaruvchi funksiyani None tekshiruvsiz ishlatib, xatoni ko'ring.
3. mypy --strict ni ishga tushirib, qo'shimcha ogohlantirishlarni kuzating.

Xulosa

Esda saqlang

• mypy — type hints ni ishga tushirmasdan tekshiradi.
• O'rnatish: pip install mypy.
• Ishlatish: mypy fayl.py.
• Xatolarni foydalanuvchiga yetib bormasdan topadi.
• --strict — jiddiy loyihalar uchun eng yaxshi tanlov.
