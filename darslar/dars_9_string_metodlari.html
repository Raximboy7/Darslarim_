String Metodi Nima?

Metod — bu stringga biriktirilgan maxsus funksiya. U matn.metod() ko'rinishida — nuqta orqali chaqiriladi. Metodlar matnni o'zgartirmaydi, balki yangi matn qaytaradi.
matn = "salom"
print(matn.upper())
# Katta harfga aylantirish
>>> SALOM

Harflarni O'zgartirish

MetodVazifasiMisol → Natija
.upper()Barchasini katta"ali" → ALI
.lower()Barchasini kichik"ALI" → ali
.title()Har so'z bosh harfi katta"ali vali" → Ali Vali
.capitalize()Faqat birinchi harf katta"ali vali" → Ali vali
.strip()Chetdagi bo'sh joylarni olib tashlaydi" ali " → ali

Qidirish va Almashtirish

MetodVazifasiNatija
.replace("a","o")So'zni almashtiradi"salom" → "solom"
.find("o")Pozitsiyani topadi"salom" → 3
.count("a")Nechta marta borligini sanaydi"banana" → 3
.startswith("sa")... bilan boshlanadimi?True / False
.endswith("om")... bilan tugaydimi?True / False

Bo'lish va Birlashtirish

.split() — matnni bo'laklarga bo'ladi

matn = "Ali,Vali,Guli"
print(matn.split(","))
# Vergul bo'yicha bo'lish
>>> ['Ali', 'Vali', 'Guli']

.join() — bo'laklarni birlashtiradi

ismlar = ["Ali", "Vali"]
print("-".join(ismlar))
# "-" bilan birlashtirish
>>> Ali-Vali

Indeksatsiya (Indexing)

Har bir harfning raqami (indeksi) bor. Sanash 0 dan boshlanadi. Manfiy indeks oxiridan sanaydi.
s = "Salom"
print(s[0], s[-1])
# Birinchi va oxirgi harf
>>> S m

Kesib Olish (Slicing)

s[boshi:oxiri] — boshidan oxirigacha (oxiri kirmaydi) kesib oladi.
KodNatijaIzoh
"Salom"[0:3]"Sal"0,1,2 indekslar
"Salom"[2:]"lom"2 dan oxirigacha
"Salom"[:3]"Sal"boshidan 3 gacha
"Salom"[::2]"Slm"har 2-harf
Eslab Qoling

• Metod chaqirish: matn.metod()
• Indeks 0 dan boshlanadi, -1 — oxirgi harf
• .split() bo'ladi, .join() birlashtiradi
• Metodlar matnni o'zgartirmaydi — yangi matn qaytaradi
