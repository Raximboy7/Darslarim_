Regular Expressions (Regex)

Modul 13 — Regex va Type Hints | Dars 74

Regex Nima?

Regular Expression (qisqacha regex) — bu matn ichida naqsh (pattern) qidirish uchun maxsus til. Naqsh — bu qidirayotgan matningiz qanday ko'rinishini tasvirlovchi qoida.
Masalan: "matn ichidagi barcha telefon raqamlarini top", yoki "elektron pochta manzilini tekshir", yoki "faqat raqamlardan iborat so'zlarni ajrat" — bularning hammasi regex bilan juda oson hal qilinadi. Oddiy .find() yoki in bilan bunday murakkab qidiruvlar qiyin bo'lar edi.

re Moduli

Pythonda regex bilan ishlash uchun re nomli modul bor. Uni ishlatishdan oldin import qilish kerak.
import re

matn = "Mening yoshim 25 da"
natija = re.findall(r"\d+", matn)
print(natija)
# re modulini chaqiramiz
# \d+ — bir yoki bir nechta raqamni qidiradi
# findall barcha mosliklarni ro'yxatda qaytaradi
>>> ['25']
r"..." nima?

Naqshni har doim r"..." (raw string) ko'rinishida yozamiz. Bu Pythonga \ belgisini maxsus emas, balki oddiy belgi sifatida o'qishni aytadi. Shuning uchun regexda doim r harfi bilan boshlang.

Metasimvollar (Metacharacters)

Regexda ba'zi belgilar maxsus ma'noga ega — ular metasimvol deyiladi. Quyida eng muhimlari:
BelgiMa'nosiMisol
.Har qanday bitta belgia.c → "abc", "axc"
*Oldingisi 0 yoki ko'p martaab* → "a", "ab", "abbb"
+Oldingisi 1 yoki ko'p martaab+ → "ab", "abbb"
?Oldingisi 0 yoki 1 martaab? → "a", "ab"
[]Belgilar to'plamidan biri[abc] → "a" yoki "b" yoki "c"
^Matnning boshi^Sa → "Sa" bilan boshlanadi
$Matnning oxiriom$ → "om" bilan tugaydi
|Yoki (biri yoki ikkinchisi)it|mushuk → "it" yoki "mushuk"

Maxsus Naqshlar: \d \w \s

Tez-tez kerak bo'ladigan belgilar uchun qisqa yozuvlar bor:
NaqshMa'nosiMisol
\dRaqam (0-9)"7", "0", "9"
\wHarf, raqam yoki "_""a", "B", "5", "_"
\sBo'sh joy (probel, tab)" "
\d+Bir yoki ko'p raqam"25", "100"
\w+Bir yoki ko'p harf/raqam"salom", "abc123"
import re
matn = "Telefon: 998 90 123"
raqamlar = re.findall(r"\d+", matn)
print(raqamlar)
# Barcha raqam guruhlarini topadi
>>> ['998', '90', '123']
Eslab Qoling

• Regex — matn ichida naqsh qidirish tili
• Avval import re qiling
• Naqshni doim r"..." ko'rinishida yozing
• \d — raqam, \w — harf/raqam, \s — bo'sh joy
• + — 1 yoki ko'p, * — 0 yoki ko'p, ? — 0 yoki 1

Misol 1: Naqsh Sinovchisi

O'zingiz naqsh va matn yozing — mos kelgan qismlar yashil rangda belgilanadi. (Brauzerda JavaScript RegExp yordamida ishlaydi — Python re moduliga juda o'xshash.)

Misol 2: Metasimvol Lug'ati

Tugmani bosing — har bir metasimvol nimani anglatishini va misolini ko'ring.

Misol 3: Tayyor Naqshlarni Sinash

Oddiy naqshlardan birini tanlang — u sizning matningizda nimani topishini ko'ring.
