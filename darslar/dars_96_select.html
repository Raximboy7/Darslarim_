SELECT Nima?

SELECT — bu ma'lumotlar bazasidan ma'lumot o'qish uchun ishlatiladigan eng muhim SQL buyrug'i. U jadvaldan kerakli qatorlarni va ustunlarni olib, sizga ko'rsatadi. Jadvalga hech narsa qo'shmaydi va o'zgartirmaydi — faqat o'qiydi.
Tasavvur qiling

Jadval — bu Excel jadvalidek. SELECT esa "menga shu ma'lumotlarni ko'rsat" degan so'rovdir. Qaysi ustunlarni, qaysi qatorlarni, qanday tartibda ko'rishni siz hal qilasiz.

SELECT * — Hamma Ustunlar

Yulduzcha * — "barcha ustunlar" degani. SELECT * FROM jadval jadvaldagi hamma narsani qaytaradi.
SELECT * FROM o'quvchilar;
# Barcha ustun va barcha qatorni o'qiydi
>>> 5 ta qator, hamma ustun bilan qaytadi

Faqat Kerakli Ustunlar

* o'rniga ustun nomlarini vergul bilan yozsangiz, faqat o'sha ustunlar qaytadi. Bu tezroq va tartibliroq.
SELECT ism, ball FROM o'quvchilar;
# Faqat ism va ball ustunlari chiqadi
>>> Ali 85
>>> Vali 72
>>> Guli 91
>>> ...

WHERE — Shart Qo'yish

WHERE bilan faqat shartni qanoatlantiradigan qatorlarni olasiz. Bu — filtrlash. Masalan, faqat Toshkentdagi o'quvchilarni ko'rish.
SELECT * FROM o'quvchilar
WHERE shahar = 'Toshkent';
# Faqat shahri Toshkent bo'lganlar
>>> Ali 20 Toshkent 85
>>> Guli 19 Toshkent 91
Diqqat

Matn qiymatlar 'tirnoq' ichida yoziladi ('Toshkent'), sonlar esa tirnoqsiz (20).

LIKE va % belgisi

% — "har qanday harflar" degani.
• 'A%' — A bilan boshlanadigan (Ali)
• '%n' — n bilan tugaydigan (Hasan)
• '%al%' — ichida "al" bor (Ali, Vali, Laylo)

ORDER BY — Tartiblash

ORDER BY natijani biror ustun bo'yicha tartiblaydi. ASC — o'sish (kichikdan kattaga, bu standart), DESC — kamayish (kattadan kichikka).
SELECT ism, ball FROM o'quvchilar
ORDER BY ball DESC;
# Eng yuqori balldan boshlab tartiblaydi
>>> Guli 91
>>> Laylo 88
>>> Ali 85
>>> Vali 72
>>> Hasan 68

LIMIT — Sonni Cheklash

LIMIT qaytadigan qatorlar sonini cheklaydi. Masalan, eng yaxshi 3 o'quvchini ko'rish uchun ORDER BY bilan birga ishlatamiz.
SELECT ism, ball FROM o'quvchilar
ORDER BY ball DESC
LIMIT 3;
# Faqat eng yuqori 3 ta natija
>>> Guli 91
>>> Laylo 88
>>> Ali 85

DISTINCT — Takrorlanmaydiganlar

DISTINCT takroriy qiymatlarni olib tashlaydi va har birini faqat bir marta ko'rsatadi. Masalan, jadvalda qaysi shaharlar borligini bilish uchun.
SELECT DISTINCT shahar FROM o'quvchilar;
# Toshkent 2 marta bor, lekin 1 marta chiqadi
>>> Toshkent
>>> Samarqand
>>> Buxoro
