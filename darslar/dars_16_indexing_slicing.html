Indexing — Indeks Orqali Kirish

Indexing — bu element raqami (indeksi) orqali bitta elementga kirish. String va ro'yxatdagi har bir element o'z raqamiga ega va sanash 0 dan boshlanadi. Birinchi element — indeks 0, ikkinchisi — 1 va hokazo.
s = "Salom"
print(s[0])
print(s[2])
# Birinchi harf

Manfiy Indeks (oxiridan sanash)

Manfiy indeks oxiridan sanaydi. -1 — eng oxirgi element, -2 — undan oldingisi. Bu oxirgi elementni topishning eng qulay usuli.
s = "Salom"
print(s[-1])
print(s[-2])
# Oxirgi harf

# Oxirdan ikkinchi
>>> m
>>> o

Ro'yxatda Indexing

Ro'yxat (list) ham xuddi string kabi indekslanadi — element raqami orqali kirasiz.
mevalar = ["olma", "banan", "uzum"]
print(mevalar[0])
print(mevalar[-1])
# Birinchi meva

# Oxirgi meva
>>> olma
>>> uzum

Slicing — Kesib Olish

s[boshi:oxiri] — boshidan oxirigacha bo'lakni kesib oladi. Muhimi: oxiri kirmaydi (boshi kiradi, oxiri kirmaydi). Natija — yangi string yoki ro'yxat bo'ladi.
KodNatijaIzoh
"Salom"[0:3]"Sal"0, 1, 2 indekslar
"Salom"[1:]"alom"1 dan oxirigacha
"Salom"[:3]"Sal"boshidan 3 gacha
"Salom"[:-1]"Salo"oxirgisidan tashqari
"Salom"[:]"Salom"to'liq nusxa
Bo'sh qoldirilsa

Boshi yozilmasa — 0 dan, oxiri yozilmasa — oxirigacha olinadi. Shuning uchun s[:] butun nusxani beradi.

Qadam Bilan Kesish (step)

To'liq ko'rinish: s[boshi:oxiri:qadam]. Uchinchi son — qadam (step) — nechta elementdan keyin sakrashni bildiradi.
s = "Salom"
print(s[::2])
# Har 2-harf: S, l, m
>>> Slm
KodNatijaIzoh
"abcdef"[::2]"ace"har 2-harf
"abcdef"[1::2]"bdf"1 dan, har 2-harf
"abcdef"[0:5:2]"ace"0 dan 5 gacha, qadam 2

Teskari Aylantirish [::-1]

Manfiy qadam -1 elementlarni oxiridan boshiga qarab oladi — ya'ni stringni yoki ro'yxatni teskari aylantiradi. Bu eng mashhur Python ixchamligi.
s = "Salom"
print(s[::-1])
# Teskari: m, o, l, a, S
>>> molaS

Keng Tarqalgan Xato

❌ Xato: indeks chegaradan tashqarida
s = "Salom" # 5 ta harf, indekslar 0..4
print(s[5]) # XATO!
>>> IndexError: string index out of range

5 ta harfning indekslari 0 dan 4 gacha. Indeks 5 mavjud emas.

✓ To'g'ri: slicing chegaradan qo'rqmaydi
s = "Salom"
print(s[-1]) # oxirgi harf
print(s[0:99]) # ortiqcha son xato bermaydi
>>> m
>>> Salom

Indexing chegaradan chiqsa xato beradi, lekin slicing ortiqcha sonni muloyim qabul qiladi.

Eslab Qoling

• Indeks 0 dan boshlanadi, -1 — oxirgi element
• Slicing: [boshi:oxiri] — oxiri KIRMAYDI
• Qadam: [boshi:oxiri:qadam], [::2] — har 2-element
• [::-1] — teskari aylantiradi

Misol 1: Indeks Vizualizatori

Manfiy indeks ham ishlaydi: -1 oxirgi harfni beradi.

Misol 2: Kesim [boshi:oxiri]

Bo'sh qoldiring: boshi yo'q = 0 dan, oxiri yo'q = oxirigacha.

Misol 3: Qadamli Kesim [boshi:oxiri:qadam]

Misol 4: Teskari Aylantirish [::-1]

Palindromni ham tekshiradi (oldidan ham, orqasidan ham bir xil o'qiladimi?).

Misol 5: Ro'yxatni Indekslash

Ro'yxat ham string kabi indekslanadi va kesiladi.
