print() Funksiyasi

Modul 1 — Asoslar | Dars 3

print() Nima?

print() — bu ekranga matn yoki ma'lumot chiqarish uchun ishlatiladi. Dasturchining eng ko'p ishlatiladigan funksiyalaridan biri. Natijalarni ko'rish uchun lozim!

Sodda qo'llanma:

print("Salom!")
>>> Salom!

print() ning Turli Usullari

1. Matnni Chiqarish

print("Assalomu alaikum")
# Matnni chiqar
>>> Assalomu alaikum

2. Raqamni Chiqarish

print(42)
# Raqamni chiqar
>>> 42

3. Bir Nechta Narsani Chiqarish

print("Ism:", "Ali")
# Ikkita narsani virgul bilan
>>> Ism: Ali

4. O'zgaruvchi Bilan

ism = "Bobur"
print(ism)
# O'zgaruvchini chiqar
>>> Bobur

5. Hisoblash Natijasini Chiqarish

print(5 + 3)
# 5 + 3 = 8 chiqadi
>>> 8

print() Parametrlari

print() funksiyasida qo'shimcha parametrlar mavjud. Bular natijaning qanday ko'rinishini boshqaradi.
Parametr Ma'nosi Misol Natija
sep Ajratuvchi belgisi print("a", "b", sep="-") a-b
end Oxirgi belgi print("Salom", end="!") Salom!

sep Parametri Misollar:

print("A", "B", "C")
# Standart: bo'sh joy
>>> A B C
print("A", "B", "C", sep="-")
# Ajratuvchi: "-"
>>> A-B-C

end Parametri Misollar:

print("Salom")
print("Dunyo")
# Har biri yangi qatorida
>>> Salom
>>> Dunyo
print("A", end="")
print("B", end="")
# Yangi qatorga o'tmaydi
>>> AB

print() Xatolaridan Qutulish

❌ Xato 1: Tirnoqsiz String
print(Salom) # XATO!

Xato: String tirnoq ichida bo'lishi kerak!
To'g'ri: print("Salom")

❌ Xato 2: Qavsni Unutib Qoldirish
print "Salom" # XATO!

Xato: print() qavslari kerak!
To'g'ri: print("Salom")

❌ Xato 3: Tirnoq Turi
print('Salom"') # XATO!

Xato: Ochuvchi va yopuvchi tirnoq bir xil bo'lishi kerak!
To'g'ri: print("Salom") yoki print('Salom')

Eslaydigan Narsa

print() — ekranga chiqarish uchun.
String — tirnoq ichida.
Raqam — tirnoqsiz.
Parametrlar — qavs ichida.

Misol 1: Oddiy Matn

Misol 2: Bir Nechta Narsani Qo'shish

Misol 3: Raqamlarni Qo'shish

Misol 4: sep Parametri

Misol: A, B, C ni kiritilgan ajratuvchi bilan chiqaradi

Ko'p Qatorli print()

Namuna 1: Har biri alohida qator
print("Qator 1")
print("Qator 2")
print("Qator 3")
>>> Qator 1
>>> Qator 2
>>> Qator 3
Namuna 2: Bir qatorga
print("A", end="")
print("B", end="")
print("C")
>>> ABC
