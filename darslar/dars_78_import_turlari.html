Modul 7 — Modullar va Fayllar | Dars 78

import ning turli usullari

Python'da modulni yuklashning bir necha usuli bor. Har birining o'z o'rni va afzalligi mavjud. Bu darsda hammasini ko'rib chiqamiz.
UsulMa'nosi
import modulButun modulni yuklaydi
from modul import nomFaqat kerakli qismini yuklaydi
import modul as mModulga qisqa nom beradi
from modul import *Hamma narsani yuklaydi (tavsiya etilmaydi)

1. import modul

Butun modul yuklanadi. Ichidagilarga modul nomi orqali murojaat qilinadi.

import math

print(math.sqrt(16))
print(math.pi)
# math.sqrt, math.pi — har safar "math." qo'shib yozamiz. Qaysi moduldan kelgani aniq ko'rinadi.
>>> 4.0
>>> 3.141592653589793

2. from modul import nom

Faqat kerakli funksiya yoki o'zgaruvchini olamiz. Endi modul nomini yozish shart emas.

from math import sqrt, pi

print(sqrt(25))
print(pi)
# sqrt() to'g'ridan-to'g'ri yoziladi. Qisqa, lekin nomlar to'qnashib qolishi mumkin (masalan o'z sqrt funksiyangiz bo'lsa).
>>> 5.0
>>> 3.141592653589793

3. import modul as nom (taxallus)

Uzun modul nomiga qisqa taxallus beramiz. Ma'lumotlar bilan ishlashda juda ko'p uchraydi.

import datetime as dt

hozir = dt.datetime.now()
print(hozir.year)
# datetime o'rniga qisqa dt yozamiz. Mashhur misollar: import numpy as np, import pandas as pd.

4. from modul import * (ehtiyot bo'ling!)

Moduldagi hamma narsani nomsiz yuklaydi.

from math import *

print(sqrt(9))
print(floor(3.7))
# Ishlaydi, lekin qaysi nom qayerdan kelganini bilib bo'lmaydi va nomlar to'qnashadi.
Nega * dan qochish kerak?

Kodni o'qish qiyinlashadi — floor() qayerdan kelgan? Bilib bo'lmaydi. Bir modulning nomi boshqasini bosib ketishi mumkin. Amaliyotda deyarli ishlatilmaydi.

Xulosa

Qaysi birini tanlash?

• Odatda import modul — eng tushunarli
• Bir-ikki narsa kerak bo'lsa from modul import nom
• Uzun nom bo'lsa as bilan taxallus bering
• import * dan iloji boricha qoching
