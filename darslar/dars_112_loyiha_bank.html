Loyiha: Bank hisobi tizimi

Modul 9 — OOP | Dars 112

Loyiha maqsadi

Bu loyihada oddiy bank hisobi tizimi yozamiz. Foydalanuvchi pul kirim qilishi (deposit), yechishi (withdraw) va balansni ko'rishi mumkin. Bir vaqtning o'zida inkapsulyatsiya (balansni himoyalash) ni ham qo'llaymiz.
Talablar 📝

• Balans tashqaridan to'g'ridan-to'g'ri o'zgartirilmasin (yashirin bo'lsin)
• Deposit — faqat musbat summa qabul qilsin
• Withdraw — balansdan ko'p pul yechilmasin
• Har amaldan keyin yangi balans ko'rsatilsin

1-qadam: klass va konstruktor

Hisob klassini yaratamiz. Balansni __balans (yashirin) qilib saqlaymiz.
class Hisob:
    def __init__(self, egasi, boshlang'ich=0):
        self.egasi = egasi
        self.__balans = boshlang'ich
# egasi — hisob egasining ismi. __balans — yashirin, himoyalangan pul

2-qadam: deposit va withdraw metodlari

Endi pul kirim va chiqim metodlarini qo'shamiz — ichida tekshiruvlar bilan.
    def deposit(self, summa):
        if summa <= 0:
            print("Summa musbat bo'lsin!")
            return
        self.__balans += summa
        print(f"+{summa}. Balans: {self.__balans}")

    def withdraw(self, summa):
        if summa > self.__balans:
            print("Balans yetarli emas!")
            return
        self.__balans -= summa
        print(f"-{summa}. Balans: {self.__balans}")
# deposit — musbat summani qo'shadi. withdraw — balansdan ko'p bo'lsa rad etadi

3-qadam: balansni ko'rish (property)

Balansni faqat o'qish uchun @property ishlatamiz — tashqaridan o'zgartirib bo'lmaydi, lekin ko'rish mumkin.
    @property
    def balans(self):
        return self.__balans
# hisob.balans → o'qiladi, lekin hisob.balans = 999 ishlamaydi

4-qadam: to'liq kod va ishlatish

Barcha qismlarni birlashtiramiz va tizimni sinab ko'ramiz.
class Hisob:
    def __init__(self, egasi, boshlang'ich=0):
        self.egasi = egasi
        self.__balans = boshlang'ich

    def deposit(self, s):
        if s > 0: self.__balans += s

    def withdraw(self, s):
        if 0 < s <= self.__balans: self.__balans -= s

    @property
    def balans(self):
        return self.__balans

h = Hisob("Ali", 100)
h.deposit(50)
h.withdraw(30)
print(h.egasi, h.balans)
# 100 + 50 - 30 = 120. Balans himoyalangan, faqat metodlar orqali o'zgaradi
>>> Ali 120

Xulosa

Ushbu loyihada qo'llandi

• Inkapsulyatsiya — __balans yashirin va himoyalangan
• Metodlar — deposit, withdraw bilan mantiq nazorati
• @property — balansni faqat o'qishga ruxsat
• Tekshiruvlar — manfiy summa va yetmagan balansdan himoya
