Default parametrlar

Modul 6 — Funksiyalar | Dars 66

Default parametr nima?

Default (standart) parametr — bu funksiya chaqirilganda qiymat berilmasa, avtomatik ishlatiladigan qiymat. Funksiyani e'lon qilishda def f(x=10) ko'rinishida beriladi.

Bu funksiyani moslashuvchan qiladi: ba'zi argumentlarni majburiy emas, ixtiyoriy qilib qo'yasiz.

Oddiy misol

Salomlashuvchi funksiya. Agar til berilmasa, o'zbekcha salom beriladi: 

def salom(ism, til="uz"):
    if til == "uz":
        return f"Salom, {ism}!"
    return f"Hello, {ism}!"

print(salom("Ali"))
print(salom("Ali", "en"))
# til=uz — standart qiymat, ikkinchisida bekor qilindi
>>> Salom, Ali!
>>> Hello, Ali!

Tartib qoidasi

Muhim qoida

Default parametrlar har doim oddiy parametrlardan keyin yozilishi kerak. Aks holda Python SyntaxError beradi. 

def hisobla(a, b, amal="+"):
    if amal == "+":
        return a + b
    return a - b

print(hisobla(5, 3))
print(hisobla(5, 3, "-"))
# a, b — majburiy; amal — ixtiyoriy
>>> 8
>>> 2

Diqqat: o'zgaruvchan default xatosi

Default qiymat sifatida ro'yxat yoki lug'at ishlatmang! U bir marta yaratiladi va har chaqiruvda bir xil obyekt qayta ishlatiladi: 

# NOTO'G'RI
def qosh(element, royxat=[]):
    royxat.append(element)
    return royxat

# TO'G'RI
def qosh2(element, royxat=None):
    if royxat is None:
        royxat = []
    royxat.append(element)
    return royxat
# None ishlatib, ichida yangi ro'yxat yaratish xavfsiz

Amaliyot

O'zingiz sinab ko'ring

1. daraja(son, n=2) — sonni n-darajaga ko'taruvchi funksiya yozing (standart kvadrat).
2. Uni daraja(5) va daraja(2, 3) bilan sinang.

>>> daraja(5) natijasi: 25
>>> daraja(2, 3) natijasi: 8

Xulosa

Eslab qoling

• Shakli: def f(x=default_qiymat)
• Default parametrlar oddiylardan keyin turadi
• Chaqiruvda qiymat berilsa, standart qiymat bekor qilinadi
• Default sifatida ro'yxat/lug'at emas, None ishlating
