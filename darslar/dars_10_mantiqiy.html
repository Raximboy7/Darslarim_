Mantiqiy Operator Nima?

Mantiqiy operatorlar bir nechta shartni birlashtirib, ulardan bitta umumiy javob — True (rost) yoki False (yolg'on) — chiqarish uchun ishlatiladi. Pythonda uchta mantiqiy operator bor: and or not.
OperatorMa'nosiQachon True bo'ladi
andVAikkala shart ham True bo'lsa
orYOKIbiror shart True bo'lsa
notEMASshartni teskari qiladi

and — VA operatori

and faqat ikkala shart ham True bo'lganda True qaytaradi. Bittasi ham False bo'lsa — natija False.
print((5 > 3) and (3 > 1))
# 5>3 True, 3>1 True → True
>>> True
print((5 > 3) and (1 > 3))
# 5>3 True, 1>3 False → False
>>> False

or — YOKI operatori

or kamida bitta shart True bo'lsa True qaytaradi. Faqat ikkala shart ham False bo'lgandagina natija False bo'ladi.
print((5 > 3) or (3 > 5))
# 5>3 True, 3>5 False → True
>>> True
print((1 > 3) or (2 > 5))
# ikkalasi ham False → False
>>> False

not — EMAS operatori

not shartning natijasini teskari qiladi: True ni False ga, False ni True ga aylantiradi.
print(not (5 > 3))
# 5>3 True, not True → False
>>> False

Bir Nechta Shartni Birlashtirish

Operatorlarni ketma-ket ishlatib, ikkitadan ko'p shartni birlashtirish mumkin. and bilan bog'langanda hammasi True bo'lishi shart.
print((5 > 3) and (3 > 1) and (2 > 0))
# uchchala shart ham True → True
>>> True
Foydali misol: yosh oralig'i

18 <= yosh and yosh <= 65 — bu ifoda yosh 18 dan 65 gacha bo'lgan oraliqdami yoki yo'qligini tekshiradi.

Keng Tarqalgan Xato

❌ Xato: har bir tomonni to'liq yozmaslik
print(yosh == 18 or 19) # XATO!

or ning ikkala tomoni ham to'liq shart bo'lishi kerak. Bu yerda 19 alohida son sifatida har doim True deb baholanadi — natija noto'g'ri bo'ladi.

✓ To'g'ri: ikkala shartni alohida yozish
print(yosh == 18 or yosh == 19)
>>> True # yosh = 18 bo'lsa
Eslab Qoling

• and — ikkala shart ham True bo'lsa True
• or — kamida bittasi True bo'lsa True
• not — natijani teskari qiladi
• Natija har doim True yoki False bo'ladi
