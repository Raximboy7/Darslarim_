Stringlar — kirish

Modul 1 — Asoslar | Dars 6

String nima?

String (matn) — bu harflar, so'zlar va jumlalardan iborat ma'lumot turi. Ismlar, xabarlar, manzillar — bularning hammasi string. Python'da string qo'shtirnoq ichiga yoziladi.
ism = "Ali"
shahar = 'Toshkent'
print(ism)
print(shahar)
# matn qo'shtirnoq ichida yoziladi
>>> Ali
>>> Toshkent

Bitta va ikkita qo'shtirnoq

Python'da 'yakka' va "juft" qo'shtirnoq bir xil ishlaydi. Farqi: agar matn ichida qo'shtirnoq bo'lsa, ikkinchi turini ishlatasiz.
gap1 = "It's a book"
gap2 = 'U "salom" dedi'
print(gap1)
print(gap2)
# apostrof bor -> juft qo'shtirnoq ishlatamiz
# juft qo'shtirnoq bor -> yakka ishlatamiz
>>> It's a book
>>> U "salom" dedi
Maslahat Odatda bitta uslubni tanlab, doim ishlatgan ma'qul. Ko'pchilik "juft qo'shtirnoq" ni afzal ko'radi.

len() — matn uzunligi

len() funksiyasi stringdagi belgilar (harflar, bo'sh joy ham) sonini qaytaradi.
soz = "Python"
print(len(soz))
print(len("Salom Dunyo"))
# "Python" da 6 ta harf
# bo'sh joy ham hisoblanadi
>>> 6
>>> 11

Stringlarni birlashtirish: +

+ belgisi ikki stringni yopishtiradi (konkatenatsiya). Bo'sh joy kerak bo'lsa, uni o'zingiz qo'shasiz.
ism = "Ali"
familiya = "Valiyev"
toliq = ism + " " + familiya
print(toliq)
# " " — orasiga bo'sh joy qo'shamiz
>>> Ali Valiyev
Diqqat! + faqat string bilan string ni qo'sha oladi. "yosh: " + 20 xatolik beradi — sonni avval str(20) ga o'girish kerak.

Stringlarni takrorlash: *

* belgisi stringni kerakli marta takrorlaydi. Chiziq yoki naqsh chizishda qulay.
print("=" * 20)
print("Ha! " * 3)
# "=" ni 20 marta takrorlaydi
# so'zni 3 marta takrorlaydi
>>> ====================
>>> Ha! Ha! Ha!

Amaliy misol: salomlashuv kartasi

ism = "Laylo"
chiziq = "-" * 15
print(chiziq)
print("Salom, " + ism + "!")
print("Ism uzunligi: " + str(len(ism)))
print(chiziq)
# str() bilan sonni matnga o'girdik
>>> ---------------
>>> Salom, Laylo!
>>> Ism uzunligi: 5
>>> ---------------
