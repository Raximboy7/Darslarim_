JSON Nima?

JSON — bu JavaScript Object Notation degani. U ma'lumotlarni saqlash va dasturlar o'rtasida almashish uchun ishlatiladigan ommabop matn formati. Eng yaxshi tomoni — JSON Python'dagi dict (lug'at) ga juda o'xshaydi, shuning uchun uni o'rganish oson.
Qayerda ishlatiladi?

Internet saytlari, mobil ilovalar, sozlamalar fayllari (config), API'lar — bularning deyarli barchasi ma'lumotni JSON ko'rinishida uzatadi. Demak JSON bilan ishlashni bilish juda muhim!

JSON Qanday Ko'rinadi?

JSON ham xuddi Python lug'ati kabi "kalit": qiymat juftliklaridan iborat. Faqat kalitlar har doim qo'sh tirnoq ichida bo'lishi shart.
{
    "ism": "Ali",
    "yosh": 20,
    "talaba": true,
    "fanlar": ["Python", "Matematika"]
}

json Moduli

JSON bilan ishlash uchun Python'da maxsus json moduli bor. Uni faqat import qilish kifoya — qo'shimcha o'rnatish kerak emas.
import json
# json moduli Python bilan birga keladi

JSON Faylni O'qish — json.load()

Faylda saqlangan JSON'ni o'qib, uni Python dict yoki list ga aylantirish uchun json.load(f) ishlatiladi.
import json

with open("data.json") as f:
    data = json.load(f)

print(data["ism"])
# json.load() faylni o'qib Python dict qaytaradi
>>> Ali

JSON Faylga Yozish — json.dump()

Python lug'atini faylga JSON ko'rinishida saqlash uchun json.dump(data, f) ishlatiladi. Faylni "w" (yozish) rejimida ochish kerak.
import json

data = {"ism": "Ali", "yosh": 20}

with open("data.json", "w") as f:
    json.dump(data, f)
# dict faylga JSON sifatida saqlanadi

String Bilan Ishlash — loads() va dumps()

Ba'zan JSON fayl emas, balki matn (string) ko'rinishida keladi (masalan internetdan). Bunda oxiriga s qo'shilgan metodlar ishlatiladi: loads = "load string", dumps = "dump string".
MetodVazifasi
json.load(f)Fayldan o'qiydi → dict
json.dump(d, f)dict ni faylga yozadi
json.loads(s)Stringdan o'qiydi → dict
json.dumps(d)dict ni stringga aylantiradi
import json

data = {"ism": "Ali", "yosh": 20}
matn = json.dumps(data)
print(matn)
# dict → JSON string
>>> {"ism": "Ali", "yosh": 20}

Chiroyli Formatlash — indent

indent= parametri JSON'ni chiroyli, o'qish oson ko'rinishda — har bir kalitni alohida qatorga chekinish bilan chiqaradi.
matn = json.dumps(data, indent=4)
print(matn)
# indent=4 — 4 ta bo'sh joy bilan chekinadi
{
    "ism": "Ali",
    "yosh": 20
}

Python ↔ JSON Turlari

Python turlari JSON turlariga avtomatik aylanadi. Mana mosliklar:
PythonJSON
dictobject { }
listarray [ ]
strstring "..."
int, floatnumber
Truetrue
Falsefalse
Nonenull
Diqqat!

JSON'da true, false, null — kichik harf bilan yoziladi. Python'da esa True, False, None — bosh harf bilan. json moduli buni o'zi to'g'rilaydi.

Misol 1: dict → JSON (json.dumps)

Quyidagi Python lug'atini JSON stringga aylantiramiz. Tugmani bosing!
data = {"ism": "Ali", "yosh": 20, "talaba": True}

Misol 2: JSON string → dict (json.loads)

JSON matnini yozing, u Python lug'atiga aylanadi va undan qiymat olinadi.

Misol 3: indent — Farqni Ko'ring

Bir xil ma'lumot indent bilan va indentsiz qanday ko'rinishini solishtiring.
data = {"ism": "Guli", "fanlar": ["Python", "Ingliz"], "faol": True}
