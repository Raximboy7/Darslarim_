super() va override

Modul 9 — OOP | Dars 105

Override — metodni qayta yozish

Override — bola klass ota klassdan meros olgan metodni o'zicha qayta yozishi. Bola klassdagi metod ota klassdagini "berkitadi" — obyekt bola klass metodini ishlatadi.
class Hayvon:
    def ovoz(self):
        print("Qandaydir ovoz")

class It(Hayvon):
    def ovoz(self):  # override
        print("Vov-vov!")

It().ovoz()
# It.ovoz() Hayvon.ovoz() ni qayta yozdi (override qildi)
>>> Vov-vov!

super() — ota klass metodini chaqirish

Ba'zan otaning metodini butunlay almashtirmasdan, unga qo'shimcha qilish kerak. super() ota klassga murojaat qilib, uning metodini chaqiradi.
class Hayvon:
    def ovoz(self):
        print("Hayvon ovoz chiqaradi")

class It(Hayvon):
    def ovoz(self):
        super().ovoz()  # otani chaqir
        print("Vov-vov!")

It().ovoz()
# super().ovoz() avval otanikini bajaradi, keyin qo'shimcha ishlaydi
>>> Hayvon ovoz chiqaradi
>>> Vov-vov!

super() __init__ ichida

Eng ko'p ishlatiladigan joy — __init__. Bola klass otaning konstruktorini chaqirib, umumiy xususiyatlarni sozlaydi, keyin o'zining qo'shimcha xususiyatini qo'shadi.
class Hayvon:
    def __init__(self, ism):
        self.ism = ism

class It(Hayvon):
    def __init__(self, ism, zot):
        super().__init__(ism) # ism ni otaga
        self.zot = zot    # o'zi qo'shadi

rex = It("Rex", "Ovcharka")
print(rex.ism, rex.zot)
# super().__init__(ism) → ism ni Hayvon o'rnatadi, zot ni It qo'shadi
>>> Rex Ovcharka

super() vs override — farqi

HolatNima bo'ladiQachon
Faqat overrideOta metodi butunlay almashtiriladiXatti-harakat tamoman boshqa bo'lsa
override + super()Ota metodi + qo'shimcha bajariladiOta ishini saqlab, ustiga qo'shsangiz
Muhim ⚠️

Agar bola klass __init__ yozib, ichida super().__init__() ni chaqirmasa — otaning xususiyatlari (masalan, ism) o'rnatilmay qoladi va xato beradi.

Xulosa

Eslab qoling

• Override — bola klass ota metodini qayta yozadi
• super() — ota klass metodini chaqiradi
• super().__init__() — otaning konstruktorini ishga tushiradi
• Ota ishini saqlab, ustiga qo'shmoqchi bo'lsangiz super() ishlating
