GUI Nima?

GUI — bu Grafik Interfeys (Graphical User Interface). Bu shunday dasturki, unda foydalanuvchi oyna, tugma, matn maydoni kabi ko'rinadigan narsalar bilan ishlaydi — buyruqlarni terib yozmaydi, balki sichqoncha bilan bosadi.
Hozirgacha biz dasturlarni terminalda (qora ekranda matn bilan) yozdik. Endi esa haqiqiy ilovalar kabi — oyna ichida ishlaydigan dasturlar yaratamiz. Masalan: kalkulyator, sekundomer, ro'yxat dasturi.
Terminal dasturi (CLI)Oynali dastur (GUI)
Faqat matn ko'rinadiOyna, tugma, rasm ko'rinadi
Klaviaturada buyruq teriladiSichqoncha bilan bosiladi
input() bilan ma'lumot olinadiTugma va maydonlar bilan olinadi
Boshlovchiga ko'rimsizroqChiroyli va qulay

tkinter Nima?

tkinter — bu Python bilan birga keladigan ichki GUI kutubxonasi. Uni alohida o'rnatish kerak emas — Python o'rnatilgan bo'lsa, tkinter ham tayyor turadi. Shuning uchun u GUI o'rganishni boshlash uchun eng qulay vositadir.
import tkinter as tk
# tkinter kutubxonasini chaqiramiz va unga qisqa nom — tk beramiz
Nega "as tk"?

Har safar tkinter.Tk() deb yozish uzun. as tk orqali biz qisqa nom beramiz va keyin tk.Tk() deb yozamiz. Bu shunchaki qulaylik uchun.

Birinchi Oyna

Bo'sh oyna ochish uchun atigi 3 ta qator kerak. Mana eng kichik tkinter dasturi:
import tkinter as tk

root = tk.Tk()    # oyna yaratamiz
root.mainloop()  # oynani ekranda ushlab turamiz
# Bu kod ishga tushsa — ekranda kichik bo'sh oyna paydo bo'ladi
root nima?

root — bu bizning asosiy oynamiz. Uni tk.Tk() yaratadi. Keyin barcha tugma va matnlarni shu root ichiga joylaymiz. Nomni boshqacha (masalan oyna) ham qo'yish mumkin, lekin odatda root deyiladi.

Sarlavha va O'lcham

Bo'sh oyna chiroyli emas. Unga sarlavha (yuqoridagi yozuv) va o'lcham (eni × bo'yi) beramiz.

.title() — oyna sarlavhasi

root.title("Mening Dasturim")
# Oynaning yuqori chizig'ida "Mening Dasturim" yozuvi chiqadi

.geometry() — oyna o'lchami

root.geometry("400x300")
# Eni 400 piksel, bo'yi 300 piksel. Format: "eniXbo'yi"
Diqqat

O'lcham matn (string) ko'rinishida beriladi: "400x300". O'rtadagi belgi — bu kichik x harfi (ko'paytirish emas). Tirnoq ichida yoziladi.

Hammasi birga

import tkinter as tk

root = tk.Tk()
root.title("Salom Dunyo")
root.geometry("400x300")
root.mainloop()
# 400x300 o'lchamli, "Salom Dunyo" sarlavhali oyna ochiladi

mainloop Nima?

root.mainloop() — bu eng muhim qator. U oynani ekranda ushlab turadi va foydalanuvchini tinmay kuzatadi: tugma bosildimi? oyna yopildimi?
mainloop bo'lmasa nima bo'ladi?

Agar mainloop() yozmasangiz, dastur oynani bir lahzada yaratadi-yu, darrov yopadi — siz hech narsani ko'rib ulgurmaysiz. mainloop() esa: "Foydalanuvchi oynani yopmaguncha, kutib tur" deydi.

"loop" — sikl

Nomidagi loop (sikl) bejiz emas. tkinter ichkarida cheksiz sikl ishlatadi: u doimo "biror narsa bo'ldimi?" deb tekshirib turadi. Shuning uchun mainloop() dan keyingi kod faqat oyna yopilgandan keyin ishlaydi.

Misol 1: Oynani O'zgartiring

Sarlavha va o'lchamni o'zgartiring — pastdagi "soxta oyna" qanday ko'rinishini o'zgartirib boring. (Bu HTML namuna, lekin tkinter ham aynan shunday ishlaydi.)

400 x 300

Misol 2: Dastur Tuzilishi (Qadamma-qadam)

Har bir tkinter dasturi bir xil tartibda yoziladi. "Keyingi qadam" tugmasini bosib, qaysi qator nima qilishini ko'ring.
import tkinter as tk
root = tk.Tk()
root.title("Salom Dunyo")
root.geometry("400x300")
root.mainloop()

Misol 3: mainloop Qanday Ishlaydi?

mainloop() — bu cheksiz kuzatuvchi sikl. "Aylantirish" tugmasini bosing va u qanday tinmay tekshirib turishini ko'ring. "Oynani yop" tugmasi siklni to'xtatadi.
Eslab Qoling

• import tkinter as tk — kutubxonani chaqirish
• root = tk.Tk() — asosiy oynani yaratish
• root.title("...") — sarlavha berish
• root.geometry("eniXbo'yi") — o'lcham berish
• root.mainloop() — oynani ekranda ushlab turish (eng oxirida!)
