Template (Shablon)

Modul 21 — Django | Dars 150

Template Nima?

Template (shablon) — bu foydalanuvchiga ko'rsatiladigan HTML sahifa. U oddiy HTML, lekin ichida bo'sh joylar (placeholderlar) bor — Django ularni haqiqiy ma'lumot bilan to'ldiradi.
Misol uchun, bitta shablon yozasiz, lekin har bir foydalanuvchi uchun uning ismi boshqacha ko'rinadi. Ya'ni shablon dinamik — bir andoza, ko'p natija.
Oddiy taqqoslash

Template — bu xat namunasi (blank). "Hurmatli ____, sizning balansingiz ____ so'm." Django bo'sh joylarni har bir mijoz ma'lumoti bilan to'ldirib, alohida xatlar yasaydi.

Django Template Language (DTL)

Django shablonlarida maxsus til ishlatiladi — Django Template Language (DTL). Uning ikkita asosiy belgisi bor:
BelgiVazifasiMisol
{{ ... }}O'zgaruvchi qiymatini chiqaradi{{ ism }}
{% ... %}Mantiq: takror, shart, meros{% if %}, {% for %}
{# ... #}Izoh (ko'rinmaydi){# eslatma #}
Eslab qoling

• {{ }} — qiymatni ko'rsatadi
• {% %} — biror amal bajaradi (takror, shart)

{{ o'zgaruvchi }} — Qiymat Chiqarish

Ikki jingalak qavs ichidagi nom — bu o'zgaruvchi. View undan kelgan qiymatni shu joyga qo'yadi.
<!-- salom.html shabloni -->
<h1>Salom, {{ ism }}!</h1>
<p>Sizning yoshingiz: {{ yosh }}</p>
# ism va yosh — view dan keladigan qiymatlar
>>> Salom, Ali!
>>> Sizning yoshingiz: 25

render() — View Bilan Bog'lash

View funksiyasi shablonni render() bilan chaqiradi va unga context (lug'at) orqali ma'lumot beradi.
# views.py
from django.shortcuts import render

def salom(request):
    context = {
        "ism": "Ali",
        "yosh": 25
    }
    return render(request, "salom.html", context)
# render(request, 'shablon.html', context)
# context — lug'at: kalit shablonda {{ kalit }} bo'ladi
Oqim

Foydalanuvchi → View (render + context) → Shablon ({{ }} to'ldiriladi) → Tayyor HTML → Brauzer

{% for %} — Ro'yxatni Aylanish

{% for %} tegi ro'yxatdagi har bir element uchun HTML bo'lagini takrorlaydi. Oxirida {% endfor %} yoziladi.
<ul>
{% for ovqat in menyu %}
    <li>{{ ovqat }}</li>
{% endfor %}
</ul>
# menyu = ["Osh", "Lag'mon", "Manti"]
# har bir element <li> ga aylanadi
>>> • Osh
>>> • Lag'mon
>>> • Manti

{% if %} — Shart Tekshirish

{% if %} shart to'g'ri bo'lsagina ichidagi HTML ni ko'rsatadi. {% else %} va {% endif %} bilan ishlatiladi.
{% if kirgan %}
    <p>Xush kelibsiz, {{ ism }}!</p>
{% else %}
    <p>Iltimos, tizimga kiring.</p>
{% endif %}
# kirgan = True bo'lsa birinchi qator,
# aks holda ikkinchi qator chiqadi

templates Papkasi

Django shablonlarni templates nomli papkadan qidiradi. Odatda har bir ilova ichida shu papka bo'ladi.
loyiha/
  └── ilova/
      ├── views.py
      └── templates/
          ├── salom.html
          └── menyu.html
# render(request, "salom.html") shu papkadan topadi

Static Fayllar (CSS / JS)

CSS, JavaScript va rasmlar static fayllar deyiladi. Ular static papkasida saqlanadi. Shablonda avval {% load static %} yoziladi, keyin {% static '...' %} bilan fayl ulanadi.
{% load static %}
<link rel="stylesheet"
    href="{% static 'style.css' %}">
# static/style.css faylini sahifaga ulaydi

Template Meros (extends / block)

Ko'p sahifalarda bir xil qism bo'ladi (menyu, footer). Ularni har safar qayta yozmaslik uchun bitta asosiy shablon (base.html) yasab, boshqa sahifalar undan meros oladi.

1. Asosiy shablon — base.html

<!-- base.html -->
<body>
    <h1>Mening Saytim</h1>
    {% block kontent %}{% endblock %}
</body>
# block — bu bolalar to'ldiradigan bo'sh joy

2. Bola shablon — uy.html

<!-- uy.html -->
{% extends 'base.html' %}
{% block kontent %}
    <p>Bu bosh sahifa.</p>
{% endblock %}
# extends — base.html ni meros oladi
# block — faqat o'z qismini to'ldiradi
>>> Mening Saytim
>>> Bu bosh sahifa.
