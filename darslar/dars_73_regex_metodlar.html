re Moduli Nima?

O'tgan darsda regex (pattern) bilan tanishdik. Endi re modulining asosiy metodlarini o'rganamiz. Bu metodlar matn ichidan pattern bo'yicha qism topadi, almashtiradi yoki bo'ladi. Avval modulni import qilamiz:
import re
# re modulini chaqirib olamiz

Asosiy Metodlar Jadvali

MetodVazifasi
re.search()Matndan birinchi mos qismni topadi
re.match()Faqat matn boshidan tekshiradi
re.findall()Barcha mos qismlarni ro'yxat qilib qaytaradi
re.sub()Mos qismlarni boshqasiga almashtiradi
re.split()Pattern bo'yicha matnni bo'laklarga bo'ladi
re.compile()Patternni oldindan tayyorlaydi (tez ishlash uchun)

re.search() — Birinchi Mosni Topish

re.search(pattern, matn) butun matn ichidan pattern mos keladigan birinchi joyni topadi. Topilsa Match obyekti, topilmasa None qaytaradi.
import re
matn = "Mening yoshim 25"
natija = re.search(r"\d+", matn)
print(natija.group())
# \d+ — bir yoki bir nechta raqam. .group() topilgan matnni beradi
>>> 25
Match obyekti va .group()

re.search() va re.match() Match obyekt qaytaradi. Topilgan matnni olish uchun .group() metodidan foydalanamiz. Hech narsa topilmasa natija None bo'ladi.

re.match() — Faqat Boshidan

re.match() patternni faqat matnning boshidan tekshiradi. Agar matn pattern bilan boshlanmasa, o'rtada mos kelsa ham None qaytaradi.
import re
print(re.match(r"\d+", "25 yosh"))  # boshida raqam bor
print(re.match(r"\d+", "yosh 25"))  # boshida raqam yo'q
# search butun matnni qaraydi, match faqat boshini
>>> <re.Match object; match='25'>
>>> None

re.findall() — Barcha Mosliklar

re.findall() matn ichidagi barcha mos qismlarni oddiy ro'yxat (list) qilib qaytaradi. Hech narsa topilmasa bo'sh ro'yxat [] beradi.
import re
matn = "Telefon: 99 123 45 67"
raqamlar = re.findall(r"\d+", matn)
print(raqamlar)
# Barcha raqam guruhlarini topadi
>>> ['99', '123', '45', '67']

re.sub() — Almashtirish

re.sub(pattern, yangi, matn) patternga mos kelgan barcha qismlarni yangi matn bilan almashtiradi. Bu matn tozalash uchun juda foydali.
import re
matn = "Narx: 100 dollar, 200 so'm"
natija = re.sub(r"\d+", "#", matn)
print(natija)
# Har bir raqamni # belgisiga almashtiradi
>>> Narx: # dollar, # so'm

re.split() — Bo'lish

re.split(pattern, matn) matnni pattern bo'yicha bo'laklarga ajratadi. Oddiy .split() dan farqi — bir nechta turdagi ajratuvchidan foydalana olamiz.
import re
matn = "olma, anor; uzum"
mevalar = re.split(r"[,;]\s*", matn)
print(mevalar)
# Vergul yoki nuqta-vergul bo'yicha bo'ladi
>>> ['olma', 'anor', 'uzum']

re.compile() — Pattern Tayyorlash

Agar bitta patternni ko'p marta ishlatadigan bo'lsak, re.compile() bilan uni oldindan tayyorlab qo'yamiz. Bu kodni tezroq va toza qiladi.
import re
pattern = re.compile(r"\d+")  # bir marta tayyorlaymiz
print(pattern.findall("2 olma 5 anor"))
print(pattern.search("narx 99").group())
# Tayyor pattern ustida findall, search ishlatamiz
>>> ['2', '5']
>>> 99
Eslab Qoling

• re.search() — birinchi mos, re.match() — faqat boshidan
• re.findall() — barcha mosliklar ro'yxati
• re.sub() — almashtiradi, re.split() — bo'ladi
• Match obyektidan matnni olish: .group()
• Patternni oldidan r"..." qo'yish odat (raw string)

Misol 1: findall — Barcha Mosliklar

Matn yozing va patternni tanlang. re.findall() barcha mos qismlarni ro'yxat qilib chiqaradi.

Misol 2: sub — Almashtirish

Patternga mos kelgan qismlarni yangi matnga almashtiramiz.

Misol 3: split — Bo'lish

Matnni pattern bo'yicha bo'laklarga ajratamiz. Vergul, nuqta-vergul yoki bo'shliq ajratuvchi bo'lishi mumkin.

Misol 4: search — Birinchi Mos

re.search() birinchi mos qismni topadi. Topilsa Match obyekti, topilmasa None.
