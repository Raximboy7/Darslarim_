Loyiha: Mini framework

Modul 10 — Iterator, Generator, Decorator | Dars 124

Loyiha maqsadi

Bu darsda biz dekoratorlar yordamida Flask'ga o'xshash mini web-frameworkning yuragini — route (marshrut) tizimini yozamiz. Maqsad: @app.route("/salom") deb yozganda funksiyani ma'lum bir manzilga bog'lash. Bu dekoratorlar amalda qanchalik kuchli ekanini ko'rsatadi.
G'oya

Framework ichida {manzil: funksiya} lug'ati bo'ladi. @app.route(...) dekoratori funksiyani shu lug'atga qo'shadi. Keyin manzil kelganda mos funksiya chaqiriladi.

1-qadam: App klassi va route dekoratori

class MiniApp:
    def __init__(self):
        self.marshrutlar = {}          # {manzil: funksiya}

    def route(self, manzil):
        def dekorator(func):
            self.marshrutlar[manzil] = func   # ro'yxatga qo'shamiz
            return func
        return dekorator
route — argument (manzil) qabul qiladigan dekorator. Ichki dekorator funksiyani marshrutlar lug'atiga yozadi va funksiyaning o'zini o'zgartirmasdan qaytaradi.

2-qadam: So'rovni qayta ishlash

    def sorov(self, manzil):
        if manzil in self.marshrutlar:
            func = self.marshrutlar[manzil]
            return func()          # mos funksiyani chaqiramiz
        return "404 — sahifa topilmadi"
sorov metodi kelgan manzilni lug'atdan qidiradi. Topsa — funksiyani chaqirib natijani qaytaradi, topmasa — 404 xabari beradi. Aynan haqiqiy framework kabi.

3-qadam: Foydalanish

app = MiniApp()

@app.route("/")
def bosh_sahifa():
    return "Assalomu alaykum!"

@app.route("/about")
def haqida():
    return "Bu mening mini framework loyiham"

print(app.sorov("/"))
print(app.sorov("/about"))
print(app.sorov("/yoq"))
@app.route("/") — bosh_sahifa funksiyasini "/" manziliga bog'laydi. Endi app.sorov("/") chaqirilganda o'sha funksiya ishlaydi.
>>> Assalomu alaykum!
>>> Bu mening mini framework loyiham
>>> 404 — sahifa topilmadi

Amaliyot

Loyihani kengaytiring

1. /vaqt manzilini qo'shing — datetime bilan joriy vaqtni qaytarsin.
2. Agar bir manzil ikki marta ro'yxatga olinsa, ogohlantirish chiqaradigan qiling.
3. metod (GET/POST) qo'shib, route("/", metod="POST") ni qo'llab-quvvatlang.

Xulosa

Esda tuting

• Argumentli dekorator = dekorator qaytaradigan funksiya (ikki qavat ichki funksiya).
• Route tizimi aslida oddiy {manzil: funksiya} lug'ati.
• Flask, Django kabi katta frameworklar ham xuddi shu g'oyaga tayanadi.
• Dekoratorlar — kodni "ro'yxatga olish" (registration) uchun kuchli vosita.
