Misol A: Log Fayl Tahlili

Shart: Server log faylida minglab qator bor. Bizga faqat xato (ERROR) qatorlarini va ularning sanasini ajratib olish kerak.

1-qadam: Namuna log qatori

2026-06-30 14:23:01 ERROR Ulanish uzildi
2026-06-30 14:23:05 INFO Foydalanuvchi kirdi
2026-06-30 14:24:10 ERROR Fayl topilmadi

2-qadam: Pattern

log = "2026-06-30 14:23:01 ERROR Ulanish uzildi"
pattern = r"(\d{4}-\d{2}-\d{2}) (\d{2}:\d{2}:\d{2}) ERROR (.+)"
moslik = re.search(pattern, log)
if moslik:
    print("Sana:", moslik.group(1))
    print("Vaqt:", moslik.group(2))
    print("Xato:", moslik.group(3))
# Uchta guruh: sana, vaqt, xato matni
>>> Sana: 2026-06-30
>>> Vaqt: 14:23:01
>>> Xato: Ulanish uzildi

3-qadam: Tushuntirish

Bo'lakMa'nosi
(\d{4}-\d{2}-\d{2})4 raqam - 2 raqam - 2 raqam = sana
(\d{2}:\d{2}:\d{2})vaqt: soat:daqiqa:soniya
ERRORFaqat ERROR so'zi bor qatorlar mos keladi
(.+)Qolgan butun matn = xato izohi

Misol B: Ma'lumot Validatsiyasi

Shart: Foydalanuvchi kiritgan ma'lumot to'g'rimi yoki yo'qligini tekshirish kerak. Masalan parol kuchi yoki ID raqam formati.

Parol kuchi qoidalari

QoidaPattern
Kamida bitta katta harf[A-Z]
Kamida bitta kichik harf[a-z]
Kamida bitta raqam\d
Kamida bitta belgi[!@#$%]
Uzunligi 8+.{8,}
import re

def parol_kuchli(parol):
    if len(parol) < 8:
        return False
    if not re.search(r"[A-Z]", parol):
        return False
    if not re.search(r"\d", parol):
        return False
    return True
# Har bir qoidani alohida search bilan tekshiramiz

ID raqam tekshiruvi (masalan AB1234567)

pattern = r"^[A-Z]{2}\d{7}$"
print(bool(re.match(pattern, "AB1234567")))
# ^ va $ — boshidan oxirigacha aniq moslik bo'lsin
>>> True

Misol C: Matndan Ma'lumot Ajratish

Shart: Web sahifa yoki HTML matnidan kerakli ma'lumotni ajratib olish (web scraping). Masalan HTML teglar ichidagi matn, email yoki havola (link).

HTML teglardan matn ajratish

import re
html = "<h1>Sarlavha</h1><p>Matn</p>"
matnlar = re.findall(r">([^<]+)<", html)
print(matnlar)
# > va < oralig'idagi matnni topadi
>>> ['Sarlavha', 'Matn']

Barcha emaillarni topish

matn = "Aloqa: ali@mail.uz va vali@shop.com"
pattern = r"[\w.]+@[\w.]+\.\w+"
print(re.findall(pattern, matn))
# findall — barcha mosliklarni ro'yxat qilib qaytaradi
>>> ['ali@mail.uz', 'vali@shop.com']
Bo'lakMa'nosi
>([^<]+)<">" va "<" oralig'idagi tegga tegmaydigan matn
[^<]+"<" dan tashqari barcha belgilar
[\w.]+@email egasi nomi, keyin "@"
\.\w+domen oxiri, masalan .uz yoki .com

Misol D: Sana va Vaqt Formatini Tekshirish

Shart: Foydalanuvchi kiritgan sana YYYY-MM-DD formatida to'g'rimi tekshirish kerak.
import re
pattern = r"^\d{4}-\d{2}-\d{2}$"
print(bool(re.match(pattern, "2026-06-30")))
print(bool(re.match(pattern, "30/06/2026")))
# Faqat YYYY-MM-DD formati mos keladi
>>> True
>>> False
Diqqat

Regex faqat formatni tekshiradi. "2026-13-45" patternga mos keladi, ammo bunday oy va kun yo'q. Mantiqiy tekshiruv uchun datetime moduli ishlatiladi.

Eslab Qoling

• re.search — birinchi moslikni topadi
• re.findall — barcha mosliklarni ro'yxat qiladi
• re.match — boshidan moslikni tekshiradi
• ^...$ — to'liq, aniq moslik uchun
• Guruhlar ( ) — ma'lumotni qismlarga ajratadi
