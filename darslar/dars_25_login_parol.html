Mini-loyiha: Login + parol tekshirish

Modul 2 — Stringlar va Formatlash | Dars 25

Loyiha Haqida

Bu darsda modul davomida o'rgangan bilimlarimizni birlashtirib, kichik login va parol tekshirish dasturini yozamiz. Foydalanuvchidan login va parol so'raymiz, ularni to'g'ri qiymat bilan solishtiramiz va 3 marta urinish beramiz.
Nimalarni ishlatamiz?

• input() — foydalanuvchidan ma'lumot olish
• if / else — tekshirish
• == — solishtirish
• while va sanoq — urinishlarni cheklash

1-qadam: Oddiy Tekshirish

Avval to'g'ri login va parolni belgilaymiz, so'ng foydalanuvchidan so'raymiz. 

# To'g'ri ma'lumotlar
togri_login = "admin"
togri_parol = "1234"

# Foydalanuvchidan so'rash
login = input("Login: ")
parol = input("Parol: ")

if login == togri_login and parol == togri_parol:
    print("Xush kelibsiz!")
else:
    print("Login yoki parol xato!")
# ikkalasi ham to'g'ri bo'lsa (and) kiritadi
>>> Login: admin
>>> Parol: 1234
>>> Xush kelibsiz!

2-qadam: Urinishlar Sonini Cheklash

Endi while sikli bilan foydalanuvchiga 3 ta urinish beramiz. Har xato urinishda urinishlar soni kamayadi. 

togri_login = "admin"
togri_parol = "1234"
urinishlar = 3

while urinishlar > 0:
    login = input("Login: ")
    parol = input("Parol: ")

    if login == togri_login and parol == togri_parol:
        print("Xush kelibsiz!")
        break
    else:
        urinishlar = urinishlar - 1
        print(f"Xato! Qolgan urinishlar: {urinishlar}")

if urinishlar == 0:
    print("Hisob bloklandi!")
# break — to'g'ri bo'lsa sikldan chiqadi
# urinishlar 0 ga tushsa bloklanadi
>>> Login: admin
>>> Parol: 0000
>>> Xato! Qolgan urinishlar: 2
>>> Login: admin
>>> Parol: 1234
>>> Xush kelibsiz!

3-qadam: Yaxshilangan To'liq Versiya

Endi .strip() bilan ortiqcha bo'shliqlarni tozalaymiz va bo'sh kiritishni tekshiramiz — bu real dasturlarga o'xshaydi. 

togri_login = "admin"
togri_parol = "1234"
urinishlar = 3

print("=== TIZIMGA KIRISH ===")

while urinishlar > 0:
    login = input("Login: ").strip()
    parol = input("Parol: ").strip()

    if login == "" or parol == "":
        print("Maydonlarni to'ldiring!")
        continue

    if login == togri_login and parol == togri_parol:
        print(f"Xush kelibsiz, {login}!")
        break
    else:
        urinishlar -= 1
        if urinishlar > 0:
            print(f"Xato! Yana {urinishlar} urinish bor.")
        else:
            print("Urinishlar tugadi. Hisob vaqtincha bloklandi.")
# strip() — bo'shliqlarni tozalaydi
# continue — bo'sh bo'lsa qaytadan so'raydi
# urinishlar -= 1 — qisqa yozuv
>>> === TIZIMGA KIRISH ===
>>> Login: admin
>>> Parol:
>>> Maydonlarni to'ldiring!
>>> Login: admin
>>> Parol: 1234
>>> Xush kelibsiz, admin!

Kod Qanday Ishlaydi?

1
To'g'ri ma'lumotlar saqlanadi

login = "admin", parol = "1234", urinishlar = 3

2
while sikli aylanadi

urinishlar 0 dan katta bo'lguncha davom etadi

3
Solishtirish

login va parol == to'g'ri qiymatlarga teng bo'lsa — kiradi (break)

4
Xato bo'lsa

urinishlar 1 ga kamayadi, 0 ga yetsa hisob bloklanadi

Vazifa — O'zingiz Rivojlantiring

Qo'shimcha mashqlar

1. Parol uzunligini tekshiring — 4 belgidan kam bo'lsa ogohlantiring.
2. Login katta-kichik harfga bog'liq bo'lmasin: login.lower() ishlating.
3. Bir nechta foydalanuvchini tekshirish uchun ma'lumotlarni kengaytiring.

Xulosa

Ushbu mini-loyihada input(), if/else, while, and/or, break/continue va string metodlarini birga ishlatdik. Login-parol tekshirish — deyarli har bir dasturda uchraydigan amaliy vazifa. Modulni tugatganingiz bilan tabriklaymiz — endi stringlar va formatlashni yaxshi bilasiz!
