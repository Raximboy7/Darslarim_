List Comprehension Nima?

List comprehension — bu yangi listni bitta qatorda, qisqa va chiroyli yaratish usuli. Odatda bir necha qator for sikli kerak bo'lgan ishni siz faqat bitta qatorda bajarasiz. Bu Python'ning eng sevimli xususiyatlaridan biri.
sonlar = [x for x in range(5)]
print(sonlar)
# 0 dan 4 gacha sonlar listi
>>> [0, 1, 2, 3, 4]

Tuzilishi (Sintaksis)

Umumiy ko'rinishi: [ ifoda for element in ketma-ketlik ]
Bu — "har bir element uchun ifodani hisobla va listga qo'sh" degani.

Qiymatni O'zgartirish

Har bir element ustida amal bajarish mumkin: ko'paytirish, daraja, qo'shish va hokazo.
ikki = [x * 2 for x in range(5)]
print(ikki)
# Har bir sonni 2 ga ko'paytirish
>>> [0, 2, 4, 6, 8]
kvadrat = [x ** 2 for x in range(6)]
print(kvadrat)
# Har bir sonning kvadrati
>>> [0, 1, 4, 9, 16, 25]

Shart bilan (if)

Oxiriga if shart qo'shsangiz — faqat shartga mos elementlar listga tushadi. Qolganlari tashlab yuboriladi.
juft = [x for x in range(10) if x % 2 == 0]
print(juft)
# Faqat juft sonlar (qoldiq 0)
>>> [0, 2, 4, 6, 8]

For Sikli bilan Taqqoslash

Bir xil natija — ikki xil yo'l. List comprehension ancha qisqa va o'qilishi oson.
Uzun usul — oddiy for sikli
kvadrat = []
for x in range(5):
    kvadrat.append(x ** 2)

3 qator: bo'sh list, sikl, append.

Qisqa usul — list comprehension
kvadrat = [x ** 2 for x in range(5)]

1 qator — natija aynan bir xil: [0, 1, 4, 9, 16].

Dictionary va Set Comprehension

Xuddi shu g'oya lug'at (dict) va to'plam (set) uchun ham ishlaydi — faqat kvadrat qavs [ ] o'rniga jingalak qavs { } ishlatiladi.

Dictionary comprehension

d = {x: x ** 2 for x in range(4)}
print(d)
# kalit: qiymat juftliklari
>>> {0: 0, 1: 1, 2: 4, 3: 9}

Set comprehension

s = {x % 3 for x in range(6)}
print(s)
# takrorlanmas qiymatlar to'plami
>>> {0, 1, 2}

Keng Tarqalgan Xato

❌ Xato: ifoda va for o'rni almashib qolgan
sonlar = [for x in range(5) x] # SyntaxError

Ifoda (nima qo'shilishi) har doim eng oldinda turishi kerak, for esa undan keyin keladi.

✓ To'g'ri: ifoda oldinda
sonlar = [x for x in range(5)]
>>> [0, 1, 2, 3, 4]
Eslab Qoling

• Tuzilishi: [ifoda for element in ketma-ketlik]
• Ifoda har doim eng oldinda turadi
• Shart qo'shish: oxiriga if shart
• { } bilan dict va set comprehension yaratiladi
• Qisqa va tushunarli — bir qatorda yangi list!
