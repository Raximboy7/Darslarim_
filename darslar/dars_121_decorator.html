Funksiya — Birinchi Darajali Obyekt

Python'da funksiya ham oddiy obyekt. Uni o'zgaruvchiga berish, boshqa funksiyaga argument qilib uzatish yoki funksiyadan qaytarish mumkin. Aynan shu xususiyat decorator'ni yaratishga imkon beradi.

Funksiyani o'zgaruvchiga berish

def salom():
    return "Salom!"

x = salom   
print(x())
# salom funksiyasini qavssiz x ga berdik (nusxa emas, xuddi o'sha funksiya)
# endi x() ham salom() bilan bir xil
>>> Salom!

Funksiyani argument qilib berish

def ikki_marta(func):
    return func() + " " + func()

print(ikki_marta(salom))
# salom funksiyasini boshqa funksiyaga argument qilib berdik
# ichida uni ikki marta chaqirdik
>>> Salom! Salom!

Decorator Nima?

Decorator — bu boshqa funksiyani o'rab oladigan funksiya. U asl funksiyaning kodini o'zgartirmasdan, unga qo'shimcha xatti-harakat (masalan, log yozish, vaqt o'lchash) qo'shadi. O'rashda asl funksiya o'zgarmaydi — atrofiga yangi qatlam qo'shiladi.
Oddiy taqqoslash

Sovg'ani tasavvur qiling: sovg'aning o'zi — bu asl funksiya. O'rash qog'ozi — bu decorator. Sovg'a o'zgarmaydi, faqat atrofiga chiroyli qog'oz (qo'shimcha xatti-harakat) qo'shiladi.

Oddiy Decorator Yaratish

Decorator ichida wrapper (o'rovchi) nomli ichki funksiya bo'ladi. Wrapper asl funksiyani chaqiradi va undan oldin/keyin qo'shimcha kod bajaradi.
def logger(func):
    def wrapper(*args):
        print("Funksiya chaqirilmoqda...")
        natija = func(*args)
        print("Funksiya tugadi.")
        return natija
    return wrapper
# logger — decorator, func — o'raladigan funksiya
# wrapper asl funksiyani chaqiradi, oldin/keyin xabar qo'shadi
# *args — har qanday argumentlarni qabul qilish uchun
# logger wrapper funksiyani qaytaradi

@decorator Sintaksisi

Funksiya ustiga @decorator yozsangiz, Python avtomatik ravishda funksiyani o'sha decorator bilan o'raydi. Bu juda qulay qisqartma.
@logger
def qoshish(a, b):
    return a + b

print(qoshish(2, 3))
# @logger = qoshish = logger(qoshish) bilan bir xil
# endi qoshish() har chaqirilganda xabarlar ham chiqadi
>>> Funksiya chaqirilmoqda...
>>> Funksiya tugadi.
>>> 5

@ Bilan va @ Holda — Bir Xil

@ holda (qo'lda)@ bilan (qisqa)
def qoshish(a, b): ... @logger
qoshish = logger(qoshish) def qoshish(a, b): ...
Eslab Qoling

@logger — bu shunchaki qoshish = logger(qoshish) ning chiroyli ko'rinishi. Ikkalasi ham aynan bir xil ishlaydi.

Amaliy Misol: Vaqt O'lchash

Decorator yordamida funksiya qancha vaqtda bajarilganini o'lchash mumkin — funksiya kodiga tegmasdan!
import time

def vaqt_olchash(func):
    def wrapper(*args):
        boshi = time.time()
        natija = func(*args)
        oxiri = time.time()
        print(f"Vaqt: {oxiri - boshi:.4f} s")
        return natija
    return wrapper
# funksiya boshlanishidan oldin va tugaganidan keyin vaqtni oladi
# ularning farqi — bajarilish vaqti
Eslab Qoling

• Funksiya — obyekt: o'zgaruvchiga, argumentga berish mumkin
• Decorator funksiyani o'rab, xatti-harakat qo'shadi
• Asl funksiyaning kodi o'zgarmaydi
• @decorator = func = decorator(func)
• Wrapper ichida *args har qanday argumentni qabul qiladi

Misol 1: Decorator Funksiyani O'raydi

Tugmani bosing — decorator asl funksiyadan oldin va keyin qanday xabar qo'shishini ko'ring.

Misol 2: @ Bilan va @ Holda

Ikki usul ham bir xil natija berishini taqqoslang.

Misol 3: Log Decorator

Bu decorator har bir funksiya chaqiruvini yozib boradi (log). Bir necha marta bosib, log ro'yxati to'lib borishini kuzating.
