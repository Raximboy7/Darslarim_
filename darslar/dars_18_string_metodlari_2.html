Yana String Metodlari

Oldingi darsda upper(), lower() kabi metodlarni ko'rdik. Endi matnni o'zgartirish, izlash, sanash va bo'laklarga ajratish uchun eng kerakli metodlar bilan tanishamiz: replace, find, count, split va join.
Eslatma

Stringlar Pythonda o'zgarmas (immutable). Shuning uchun bu metodlar stringni o'zgartirmaydi, balki yangi natija qaytaradi. Natijani o'zgaruvchiga saqlashni unutmang!

replace() — Almashtirish

replace(eski, yangi) — matndagi barcha "eski" bo'laklarni "yangi" bilan almashtiradi. 

matn = "Men olma yaxshi ko'raman"
yangi = matn.replace("olma", "banan")
print(yangi)
# "olma" so'zi "banan" bilan almashadi
>>> Men banan yaxshi ko'raman

Barcha mos keladigan joylar almashadi: 

gap = "aa-aa-aa"
print(gap.replace("a", "b"))
# har bir "a" "b" ga aylanadi
>>> bb-bb-bb

find() — Izlash

find(bo'lak) — bo'lak birinchi marta qaysi indeksda uchraganini qaytaradi. Agar topilmasa -1 qaytaradi. 

matn = "salom dunyo"
print(matn.find("dunyo"))
print(matn.find("lom"))
print(matn.find("kompyuter"))
# indeks 6 dan boshlanadi
# indeks 2
# topilmadi -> -1
>>> 6
>>> 2
>>> -1
Maslahat

Indekslar 0 dan boshlanadi. "salom dunyo" da "d" harfi 6-indeksda turadi (s=0, a=1, l=2, o=3, m=4, bo'shliq=5, d=6).

count() — Sanash

count(bo'lak) — bo'lak matnda necha marta uchraganini sanaydi. 

matn = "ananas"
print(matn.count("a"))
print(matn.count("an"))
print(matn.count("z"))
# "a" 3 marta
# "an" 2 marta
# "z" yo'q -> 0
>>> 3
>>> 2
>>> 0

split() va join() — Bo'lish va Birlashtirish

split(ajratuvchi) — matnni ro'yxatga (list) bo'laklarga ajratadi. join(list) — aksincha, ro'yxatni bitta matnga birlashtiradi.

split misoli

gap = "olma,banan,uzum"
mevalar = gap.split(",")
print(mevalar)
# vergul bo'yicha ajratadi
>>> ['olma', 'banan', 'uzum']

Ajratuvchi berilmasa, bo'shliq bo'yicha ajratadi: 

jumla = "Men Python o'rganaman"
print(jumla.split())
# so'zlarga ajratadi
>>> ['Men', 'Python', "o'rganaman"]

join misoli

mevalar = ["olma", "banan", "uzum"]
natija = " - ".join(mevalar)
print(natija)
# ro'yxatni " - " bilan birlashtiradi
>>> olma - banan - uzum

Amaliy Mashq

Sinab ko'ring

1. Bir jumlani split() bilan so'zlarga ajrating va so'zlar sonini toping.
2. Ismingizdagi biror harf necha marta uchrashini count() bilan sanang.
3. Telefon raqamidagi "-" belgilarni replace() bilan olib tashlang.

Xulosa

Bu darsda 5 ta kuchli string metodini o'rgandik: replace (almashtirish), find (izlash), count (sanash), split (bo'lish) va join (birlashtirish). Bu metodlar matn bilan ishlashda eng ko'p qo'llaniladiganlaridan hisoblanadi.
