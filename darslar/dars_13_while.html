While Sikli Nima?

While sikli — bu biror shart to'g'ri (True) bo'lib turgan vaqtda kodni qayta-qayta takrorlaydigan tuzilma. "Sikl" so'zi "aylanma, takror" degani. Shart yolg'on (False) bo'lishi bilan sikl to'xtaydi.
while shart:
    # takrorlanadigan kod
# shart True bo'lguncha takrorlanadi
Diqqat: ikki nuqta va bo'sh joy

while qatori : bilan tugaydi, takrorlanadigan kod esa ichkariga (4 bo'sh joy) suriladi — xuddi if dagidek.

Birinchi Misol: 0 dan 4 gacha

i = 0
while i < 5:
    print(i)
    i = i + 1
# i ni 0 dan boshlaymiz
# i 5 dan kichik bo'lguncha
# i ni chiqaramiz
# i ni 1 ga oshiramiz (MUHIM!)
>>> 0
>>> 1
>>> 2
>>> 3
>>> 4

i — bu hisoblagich (counter). Har aylanishda u 1 ga oshadi: 0, 1, 2, 3, 4. i 5 ga teng bo'lganda i < 5 sharti False bo'ladi va sikl to'xtaydi.

⚠️ Eng Xavfli Xato: Cheksiz Sikl

⚠️ Hisoblagichni o'zgartirishni unutmang!

Agar i ni oshirmasangiz, shart hech qachon False bo'lmaydi — sikl cheksiz (infinite loop) takrorlanadi va dastur muzlab qoladi. To'xtatish uchun Ctrl + C bosiladi.

❌ Xato: cheksiz sikl
i = 0
while i < 5:
    print(i)  # i o'zgarmayapti!

i doim 0 bo'lib qoladi → 0 0 0 0 ... cheksiz chiqaveradi.

✓ To'g'ri: hisoblagich oshiriladi
i = 0
while i < 5:
    print(i)
    i += 1  # i = i + 1
>>> 0 1 2 3 4 — keyin to'xtaydi

break — Sikldan Chiqish

break sikldan darhol chiqib ketadi, shart hali True bo'lsa ham.
i = 1
while True:
    print(i)
    if i == 3:
        break
    i += 1
# cheksizdek ko'rinadi, lekin...
# i 3 ga yetganda
# break sikldan chiqaradi
>>> 1
>>> 2
>>> 3

continue — Keyingisiga O'tish

continue joriy aylanishning qolgan kodini tashlab ketib, to'g'ridan-to'g'ri keyingi aylanishga o'tadi.
i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)
# i 3 bo'lganda
# print tashlab ketiladi
# 3 chiqmaydi
>>> 1
>>> 2
>>> 4
>>> 5  (3 yo'q!)

while-else Bloki

else bloki sikl oddiy yo'l bilan (shart False bo'lib) tugaganda ishlaydi. Agar sikl break bilan to'xtasa, else ishlamaydi.
i = 0
while i < 3:
    print(i)
    i += 1
else:
    print("Sikl tugadi")
# 0, 1, 2 chiqadi
# shart False bo'ldi
# else ishlaydi
>>> 0
>>> 1
>>> 2
>>> Sikl tugadi

break, continue — Solishtirish

Eslab Qoling

• while shart: — shart True bo'lguncha takrorlaydi
• Hisoblagichni o'zgartirishni UNUTMANG, aks holda cheksiz sikl bo'ladi
• break sikldan chiqaradi, continue keyingisiga o'tadi
• else bloki sikl break'siz tugaganda ishlaydi
