Boshqa formatlash — .format() va %

Formatlashning Boshqa Usullari

Oldingi darsda zamonaviy f-string ni ko'rdik. Lekin eski kodlarda hali ham ikkita usul ko'p uchraydi: str.format() metodi va eski % operatori. Ularni bilish, o'zgalarning kodini o'qishda yordam beradi.

.format() Metodi

Matnda {} qavslar qo'yiladi, keyin .format() ichiga qiymatlar beriladi. Qiymatlar tartib bo'yicha qavslarga joylashadi. 

ism = "Ali"
yosh = 20
matn = "Mening ismim {}, yoshim {}".format(ism, yosh)
print(matn)
# {} joylariga ism va yosh joylashadi
>>> Mening ismim Ali, yoshim 20

Pozitsion va Nomli Argumentlar

Qavs ichiga indeks raqami yozib, tartibni o'zgartirish mumkin (pozitsion): 

matn = "{0} va {1}, yana {0}".format("olma", "banan")
print(matn)
# {0} = birinchi, {1} = ikkinchi qiymat
>>> olma va banan, yana olma

Yoki nom berib ham yozish mumkin (nomli): 

matn = "{shahar}da harorat {temp} daraja".format(
    shahar="Toshkent", temp=30)
print(matn)
# nom bo'yicha joylashadi
>>> Toshkentda harorat 30 daraja

Eski % Usuli

Juda eski usul — %s (string uchun), %d (butun son), %f (kasr son) belgilaridan foydalaniladi. 

ism = "Vali"
yosh = 25
print("Ism: %s, Yosh: %d" % (ism, yosh))
# %s -> string, %d -> butun son
>>> Ism: Vali, Yosh: 25

Kasr sonlarni belgilangan aniqlikda chiqarish: 

narx = 19.9876
print("Narx: %.2f" % narx)
# %.2f -> 2 xona kasr
>>> Narx: 19.99

Uch Usulni Solishtirish

Bir xil natijani uch xil usulda yozamiz: 

ism = "Ali"

# 1. f-string (eng zamonaviy)
print(f"Salom, {ism}")

# 2. .format()
print("Salom, {}".format(ism))

# 3. % operator (eski)
print("Salom, %s" % ism)
# uchalasi ham bir xil natija beradi
>>> Salom, Ali
>>> Salom, Ali
>>> Salom, Ali

Xulosa

Nimani eslab qolish kerak?

• .format() — {} qavslar, tartib yoki nom bo'yicha ishlaydi.
• % — eski usul, %s / %d / %f belgilar.
• Yangi kod yozayotganda f-string ni tanlang — u eng qisqa va tushunarli. Lekin eski kodni o'qish uchun bu ikki usulni ham bilib qo'ying.
