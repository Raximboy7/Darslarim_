Flask Templates (Jinja2)

Modul 13 — Web Asoslari | Dars 86

Template Nima?

Template (shablon) — bu tayyor HTML sahifa, lekin uning ichida bo'sh joylar bor. Bu bo'sh joylarga Python kodi keyinroq qiymat qo'yib beradi. Ya'ni HTML ko'rinishi bir marta yoziladi, ammo ichidagi ma'lumot har safar dinamik o'zgarib turadi.
Oldingi darsda biz return "Salom" deb matnni to'g'ridan-to'g'ri qaytargandik. Lekin butun bir HTML sahifani Python ichida yozish juda noqulay. Shuning uchun HTML ni alohida faylga ajratamiz — bu fayl template deyiladi.

Jinja2 — Template Dvigateli

Jinja2 — bu Flask bilan birga keladigan template engine (template dvigateli). Uning vazifasi: HTML shablonni o'qib, ichidagi maxsus belgilarni Python qiymatlari bilan to'ldirib, tayyor HTML sahifani qaytarish.
BelgiVazifasiMisol
{{ ... }}Qiymat chiqaradi (o'zgaruvchi){{ ism }}
{% ... %}Mantiq: for, if{% if yosh > 18 %}
{# ... #}Izoh (ko'rinmaydi){# bu izoh #}
Oddiy qoida

• Ikki gulqavs {{ }} — biror narsani ko'rsatish uchun.
• Gulqavs va foiz {% %} — amal bajarish (sikl, shart) uchun.

templates Papkasi

Flask barcha HTML shablonlarni templates nomli papkadan qidiradi. Bu papka nomini AYNAN shunday yozish kerak. Loyiha tuzilishi quyidagicha bo'ladi:
loyiha/
  ├── app.py  # Python kodi
  └── templates/
      ├── index.html
      └── salom.html

render_template() Funksiyasi

Shablonni ko'rsatish uchun render_template() funksiyasidan foydalanamiz. U templates papkasidan HTML faylni topadi, Jinja2 yordamida to'ldiradi va brauzerga qaytaradi.
from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def bosh_sahifa():
    return render_template("index.html")
# templates/index.html faylini topib, brauzerga qaytaradi

Dinamik Mazmun: {{ o'zgaruvchi }}

Eng kuchli imkoniyat — Python'dan HTML'ga ma'lumot uzatish. Buni render_template() ga qo'shimcha argument berib qilamiz. HTML ichida esa {{ }} bilan o'sha qiymatni chiqaramiz.

1) Python tomoni (app.py)

@app.route("/salom")
def salom():
    return render_template("salom.html", ism="Ali")
# ism="Ali" — shablonga "ism" nomli qiymat uzatildi

2) HTML tomoni (templates/salom.html)

<h1>Salom, {{ ism }}!</h1>
# {{ ism }} o'rniga "Ali" qo'yiladi
>>> Brauzerda: Salom, Ali!

Jinja2 da for Sikl: {% for %}

Ro'yxatdagi har bir elementni HTML'da ko'rsatish uchun {% for %} ishlatamiz. U xuddi Python'dagi for kabi ishlaydi, lekin oxirida {% endfor %} bilan yopiladi.

Python tomoni

@app.route("/mevalar")
def mevalar():
    royxat = ["Olma", "Banan", "Uzum"]
    return render_template("mevalar.html", mevalar=royxat)
# mevalar nomli ro'yxat shablonga uzatildi

HTML tomoni

<ul>
  {% for meva in mevalar %}
    <li>{{ meva }}</li>
  {% endfor %}
</ul>
# Har bir meva uchun bitta <li> hosil bo'ladi
>>> • Olma
>>> • Banan
>>> • Uzum

Jinja2 da Shart: {% if %}

Biror narsani faqat shart bajarilganda ko'rsatish uchun {% if %} ishlatamiz. U {% endif %} bilan yopiladi. Kerak bo'lsa {% else %} ham qo'shamiz.
{% if yosh >= 18 %}
  <p>Siz kattasiz</p>
{% else %}
  <p>Siz bolasiz</p>
{% endif %}
# yosh 18 dan katta yoki teng bo'lsa birinchi matn chiqadi
Diqqat: yopishni unutmang!

Har bir {% for %} uchun {% endfor %}, har bir {% if %} uchun {% endif %} bo'lishi SHART.

Misol 1: {{ ism }} O'rniga Qiymat

Ismni yozing va tugmani bosing — Jinja2 {{ ism }} o'rniga sizning qiymatingizni qanday qo'yishini ko'ring.
🌐 localhost:5000/salom

Misol 2: {% for %} bilan Ro'yxat → HTML

Vergul bilan ajratilgan ro'yxat yozing. {% for %} har bir elementdan bitta <li> yasab beradi.
🌐 localhost:5000/mevalar

Misol 3: {% if %} bilan Shartli Ko'rsatish

Yoshni kiriting. {% if yosh >= 18 %} qaysi matnni ko'rsatishni o'zi hal qiladi.
🌐 localhost:5000/yosh

Misol 4: O'z Shablonimni Sinab Ko'rish

Quyidagi shablonni o'zgartiring. {{ ism }} va {{ shahar }} qiymatlari qo'yilishini ko'ring.
🌐 localhost:5000/
Eslab Qoling

• HTML shablonlar templates/ papkasida turadi
• render_template("fayl.html") shablonni ko'rsatadi
• {{ qiymat }} — ma'lumot chiqaradi
• {% for %} ... {% endfor %} — sikl
• {% if %} ... {% endif %} — shart
• Python'dan qiymat uzatish: render_template("a.html", ism="Ali")
